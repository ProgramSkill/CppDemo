// Calculate n! = n * (n-1) * (n-2) * ... * 1
long long factorial(int n) {
    // [Base Case] When does the recursion stop?
    if (n == 0 || n == 1) {
        return 1;
    }

    // [Recursive Step] Break down the problem
    return n * factorial(n - 1);
}

Understanding Recursive Factorial

Descending Phase (Recursive Calls): The function calls itself with smaller values until reaching the base case (n = 0 or n = 1).

Base Case: The stopping condition that prevents infinite recursion. Returns 1 directly.

Ascending Phase (Returning Values): Each call waits for the result of (n-1)!, then multiplies it by n and returns.

  • Time Complexity: O(n) - n recursive calls
  • Space Complexity: O(n) - call stack depth
  • Recursion Type: Single-Branch Linear Recursion (one recursive call per function)
