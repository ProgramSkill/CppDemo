Factorial - Single-Branch Linear Recursion

01_factorial.cpp
1// recursion factorial -- Single-Branch Linear Recursion 2#include <iostream> 3using namespace std; 4 5// Calculate n! = n * (n-1) * (n-2) * ... * 1 6// Example: 5! = 5 * 4 * 3 * 2 * 1 = 120 7long long factorial(int n) { 8 // [Error Check] Negative numbers are invalid 9 if (n < 0) { 10 cout << "Error: factorial of negative number is undefined" << endl; 11 return -1; 12 } 13 14 // [Base Case] When does the recursion stop? 15 if (n == 0 || n == 1) { 16 cout << "Reached base case, n = " << n << ", returning 1" << endl; 17 return 1; 18 } 19 20 // [Recursive Step] Break down the problem 21 cout << "Computing " << n << "! = " << n << " * " << (n - 1) << "!" << endl; 22 long long result = n * factorial(n - 1); // Recursive call 23 cout << n << "! = " << result << endl; 24 25 return result; 26} 27 28int main() { 29 int num = 5; 30 long long answer = factorial(num); 31 cout << "\nFinal result: " << num << "! = " << answer << endl; 32 return 0; 33}

Algorithm Description

Definition: n! = n * (n-1) * ... * 1
Recursive Formula: f(n) = n * f(n-1)
Base Case: f(0) = f(1) = 1

Complexity Analysis

Time: O(n) - n recursive calls
Space: O(n) - recursion stack depth

Execution Flow (n=5)

+-----------------------------------------------------------------------------+
| Descending Phase                       | Ascending Phase                    |
+-----------------------------------------------------------------------------+
| factorial(5) called                    | 5 * 24 = 120 -> Final Result        |
|   -> 5 * factorial(4)                  |   ^                                  |
|        -> 4 * factorial(3)             | 4 * 6 = 24                           |
|             -> 3 * factorial(2)        |   ^                                  |
|                  -> 2 * factorial(1)   | 3 * 2 = 6                            |
|                       -> [Base] = 1    |   ^                                  |
|                                        | 2 * 1 = 2                            |
+-----------------------------------------------------------------------------+

Factorial Table

0! = 1      3! = 6       6! = 720
1! = 1      4! = 24      7! = 5040
2! = 2      5! = 120     8! = 40320

Key Points

1. Base Case: Prevents infinite recursion
2. Problem Decomposition: n! = n * (n-1)!
3. Stack Overflow Risk: Use iteration for large n
