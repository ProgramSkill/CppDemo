# NumPy

NumPy数值计算模块，涵盖数组操作、广播和线性代数。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| 数组创建 | array, zeros, ones, arange | ⭐ |
| 数组操作 | 索引、切片、重塑 | ⭐⭐ |
| 广播机制 | 不同形状数组运算 | ⭐⭐⭐ |
| 线性代数 | 矩阵乘法、特征值 | ⭐⭐⭐ |

## 🎯 学习目标

完成本模块后，你将能够：
- 创建和操作NumPy数组
- 使用向量化操作提高性能
- 理解并应用广播规则
- 进行基本的线性代数运算

## 💡 核心示例

```python
import numpy as np

# 创建数组
arr = np.array([[1, 2, 3], [4, 5, 6]])

# 数组运算
print(arr.mean())       # 平均值
print(arr.sum(axis=0))  # 列求和
print(arr @ arr.T)      # 矩阵乘法
```
