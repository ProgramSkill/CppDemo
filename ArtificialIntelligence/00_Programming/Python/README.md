# Python Basics

Python基础知识模块，涵盖语法、数据类型和控制流。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| 语法基础 | 变量、运算符、表达式 | ⭐ |
| 数据类型 | int, float, str, bool | ⭐ |
| 控制流 | if/else, for, while | ⭐⭐ |
| 函数 | 定义、参数、返回值 | ⭐⭐ |

## 🎯 学习目标

完成本模块后，你将能够：
- 编写基本的Python程序
- 理解并使用各种数据类型
- 使用条件语句和循环
- 定义和调用函数
