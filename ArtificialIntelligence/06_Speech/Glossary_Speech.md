# Speech Processing Glossary (语音处理词汇表)

## English-Chinese Technical Terms

---

## 1. 基础概念 (Fundamental Concepts)

| English | 中文 | 说明 |
|---------|------|------|
| **Speech Processing** | 语音处理 | 处理人类语音 |
| **Audio** | 音频 | 声音信号 |
| **Sample Rate** | 采样率 | 每秒采样次数 |
| **Waveform** | 波形 | 声音的时域表示 |
| **Spectrogram** | 频谱图 | 频率-时间表示 |
| **Frequency** | 频率 | 声音的高低 |
| **Amplitude** | 振幅 | 声音的响度 |

---

## 2. 特征提取 (Feature Extraction)

| English | 中文 | 说明 |
|---------|------|------|
| **MFCC** | 梅尔频率倒谱系数 | 常用语音特征 |
| **Mel Spectrogram** | 梅尔频谱图 | 模拟人耳的频谱 |
| **Mel Scale** | 梅尔刻度 | 模拟人耳感知 |
| **Filter Bank** | 滤波器组 | 频率分析 |
| **STFT** | 短时傅里叶变换 | 时频分析 |
| **Pitch** | 音高 | 基频 |
| **Formant** | 共振峰 | 声道共振 |

---

## 3. 语音识别 (Speech Recognition)

| English | 中文 | 说明 |
|---------|------|------|
| **ASR** | 自动语音识别 | 语音转文字 |
| **Speech-to-Text** | 语音转文本 | 同ASR |
| **Acoustic Model** | 声学模型 | 建模语音特征 |
| **Language Model** | 语言模型 | 建模语言规律 |
| **CTC** | 连接时序分类 | 序列标注损失 |
| **Attention** | 注意力 | 对齐机制 |
| **End-to-End** | 端到端 | 直接语音到文字 |
| **WER** | 词错误率 | ASR评估指标 |

---

## 4. 语音合成 (Speech Synthesis)

| English | 中文 | 说明 |
|---------|------|------|
| **TTS** | 文本转语音 | 语音合成 |
| **Text-to-Speech** | 文本转语音 | 同TTS |
| **Vocoder** | 声码器 | 生成波形 |
| **Mel-to-Audio** | 梅尔到音频 | 频谱转波形 |
| **Prosody** | 韵律 | 语调、节奏 |
| **Duration** | 时长 | 音素持续时间 |
| **Neural Vocoder** | 神经声码器 | WaveNet等 |

---

## 5. 说话人识别 (Speaker Recognition)

| English | 中文 | 说明 |
|---------|------|------|
| **Speaker Recognition** | 说话人识别 | 识别说话人 |
| **Speaker Verification** | 说话人验证 | 确认身份 |
| **Speaker Identification** | 说话人辨认 | 识别是谁 |
| **Speaker Diarization** | 说话人分离 | 区分不同说话人 |
| **Speaker Embedding** | 说话人嵌入 | 说话人向量表示 |
| **Voice Print** | 声纹 | 声音指纹 |

---

## 6. 常见模型 (Common Models)

| English | 中文 | 说明 |
|---------|------|------|
| **Whisper** | Whisper | OpenAI ASR模型 |
| **Wav2Vec** | Wav2Vec | 自监督语音模型 |

| **VITS** | VITS | 端到端TTS |

## 常用缩写 (Common Abbreviations)

| 缩写 | 英文全称 | 中文 |
|------|----------|------|
| **ASR** | Automatic Speech Recognition | 自动语音识别 |
| **TTS** | Text-to-Speech | 文本转语音 |
| **STT** | Speech-to-Text | 语音转文本 |
| **MFCC** | Mel-Frequency Cepstral Coefficients | 梅尔频率倒谱系数 |
| **CTC** | Connectionist Temporal Classification | 连接时序分类 |
| **WER** | Word Error Rate | 词错误率 |
| **VAD** | Voice Activity Detection | 语音活动检测 |
