# Recurrent Neural Networks (循环神经网络)

用于处理序列数据的神经网络。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| 基础RNN | 时序建模 | ⭐⭐ |
| LSTM | 长期依赖 | ⭐⭐⭐ |
| GRU | LSTM简化版 | ⭐⭐⭐ |
| 双向RNN | 前后文信息 | ⭐⭐⭐ |

## 💡 LSTM门机制

```
遗忘门: fₜ = σ(Wf·[hₜ₋₁, xₜ] + bf)
输入门: iₜ = σ(Wi·[hₜ₋₁, xₜ] + bi)
输出门: oₜ = σ(Wo·[hₜ₋₁, xₜ] + bo)
```
