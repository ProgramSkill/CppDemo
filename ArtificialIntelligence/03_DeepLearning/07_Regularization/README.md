# Regularization (正则化)

防止深度学习模型过拟合的技术。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| Dropout | 随机丢弃 | ⭐⭐ |
| Batch Norm | 批归一化 | ⭐⭐ |
| Layer Norm | 层归一化 | ⭐⭐ |
| 数据增强 | 扩充数据 | ⭐⭐ |

## 💡 技术对比

| 技术 | 作用 | 常用位置 |
|------|------|----------|
| Dropout | 防过拟合 | 全连接层 |
| BatchNorm | 稳定训练 | 卷积后 |
| LayerNorm | 稳定训练 | Transformer |
