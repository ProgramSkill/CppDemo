# Convolutional Neural Networks (卷积神经网络)

用于处理图像等网格数据的神经网络。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| 卷积操作 | 特征提取 | ⭐⭐ |
| 池化层 | 下采样 | ⭐⭐ |
| 经典架构 | VGG, ResNet | ⭐⭐⭐ |
| 迁移学习 | 预训练模型 | ⭐⭐ |

## 💡 关键组件

| 组件 | 作用 |
|------|------|
| Conv2d | 特征提取 |
| MaxPool | 下采样 |
| BatchNorm | 稳定训练 |
| Dropout | 正则化 |
