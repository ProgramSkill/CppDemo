# Image Segmentation (图像分割)

像素级别的图像理解。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| 语义分割 | 每个像素分类 | ⭐⭐⭐ |
| 实例分割 | 区分不同实例 | ⭐⭐⭐ |
| 全景分割 | 语义+实例 | ⭐⭐⭐⭐ |

## 💡 经典模型

| 模型 | 任务 | 特点 |
|------|------|------|
| FCN | 语义分割 | 全卷积 |
| U-Net | 语义分割 | 编码-解码 |
| Mask R-CNN | 实例分割 | 检测+分割 |
| DeepLab | 语义分割 | 空洞卷积 |
