# Image Classification (图像分类)

将整张图像分类到预定义的类别中。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| CNN基础 | 卷积神经网络 | ⭐⭐ |
| 经典架构 | VGG, ResNet | ⭐⭐⭐ |
| 迁移学习 | 使用预训练模型 | ⭐⭐ |
| ViT | 视觉Transformer | ⭐⭐⭐ |

## 💡 评估指标

| 指标 | 说明 |
|------|------|
| Accuracy | 正确分类比例 |
| Top-5 Accuracy | 前5预测包含正确类 |
| F1-Score | 精确率和召回率的调和平均 |
