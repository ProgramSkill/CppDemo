# Object Detection (目标检测)

定位并识别图像中的多个物体。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| 两阶段检测 | R-CNN系列 | ⭐⭐⭐ |
| 单阶段检测 | YOLO, SSD | ⭐⭐⭐ |
| Anchor-Free | FCOS, CenterNet | ⭐⭐⭐ |
| Transformer检测 | DETR | ⭐⭐⭐⭐ |

## 💡 关键概念

| 概念 | 说明 |
|------|------|
| IoU | 预测框与真实框的重叠度 |
| NMS | 去除重叠的检测框 |
| mAP | 平均精度均值 |
