# LLM Agents (LLM代理)

能够自主执行任务和使用工具的LLM系统。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| 工具使用 | 调用外部API | ⭐⭐ |
| ReAct | 推理+行动 | ⭐⭐⭐ |
| 规划 | 任务分解 | ⭐⭐⭐ |
| 多代理 | 代理协作 | ⭐⭐⭐⭐ |

## 💡 Agent架构

```
用户查询 → LLM思考 → 选择工具 → 执行 → 观察 → 继续/完成
              ↑                          ↓
              ←────── 迭代循环 ──────────←
```

## 📖 常用工具

| 工具 | 功能 |
|------|------|
| 搜索 | 获取实时信息 |
| 计算器 | 数学计算 |
| 代码执行 | 运行代码 |
| 数据库 | 查询数据 |
