# RAG (Retrieval Augmented Generation)

结合检索和生成的LLM应用模式。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| 向量数据库 | 存储嵌入 | ⭐⭐ |
| 文档处理 | 分块、嵌入 | ⭐⭐ |
| 检索策略 | 语义搜索 | ⭐⭐ |
| 生成整合 | 上下文注入 | ⭐⭐ |

## 💡 RAG流程

```
文档 → 分块 → 嵌入 → 向量数据库
                         ↓
查询 → 嵌入 → 检索 → 上下文 → LLM → 回答
```

## 📖 工具推荐

| 工具 | 用途 |
|------|------|
| LangChain | RAG框架 |
| LlamaIndex | 数据框架 |
| ChromaDB | 向量数据库 |
| Pinecone | 托管向量数据库 |
