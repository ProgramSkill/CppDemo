# Association Rule Learning (关联规则学习)

关联规则学习用于发现数据中项目之间的有趣关系。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| Apriori | 频繁项集挖掘 | ⭐⭐ |
| FP-Growth | 高效模式挖掘 | ⭐⭐ |
| 规则评估 | 支持度、置信度、提升度 | ⭐⭐ |

## 🎯 学习目标

- 理解关联规则的概念
- 使用Apriori算法挖掘规则
- 评估和解释关联规则

## 💡 核心指标

```
支持度: Support(A→B) = P(A∪B)
置信度: Confidence(A→B) = P(B|A)
提升度: Lift(A→B) = P(B|A)/P(B)
```

## 📖 应用场景

- 购物篮分析
- 推荐系统
- 医疗诊断
- 网页点击分析
