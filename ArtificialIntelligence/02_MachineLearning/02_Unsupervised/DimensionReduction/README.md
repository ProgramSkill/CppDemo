# Dimensionality Reduction (降维)

降维是减少数据特征数量同时保留重要信息的技术。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| PCA | 线性降维 | ⭐⭐ |
| t-SNE | 非线性可视化 | ⭐⭐ |
| UMAP | 快速非线性降维 | ⭐⭐ |
| LDA | 监督降维 | ⭐⭐⭐ |

## 🎯 学习目标

- 理解降维的数学原理
- 选择合适的降维方法
- 应用降维进行可视化和预处理

## 💡 方法对比

| 方法 | 类型 | 速度 | 保留全局结构 |
|------|------|------|-------------|
| PCA | 线性 | 快 | 是 |
| t-SNE | 非线性 | 慢 | 部分 |
| UMAP | 非线性 | 中等 | 是 |
