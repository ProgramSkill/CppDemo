# Clustering (聚类)

聚类是将相似数据点分组的无监督学习方法。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| K-Means | 基于距离的聚类 | ⭐ |
| 层次聚类 | 构建聚类树 | ⭐⭐ |
| DBSCAN | 基于密度的聚类 | ⭐⭐ |
| 谱聚类 | 基于图的聚类 | ⭐⭐⭐ |

## 🎯 学习目标

- 理解不同聚类算法的原理
- 选择合适的聚类方法
- 评估聚类质量

## 💡 算法对比

| 算法 | 簇形状 | 需要K | 处理噪声 |
|------|--------|-------|---------|
| K-Means | 球形 | 是 | 差 |
| DBSCAN | 任意 | 否 | 好 |
| 层次聚类 | 任意 | 否* | 一般 |
