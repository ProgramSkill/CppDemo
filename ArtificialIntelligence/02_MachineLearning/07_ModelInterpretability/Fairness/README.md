# Fairness (公平性)

确保模型对不同群体公平。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| 公平性指标 | 定义和度量 | ⭐⭐ |
| 偏见检测 | 识别模型偏见 | ⭐⭐ |
| 偏见缓解 | 减少不公平 | ⭐⭐⭐ |

## 💡 常见公平性指标

| 指标 | 定义 |
|------|------|
| Demographic Parity | 各组预测正例比例相等 |
| Equalized Odds | 各组TPR和FPR相等 |
| Equal Opportunity | 各组TPR相等 |
| Calibration | 各组预测概率准确 |
