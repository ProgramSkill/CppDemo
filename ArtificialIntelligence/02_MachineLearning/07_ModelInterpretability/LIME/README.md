# LIME (Local Interpretable Model-agnostic Explanations)

局部可解释的模型无关解释方法。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| 局部近似 | 用简单模型近似 | ⭐⭐ |
| 扰动采样 | 生成邻域样本 | ⭐⭐ |
| 表格数据 | LimeTabular | ⭐⭐ |
| 文本数据 | LimeText | ⭐⭐ |
| 图像数据 | LimeImage | ⭐⭐⭐ |

## 💡 工作流程

```
1. 在实例周围生成扰动样本
2. 用黑箱模型预测扰动样本
3. 按距离加权拟合简单模型
4. 解释简单模型的系数
```
