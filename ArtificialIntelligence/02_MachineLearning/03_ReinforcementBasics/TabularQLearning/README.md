## 💡 Q-Learning更新

```
Q(s,a) ← Q(s,a) + α[r + γ max_a' Q(s',a') - Q(s,a)]

- α: 学习率
- γ: 折扣因子
- max_a' Q(s',a'): 下一状态最大Q值
```
