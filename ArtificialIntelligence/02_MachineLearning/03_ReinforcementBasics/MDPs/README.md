# Markov Decision Processes (马尔可夫决策过程)

MDP是强化学习的数学基础框架。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| MDP定义 | 状态、动作、转移、奖励 | ⭐⭐ |
| 马尔可夫性质 | 无记忆性质 | ⭐⭐ |
| 贝尔曼方程 | 价值函数递归 | ⭐⭐⭐ |

## 💡 MDP元组

```
MDP = (S, A, P, R, γ)
- S: 状态空间
- A: 动作空间
- P: 转移概率 P(s'|s,a)
- R: 奖励函数 R(s,a,s')
- γ: 折扣因子
```
