# Bagging (装袋法)

Bootstrap Aggregating - 通过自助采样训练多个模型并聚合。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| Bootstrap采样 | 有放回随机抽样 | ⭐ |
| 聚合方法 | 投票/平均 | ⭐ |
| 随机森林 | Bagging + 随机特征 | ⭐⭐ |
| OOB评估 | 袋外误差估计 | ⭐⭐ |

## 💡 核心思想

```
原始数据
    │
    ├── Bootstrap 1 → 模型1 ──┐
    ├── Bootstrap 2 → 模型2 ──┼── 聚合 → 最终预测
    └── Bootstrap 3 → 模型3 ──┘
```
