# Feature Encoding (特征编码)

将类别变量转换为数值表示。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| 标签编码 | 整数编码 | ⭐ |
| 独热编码 | 二进制向量 | ⭐ |
| 目标编码 | 目标均值 | ⭐⭐ |
| 频率编码 | 出现频率 | ⭐⭐ |

## 💡 选择指南

| 编码方法 | 适用场景 |
|----------|----------|
| Label | 有序类别，树模型 |
| One-Hot | 无序类别，少量类别 |
| Target | 高基数类别 |
| Frequency | 频率有意义时 |
