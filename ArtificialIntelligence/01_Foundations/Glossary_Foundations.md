# Mathematical Foundations Glossary (数学基础词汇表)

## 1. 线性代数 (Linear Algebra)

| English | 中文 | 说明 |
|---------|------|------|
| **Scalar** | 标量 | 单个数值 |
| **Vector** | 向量 | 有序数字列表 |
| **Matrix** | 矩阵 | 二维数组 |
| **Tensor** | 张量 | n维数组 |
| **Dimension** | 维度 | 数组的轴数 |
| **Row** | 行 | 矩阵的水平方向 |
| **Column** | 列 | 矩阵的垂直方向 |
| **Transpose** | 转置 | 交换行列 |
| **Dot Product** | 点积 | 向量内积 |
| **Cross Product** | 叉积 | 向量外积 |
| **Matrix Multiplication** | 矩阵乘法 | 矩阵间的乘法运算 |
| **Identity Matrix** | 单位矩阵 | 对角线为1的矩阵 |
| **Inverse Matrix** | 逆矩阵 | A⁻¹使得AA⁻¹=I |
| **Determinant** | 行列式 | 矩阵的标量属性 |
| **Trace** | 迹 | 对角元素之和 |
| **Rank** | 秩 | 线性无关行/列的数量 |
| **Eigenvalue** | 特征值 | Av=λv中的λ |
| **Eigenvector** | 特征向量 | Av=λv中的v |
| **Singular Value** | 奇异值 | SVD分解中的值 |
| **Orthogonal** | 正交 | 垂直，内积为0 |
| **Orthonormal** | 标准正交 | 正交且长度为1 |
| **Span** | 张成空间 | 向量线性组合的集合 |
| **Basis** | 基 | 张成空间的最小向量集 |
| **Linear Independence** | 线性无关 | 无法互相表示 |
| **Norm** | 范数 | 向量的长度度量 |

| English | 中文 | 说明 |
|---------|------|------|
| **Derivative** | 导数 | 函数的变化率 |
| **Partial Derivative** | 偏导数 | 对单一变量的导数 |
| **Gradient** | 梯度 | 偏导数组成的向量 |
| **Integral** | 积分 | 曲线下的面积 |
| **Differentiation** | 微分 | 求导的过程 |
| **Chain Rule** | 链式法则 | 复合函数的求导规则 |
| **Product Rule** | 乘法法则 | 乘积的求导规则 |
| **Quotient Rule** | 除法法则 | 商的求导规则 |
| **Jacobian** | 雅可比矩阵 | 向量函数的导数矩阵 |
| **Hessian** | 海森矩阵 | 二阶偏导数矩阵 |
| **Taylor Series** | 泰勒级数 | 函数的多项式近似 |

| **Continuity** | 连续性 | 函数无跳跃 |
| **Critical Point** | 临界点 | 导数为零的点 |
| **Local Minimum** | 局部最小值 | 邻域内的最小值 |
| **Global Minimum** | 全局最小值 | 整个定义域的最小值 |
| **Saddle Point** | 鞍点 | 既非最大也非最小 |
| **Convex Function** | 凸函数 | 局部最小=全局最小 |
| **Concave Function** | 凹函数 | 局部最大=全局最大 |

## 3. 概率论 (Probability Theory)

| English | 中文 | 说明 |
|---------|------|------|
| **Probability** | 概率 | 事件发生的可能性 |
| **Random Variable** | 随机变量 | 随机结果的数值表示 |
| **Sample Space** | 样本空间 | 所有可能结果的集合 |
| **Event** | 事件 | 样本空间的子集 |
| **Probability Distribution** | 概率分布 | 概率的函数表示 |
| **Probability Mass Function (PMF)** | 概率质量函数 | 离散分布的概率函数 |
| **Probability Density Function (PDF)** | 概率密度函数 | 连续分布的密度函数 |
| **Cumulative Distribution Function (CDF)** | 累积分布函数 | P(X≤x)的函数 |
| **Expected Value** | 期望值 | 平均值，E[X] |
| **Variance** | 方差 | 离散程度，Var(X) |
| **Standard Deviation** | 标准差 | 方差的平方根 |
| **Covariance** | 协方差 | 两变量的联合变化 |
| **Correlation** | 相关性 | 标准化的协方差 |
| **Conditional Probability** | 条件概率 | P(A|B) |
| **Joint Probability** | 联合概率 | P(A∩B) |
| **Marginal Probability** | 边缘概率 | 单一变量的概率 |
| **Independence** | 独立性 | P(A∩B)=P(A)P(B) |
| **Bayes' Theorem** | 贝叶斯定理 | 条件概率的转换公式 |
| **Prior Probability** | 先验概率 | 观察前的概率 |
| **Posterior Probability** | 后验概率 | 观察后的概率 |
| **Likelihood** | 似然 | 数据给定参数的概率 |
