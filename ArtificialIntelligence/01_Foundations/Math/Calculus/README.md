# Calculus (微积分)

微积分为机器学习中的优化算法提供理论基础。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| 导数与偏导数 | 变化率、梯度 | ⭐ |
| 链式法则 | 复合函数求导 | ⭐⭐ |
| 多元微积分 | 雅可比、海森矩阵 | ⭐⭐⭐ |
| 泰勒展开 | 函数近似 | ⭐⭐⭐ |

## 🎯 学习目标

- 掌握求导和梯度计算
- 理解链式法则在反向传播中的应用
- 应用泰勒展开理解优化算法

## 💡 核心公式

```
梯度: ∇f = [∂f/∂x₁, ∂f/∂x₂, ...]
链式法则: ∂L/∂x = (∂L/∂y)(∂y/∂x)
```
