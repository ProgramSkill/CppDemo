# Probability Theory (概率论)

概率论是理解机器学习中不确定性的基础。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| 基础概率 | 条件概率、独立性 | ⭐ |
| 贝叶斯定理 | 先验、后验、似然 | ⭐⭐ |
| 概率分布 | 常见分布及性质 | ⭐⭐ |
| 期望与方差 | 随机变量的统计量 | ⭐⭐ |

## 🎯 学习目标

- 理解概率的基本概念
- 掌握贝叶斯定理及其应用
- 熟悉常见概率分布

## 💡 核心公式

```
贝叶斯定理: P(A|B) = P(B|A)P(A)/P(B)
期望值: E[X] = Σ xᵢP(xᵢ)
```
