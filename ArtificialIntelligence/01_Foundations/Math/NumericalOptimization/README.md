# Numerical Optimization (数值优化)

数值优化是训练机器学习模型的核心技术。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| 梯度下降 | 基础优化算法 | ⭐ |
| 随机梯度下降 | SGD及其变体 | ⭐⭐ |
| 动量与自适应 | Momentum, Adam | ⭐⭐⭐ |
| 凸优化 | 凸函数性质与收敛 | ⭐⭐⭐ |

## 🎯 学习目标

- 理解梯度下降的原理
- 掌握各种优化器的特点
- 理解凸优化的理论基础

## 💡 核心公式

```
梯度下降: θ = θ - α∇L(θ)
Adam: θ = θ - αm/(√v + ε)
```
