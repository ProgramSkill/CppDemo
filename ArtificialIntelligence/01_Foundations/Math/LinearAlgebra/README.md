# Linear Algebra (线性代数)

线性代数是机器学习的数学基础，涵盖向量、矩阵和线性变换。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| 向量与矩阵 | 基本运算、转置、逆 | ⭐ |
| 线性方程组 | 高斯消元、LU分解 | ⭐⭐ |
| 特征值与特征向量 | 矩阵分解、PCA基础 | ⭐⭐⭐ |
| 奇异值分解 | SVD及其应用 | ⭐⭐⭐ |

## 🎯 学习目标

- 理解向量空间和线性变换
- 掌握矩阵运算和分解
- 应用特征值分解到ML问题

## 💡 核心公式

```
特征值方程: Av = λv
SVD分解: A = UΣVᵀ
```
