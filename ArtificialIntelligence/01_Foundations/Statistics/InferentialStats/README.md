# Inferential Statistics (推断统计)

推断统计用于从样本数据推断总体特征。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| 参数估计 | 点估计、区间估计 | ⭐⭐ |
| 假设检验 | t检验、卡方检验 | ⭐⭐ |
| 置信区间 | 参数的可信范围 | ⭐⭐ |
| 回归分析 | 变量关系建模 | ⭐⭐⭐ |

## 🎯 学习目标

- 理解抽样分布
- 掌握假设检验流程
- 构建和解释置信区间

## 💡 核心概念

```
置信区间: x̄ ± z(σ/√n)
t统计量: t = (x̄ - μ)/(s/√n)
p值: 观察到结果的概率
```
