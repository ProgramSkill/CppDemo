# Descriptive Statistics (描述统计)

描述统计用于总结和描述数据的基本特征。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| 集中趋势 | 均值、中位数、众数 | ⭐ |
| 离散程度 | 方差、标准差、范围 | ⭐ |
| 分布形状 | 偏度、峰度 | ⭐⭐ |
| 数据可视化 | 直方图、箱线图 | ⭐ |

## 🎯 学习目标

- 计算和解释描述统计量
- 理解数据分布的特征
- 使用可视化展示数据

## 💡 核心公式

```
均值: μ = (1/n)Σxᵢ
方差: σ² = (1/n)Σ(xᵢ-μ)²
```
