# Data Structures (数据结构)

数据结构是算法效率的基础。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| 数组与链表 | 线性数据结构 | ⭐ |
| 栈与队列 | LIFO和FIFO结构 | ⭐ |
| 树与图 | 层次和网络结构 | ⭐⭐ |
| 哈希表 | 快速查找结构 | ⭐⭐ |

## 🎯 学习目标

- 理解各种数据结构的特点
- 选择合适的数据结构解决问题
- 分析数据结构的时空复杂度

## 💡 时间复杂度对比

| 操作 | 数组 | 链表 | 哈希表 |
|------|------|------|--------|
| 访问 | O(1) | O(n) | O(1) |
| 搜索 | O(n) | O(n) | O(1) |
| 插入 | O(n) | O(1) | O(1) |
