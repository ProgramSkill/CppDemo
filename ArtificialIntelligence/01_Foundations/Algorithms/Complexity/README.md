# Algorithm Complexity (算法复杂度)

算法复杂度分析用于评估算法的效率。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| 大O表示法 | 复杂度上界 | ⭐ |
| 时间复杂度 | 执行时间分析 | ⭐⭐ |
| 空间复杂度 | 内存使用分析 | ⭐⭐ |
| 渐进分析 | 最坏/平均/最好情况 | ⭐⭐⭐ |

## 🎯 学习目标

- 掌握大O表示法
- 分析算法的时空复杂度
- 比较不同算法的效率

## 💡 常见复杂度

| 复杂度 | 名称 | 示例 |
|--------|------|------|
| O(1) | 常数 | 数组访问 |
| O(log n) | 对数 | 二分查找 |
| O(n) | 线性 | 遍历数组 |
| O(n log n) | 线性对数 | 归并排序 |
| O(n²) | 平方 | 冒泡排序 |
| O(2ⁿ) | 指数 | 递归斐波那契 |
