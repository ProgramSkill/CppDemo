# Time Series Forecasting (时间序列预测)

预测未来的数值或趋势。

## 📚 内容概览

| 主题 | 描述 | 难度 |
|------|------|------|
| 朴素方法 | 简单基线 | ⭐ |
| ARIMA | 统计方法 | ⭐⭐⭐ |
| Prophet | Facebook模型 | ⭐⭐ |
| 深度学习 | LSTM, Transformer | ⭐⭐⭐ |

## 💡 方法选择

| 数据特点 | 推荐方法 |
|----------|----------|
| 简单趋势 | 指数平滑 |
| 强季节性 | SARIMA, Prophet |
| 复杂模式 | LSTM, Transformer |
