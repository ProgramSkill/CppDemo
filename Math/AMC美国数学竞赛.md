# AMC 美国数学竞赛

> **重要提示（2025年1月）**
>
> AMC（American Mathematics Competitions）是美国最主要的数学竞赛系列，由中国区授权机构组织。**不受中国教育部"白名单"限制**，是目前国内学生仍可参加的权威国际数学竞赛。

---

## 一、竞赛简介

AMC 是美国数学协会（MAA）组织的美国数学竞赛系列，是全球最具影响力的数学竞赛之一。

### 1.1 竞赛体系

AMC 竞赛系列构成了完整的数学人才培养通道：

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│  AMC 8  │ →  │ AMC 10  │ →  │  AIME   │ →  │  USAMO  │
│ (小学/  │    │ (初中/  │    │ (美国   │    │ (美国   │
│  初中)  │    │  高中)  │    │ 邀请赛) │    │ 数学奥赛)│
└─────────┘    └─────────┘    └─────────┘    └─────────┘
     │              │
     │              └─────────→ │ AMC 12 │ → ... (路径相同)
     │                        │ (高中)  │
     │                        └─────────┘
```

### 1.2 竞赛概览

| 竞赛 | 全称 | 适合年级 | 考试时间 | 题目数量 |
|------|------|----------|----------|----------|
| **AMC 8** | American Mathematics Competition 8 | 8年级及以下（14.5周岁内） | 40分钟 | 25题 |
| **AMC 10** | American Mathematics Competition 10 | 10年级及以下（17.5周岁内） | 75分钟 | 25题 |
| **AMC 12** | American Mathematics Competition 12 | 12年级及以下（19.5周岁内） | 75分钟 | 25题 |
| **AIME** | American Invitational Mathematics Examination | 邀请制 | 3小时 | 15题 |
| **USAMO** | USA Mathematical Olympiad | 邀请制 | 2天×4.5小时 | 6题 |

### 1.3 竞赛历史

| 时间节点 | 事件 |
|---------|------|
| **1950年** | 美国数学竞赛开始举办 |
| **2000年** | AMC 正式成立，整合原有竞赛 |
| **2000年代** | 逐步开放给国际学生参与 |
| **2010年代** | 中国学生参与人数快速增长 |
| **2020年代** | 线上考试形式普及 |

## 六、典型题型与解题方法

### 6.1 AMC 8 题型

#### 题型1：计算题

**例题**：计算 $1 + 2 - 3 + 4 + 5 - 6 + 7 + 8 - 9 + \cdots + 97 + 98 - 99$

**解**：每三项一组，$(1 + 2 - 3) + (4 + 5 - 6) + \cdots$

每组为 $0$，共 $99 \div 3 = 33$ 组

∴ 原式 $= 0 \times 33 = 0$

#### 题型2：几何题

**例题**：正方形面积为 36，以其四边中点为顶点作新的正方形，求新正方形的面积。

**解**：设原正方形边长为 $a = 6$

新正方形对角线长为 $a = 6$，边长为 $\frac{6}{\sqrt{2}} = 3\sqrt{2}$

∴ 新正方形面积 = $(3\sqrt{2})^2 = 18$

### 6.2 AMC 10/12 题型

#### 题型1：函数问题

**例题**：设 $f(x) = ax + b$，且 $f(1) = 3$，$f(3) = 7$，求 $f(5)$。

**解**：
$$
\begin{cases}
a + b = 3 \\
3a + b = 7
\end{cases}
$$

解得 $a = 2$，$b = 1$

∴ $f(5) = 2 \times 5 + 1 = 11$

#### 题型2：数列问题

**例题**：数列 $\{a_n\}$ 满足 $a_1 = 1$，$a_{n+1} = 2a_n + 1$，求 $a_5$。

**解**：
$$
a_1 = 1, \quad a_2 = 3, \quad a_3 = 7, \quad a_4 = 15, \quad a_5 = 31
$$

或用通项公式：$a_n = 2^n - 1$

∴ $a_5 = 2^5 - 1 = 31$

#### 题型3：组合计数

**例题**：从 1 到 100 的正整数中，有多少个数既不是 2 的倍数也不是 3 的倍数？

**解**：使用容斥原理

- 2 的倍数：$\lfloor \frac{100}{2} \rfloor = 50$ 个
- 3 的倍数：$\lfloor \frac{100}{3} \rfloor = 33$ 个
- 6 的倍数（重复计算）：$\lfloor \frac{100}{6} \rfloor = 16$ 个

是 2 或 3 的倍数的数：$50 + 33 - 16 = 67$ 个

∴ 既不是 2 也不是 3 的倍数的数：$100 - 67 = 33$ 个

## 九、注意事项

### 9.1 参赛须知

1. **报名渠道**：必须通过官方授权机构报名
2. **考试证件**：携带有效身份证件
3. **考试用具**：准备计算器（AMC 8/10/12不允许使用）
4. **诚信考试**：严格遵守考试纪律

### 9.2 家长建议

1. **理性看待**：竞赛是培养能力的手段，不是目的
2. **量力而行**：根据孩子实际情况选择参赛级别
3. **长期规划**：注重数学能力的长期培养
4. **全面发展**：不要偏科，保持各学科均衡发展
