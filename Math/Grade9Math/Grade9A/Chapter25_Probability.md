# 第二十五章 概率初步

## 📚 知识要点

### 25.1 随机事件

#### 25.1.1 随机事件的概念

**必然事件**：在一定条件下，一定会发生的事件
- 例如：太阳从东边升起
- 概率：P = 1

**不可能事件**：在一定条件下，一定不会发生的事件
- 例如：在标准大气压下水在100°C沸腾
- 概率：P = 0

**随机事件**：在一定条件下，可能发生也可能不发生的事件
- 例如：抛掷一枚硬币，正面朝上
- 概率：0 < P < 1

#### 25.1.2 概率

**定义**：一般地，对于一个随机事件A，我们把刻画其发生的可能性的大小的数，叫做事件A的概率，记作P(A)

**性质**：
- 0 ≤ P(A) ≤ 1
- P(必然事件) = 1
- P(不可能事件) = 0
- P(随机事件)：0 < P < 1

### 25.2 用列举法求概率

#### 25.2.1 列表法

**方法**：
1. 列出所有可能的结果
2. 找出满足条件的結果
3. 计算概率：P = 满足条件的结果数/所有可能的结果数

**例子**：抛掷一枚硬币，求正面朝上的概率
- 所有可能的结果：正面、反面（2个）
- 满足条件的结果：正面（1个）
- P(正面) = 1/2

#### 25.2.2 树状图

**定义**：用树状图列出所有可能的结果的方法

**优点**：
- 直观展示所有可能的结果
- 适合多步随机事件

**例子**：同时抛掷两枚硬币
- 第一步：第一枚硬币（正面、反面）
- 第二步：第二枚硬币（正面、反面）
- 共4种结果：(正,正)、(正,反)、(反,正)、(反,反)

#### 25.2.3 古典概型

**特点**：
1. 所有可能的结果有限
2. 各个结果发生的可能性相等（等可能性）

**常见类型**：
1. 抛硬币问题
2. 摸球问题
3. 抛骰子问题
4. 卡片摸球问题

### 25.3 用频率估计概率

#### 25.3.1 频率

**定义**：在相同条件下，进行n次试验，事件A发生的次数m，则比值m/n叫做事件A发生的频率

**公式**：频率 = 事件A发生的次数/试验总次数

#### 25.3.2 大数定律

**内容**：在大量重复试验中，频率会逐渐稳定于某个常数，这个常数就是概率

**应用**：用频率估计概率

**公式**：当试验次数很大时，可以用频率估计概率

## 📖 典型例题

### 例题1：随机事件的判断

**题目**：下列事件中，是随机事件的是（ ）
A. 太阳从西边升起
B. 抛掷一枚硬币，正面朝上
C. 在标准大气压下水在100°C沸腾
D. 购买一张彩票，中奖

**解答**：
**答案：B、D**

解析：
- A：必然事件
- B：随机事件
- C：不可能事件
- D：随机事件

---

### 例题2：列举法求概率

**题目**：袋中有3个红球，2个白球
    （1）求随机摸出一个红球的概率
    （2）求随机摸出一个白球的概率

**解答**：
（1）总球数：3 + 2 = 5个
    红球数：3个
    P(红球) = 3/5

（2）白球数：2个
    P(白球) = 2/5

---

### 例题3：树状图求概率

**题目**：同时抛掷两枚硬币，求两枚硬币都是正面朝上的概率。

**解答**：
用树状图表示：
第一枚：正、反
第二枚：正、反

所有结果：(正,正)、(正,反)、(反,正)、(反,反)共4种
满足条件的结果：(正,正)1种

P(两正) = 1/4

---

### 例题4：古典概型

**题目**：袋子中有5个球：2个红球，3个白球
    随机摸出2个球，求两个球颜色相同的概率。

**解答**：
总数：C(5,2) = 10
同色情况：
- 都是红球：C(2,2) = 1
- 都是白球：C(3,2) = 3
P(同色) = (1 + 3)/10 = 4/10 = 2/5
