# 第二十一章 一元二次方程

## 📚 知识要点

### 21.1 一元二次方程的概念

**定义**：等号两边都是整式，只含有一个未知数，并且未知数的最高次数是2的方程叫做一元二次方程

**一般形式**：ax² + bx + c = 0（a ≠ 0）

**其中**：
- a、b、c 是常数
- a 是二次项系数
- b 是一次项系数
- c 是常数项

**例子**：
- 2x² + 3x - 5 = 0（是一元二次方程）
- x² - 4 = 0（是一元二次方程，b = 0）
- 2x + 3 = 0（不是，最高次数是1）
- x² + y = 5（不是，含有两个未知数）

### 21.2 解一元二次方程

#### 21.2.1 直接开平方法

**适用**：形如 x² = k（k ≥ 0）或 (x + m)² = k（k ≥ 0）的方程

**步骤**：
1. 将方程化为 x² = k 或 (x + m)² = k 的形式
2. 两边开平方：x = ±√k 或 x + m = ±√k
3. 求出方程的解

**例子**：
- x² = 4 → x = ±2
- (x + 1)² = 9 → x + 1 = ±3 → x = -1 ± 3 → x₁ = 2，x₂ = -4

#### 21.2.2 配方法

**适用**：任何一元二次方程

**步骤**：
1. 化为一般形式：ax² + bx + c = 0
2. 移项：ax² + bx = -c
3. 二次项系数化为1：x² + (b/a)x = -c/a
4. 配方：x² + (b/a)x + (b/2a)² = (b/2a)² - c/a
5. 化为完全平形式：(x + b/2a)² = (b² - 4ac)/(4a²)
6. 开平方求解

**例子**：解方程 x² - 6x - 7 = 0
解：移项：x² - 6x = 7
    配方：x² - 6x + 9 = 7 + 9
    (x - 3)² = 16
    x - 3 = ±4
    x₁ = 7，x₂ = -1

#### 21.2.3 公式法

**求根公式**：对于一元二次方程 ax² + bx + c = 0（a ≠ 0）

**判别式**：Δ = b² - 4ac

**求根公式**：
$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

**根的判别式**：
- Δ > 0：有两个不相等的实数根
- Δ = 0：有两个相等的实数根
- Δ < 0：没有实数根

**例子**：解方程 2x² - 5x + 2 = 0
解：a = 2，b = -5，c = 2
    Δ = b² - 4ac = 25 - 16 = 9 > 0
    $x = \frac{5 \pm \sqrt{9}}{4} = \frac{5 \pm 3}{4}$
    $x_1 = \frac{8}{4} = 2$
    $x_2 = \frac{2}{4} = \frac{1}{2}$

### 21.3 实际问题与一元二次方程

#### 21.3.1 列方程解应用题的步骤

1. **审题**：理解题意，明确已知量和未知量
2. **设未知数**：选择适当的未知数
3. **列方程**：根据等量关系列出一元二次方程
4. **解方程**：求出方程的解
5. **检验**：检验解是否符合题意
6. **作答**：写出答案

#### 21.3.2 常见应用题型

**类型1：增长问题**
- 原数 → 增长后的数

**类型2：面积问题**
- 长方形的面积、周长
- 直角三角形的边长

**类型3：运动问题**
- 追及问题
- 相遇问题

**类型4：销售问题**
- 利润、价格、数量

## 📖 典型例题

### 例题1：一元二次方程的概念

**题目**：下列方程中，是一元二次方程的是（  ）
A. 2x + 3 = 5
B. x² + y = 4
C. x² - 4 = 0
D. x² - 2x = x² - 4

**解答**：
**答案：C**

解析：
- A：最高次数是1，不是
- B：含有两个未知数，不是
- C：是一元二次方程（a=1, b=0, c=-4）
- D：化简后是 -2x = -4，不是

---

### 例题2：直接开平方法

**题目**：解下列方程
（1）x² = 9
（2）(x + 2)² = 16
（3）2(x - 1)² = 8

**解答**：
（1）x² = 9
    ∴ x = ±**3**

（2）(x + 2)² = 16
    x + 2 = ±4
    ∴ x₁ = 2，x₂ = **-6**

（3）2(x - 1)² = 8
    (x - 1)² = 4
    x - 1 = ±2
    ∴ x₁ = 3，x₂ = **-1**

---

### 例题3：配方法

**题目**：用配方法解方程 x² - 6x - 7 = 0

**解答**：
移项：x² - 6x = 7
配方：x² - 6x + 9 = 7 + 9
     (x - 3)² = 16
     x - 3 = ±4
∴ x₁ = 7，x₂ = **-1**

---

### 例题4：公式法

**题目**：用公式法解方程 x² - 2x - 3 = 0

**解答**：
a = 1，b = -2，c = -3
Δ = b² - 4ac = (-2)² - 4×1×(-3) = 4 + 12 = 16 > 0

$x = \frac{2 \pm \sqrt{16}}{2} = \frac{2 \pm 4}{2}$

∴ $x_1 = 3$，x₂ = **-1**

---

### 例题5：根的判别式

**题目**：不解方程，判断下列方程根的情况
（1）x² - 4x + 4 = 0
（2）x² - 4x + 5 = 0
（3）2x² + 3x - 4 = 0

**解答**：
（1）a = 1，b = -4，c = 4
    Δ = b² - 4ac = 16 - 16 = 0
    ∴ 有**两个相等**的实数根

（2）a = 1，b = -4，c = 5
    Δ = 16 - 20 = -4 < 0
    ∴ **没有**实数根

（3）a = 2，b = 3，c = -4
    Δ = 9 + 32 = 41 > 0
    ∴ 有**两个不等**的实数根

---

### 例题6：根与系数的关系

**题目**：已知方程 x² + 3x - 4 = 0 的两个根是 x₁ 和 x₂
求（1）x₁ + x₂；（2）x₁·x₂；（3）1/x₁ + 1/x₂

**解答**：
由韦达定理：
（1）$x_1 + x_2 = -3$

（2）$x_1 \cdot x_2 = -4$

（3）$\frac{1}{x_1} + \frac{1}{x_2} = \frac{x_1 + x_2}{x_1 \cdot x_2} = \frac{-3}{-4} = \frac{3}{4}$

---

### 例题7：面积问题

**题目**：一个长方形的长比宽多4cm，面积是21cm²，求这个长方形的长和宽。

**解答**：
设宽为 x cm，则长为 (x + 4) cm
根据题意：x(x + 4) = 21
x² + 4x - 21 = 0
(x + 7)(x - 3) = 0
∴ x₁ = -7（舍去），x₂ = 3

∴ 宽 = **3 cm**，长 = 3 + 4 = **7 cm**

---

### 例题：增长率问题

**题目**：某工厂两年的利润增长率为20%，已知两年的总利润为242万元，求第一年的利润。

**解答**：
设第一年的利润为 x 万元
根据题意：x(1 + 20%)² = 242
x × 1.44 = 242
x = 242 ÷ 1.44 ≈ **168.06（万元）**
