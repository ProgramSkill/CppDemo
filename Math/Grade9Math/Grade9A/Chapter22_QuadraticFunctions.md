# 第二十二章 二次函数

## 📚 知识要点

### 22.1 二次函数的概念

**定义**：一般地，形如 y = ax² + bx + c（a、b、c是常数，a ≠ 0）的函数，叫做二次函数

**自变量**：x
**函数值**：y
**二次项系数**：a
**一次项系数**：b
**常数项**：c

**特例**：y = ax²（a ≠ 0）叫做简单的二次函数

### 22.2 二次函数的图象和性质

#### 22.2.1 二次函数的图象

**图象**：二次函数 y = ax² + bx + c 的图象是一条抛物线

**画法**：
1. 列表：取x、y的对应值
2. 描点：在坐标系中描点
3. 连线：用平滑的曲线连接

**顶点坐标公式**：
$$x = -\frac{b}{2a}$$
$$y = \frac{4ac - b^2}{4a}$$

#### 22.2.2 二次函数的性质

**性质1：开口方向**
- a > 0 时，抛物线开口向上
- a < 0 时，抛物线开口向下

**性质2：对称轴**
- 对称轴是直线 $x = -\frac{b}{2a}$

**性质3：顶点坐标**
- 顶点坐标是 $\left(-\frac{b}{2a}, \frac{4ac - b^2}{4a}\right)$

**性质4：最值**
- a > 0 时，函数有最小值：$y_{min} = \frac{4ac - b^2}{4a}$
- a < 0 时，函数有最大值：$y_{max} = \frac{4ac - b^2}{4a}$

**性质5：增减性**
- a > 0 时：对称轴左侧，y随x的增大而减小；对称轴右侧，y随x的增大而增大
- a < 0 时：对称轴左侧，y随x的增大而增大；对称轴右侧，y随x的增大而减小

### 22.3 二次函数与一元二次方程

**关系**：
二次函数 y = ax² + bx + c 与 x 轴的交点的横坐标，就是方程 ax² + bx + c = 0 的实数根

**交点个数**：
- Δ > 0：有两个交点
- Δ = 0：有一个交点（顶点在x轴上）
- Δ < 0：没有交点

### 22.4 实际问题与二次函数

#### 22.4.1 实际问题中的函数关系

**类型1：面积问题**
- 矩形面积、图形面积
- 圆的面积

**类型2：运动问题**
- 抛物体运动
- 追及问题

**类型3：最值问题**
- 最大利润
- 最小成本

**类型4：利润问题**
- 销售问题
- 价格问题

## 📖 典型例题

### 例题1：二次函数的概念

**题目**：下列函数中，是二次函数的是（  ）
A. y = 2x + 1
B. y = x²
C. y = 1/x
D. y = x² + 1/x

**解答**：
**答案：B**

解析：
- A：是一次函数
- B：是二次函数（a=1, b=0, c=0）
- C：是反比例函数
- D：不是整式函数

---

### 例题2：二次函数的图象

**题目**：画出函数 y = x² - 2x - 3 的图象。

**解答**：
1. 对称轴：x = -(-2)/2 = 1
2. 顶点坐标：x = 1，y = 1 - 2 - 3 = -4，即(1, -4)
3. 与x轴交点：x² - 2x - 3 = 0
   x = [2 ± √(4 + 12)]/2 = 1 ± 2
   x₁ = 3，x₂ = -1
   交点：(3, 0)和(-1, 0)

4. 与y轴交点：(0, -3)

---

### 例题3：二次函数的性质

**题目**：函数 y = -2x² + 4x + 1
（1）求开口方向、对称轴、顶点坐标、最值
（2）求对称轴右侧的增减性

**解答**：
（1）a = -2 < 0，开口**向下**
    对称轴：x = -4/(-4) = **1**
    顶点：x = 1，y = -2 + 4 + 1 = **3**
    ∵ a < 0，函数有最大值：**y_max = 3**

（2）a = -2 < 0，对称轴右侧，y随x的增大而**减小**

---

### 例题4：二次函数与一元二次方程

**题目**：求函数 y = x² - 2x - 3 与 x 轴的交点坐标。

**解答**：
令 y = 0，即 x² - 2x - 3 = 0
Δ = 4 + 12 = 16 > 0
x = [2 ± 4]/2 = 1 ± 2
∴ 交点坐标为 **(3, 0)** 和 **(-1, 0)**

---

### 例题5：最值问题

**题目**：求函数 y = 2x² - 4x + 1 的最小值。

**解答**：
a = 2 > 0，函数有最小值
对称轴：x = -(-4)/(2×2) = 1
顶点纵坐标：y = 2 - 4 + 1 = -1
∴ 最小值是 **-1**

---

### 例题6：实际应用

**题目**：某商品现在的售价是100元，计划每降价x元，销售量就增加x件
设每天的利润为y元，写出 y 与 x 的函数关系式。

**解答**：
售价：(100 - x) 元
销售量：(基量 + x) 件
成本：假设每件成本为 c 元（常数）
利润 y = (100 - x - c)(基量 + x)

这题缺少信息，无法给出具体函数式。
