# 第二十四章 圆

## 📚 知识要点

### 24.1 圆的有关性质

#### 24.1.1 圆的定义

**定义**：平面上到定点的距离等于定长的所有点组成的图形叫做圆

**圆心**：定点
**半径**：定长
**直径**：过圆心的弦（d = 2r）

#### 24.1.2 圆的对称性

**性质**：圆是轴对称图形，任何一条直径所在的直线都是圆的对称轴

### 24.2 点和圆的位置关系

#### 24.2.1 点和圆的三种位置关系

设圆的半径为r，点P到圆心的距离为d：

**位置关系**：
1. **点在圆上**：d = r
2. **点在圆内**：d < r
3. **点在圆外**：d > r

#### 24.2.2 反过来也成立

**判定**：
- d = r ⇔ 点在圆上
- d < r ⇔ 点在圆内
- d > r ⇔ 点在圆外

### 24.3 圆心角、圆周角

#### 24.3.1 圆心角

**定义**：顶点在圆心的角叫做圆心角

**性质**：圆心角的度数等于它所对的弧的度数

#### 24.3.2 圆周角

**定义**：顶点在圆上，并且两边都与圆相交的角叫做圆周角

**定理**：一条弧所对的圆周角等于它所对的圆心角的一半

**推论1**：同弧或等弧所对的圆周角相等

**推论2**：半圆（或直径）所对的圆周角是直角

**推论3**：90°的圆周角所对的弦是直径

#### 24.3.3 圆心角和圆周角的关系

**定理**：同弧所对的圆心角是圆周角的2倍

### 24.4 弧长和扇形面积公式

#### 24.4.1 弧长公式

**公式**：l = (nπr)/180

其中：
- l 是弧长
- n 是圆心角的度数
- r 是圆的半径

#### 24.4.2 扇形面积公式

**公式**：S = (nπr²)/360

其中：
- S 是扇形面积
- n 是圆心角的度数
- r 是圆的半径

## 📖 典型例题

### 例题1：点和圆的位置关系

**题目**：⊙O的半径是5cm，点P到圆心O的距离OP = 3cm
    则点P在____

**解答**：
∵ OP = 3 < 5 = r
∴ 点P在**圆内**

---

### 例题2：圆周角定理

**题目**：如图，在⊙O中，∠AOB = 80°
    求：∠ACB的度数

**解答**：
∠ACB = ½∠AOB = ½ × 80° = **40°**

---

### 例题3：垂径定理

**题目**：如图，在⊙O中，AB是直径，CD⊥AB于E，AC = 6，BD = 8
    求CD的长。

**解答**：
∵ CD⊥AB
∴ CE = AE = ½AC = 3
   ED = BE = ½BD = 4
∴ CD = CE + ED = 3 + 4 = **7**

---

### 例题4：切线的性质

**题目**：如图，PA切⊙O于A，PO交⊙O于C，OA = 5，PC = 2
    求PA的长。

**解答**：
∵ PA 是切线，OA ⊥ PA
∴ ∠OAP = 90°

在Rt△OAP中：
OA² + PA² = OP²
5² + PA² = (5 + 2)² = 49
PA² = 49 - 25 = 24
PA = **2√6**

---

### 例题5：弧长公式

**题目**：已知圆的半径是6cm，圆心角为60°
    求：（1）弧长；（2）扇形面积

**解答**：
（1）弧长 l = (60π × 6)/180 = **2π cm**

（2）面积 S = (60π × 6²)/360 = **6π cm²**

---

### 例题6：综合应用

**题目**：如图，AB是⊙O的直径，CD是弦，AE⊥CD于E，CF⊥CD于F
    求证：CE = DF

**证明**：
∵ AB是直径
∴ ∠AOB = 180°
又∵ AE ⊥ CD，CF ⊥ CD
∴ ∠AEC = 90°，∠BFC = 90°
∴ AE ∥ CF
∴ 四边形AECF是平行四边形
∴ CE = AF

又∵ AE ⊥ CD
∴ CE = AF = ½AB
