# 第二十七章 锐角三角函数

## 📚 知识要点

### 27.1 锐角三角函数

#### 27.1.1 正弦

**定义**：在直角三角形中，锐角A的对边与斜边的比叫做∠A的正弦

**符号**：sin A

**公式**：sin A = ∠A的对边/斜边 = a/c

**性质**：
- 0 < sin A < 1
- sin 0° = 0
- sin 90° = 1
- 在锐角范围内，角度越大，正弦值越大

#### 27.1.2 余弦

**定义**：在直角三角形中，锐角A的邻边与斜边的比叫做∠A的余弦

**符号**：cos A

**公式**：cos A = ∠A的邻边/斜边 = b/c

**性质**：
- 0 < cos A < 1
- cos 0° = 1
- cos 90° = 0
- 在锐角范围内，角度越大，余弦值越小

#### 27.1.3 正切

**定义**：在直角三角形中，锐角A的对边与邻边的比叫做∠A的正切

**符号**：tan A

**公式**：tan A = ∠A的对边/∠A的邻边 = a/b

**性质**：
- tan A > 0
- tan 0° = 0
- tan 90°不存在
- 在锐角范围内，角度越大，正切值越大

#### 27.1.4 余切

**定义**：在直角三角形中，锐角A的邻边与对边的比叫做∠A的余切

**符号**：cot A

**公式**：cot A = ∠A的邻边/∠A的对边 = b/a = 1/tan A

**关系**：tan A · cot A = 1

### 27.2 特殊角的三角函数值

#### 27.2.1 30°、45°、60°的三角函数值

**三角函数值表**：

| 角度α | sin α | cos α | tan α |
|-------|-------|-------|-------|
| 30° | 1/2 | √3/2 | √3/3 |
| 45° | √2/2 | √2/2 | 1 |
| 60° | √3/2 | 1/2 | √3 |

**记忆方法**：
- 30°、60°：正弦值小到大，余弦值大到小
- 45°：正弦值等于余弦值
- 正切值：30°→√3/3，45°→1，60°→√3

#### 27.2.2 互余两角的关系

**关系1**：sin A = cos(90° - A)

**关系2**：cos A = sin(90° - A)

**关系3**：tan A · cot A = 1

### 27.3 解直角三角形

#### 27.3.1 解直角三角形的概念

**定义**：在直角三角形中，除直角外，一共有5个元素（3条边和2个锐角），由已知元素求未知元素的过程，叫做解直角三角形

**基本关系**：
1. 三边之间的关系：a² + b² = c²（勾股定理）
2. 锐角之间的关系：∠A + ∠B = 90°
3. 边角之间的关系：
   - sin A = a/c，cos A = b/c，tan A = a/b
   - sin B = b/c，cos B = a/c，tan B = b/a

#### 27.3.2 解直角三角形的基本类型

**类型1：已知一条边和一个锐角**
- 利用三角函数求其他边
- 利用互余关系求另一个角

**类型2：已知两条边**
- 利用勾股定理求第三边
- 利用三角函数求角

**类型3：已知斜边和一条直角边**
- 利用勾股定理求另一条直角边
- 利用三角函数求角

#### 27.3.3 解直角三角形的步骤

**步骤1**：画出图形，标注已知条件

**步骤2**：根据已知条件，选择合适的边角关系

**步骤3**：利用三角函数、勾股定理、互余关系求未知元素

**步骤4**：检验结果

### 27.4 解直角三角形的应用

#### 27.4.1 仰角、俯角

**仰角**：视线在水平线上方，与水平线的夹角

**俯角**：视线在水平线下方，与水平线的夹角

#### 27.4.2 坡度、坡角

**坡度（坡比）**：坡面的垂直高度与水平宽度的比

**公式**：i = 竖直高度/水平宽度

**坡角**：坡面与水平面的夹角

**关系**：i = tan α

#### 27.4.3 方位角

**方位角**：从正北方向顺时针转到目标方向线的水平角

**表示方法**：北偏东、北偏西、南偏东、南偏西等

#### 27.4.4 应用问题的类型

**类型1：测量高度**
- 楼房高度
- 山的高度
- 旗杆高度

**类型2：测量距离**
- 河宽
- 两点间距离
- 航行距离

**类型3：工程问题**
- 坡度计算
- 路基设计
- 大坝设计

## 📖 典型例题

### 例题1：锐角三角函数的定义

**题目**：在Rt△ABC中，∠C = 90°，AC = 3，BC = 4，AB = 5
    求：sin A、cos A、tan A的值

**解答**：
∵ ∠C = 90°
∴ sin A = BC/AB = 4/5
   cos A = AC/AB = 3/5
   tan A = BC/AC = 4/3

---

### 例题2：特殊角的三角函数值

**题目**：求下列各式的值
（1）sin²30° + cos²30°
（2）tan 45° · sin 60° - cos 60°

**解答**：
（1）sin²30° + cos²30°
    = (1/2)² + (√3/2)²
    = 1/4 + 3/4
    = **1**

（2）tan 45° · sin 60° - cos 60°
    = 1 · (√3/2) - 1/2
    = **√3/2 - 1/2**

---

### 例题3：互余两角的关系

**题目**：已知 sin A = 3/5，∠A为锐角，求 cos(90° - A)的值

**解答**：
∵ sin A = cos(90° - A)
∴ cos(90° - A) = sin A = **3/5**

---

### 例题4：解直角三角形

**题目**：在Rt△ABC中，∠C = 90°，已知 a = 3，b = 4
    求：c、∠A、∠B

**解答**：
（1）由勾股定理：
c = √(a² + b²) = √(9 + 16) = **5**

（2）tan A = a/b = 3/4
∴ ∠A ≈ **37°**

（3）∠B = 90° - ∠A ≈ **53°**

---

### 例题5：解直角三角形

**题目**：在Rt△ABC中，∠C = 90°，已知 c = 10，∠A = 30°
    求：a、b、∠B

**解答**：
（1）a = c · sin A = 10 × sin 30° = 10 × 1/2 = **5**

（2）b = c · cos A = 10 × cos 30° = 10 × (√3/2) = **5√3**

（3）∠B = 90° - 30° = **60°**

---

### 例题6：仰角问题

**题目**：如图，从地面上D点看山顶A的仰角为30°
    已知D点到山脚B的距离为100米，山高AB为50米
    求：点A到点D的距离AD（精确到1米）

**解答**：
在Rt△ABC中：
BC = BD = 100米
AC = AB + BC = 50 + 100 = 150米

在Rt△ACD中：
tan ∠ADC = AC/CD
tan 30° = 150/CD
CD = 150/(√3/3) = 150√3 ≈ 260（米）

由勾股定理：
AD = √(AC² + CD²) = √(150² + 260²)
   = √(22500 + 67600) = √90100 ≈ **300米**

---

### 例题7：坡度问题

**题目**：如图，一水坝的横断面是梯形ABCD
    坝顶宽CD = 4米，坝高DE = 6米
    迎水坡的坡度i = 1:2，背水坡的坡度i = 1:1
    求坝底宽AB。

**解答**：
由坡度定义：
迎水坡：i₁ = DE/AE = 1/2
∴ AE = 2DE = 2 × 6 = 12米

背水坡：i₂ = CF/BF = 1/1
∴ BF = CF = 6米

坝底宽：
AB = AE + EF + FB = 12 + 4 + 6 = **22米**

---

### 例题8：方位角问题

**题目**：如图，海上有两座灯塔A、B
    从灯塔A看灯塔B的方位角是北偏东60°
    从灯塔B看灯塔A的方位角是南偏西30°
    两灯塔相距10海里
    求：从灯塔A向北行驶多少海里后，到灯塔B的距离最短。

**解答**：
最短距离时，船在B的正南方

设船行驶到C点，则AC ⊥ BC
在Rt△ABC中：
∠BAC = 90° - 60° = 30°
sin 30° = BC/AB
BC = AB · sin 30° = 10 × 1/2 = **5海里**

---

### 例题9：综合应用

**题目**：如图，在山脚A处测得山顶P的仰角为30°
    向山前进100米到B点，测得山顶P的仰角为45°
    求山高。

**解答**：
设山高PH = h米

在Rt△PAH中：
AH = PH/tan 30° = h/(√3/3) = h√3

在Rt△PBH中：
BH = PH/tan 45° = h/1 = h

由题意：AH - BH = 100
h√3 - h = 100
h(√3 - 1) = 100
h = 100/(√3 - 1) = 100(√3 + 1)/2
  = **50(√3 + 1)米** ≈ **137米**

---

### 例题10：综合应用

**题目**：如图，一艘船以每小时20海里的速度向正北航行
    在A处看灯塔P的方位角是北偏东30°
    航行2小时后到B处，看灯塔P的方位角是北偏东60°
    求：B处到灯塔P的距离。

**解答**：
AB = 20 × 2 = 40海里

在Rt△PAB中：
∠PAB = 90° - 30° = 60°
∠PBA = 90° - 60° = 30°
∠APB = 180° - 60° - 30° = 90°

∴ △PAB是直角三角形

在Rt△PAB中：
tan ∠PAB = PB/AB
tan 60° = PB/40
PB = 40√3 ≈ **69海里**
