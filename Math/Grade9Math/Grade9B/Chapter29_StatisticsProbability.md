# 第二十九章 统计与概率综合

## 📚 知识要点

### 29.1 数据的收集与整理

#### 29.1.1 数据的收集

**全面调查（普查）**：
- 定义：考察总体中的每一个对象
- 优点：准确、全面
- 缺点：工作量大、有时不可能实现
- 适用：总体数量较少，或要求精度高

**抽样调查**：
- 定义：从总体中抽取一部分对象进行调查
- 优点：省时、省力、破坏性小
- 缺点：结果可能不准确
- 适用：总体数量较大，或有破坏性

**总体**：要考察的全体对象

**个体**：总体中的每一个考察对象

**样本**：从总体中抽取的一部分个体

**样本容量**：样本中个体的数量（不带单位）

#### 29.1.2 数据的整理

**整理方法**：
1. 列表法：用统计表整理数据
2. 图形法：用统计图表示数据
3. 分组法：将数据分组整理

**统计图**：
- 条形图：显示具体数量
- 扇形图：显示部分占总体的比例
- 折线图：显示变化趋势
- 直方图：显示分布情况

### 29.2 数据的描述

#### 29.2.1 代表性数据

**平均数**：
- 定义：所有数据的和除以数据的个数
- 公式：$\bar{x} = \frac{x_1 + x_2 + ... + x_n}{n}$
- 特点：反映数据的平均水平，受极端值影响大

**中位数**：
- 定义：将数据按大小排列，位于中间位置的数据
- 求法：奇数个数据取中间，偶数个数据取中间两个的平均
- 特点：反映数据的中等水平，不受极端值影响

**众数**：
- 定义：数据中出现次数最多的数据
- 特点：反映数据的集中趋势，可能不唯一

**极差**：
- 定义：最大值与最小值的差
- 公式：极差 = 最大值 - 最小值
- 特点：反映数据的离散程度

**方差**：
- 定义：各数据与平均数的差的平方的平均数
- 公式：$s^2 = \frac{(x_1-\bar{x})^2 + (x_2-\bar{x})^2 + ... + (x_n-\bar{x})^2}{n}$
- 特点：反映数据的波动大小，方差越大，波动越大

**标准差**：
- 定义：方差的算术平方根
- 公式：$s = \sqrt{s^2}$
- 特点：与方差相同，但单位与原数据相同

#### 29.2.2 频数分布

**频数**：数据分组后，各小组内数据的个数

**频率**：各小组的频数与数据总数的比值

**频率分布表**：表示各组频数和频率的表格

**频率分布直方图**：用长方形的面积表示各组频率的图形

**画法步骤**：
1. 计算极差
2. 决定组距和组数
3. 决定分点
4. 列频率分布表
5. 画频率分布直方图

### 29.3 概率的计算

#### 29.3.1 概率的基本性质

**性质1**：0 ≤ P(A) ≤ 1

**性质2**：P(必然事件) = 1

**性质3**：P(不可能事件) = 0

**性质4**：P(随机事件)：0 < P < 1

**性质5**：若A、B互斥，则 P(A∪B) = P(A) + P(B)

**性质6**：P(A') = 1 - P(A)（A'表示A的对立事件）

#### 29.3.2 古典概型

**特点**：
1. 所有可能的结果有限
2. 各个结果发生的可能性相等

**公式**：$P(A) = \frac{A包含的结果数}{所有可能的结果数}$

**常见类型**：
1. 抛硬币问题
2. 摸球问题
3. 抛骰子问题
4. 卡片问题
5. 坐标问题

#### 29.3.3 列举法求概率

**列表法**：
- 适合两步完成的随机事件
- 列出所有可能的结果
- 找出满足条件的结果
- 计算概率

**树状图法**：
- 适合三步或更多步的随机事件
- 用树状图表示所有可能的结果
- 找出满足条件的结果
- 计算概率

### 29.4 统计与概率的综合应用

#### 29.4.1 用样本估计总体

**方法**：
1. 用样本的平均数估计总体的平均数
2. 用样本的频率估计总体的概率
3. 用样本的方差估计总体的方差

**应用**：
- 产品质量检测
- 民意调查
- 科学实验
- 社会调查

#### 29.4.2 决策问题

**决策依据**：
1. 期望值：平均收益
2. 方差：风险大小
3. 概率：可能性大小

**决策原则**：
1. 期望值越大越好
2. 方差越小越好（风险越小）
3. 综合考虑期望值和方差

## 📖 典型例题

### 例题1：平均数、中位数、众数

**题目**：某班10名学生的数学成绩如下：
85、92、78、90、85、88、92、85、87、90
求：（1）平均数；（2）中位数；（3）众数

**解答**：
（1）平均数
$\bar{x} = \frac{85+92+78+90+85+88+92+85+87+90}{10}$
    = $\frac{872}{10}$ = **87.2**

（2）将数据从小到大排列：
78、85、85、85、87、88、90、90、92、92
中位数 = $\frac{87+88}{2}$ = **87.5**

（3）85出现3次，92出现2次，90出现2次
众数 = **85**

### 例题2：方差和标准差

**题目**：甲、乙两台机床生产同种零件，抽出10个零件，测量直径（单位：mm）如下：
甲：10、10.1、9.9、10.2、9.8、10、10.1、9.9、10.2、9.8
乙：10、10、10、9.9、10.1、10、10、9.9、10.1、10
判断哪台机床生产更稳定。

**解答**：
$\bar{x}_{甲} = \bar{x}_{乙} = 10$

方差：
$s_{甲}^2 = \frac{(10-10)^2 + (10.1-10)^2 + (9.9-10)^2 + ...}{10}$
       = **0.016**

$s_{乙}^2 = \frac{(10-10)^2 + (10-10)^2 + (10-10)^2 + ...}{10}$
       = **0.004**

∵ $s_{乙}^2 < s_{甲}^2$
∴ **乙机床**生产更稳定

### 例题4：列举法求概率

**题目**：袋中有3个红球，2个白球
（1）随机摸出1个球，求摸到红球的概率
（2）随机摸出2个球，求2个球颜色相同的概率

**解答**：
（1）总球数 = 3 + 2 = 5个
    红球数 = 3个
    P(红球) = 3/5

（2）总数 = C(5,2) = 10
    同色情况：
    - 都是红球：C(3,2) = 3
    - 都是白球：C(2,2) = 1
    P(同色) = (3 + 1)/10 = **2/5**

### 例题5：树状图求概率

**题目**：同时抛掷两枚硬币，求两枚硬币都是正面朝上的概率。

**解答**：
用树状图表示：
第一枚：正、反
第二枚：正、反

所有结果：(正,正)、(正,反)、(反,正)、(反,反)共4种
满足条件的结果：(正,正)1种

P(两正) = **1/4**

### 例题6：用样本估计总体

**题目**：从某厂生产的10000件产品中随机抽取100件进行检查，发现有5件次品
估计该厂这批产品中次品的件数。

**解答**：
样本的次品率 = 5/100 = 0.05 = 5%

估计总体的次品数 = 10000 × 5% = **500件**

### 例题7：决策问题

**题目**：某公司有甲、乙两个投资方案，收益（单位：万元）如下：
甲方案：50、60、70的概率分别为0.2、0.5、0.3
乙方案：40、70、80的概率分别为0.3、0.4、0.3
应该选择哪个方案？

**解答**：
甲方案的期望收益：
E(甲) = 50×0.2 + 60×0.5 + 70×0.3
      = 10 + 30 + 21 = **61万元**

乙方案的期望收益：
E(乙) = 40×0.3 + 70×0.4 + 80×0.3
      = 12 + 28 + 24 = **64万元**

方差：
$s_{甲}^2 = (50-61)^2×0.2 + (60-61)^2×0.5 + (70-61)^2×0.3$
        = 24.2 + 0.5 + 24.3 = **49**

$s_{乙}^2 = (40-64)^2×0.3 + (70-64)^2×0.4 + (80-64)^2×0.3$
        = 172.8 + 14.4 + 76.8 = **264**

∵ E(乙) > E(甲)，且甲方案风险更小
∴ 根据不同决策目标：
- 追求高收益：选**乙方案**
- 追求稳定：选**甲方案**

### 例题8：综合应用

**题目**：某校九年级共有500名学生，为了解这些学生的视力情况，随机抽取了50名学生进行检查，检查结果如下表：

| 视力 | 人数 |
|------|------|
| 4.0以下 | 5 |
| 4.0~4.5 | 10 |
| 4.6~5.0 | 20 |
| 5.0以上 | 15 |

（1）求样本的平均视力（假设各组视力分别为4.0、4.3、4.8、5.2）
（2）估计全校视力在4.6以下的学生人数
（3）求随机抽取1名学生，视力在5.0以上的概率

**解答**：
（1）$\bar{x} = \frac{4.0×5 + 4.3×10 + 4.8×20 + 5.2×15}{50}$
         = $\frac{20 + 43 + 96 + 78}{50}$
         = $\frac{237}{50}$ = **4.74**

（2）样本中视力在4.6以下的频率 = $\frac{5+10}{50}$ = 0.3 = 30%

估计全校人数 = 500 × 30% = **150人**

（3）P(5.0以上) = $\frac{15}{50}$ = **0.3**

### 例题9：概率与统计

**题目**：某商场进行抽奖活动，规则如下：
在一个不透明的箱子里装有3个红球和2个白球（除颜色外其余都相同）
每次抽奖需付10元，从中随机摸出1个球
若摸到红球，奖励15元；若摸到白球，无奖励
求：（1）摸到红球的概率
（2）抽奖一次的平均收益

**解答**：
（1）总球数 = 3 + 2 = 5个
    红球数 = 3个
    P(红球) = **3/5**

（2）收益分析：
- 摸到红球：收益 = 15 - 10 = 5元，概率 = 3/5
- 摸到白球：收益 = 0 - 10 = -10元，概率 = 2/5

平均收益 E = 5 × (3/5) + (-10) × (2/5)
            = 3 - 4 = **-1元**

即平均每次抽奖**亏损1元**

### 例题10：综合应用

**题目**：某射击运动员在100次射击中，成绩如下：

| 环数 | 频数 |
|------|------|
| 10环 | 15 |
| 9环 | 25 |
| 8环 | 30 |
| 7环 | 20 |
| 6环及以下 | 10 |

（1）求该运动员的平均成绩
（2）求该运动员成绩的中位数
（3）求该运动员成绩的众数
（4）若该运动员再射击一次，求命中10环的概率（用频率估计）

**解答**：
（1）$\bar{x} = \frac{10×15 + 9×25 + 8×30 + 7×20 + 6×10}{100}$
         = $\frac{150 + 225 + 240 + 140 + 60}{100}$
         = $\frac{815}{100}$ = **8.15环**

（2）中位数是第50、51名的平均
第1~10名：6环及以下
第11~30名：7环
第31~60名：8环
第50、51名都是8环
∴ 中位数 = **8环**

（3）30人得8环，次数最多
∴ 众数 = **8环**

（4）命中10环的频率 = 15/100 = 0.15
估计概率 P(10环) = **0.15**
