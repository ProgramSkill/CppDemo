## 🎯 学习目标

### 知识与技能
1. **数与代数**：掌握有理数、实数的概念和运算；理解代数式、整式、方程、不等式等概念；能熟练进行相关运算和变形。
2. **图形与几何**：认识基本几何图形；理解相交线、平行线的性质；掌握平面直角坐标系的应用。
3. **统计与概率**：学会数据的收集、整理和描述方法；能绘制和解读统计图表。

### 过程与方法
- 通过观察、实验、推理等活动，发展数学思维
- 运用数学知识解决实际问题，培养应用意识
- 学会与他人合作交流，提高表达能力

### 情感态度与价值观
- 感受数学的严谨性和逻辑性
- 体验数学与生活的密切联系
- 培养实事求是、勇于探索的科学精神

## 📝 使用说明

### 文档结构
每个章节的 Markdown 文档包含以下部分：
1. **知识要点** - 核心概念和定义
2. **重点难点** - 学习要点提示
3. **典型例题** - 带详细解答过程
4. **练习题** - 巩固练习
5. **知识结构** - 章节内容梳理

### 学习建议
1. **课前预习**：先阅读知识要点，了解主要内容
2. **课中学习**：结合典型例题，理解解题方法
3. **课后复习**：完成练习题，检验掌握程度
4. **定期回顾**：使用知识结构图进行系统复习
