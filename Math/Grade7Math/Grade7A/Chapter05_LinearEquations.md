## 📖 引言：从算术到方程

### 5.1 生活中的问题

**问题1**：小明买了3支铅笔和2本笔记本，一共花了15元。已知铅笔每支1元，笔记本每本多少元？

**算术方法**（小学思维）：
- 3支铅笔花了：3 × 1 = 3元
- 2本笔记本花了：15 - 3 = 12元
- 笔记本每本：12 ÷ 2 = 6元

**方程方法**（代数思维）：
- 设笔记本每本 x 元
- 列方程：3 × 1 + 2x = 15
- 解方程：2x = 12，x = 6

**对比两种方法**：
- 算术方法：从已知出发，逆向推理
- 方程方法：设未知数，建立等量关系，正向求解

**问题2**：一个数乘以3再加5等于20，这个数是多少？

**算术方法**：
- (20 - 5) ÷ 3 = 5

**方程方法**：
- 设这个数为 x
- 3x + 5 = 20
- 3x = 15
- x = 5

### 5.2 为什么要学习方程？

**方程的优势**：

1. **思维更直接**
   - 算术：需要逆向思考
   - 方程：顺着题意直接列式

2. **解决更复杂的问题**
   - 对于复杂问题，算术方法很难思考
   - 方程可以系统地解决

3. **培养抽象思维**
   - 用字母表示未知数
   - 建立等量关系
   - 这是一种重要的数学思想

### 5.3 方程的历史

**古代的方程思想**：

- **中国**：《九章算术》中就有方程章
- **希腊**：丢番图研究代数方程
- **阿拉伯**：花拉子米《代数学》系统研究方程

**从算术到代数是数学发展的重要飞跃**

## 5.1 一元一次方程的概念

### 5.1.1 什么是一元一次方程？

让我们通过观察来理解概念。

**观察下列各式**：

（1）2 + 3 = 5（这是等式，但不含未知数）
（2）x + 3 = 5（含有未知数x）
（3）x + y = 5（含有两个未知数）
（4）x² + 2x = 3（未知数的最高次数是2）
（5）3x - 5 = x + 2（只含一个未知数，最高次数是1）

**定义**：
- **方程**：含有未知数的等式叫做方程
- **一元**：只含有一个未知数
- **一次**：未知数的次数都是1（即未知数的指数是1）
- **整式方程**：方程两边都是整式

**一元一次方程的定义**：
**只含有一个未知数（元），未知数的次数都是1（次），等号两边都是整式的方程叫做一元一次方程。**

**标准形式**：ax + b = 0（其中 a ≠ 0）

**理解标准形式**：
- a：未知数的系数（不能为0）
- b：常数项
- ax + b = 0：任何一元一次方程都可以化成这个形式

### 5.1.2 判断一元一次方程

**判断方法**：
1. ✅ 含有一个未知数
2. ✅ 未知数的次数都是1
3. ✅ 方程两边都是整式

**例1**：判断下列各式是否为一元一次方程

（1）2x + 3 = 7
- 含有未知数x ✓
- 只有一个未知数 ✓
- x的次数是1 ✓
- 两边都是整式 ✓
- **结论：是一元一次方程**

（2）x² + 2x = 3
- 未知数的最高次数是2 ✗
- **结论：不是一元一次方程（是一元二次方程）**

（3）x + y = 5
- 含有两个未知数x和y ✗
- **结论：不是一元一次方程（是二元一次方程）**

（4）x/2 + 1 = 3
- 可以化成 ½x + 1 = 3
- 含有一个未知数，次数是1 ✓
- **结论：是一元一次方程**

（5）2x - 3 = x + 5
- 化简后：x - 8 = 0
- 含有一个未知数，次数是1 ✓
- **结论：是一元一次方程**

（6）1/x = 2
- 左边1/x不是整式（分母含有未知数）✗
- **结论：不是一元一次方程（是分式方程）**

（7）√x = 3
- √x不是整式（根号下含有未知数）✗
- **结论：不是一元一次方程（是无理方程）**

（8）3x = 0
- a = 3，b = 0 ✓
- **结论：是一元一次方程**

（9）0x + 5 = 0
- a = 0，不符合 a ≠ 0 的要求 ✗
- 化简后：5 = 0（不成立）
- **结论：不是一元一次方程**

**练习**：判断下列各式是否为一元一次方程
1. x + 2 = 5 ✓
2. x² = 4 ✗（次数是2）
3. 2x - 3y = 1 ✗（两个未知数）
4. 3/x = 2 ✗（不是整式方程）
5. 5x - 3 = 2x + 4 ✓

### 5.1.3 方程的解

**定义**：**使方程左右两边相等的未知数的值叫做方程的解。**

**理解**：
- "解"是名词，表示使方程成立的那个未知数的值
- 一元一次方程通常只有一个解

**例2**：检验下列各数是否是方程 2x - 3 = 5 的解

（1）x = 4

**检验方法**：把x = 4代入方程
- 左边 = 2 × 4 - 3 = 8 - 3 = 5
- 右边 = 5
- 左边 = 右边 ✓
- **结论：x = 4 是方程的解**

（2）x = 3

**检验**：
- 左边 = 2 × 3 - 3 = 6 - 3 = 3
- 右边 = 5
- 左边 ≠ 右边 ✗
- **结论：x = 3 不是方程的解**

**例3**：已知 x = -2 是方程 3x + a = 5 的解，求 a 的值

**分析**：既然 x = -2 是方程的解，把它代入方程应该成立。

**解**：把 x = -2 代入方程
3 × (-2) + a = 5
-6 + a = 5
a = 5 + 6
a = 11

**检验**：当 a = 11 时，方程变为 3x + 11 = 5
代入 x = -2：3 × (-2) + 11 = -6 + 11 = 5 ✓

**答案**：a = 11

**练习**：
1. 检验 x = 5 是否为方程 2x + 3 = 13 的解
   解：左边 = 2 × 5 + 3 = 13 = 右边 ✓
   所以 x = 5 是方程的解

2. 已知 x = 3 是方程 2x + a = 10 的解，求 a
   解：2 × 3 + a = 10，6 + a = 10，a = 4

## 5.2 等式的性质

### 5.2.1 什么叫做等式？

**定义**：**用等号"="表示相等关系的式子叫做等式。**

**例子**：
- 1 + 2 = 3（数字等式）
- a + b = b + a（表示加法交换律）
- 3x + 2 = 8（方程也是等式）

**等式的结构**：
- 左边：等号左边的部分
- 右边：等号右边的部分
- 等号：表示左右相等

**例如**：在等式 3x + 2 = 8 中
- 左边是 3x + 2
- 右边是 8
- 中间是等号

### 5.2.2 等式的性质

**性质一：等式两边同时加（或减）同一个数或代数式，结果仍相等**

**数学表达**：
如果 a = b，那么 a ± c = b ± c

**直观理解**：
想象天平，两边平衡
- 左边放a，右边放b，天平平衡（a = b）
- 两边同时加c，仍然平衡
- 两边同时减c，仍然平衡

**例4**：用适当的数或代数式填空，使结果仍是等式

（1）如果 x + 3 = 5，那么 x + 3 - 3 = 5 - __

**分析**：两边同时减3
**答案**：x + 3 - 3 = 5 - 3

（2）如果 2x = x + 5，那么 2x - __ = x + 5 - x

**分析**：两边同时减x
**答案**：2x - x = x + 5 - x

（3）如果 ½x = 3，那么 __ = 6

**分析**：两边同时乘2
**答案**：x = 6

**性质二：等式两边同时乘同一个数，或除以同一个不为0的数，结果仍相等**

**数学表达**：
- 如果 a = b，那么 ac = bc
- 如果 a = b，c ≠ 0，那么 a/c = b/c

**注意**：
- 除以一个数时，这个数不能为0！
- 因为0不能作除数

**直观理解**：
- 天平两边质量同时扩大相同的倍数，仍然平衡
- 天平两边质量同时缩小相同的倍数（非零），仍然平衡

**例5**：用适当的数或代数式填空

（1）如果 3x = 12，那么 x = 12 ÷ __

**分析**：两边同时除以3
**答案**：x = 12 ÷ 3 = 4

（2）如果 -2x = 6，那么 x = 6 ÷ __

**分析**：两边同时除以-2
**答案**：x = 6 ÷ (-2) = -3

（3）如果 x/5 = 3，那么 x = __ × 3

**分析**：两边同时乘5
**答案**：x = 5 × 3 = 15

（4）如果 x = -2，那么 -x = __

**分析**：两边同时乘-1
**答案**：-x = -(-2) = 2

### 5.2.3 等式的性质的应用

**应用1：解简单的方程**

**例6**：利用等式的性质解下列方程

（1）x + 5 = 12

**解法**：目标是把x单独留在等号左边
- 两边同时减5
- x + 5 - 5 = 12 - 5
- x = 7

**检验**：左边 = 7 + 5 = 12 = 右边 ✓

（2）x - 3 = 8

**解法**：
- 两边同时加3
- x - 3 + 3 = 8 + 3
- x = 11

（3）3x = 15

**解法**：
- 两边同时除以3
- 3x ÷ 3 = 15 ÷ 3
- x = 5

（4）x/4 = -2

**解法**：
- 两边同时乘4
- x/4 × 4 = -2 × 4
- x = -8

（5）-x = 5

**解法**：
- 两边同时乘-1
- -x × (-1) = 5 × (-1)
- x = -5

（6）2x + 3 = 11

**解法**：分两步
- 第一步：两边同时减3
  2x + 3 - 3 = 11 - 3
  2x = 8
- 第二步：两边同时除以2
  2x ÷ 2 = 8 ÷ 2
  x = 4

**应用2：求代数式的值**

**例7**：已知 x + y = 10，xy = 3
求：2(x + y) - 3xy 的值

**解**：
2(x + y) - 3xy
= 2 × 10 - 3 × 3（代入已知值）
= 20 - 9
= 11

### 5.2.4 等式的性质的推广

**推论1：等式的对称性**

如果 a = b，那么 b = a

**理解**：相等是相互的

**推论2：等式的传递性**

如果 a = b，b = c，那么 a = c

**理解**：如果a等于b，b等于c，那么a也等于c

**推论3：等量代换**

如果 a = b，那么可以用 b 代替 a，或用 a 代替 b

**应用**：
已知 a + b = 10，b = 3，求 a

解：因为 a + b = 10，b = 3
所以 a + 3 = 10（等量代换）
a = 7

## 5.3 解一元一次方程（一）——合并同类项与移项

### 5.3.1 问题的引入

**问题**：某工厂的足球车间，工人分为两组，第一组有x人，第二组人数比第一组的2倍少5人，已知两组共有55人，求两组各有多少人？

**分析**：
- 第一组：x人
- 第二组：2x - 5人
- 两组共：x + (2x - 5) = 55

**列方程**：
x + 2x - 5 = 55

**问题**：这个方程怎么解？
小学里没学过，现在我们学习新的方法。

### 5.3.2 合并同类项

**回顾**：在整式加减中，我们学过合并同类项
- 3x + 2x = 5x
- 2a - a = a

**在方程中的应用**：

**例8**：解方程 x + 2x - 5 = 55

**解**：
x + 2x - 5 = 55
3x - 5 = 55（合并同类项：x + 2x = 3x）
3x = 55 + 5（移项：-5移到右边变+5）
3x = 60（合并同类项）
x = 20（系数化为1：两边除以3）

**检验**：左边 = 20 + 2×20 - 5 = 20 + 40 - 5 = 55 = 右边 ✓

**答**：第一组有20人，第二组有35人。

**合并同类项的要点**：
- 只含未知数的项合并
- 常数项合并
- 系数相加，字母和指数不变

**例9**：解下列方程

（1）2x - 5x = 9

**解**：
-3x = 9（合并：2x - 5x = -3x）
x = -3（系数化为1）

（2）7x - 2x = 15

**解**：
5x = 15（合并：7x - 2x = 5x）
x = 3（系数化为1）

（3）⅔x + ⅓x = 5

**解**：
x = 5（合并：⅔x + ⅓x = x）

（4）-5x + 6x - 2x = 12

**解**：
-x = 12（合并：-5x + 6x - 2x = -x）
x = -12（系数化为1）

### 5.3.3 移项

**问题**：解方程 3x + 5 = 2x - 7

**思考**：
- 未知数在等号两边都有
- 能不能把含x的项都移到一边，常数项移到另一边？

**移项法则**：**把方程中的某一项改变符号后，从方程的一边移到另一边，这种变形叫做移项。**

**关键**：移项要变号！

**例10**：解方程 3x + 5 = 2x - 7

**解法**：
3x + 5 = 2x - 7
3x - 2x = -7 - 5（移项：2x移到左边变-2x，5移到右边变-5）
x = -12（合并同类项）

**检验**：左边 = 3×(-12) + 5 = -36 + 5 = -31
右边 = 2×(-12) - 7 = -24 - 7 = -31
左边 = 右边 ✓

**为什么要变号？**

**理解1：利用等式的性质**
3x + 5 = 2x - 7
两边同时减5：3x = 2x - 7 - 5
两边同时减2x：3x - 2x = -7 - 5
x = -12

**理解2：移项是等式性质的简化**
移项 = 两边同时加减的简化写法

**例11**：解下列方程

（1）4x - 3 = 3x + 2

**解**：
4x - 3x = 2 + 3（移项）
x = 5（合并）

（2）5x + 1 = 6 - 2x

**解**：
5x + 2x = 6 - 1（移项）
7x = 5（合并）
x = 5/7（系数化为1）

（3）-2x = 5 - 3x

**解**：
-2x + 3x = 5（移项：-3x移到左边变+3x）
x = 5（合并）

（4）x - 7 = 2x - 8

**解**：
x - 2x = -8 + 7（移项）
-x = -1（合并）
x = 1（系数化为1）

### 5.3.4 移项与合并同类项的综合应用

**例12**：解方程 5x - 8 = 2x + 7

**解**：
5x - 8 = 2x + 7
5x - 2x = 7 + 8（移项）
3x = 15（合并同类项）
x = 5（系数化为1）

**例13**：解方程 2x + 3 = 5x - 9

**解**：
2x + 3 = 5x - 9
2x - 5x = -9 - 3（移项）
-3x = -12（合并同类项）
x = 4（系数化为1）

**例14**：解方程 ⅓x - 2 = ⅔x + 1

**解**：
⅓x - 2 = ⅔x + 1
⅓x - ⅔x = 1 + 2（移项）
-⅓x = 3（合并同类项）
x = -9（系数化为1：两边除以-⅓，即乘-3）

**练习**：解下列方程
1. 3x + 5 = 2x + 9
   解：3x - 2x = 9 - 5，x = 4

2. 5x - 3 = 2x + 6
   解：5x - 2x = 6 + 3，3x = 9，x = 3

3. 4x + 7 = 6 - x
   解：4x + x = 6 - 7，5x = -1，x = -⅕

### 5.3.5 去括号

**问题**：解方程 2(x - 3) = 8

**思考**：
- 能不能先去掉括号？
- 去括号法则还记得吗？

**去括号法则回顾**：
- 括号前是"+"：去掉括号，各项不变号
- 括号前是"-"：去掉括号，各项都变号

**分配律**：a(b + c) = ab + ac

**例15**：解方程 2(x - 3) = 8

**解法**：
2(x - 3) = 8
2x - 6 = 8（去括号：2×x - 2×3）
2x = 8 + 6（移项）
2x = 14（合并）
x = 7（系数化为1）

**检验**：左边 = 2×(7 - 3) = 2×4 = 8 = 右边 ✓

**例16**：解方程 3(x + 2) - 5 = 2(x - 1)

**解**：
3(x + 2) - 5 = 2(x - 1)
3x + 6 - 5 = 2x - 2（去括号）
3x + 1 = 2x - 2（合并）
3x - 2x = -2 - 1（移项）
x = -3（合并同类项）

**例17**：解方程 -2(x - 1) = 4

**解**：
-2(x - 1) = 4
-2x + 2 = 4（去括号：-2×x + (-2)×(-1)）
-2x = 4 - 2（移项）
-2x = 2（合并）
x = -1（系数化为1）

**注意**：负号乘负数得正数！

**例18**：解方程 4(x - 1) - 2(x + 3) = 0

**解**：
4(x - 1) - 2(x + 3) = 0
4x - 4 - 2x - 6 = 0（去括号）
2x - 10 = 0（合并同类项）
2x = 10（移项）
x = 5（系数化为1）

**练习**：解下列方程
1. 3(x - 2) = 9
   解：3x - 6 = 9，3x = 15，x = 5

2. 2(x + 3) = x - 5
   解：2x + 6 = x - 5，x = -11

3. 5(x - 1) - 3(x + 2) = 6
   解：5x - 5 - 3x - 6 = 6，2x - 11 = 6，2x = 17，x = 8.5

## 5.4 解一元一次方程（二）——去分母

### 5.4.1 问题的引入

**问题**：解方程 x/2 + 1 = x/3 - 2

**观察**：
- 方程中有分母
- 如果先通分，会很麻烦
- 能不能去掉分母？

**方法**：方程两边同时乘各分母的最小公倍数

### 5.4.2 去分母的方法

**去分母法则**：
**方程两边同时乘各分母的最小公倍数，去掉分母。**

**注意**：
- 不要漏乘不含分母的项！
- 分子是多项式时要加括号！

**例19**：解方程 x/2 + 1 = x/3 - 2

**分析**：
- 分母分别是2和3
- 最小公倍数是6

**解**：
x/2 + 1 = x/3 - 2
6 × (x/2 + 1) = 6 × (x/3 - 2)（两边同乘6）
6 × x/2 + 6 × 1 = 6 × x/3 - 6 × 2（分配律）
3x + 6 = 2x - 12（计算）
3x - 2x = -12 - 6（移项）
x = -18（合并同类项）

**检验**：左边 = -18/2 + 1 = -9 + 1 = -8
右边 = -18/3 - 2 = -6 - 2 = -8
左边 = 右边 ✓

**例20**：解方程 (x + 1)/2 - (x - 1)/3 = 1

**分析**：
- 分母分别是2和3
- 最小公倍数是6
- 注意分子是多项式，去分母后要加括号！

**解**：
(x + 1)/2 - (x - 1)/3 = 1
6 × [(x + 1)/2 - (x - 1)/3] = 6 × 1（两边同乘6）
3(x + 1) - 2(x - 1) = 6（注意：分子要加括号！）
3x + 3 - 2x + 2 = 6（去括号）
x + 5 = 6（合并同类项）
x = 1（移项）

**检验**：左边 = (1 + 1)/2 - (1 - 1)/3 = 1 - 0 = 1 = 右边 ✓

**常见错误**：
❌ 错误：6 × (x + 1)/2 - 6 × (x - 1)/3 = 6
      3x + 1 - 2x - 1 = 6（错误！没有加括号）

✅ 正确：3(x + 1) - 2(x - 1) = 6
      3x + 3 - 2x + 2 = 6

**例21**：解方程 (2x - 1)/4 - (5x + 1)/6 = 1

**分析**：
- 分母分别是4和6
- 最小公倍数是12

**解**：
(2x - 1)/4 - (5x + 1)/6 = 1
12 × [(2x - 1)/4 - (5x + 1)/6] = 12 × 1
3(2x - 1) - 2(5x + 1) = 12
6x - 3 - 10x - 2 = 12（去括号）
-4x - 5 = 12（合并同类项）
-4x = 17（移项）
x = -17/4（系数化为1）

**例22**：解方程 x - (x - 1)/2 = 2 - (x + 2)/3

**分析**：
- 分母是2和3
- 最小公倍数是6
- 注意：x不含分母，但也要乘6！

**解**：
x - (x - 1)/2 = 2 - (x + 2)/3
6 × [x - (x - 1)/2] = 6 × [2 - (x + 2)/3]
6x - 3(x - 1) = 12 - 2(x + 2)
6x - 3x + 3 = 12 - 2x - 4（去括号）
3x + 3 = 8 - 2x（合并同类项）
5x = 5（移项）
x = 1（系数化为1）

**练习**：解下列方程
1. x/2 - x/3 = 1
   解：3x - 2x = 6，x = 6

2. (x + 1)/3 - (x - 1)/2 = 2
   解：2(x + 1) - 3(x - 1) = 12，2x + 2 - 3x + 3 = 12，-x = 7，x = -7

3. (2x + 1)/5 - (x - 1)/3 = 0
   解：3(2x + 1) - 5(x - 1) = 0，6x + 3 - 5x + 5 = 0，x + 8 = 0，x = -8

## 5.5 解一元一次方程的一般步骤

### 5.5.1 解方程的五个步骤

**标准步骤**：

1. **去分母**：方程两边同乘各分母的最小公倍数
2. **去括号**：根据去括号法则和分配律
3. **移项**：把含有未知数的项移到方程的一边，常数项移到另一边
4. **合并同类项**：将方程化为 ax = b（a ≠ 0）的形式
5. **系数化为1**：方程两边同除以未知数的系数

**记忆口诀**：
去分母，去括号
移项，合并同类项
系数化为1
检验答案别忘记

### 5.5.2 综合例题

**例23**：解方程 (2x - 1)/3 - (5x + 1)/6 = 1

**完整步骤**：

**步骤1：去分母**
两边同乘6（3和6的最小公倍数）
2(2x - 1) - (5x + 1) = 6

**步骤2：去括号**
4x - 2 - 5x - 1 = 6

**步骤3：移项**
4x - 5x = 6 + 2 + 1

**步骤4：合并同类项**
-x = 9

**步骤5：系数化为1**
x = -9

**检验**：左边 = (2×(-9) - 1)/3 - (5×(-9) + 1)/6
            = (-18 - 1)/3 - (-45 + 1)/6
            = -19/3 - (-44)/6
            = -19/3 + 44/6
            = -38/6 + 44/6
            = 6/6
            = 1
       右边 = 1
       左边 = 右边 ✓

**例24**：解方程 2[x - 3(x - 1)] = 4

**步骤1：去小括号**
2[x - 3x + 3] = 4

**步骤2：去中括号**
2x - 6x + 6 = 4

**步骤3：移项**
2x - 6x = 4 - 6

**步骤4：合并同类项**
-4x = -2

**步骤5：系数化为1**
x = 1/2

**例25**：解方程 x/2 - (x - 1)/3 = 1

**步骤1：去分母（同乘6）**
3x - 2(x - 1) = 6

**步骤2：去括号**
3x - 2x + 2 = 6

**步骤3：移项**
3x - 2x = 6 - 2

**步骤4：合并同类项**
x = 4

**检验**：左边 = 4/2 - (4 - 1)/3 = 2 - 1 = 1 = 右边 ✓

### 5.5.3 特殊情况的处理

**情况1：分数的系数**

**例26**：解方程 ⅔x - ½ = ⅕x + 1

**分析**：分母分别是2、3、5，最小公倍数是30

**解**：
30 × (⅔x - ½) = 30 × (⅕x + 1)
20x - 15 = 6x + 30
20x - 6x = 30 + 15
14x = 45
x = 45/14

**情况2：小数的系数**

**例27**：解方程 0.5x - 0.3 = 0.2x + 0.7

**方法1**：直接解
0.5x - 0.2x = 0.7 + 0.3
0.3x = 1.0
x = 1/0.3 = 10/3

**方法2**：化整数（两边同乘10）
5x - 3 = 2x + 7
5x - 2x = 7 + 3
3x = 10
x = 10/3

**情况3：含有多个括号**

**例28**：解方程 3(2x - 1) - 2(x + 3) = 4(x - 2)

**解**：
6x - 3 - 2x - 6 = 4x - 8（去括号）
4x - 9 = 4x - 8（合并）
4x - 4x = -8 + 9（移项）
0 = 1（矛盾！）

**结论**：此方程无解！

**分析**：为什么无解？
- 化简后得到 0 = 1
- 这是不成立的
- 说明原方程无论x取何值，都不成立

**例29**：解方程 2(x + 3) - 3 = 2x + 3

**解**：
2x + 6 - 3 = 2x + 3（去括号）
2x + 3 = 2x + 3（合并）
2x - 2x = 3 - 3（移项）
0 = 0（恒成立！）

**结论**：此方程有无数解！

**分析**：为什么有无数解？
- 化简后得到 0 = 0
- 这是恒成立的
- 说明x可以取任何值

## 5.6 实际问题与一元一次方程

### 5.6.1 列方程解应用题的一般步骤

**七个步骤**：

**步骤1：审题**
- 认真阅读题目
- 理解题意
- 明确已知量和未知量

**步骤2：设未知数**
- 用字母（通常是x）表示未知数
- 有时设直接未知数
- 有时设间接未知数

**步骤3：找等量关系**
- 这是最关键的一步！
- 找出题目中相等的量
- 可以用文字或符号表示

**步骤4：列方程**
- 根据等量关系
- 用代数式表示各量
- 列出方程

**步骤5：解方程**
- 运用学过的方法
- 求出未知数的值

**步骤6：检验**
- 检验求得的值是否符合题意
- 检验计算是否正确

**步骤7：作答**
- 写出完整的答案
- 不要忘记单位！

### 5.6.2 类型一：和差倍分问题

**特征**：已知几个量之间的关系，求各量

**例30**：某班男生人数是女生人数的2倍，女生比男生少15人，求男生和女生各有多少人？

**分析**：
- 设：设女生有 x 人
- 则：男生有 2x 人
- 等量关系：男生 - 女生 = 15

**解**：设女生有 x 人，则男生有 2x 人
根据题意：2x - x = 15
解得：x = 15
所以：2x = 30

**答**：男生有30人，女生有15人。

**检验**：男生30人是女生15人的2倍 ✓
女生比男生少：30 - 15 = 15人 ✓

**例31**：一个数的3倍比它的2倍多5，求这个数。

**分析**：
- 设：设这个数为 x
- 3倍：3x
- 2倍：2x
- 等量关系：3x - 2x = 5

**解**：设这个数为 x
根据题意：3x - 2x = 5
解得：x = 5

**答**：这个数是5。

**检验**：3×5 - 2×5 = 15 - 10 = 5 ✓

**例32**：甲、乙两仓库共有粮食200吨，甲仓库运出40吨，乙仓库运进10吨后，两仓库粮食一样多。求甲、乙两仓库原来各有粮食多少吨？

**分析**：
- 设：设甲仓库原来有 x 吨
- 则：乙仓库原来有 (200 - x) 吨
- 运出/运进后：
  - 甲：x - 40
  - 乙：(200 - x) + 10 = 210 - x
- 等量关系：x - 40 = 210 - x

**解**：设甲仓库原来有 x 吨，则乙仓库原来有 (200 - x) 吨
根据题意：x - 40 = 210 - x
2x = 250
x = 125
所以：200 - x = 75

**答**：甲仓库原来有125吨，乙仓库原来有75吨。

### 5.6.3 类型二：行程问题

**基本公式**：
- 路程 = 速度 × 时间
- 速度 = 路程 ÷ 时间
- 时间 = 路程 ÷ 速度

**常见类型**：
1. 相遇问题
2. 追及问题
3. 往返问题
4. 航行问题

**例33**（相遇问题）：甲、乙两车同时从相距300km的两地相向而行，甲车速度为60km/h，乙车速度为40km/h，经过几小时两车相遇？

**分析**：
- 设：设经过 x 小时两车相遇
- 甲车行驶路程：60x km
- 乙车行驶路程：40x km
- 等量关系：甲路程 + 乙路程 = 总路程

**解**：设经过 x 小时两车相遇
根据题意：60x + 40x = 300
100x = 300
x = 3

**答**：经过3小时两车相遇。

**检验**：甲行驶：60×3 = 180km
乙行驶：40×3 = 120km
总共：180 + 120 = 300km ✓

**例34**（追及问题）：小明步行每分钟60米，小红骑自行车每分钟180米。小明先出发10分钟后，小红从同一地点出发去追小明，问小红出发后几分钟能追上小明？

**分析**：
- 设：小红出发后 x 分钟追上小明
- 小红行驶路程：180x 米
- 小明已走了：(10 + x) 分钟
- 小明行驶路程：60(10 + x) 米
- 等量关系：两人路程相同

**解**：设小红出发后 x 分钟追上小明
根据题意：180x = 60(10 + x)
180x = 600 + 60x
120x = 600
x = 5

**答**：小红出发后5分钟能追上小明。

**检验**：小红：180×5 = 900米
小明：60×(10 + 5) = 900米 ✓

**例35**（往返问题）：一辆汽车从A地开往B地，速度为60km/h，从B地返回A地，速度为40km/h，已知往返共用了5小时，求A、B两地的距离。

**分析**：
- 设：设A、B两地距离为 x km
- 去程时间：x/60 小时
- 返程时间：x/40 小时
- 等量关系：去程时间 + 返程时间 = 总时间

**解**：设A、B两地距离为 x km
根据题意：x/60 + x/40 = 5
去分母（同乘120）：2x + 3x = 600
5x = 600
x = 120

**答**：A、B两地的距离是120km。

**检验**：去程时间：120/60 = 2小时
返程时间：120/40 = 3小时
总共：2 + 3 = 5小时 ✓

### 5.6.4 类型三：工程问题

**基本公式**：
- 工作总量 = 工作效率 × 工作时间
- 工作效率 = 工作总量 ÷ 工作时间
- 工作时间 = 工作总量 ÷ 工作效率

**技巧**：通常把工作总量看作"1"

**例36**：某项工程，甲单独做需要10天完成，乙单独做需要15天完成，两人合作需要多少天完成？

**分析**：
- 设：设两人合作需要 x 天完成
- 甲的效率：1/10（每天完成工程的1/10）
- 乙的效率：1/15
- 两人合作的效率：1/10 + 1/15
- 等量关系：(1/10 + 1/15) × x = 1

**解**：设两人合作需要 x 天完成
根据题意：x/10 + x/15 = 1
去分母（同乘30）：3x + 2x = 30
5x = 30
x = 6

**答**：两人合作需要6天完成。

**检验**：甲6天完成：6/10 = 3/5
乙6天完成：6/15 = 2/5
总共：3/5 + 2/5 = 1 ✓

**例37**：一项工程，甲单独做需要12天完成，乙单独做需要20天完成，现在先由甲做3天，剩下的由甲、乙合作，还需要多少天完成？

**分析**：
- 设：设还需要 x 天完成
- 甲的效率：1/12，乙的效率：1/20
- 甲3天完成：3/12 = 1/4
- 还剩：1 - 1/4 = 3/4
- 等量关系：x/12 + x/20 = 3/4

**解**：设还需要 x 天完成
根据题意：x/12 + x/20 = 3/4
去分母（同乘60）：5x + 3x = 45
8x = 45
x = 45/8

**答**：还需要45/8天（即5又5/8天）完成。

### 5.6.5 类型四：销售问题

**基本公式**：
- 利润 = 售价 - 进价
- 利润率 = (利润 ÷ 进价) × 100%
- 售价 = 进价 × (1 + 利润率)
- 标价 × 折扣 = 售价

**例38**：某商品进价为200元，售价为240元，求利润率。

**分析**：
- 利润 = 售价 - 进价
- 利润率 = (利润 ÷ 进价) × 100%

**解**：
利润 = 240 - 200 = 40（元）
利润率 = 40 ÷ 200 × 100% = 20%

**答**：利润率为20%。

**例39**：某商品进价为1000元，按标价的8折出售，利润率为20%，求该商品的标价。

**分析**：
- 设：设标价为 x 元
- 售价：0.8x 元
- 利润：0.8x - 1000
- 利润率：(0.8x - 1000) ÷ 1000
- 等量关系：(0.8x - 1000)/1000 = 20%

**解**：设标价为 x 元
根据题意：(0.8x - 1000)/1000 = 20%
0.8x - 1000 = 200
0.8x = 1200
x = 1500

**答**：该商品的标价为1500元。

**检验**：售价：1500 × 0.8 = 1200元
利润：1200 - 1000 = 200元
利润率：200/1000 = 20% ✓

### 5.6.6 类型五：储蓄问题

**基本公式**：
- 利息 = 本金 × 利率 × 期数
- 本息和 = 本金 + 利息

**例40**：爸爸存入银行20000元，年利率为3.25%，利息税为20%，一年后爸爸实际能拿到多少利息？

**分析**：
- 利息 = 本金 × 利率 × 时间
- 利息税 = 利息 × 20%
- 实际利息 = 利息 - 利息税

**解**：
利息 = 20000 × 3.25% × 1 = 650（元）
利息税 = 650 × 20% = 130（元）
实际利息 = 650 - 130 = 520（元）

**答**：一年后爸爸实际能拿到520元利息。

### 5.6.7 列方程解应用题的技巧

**技巧1：如何设未知数**

**直接设元法**：问什么设什么
- 例：求x是多少？就设x

**间接设元法**：设与所求相关的量
- 例：年龄问题中，设中间量有时更方便

**技巧2：如何找等量关系**

**方法1：从关键字找**
- "是"、"等于"、"相同"
- "比...多"、"比...少"
- "一共"、"剩余"

**方法2：从不变量找**
- 行程问题：路程不变
- 工程问题：工作总量不变
- 浓度问题：溶质不变

**方法3：从公式找**
- 路程 = 速度 × 时间
- 工作总量 = 效率 × 时间
- 售价 = 标价 × 折扣

**技巧3：如何检验答案**

**检验1**：代入方程检验
- 把求得的值代入方程
- 看左右两边是否相等

**检验2**：是否符合实际意义
- 人数不能是分数
- 时间不能是负数
- 年龄要合理

## 📚 学习建议

### 🎯 学习方法

1. **理解方程思想**
   - 从算术思维转向代数思维
   - 用字母表示未知数
   - 建立等量关系

2. **掌握解法步骤**
   - 熟记五个步骤
   - 注意每一步的要点
   - 多做练习

3. **重视移项**
   - 移项要变号！
   - 这是最容易错的地方
   - 养成检查习惯

4. **注意去分母**
   - 不要漏乘！
   - 分子是多项式要加括号
   - 找对最小公倍数

5. **学会检验**
   - 代入原方程检验
   - 检验计算是否正确
   - 检验是否符合实际

### ⚠️ 常见错误提醒

1. ❌ 移项忘记变号
   ✅ 移项一定要变号

2. ❌ 去分母时漏乘
   ✅ 每一项都要乘

3. ❌ 去括号时符号错误
   ✅ 负号要去括号，各项都变号

4. ❌ 系数化为1时除错
   ✅ 认真计算，不要算错

5. ❌ 列方程时等量关系找错
   ✅ 仔细审题，找准等量关系
