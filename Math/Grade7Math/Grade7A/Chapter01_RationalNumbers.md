## 📖 引言：我们为什么要学习负数？

### 1.1 生活中的困惑

在小学里，我们学习了自然数（0、1、2、3...）和小数、分数。这些数在日常生活中足够用了，比如：
- 苹果有5个
- 铅笔有2.5元
- 吃了蛋糕的½

但是，生活中还有很多问题仅用这些数无法解决：

**问题1**：冬天到了，北京的温度是零下10度，我们该怎么表示？
- 如果写成"10度"，会让人误解为零上10度
- 我们需要一种方法来区分"零上"和"零下"

**问题2**：小明的爸爸做生意，今天赚了500元，明天亏了300元，我们该怎么记录？
- 如果都记作"500元"和"300元"，就分不清赚还是亏
- 我们需要一种方法来区分"收入"和"支出"

**问题3**：珠穆朗玛峰海拔8848.86米，而吐鲁番盆地海拔-154米，这个"-"是什么意思？
- 为什么要用负数来表示低于海平面的高度？

**问题4**：如果你在学校的操场中心，向东走5米记作5米，那么向西走3米该怎么记？
- 如果都记作"3米"，就分不清方向了

为了解决这些问题，数学家们发明了**负数**。

### 1.2 负数的诞生

负数的概念最早出现在中国（公元1世纪《九章算术》），后来传入印度和阿拉伯，最终在欧洲被广泛接受。

**核心思想**：用符号来区分具有相反意义的量
- 正号"+"（可以省略）：表示一种意义
- 负号"-"：表示相反的意义

## 1.1 正数和负数

### 1.1.1 正数的认识

**定义**：**大于0的数**叫做正数。

**表示方法**：
- 正数前面可以放"+"号，如：+5、+3.2、+½
- 也可以省略"+"号，直接写成：5、3.2、½

**为什么要用正号？**
- 当需要明确表示"正"的意义时，比如比较温度时：+5℃和-5℃
- 在数学运算中，强调符号时

**例1**：判断下列各数是否为正数
- 5 → 是正数（大于0）
- 0 → 不是正数（0既不是正数也不是负数）
- 3.14 → 是正数
- ¼ → 是正数
- -2 → 不是正数（是负数）

### 1.1.2 负数的认识

**定义**：**小于0的数**叫做负数。

**表示方法**：
- 负数前面必须放"-"号
- 如：-5、-3.2、-½

**负号的读法**：
- -5 读作"负五"
- 在日常生活中也可以读作"减五"，但在数学中应读作"负五"

**例2**：判断下列各数是否为负数
- -5 → 是负数
- -0.3 → 是负数
- -¼ → 是负数
- 0 → 不是负数
- 3 → 不是负数（是正数）

### 1.1.3 0的特殊地位

**0既不是正数，也不是负数**。

**为什么0很特殊？**
- 0是正数和负数的分界点
- 在数轴上，0位于正中间
- 0表示"没有"或"起点"

**在实际生活中，0的意义**：
- 温度0℃：水结冰的温度
- 海拔0米：海平面的高度
- 账户余额0元：既不欠钱也没有存款

### 1.1.4 正负数的实际应用

#### 应用1：温度的表示

**规则**：零上温度为正，零下温度为负

**例子**：
- 零上10度：+10℃ 或 10℃
- 零下5度：-5℃
- 0度：0℃

**练习**：
1. 哈尔滨冬天的温度是零下20度，记作____℃
2. 三亚夏天的温度是零上35度，记作____℃
3. 今天气温0度，记作____℃

**答案**：1. -20  2. 35 或 +35  3. 0

#### 应用2：海拔高度的表示

**规则**：海平面以上为正，海平面以下为负

**例子**：
- 珠穆朗玛峰：+8848.86米（或8848.86米）
- 吐鲁番盆地：-154米
- 死海：-422米

**为什么要这样规定？**
- 海平面是一个自然的分界线
- 这样可以直观地看出各地相对于海平面的高度

**练习**：
1. 泰山海拔1545米，记作____米
2. 马里亚纳海沟海拔-11034米，它的含义是____

**答案**：1. +1545 或 1545  2. 低于海平面11034米

#### 应用3：财务收支的表示

**规则**：收入为正，支出为负

**例子**：
- 小明爸爸今天赚了500元：+500元
- 明天买菜花了80元：-80元
- 月工资3000元：+3000元
- 房租支出1500元：-1500元

**练习**：
某家庭一周的收支记录如下：
周一：+200元（工资收入）
周二：-50元（买菜）
周三：-30元（水电费）
周四：+100元（兼职收入）
周五：-200元（还信用卡）
周六：-80元（购物）
周日：-40元（娱乐）

这一周的结余是多少？

**解答**：
结余 = +200 + (-50) + (-30) + (+100) + (-200) + (-80) + (-40)
     = 200 - 50 - 30 + 100 - 200 - 80 - 40
     = (200 + 100) - (50 + 30 + 200 + 80 + 40)
     = 300 - 400
     = -100元

说明这一周支出了100元。

#### 应用4：方向的表示

**规则**：通常规定向东、向北、向上为正，向西、向南、向下为负

**例子**：
- 从学校向东走500米：+500米
- 从学校向西走300米：-300米
- 电梯上升10层：+10层
- 电梯下降5层：-5层

**练习**：
小明在操场中心：
1. 向东走60米，记作____米
2. 然后向西走80米，记作____米
3. 现在小明距离出发点多少米？在什么方向？

**解答**：
1. +60米
2. -80米
3. 位置 = +60 + (-80) = -20米，说明在西边20米处

### 1.1.5 常见错误提醒

❌ **错误1**：认为带正号的数就是正数，带负号的数就是负数
- 反例：+0既不是正数也不是负数

✅ **正确**：判断正负数要看数值本身，与符号无关
- 正数：大于0的数
- 负数：小于0的数

❌ **错误2**：认为0是最小的数
- 事实：负数比0更小，如-1 < 0

✅ **正确**：负数 < 0 < 正数

## 1.2 有理数

### 1.2.1 为什么要引入有理数？

在上一节，我们认识了正数和负数。现在我们有以下这些数：

**正数**：5、3.2、¼、100...
**负数**：-5、-3.2、-¼、-100...
**既不是正数也不是负数**：0

数学上，为了方便研究和表达，我们需要给这些数一个统一的名称。

**定义**：**整数和分数统称为有理数**。

### 1.2.2 有理数的分类

有理数可以按照不同的标准进行分类：

#### 分类一：按数的定义（是否为整数）

```
有理数
├── 整数
│   ├── 正整数：1、2、3、100...（也叫自然数）
│   ├── 0
│   └── 负整数：-1、-2、-3、-100...
└── 分数
    ├── 正分数：½、0.3、5⅓...
    └── 负分数：-½、-0.3、-5⅓...
```

**说明**：
- **整数**包括正整数、0、负整数
- **分数**包括正分数和负分数
- 注意：有限小数和无限循环小数都可以化为分数，所以它们也是有理数

**例3**：把下列各数填入相应的集合
-3，½，0，-0.3，5，-7，¾，-100

**解答**：
- 正整数集合：{5}
- 负整数集合：{-3，-7，-100}
- 正分数集合：{½，¾}
- 负分数集合：{-0.3}

**注意**：0不是正整数也不是负整数，0就是0

#### 分类二：按数的性质符号

```
有理数
├── 正有理数（正整数和正分数）
├── 0
└── 负有理数（负整数和负分数）
```

**例4**：把下列各数填入相应的集合
-5，10，-0.3，0，¼，-8，-1½，100

**解答**：
- 正有理数集合：{10，¼，100}
- 负有理数集合：{-5，-0.3，-8，-1½}
- {0}就是0

### 1.2.3 重要概念辨析

#### 整数包括哪些？

**整数 = 正整数 + 0 + 负整数**

- 正整数：1、2、3...（小学学的自然数）
- 0
- 负整数：-1、-2、-3...

**注意**：整数不只是正整数！

#### 分数包括哪些？

**分数 = 正分数 + 负分数**

- 正分数：½、0.3、5⅓...
- 负分数：-½、-0.3、-5⅓...

**小问题**：0.3是分数吗？
**答**：是！因为0.3 = 3/10，可以写成分子分母都是整数的形式。

**小问题**：π是分数吗？
**答**：不是！π = 3.1415926...是无限不循环小数，不能化为分数，所以π不是有理数（是无理数）。

### 1.2.4 有理数的表示形式

任何一个有理数都可以写成 **p/q** 的形式，其中：
- p是整数（分子）
- q是正整数（分母，q ≠ 0）

**例子**：
- 5 = 5/1
- -3 = -3/1
- 0.3 = 3/10
- -½ = -1/2

**练习**：判断下列说法是否正确
1. 所有整数都是有理数 ✓
2. 所有分数都是有理数 ✓
3. 有理数都是整数 ✗（还有分数）
4. 0是整数，也是有理数 ✓
5. π是有理数 ✗（π是无理数）

## 1.3 数轴

### 1.3.1 为什么要学习数轴？

**问题**：如果我们只看数字 -3 和 2，你能直观地看出它们的大小关系和相对位置吗？

**答案**：不太容易。但如果画在一条直线上，就一目了然了！

```
     -3        0        2
  ←━━━━┯━━━━━━━┯━━━━━━━┯→
      -2      -1       1
```

**数轴的作用**：
1. 直观地表示有理数
2. 比较数的大小（右边的数总比左边的大）
3. 帮助理解相反数、绝对值等概念

### 1.3.2 数轴的定义

**定义**：**规定了原点、正方向和单位长度的直线**叫做数轴。

### 1.3.3 数轴的三要素

数轴必须同时具备三个要素，缺一不可：

#### 要素1：原点

**定义**：在直线上取一点表示0，这个点叫做原点。

**作用**：
- 原点是正数和负数的分界点
- 原点左边表示负数
- 原点右边表示正数

#### 要素2：正方向

**定义**：一般规定箭头指向的方向（通常向右）为正方向。

**作用**：
- 正方向决定了哪边是正数，哪边是负数
- 通常规定向右为正，向左为负

**注意**：正方向是可以任意规定的，但通常向右为正

#### 要素3：单位长度

**定义**：选取适当的长度作为单位长度。

**作用**：
- 单位长度用于在数轴上标出刻度
- 单位长度要统一，不能忽大忽小

**例5**：判断下列图形是否为数轴

（1）
```
  ←━━━┯━━━┯━━━┯→
    -1   0   1
```
**分析**：有原点、正方向、单位长度 → ✓ 是数轴

（2）
```
  ←━━━┯━━━┯━━━→
    -1   1
```
**分析**：没有标出原点 → ✗ 不是数轴

（3）
```
  ←━━━┯━━━┯━━━┯
    -1   0   1
```
**分析**：没有箭头，缺少正方向 → ✗ 不是数轴

（4）
```
  ←━━━━━┯━━━┯━━━┯→
     -1  0   1
```
**分析**：单位长度不统一 → ✗ 不是数轴

### 1.3.4 数轴的画法

**步骤1**：画一条水平的直线

**步骤2**：在直线上取一点作为原点（通常画在中间）

**步骤3**：确定正方向（通常向右为正，画箭头）

**步骤4**：选取单位长度，在原点两侧标出刻度

**注意事项**：
- 单位长度要统一
- 刻度要均匀分布
- 根据需要标注适当的刻度（不需要标出所有整数）

**例6**：画一条数轴，并在数轴上标出下列各数
-2，0，3.5，-1.5，4

**画法**：
1. 画水平直线
2. 取原点
3. 向右为正方向
4. 取适当单位长度，标出刻度

```
        -2       0    3.5  4
    ←━━━┯━━━┯━━━┯━━━┯━━━┯→
       -3 -1.5 -1   1   2   3
```

### 1.3.5 数轴上的点与有理数

**结论**：**所有的有理数都可以用数轴上的点来表示**。

**理解**：
- 原点右边的点表示正数
- 原点左边的点表示负数
- 原点表示0

**例7**：指出数轴上A、B、C、D各点表示什么数？

```
        D    C        B    A
    ←━━━┯━━━┯━━━┯━━━┯━━━┯→
       -3  -2  -1   0   1   2
```

**解答**：
- A点在原点右边2个单位长度 → A表示：2
- B点在原点右边1个单位长度 → B表示：1
- C点在原点左边1个单位长度 → C表示：-1
- D点在原点左边2个单位长度 → D表示：-2

**练习**：在数轴上标出下列各数
-3，1.5，-0.5，2，-2.5

**答案**：
```
      -3    -0.5   1.5    2
  ←━━━┯━━━┯━━━┯━━━┯━━━┯━→
     -4  -2.5 -1   0   1
```

### 1.3.6 数轴的应用

#### 应用1：比较有理数的大小

**法则**：**在数轴上，右边的数总比左边的数大**。

**例8**：比较 -3 和 1 的大小

**方法1**：画数轴
```
     -3        1
  ←━━━┯━━━━━━━┯→
      0
```
1在-3的右边 → 1 > -3

**方法2**：直接比较
正数大于一切负数 → 1 > -3

#### 应用2：表示数与数之间的距离

**例9**：在数轴上，点A表示-2，点B表示3，求A、B两点之间的距离。

**解法1**：画数轴
```
     A       0     B
  ←━━━┯━━━┯━━━┯━━━┯→
     -2  -1   1   2
```
数一数：A、B之间有5个单位长度

**解法2**：计算
距离 = 大数 - 小数 = 3 - (-2) = 3 + 2 = 5

## 1.4 相反数

### 1.4.1 相反数的直观认识

**观察**：在数轴上，找出下列各组数的位置：

（1）3 和 -3
```
      -3        0        3
  ←━━━┯━━━━━━━┯━━━━━━━┯→
```
-3和3关于原点对称

（2）-2.5 和 2.5
```
    -2.5       0       2.5
  ←━━━━┯━━━━━━━┯━━━━━━━┯→
```
-2.5和2.5关于原点对称

**发现**：这两组数有什么共同特点？
1. 符号相反（一个正，一个负）
2. 数字相同
3. 在数轴上到原点的距离相等
4. 位于原点的两侧

### 1.4.2 相反数的定义

**定义**：**只有符号不同的两个数叫做互为相反数**。

**理解**：
- "互为"的意思是相互的，a的相反数是b，则b的相反数是a
- "只有符号不同"意味着数字部分完全相同

**例10**：说出下列各数的相反数
（1）5的相反数是-5
（2）-3的相反数是3
（3）0的相反数是0
（4）½的相反数是-½
（5）-0.3的相反数是0.3

**特别说明**：0的相反数是0
- 因为0既不是正数也不是负数
- 在数轴上，0关于原点对称的点是它本身

### 1.4.3 相反数的表示方法

**符号表示**：数a的相反数记作 **-a**

**注意**：
- 这里的"-"不是运算符号，而是相反数的符号
- -a不一定是负数（当a是负数时，-a是正数）

**例11**：
- a = 5 时，-a = -5
- a = -3 时，-a = -(-3) = 3
- a = 0 时，-a = -0 = 0

### 1.4.4 相反数的性质

#### 性质1：互为相反数的两个数和为0

**数学表达**：a + (-a) = 0

**例子**：
- 5 + (-5) = 0
- -3 + 3 = 0
- 0 + 0 = 0

#### 性质2：相反数的相反数等于它本身

**数学表达**：-(-a) = a

**理解**：
- 5的相反数是-5，-5的相反数是5
- -3的相反数是3，3的相反数是-3

**例12**：化简
- -(-5) = 5
- -(-3) = 3
- -(7) = -7

#### 性质3：互为相反数的两个数在数轴上位于原点两侧，且到原点的距离相等

**几何意义**：关于原点对称

**例子**：
```
      -3        0        3
  ←━━━┯━━━━━━━┯━━━━━━━┯→
```
-3和3关于原点对称，距离都是3个单位长度

### 1.4.5 相反数的应用

#### 应用1：化简多重符号

**法则**：
- 一个负号：负号
- 两个负号：正号（负负得正）
- 三个负号：负号
- ...以此类推：奇数个负号为负，偶数个负号为正

**例13**：化简下列各式
- -(5) = -5
- -(-5) = 5
- -(-(-5)) = -5
- -(-(-(-5))) = 5

**记忆口诀**：负负得正，负负负得负...

#### 应用2：求代数式的相反数

**例14**：
（1）a + b的相反数是 -(a + b) = -a - b
（2）a - b的相反数是 -(a - b) = -a + b = b - a

**注意**：求代数式的相反数时，要给整个式子加括号

### 1.4.6 常见错误

❌ **错误1**：认为带负号的数一定是负数
- 反例：-(-5) = 5（正数）

✅ **正确**：要看化简后的结果

❌ **错误2**：认为任何数的相反数都是负数
- 反例：-5的相反数是5（正数）

✅ **正确**：正数的相反数是负数，负数的相反数是正数

## 1.5 绝对值

### 1.5.1 绝对值的直观认识

**问题情境**：

小明和小红从学校出发：
- 小明向东走了3千米
- 小红向西走了3千米

如果我们只关心"走了多远"，而不关心"方向"，那么他们走的距离是一样的，都是3千米。

在数学上，这个"不考虑方向的距离"就是**绝对值**。

### 1.5.2 绝对值的定义

#### 代数定义

**绝对值的几何意义**：一个数a的绝对值就是数轴上表示数a的点到原点的距离。

**数学表达**：
```
|a| = a      （当 a > 0 时，即正数的绝对值是它本身）
|a| = 0      （当 a = 0 时，即0的绝对值是0）
|a| = -a     （当 a < 0 时，即负数的绝对值是它的相反数）
```

**重要理解**：
- |a| ≥ 0（绝对值是非负数）
- 距离不可能为负数

#### 几何定义

在数轴上，一个数的绝对值就是表示这个数的点到原点的距离。

**例子**：
```
      -3        0        3
  ←━━━┯━━━━━━━┯━━━━━━━┯→
```
|-3| = 3（-3到原点的距离）
|3| = 3（3到原点的距离）

### 1.5.3 绝对值的计算

**例15**：求下列各数的绝对值
（1）|+5| = 5（正数的绝对值是它本身）
（2）|-8| = 8（负数的绝对值是它的相反数）
（3）|0| = 0（0的绝对值是0）
（4）|-3.5| = 3.5
（5）|0.001| = 0.001

**例16**：已知 |a| = 3，求a的值

**分析**：
- 如果a是正数，a = 3
- 如果a是负数，a = -3
- 如果a = 0，|a| = 0 ≠ 3，不满足

**解答**：a = 3 或 a = -3

**结论**：绝对值为同一个正数的有理数有两个，它们互为相反数。

**推广**：
- 如果 |a| = 0，则 a = 0（唯一）
- 如果 |a| = m（m > 0），则 a = m 或 a = -m（两个值）

### 1.5.4 绝对值的性质

#### 性质1：非负性

**性质**：任何有理数的绝对值都是非负数，即 |a| ≥ 0

**理解**：
- 绝对值表示距离，距离不可能为负
- 绝对值最小的数是0

**例17**：判断下列说法是否正确
（1）|a| ≥ 0  ✓
（2）|a| > 0  ✗（当a=0时，|a|=0）

#### 性质2：互为相反数的两个数的绝对值相等

**性质**：|a| = |-a|

**例子**：
- |5| = |-5| = 5
- |3.2| = |-3.2| = 3.2

**几何理解**：互为相反数的两个数在数轴上到原点的距离相等

#### 性质3：绝对值相等的两个数的关系

**性质**：若 |a| = |b|，则 a = b 或 a = -b

**例子**：
- 若 |a| = |5|，则 a = 5 或 a = -5
- 若 |a| = |b| = 3，则 a = b = 3 或 a = b = -3 或 a = 3, b = -3 或 a = -3, b = 3

### 1.5.5 绝对值的应用

#### 应用1：求有理数的范围

**例18**：已知 |a| < 3，求整数a的值

**分析**：
|a| < 3 表示a到原点的距离小于3

**解法1**：在数轴上表示
```
      -3        0        3
  ←━━━┯━━━━━━━┯━━━━━━━┯→
```
符合条件的整数：-2, -1, 0, 1, 2

**解法2**：利用绝对值定义
|a| < 3 ⇒ -3 < a < 3
整数a的值：-2, -1, 0, 1, 2

#### 应用2：绝对值的化简

**例19**：化简 |a - 2|

**分析**：需要讨论a - 2的正负
- 当 a > 2 时，a - 2 > 0，|a - 2| = a - 2
- 当 a = 2 时，a - 2 = 0，|a - 2| = 0
- 当 a < 2 时，a - 2 < 0，|a - 2| = -(a - 2) = 2 - a

**结论**：
```
|a - 2| = {
    a - 2    (当 a > 2 时)
    0        (当 a = 2 时)
    2 - a    (当 a < 2 时)
}
```

#### 应用3：绝对值的方程

**例20**：解方程 |x - 1| = 2

**分析**：|x - 1| = 2 表示x到1的距离是2

**解法1**：几何意义
在数轴上，到1的距离是2的点有两个：
- 1的左边2个单位：1 - 2 = -1
- 1的右边2个单位：1 + 2 = 3

**解法2**：代数解法
|x - 1| = 2 ⇒ x - 1 = 2 或 x - 1 = -2
- 若 x - 1 = 2，则 x = 3
- 若 x - 1 = -2，则 x = -1

**答案**：x = 3 或 x = -1

#### 应用4：绝对值的不等式

**例21**：解不等式 |x| < 3

**几何理解**：|x| < 3 表示x到原点的距离小于3

在数轴上：
```
      -3        0        3
  ←━━━━┯━━━━━━━┯━━━━━━━┯→
```
符合条件的x在-3和3之间

**答案**：-3 < x < 3

**推广**：
- |x| < a（a > 0）⇒ -a < x < a
- |x| > a（a > 0）⇒ x < -a 或 x > a

### 1.5.6 绝对值的常见错误

❌ **错误1**：认为绝对值一定是正数
- 反例：|0| = 0（0不是正数）

✅ **正确**：绝对值是非负数（|a| ≥ 0）

❌ **错误2**：认为|-a| = -a
- 反例：|-5| = 5 ≠ -5

✅ **正确**：|-a| = |a|

## 1.6 有理数的大小比较

### 1.6.1 有理数大小比较的基本法则

#### 法则1：正数、0、负数的大小关系

**结论**：**正数 > 0 > 负数**

**理解**：
- 一切正数都大于0
- 0大于一切负数
- 正数大于一切负数

**例22**：比较下列各组数的大小
（1）5 和 0 → 5 > 0
（2）0 和 -3 → 0 > -3
（3）5 和 -3 → 5 > -3

#### 法则2：两个负数比较大小

**法则**：**两个负数比较，绝对值大的反而小**

**为什么？**

在数轴上观察：
```
      -5        -3       0
  ←━━━┯━━━━━━━┯━━━━━━━┯→
```
-5在-3的左边，所以 -5 < -3
但 |-5| = 5 > |-3| = 3

**结论**：绝对值大的负数反而小

**记忆口诀**："负数比大小，绝对值大的反而小"

**例23**：比较下列各组数的大小
（1）-3 和 -5
|-3| = 3，|-5| = 5
因为 3 < 5，所以 -3 > -5

（2）-0.9 和 -1.1
|-0.9| = 0.9，|-1.1| = 1.1
因为 0.9 < 1.1，所以 -0.9 > -1.1

（3）-½ 和 -⅓
|-½| = ½ = 3/6，|-⅓| = ⅓ = 2/6
因为 3/6 > 2/6，所以 |-½| > |-⅓|，因此 -½ < -⅓

### 1.6.2 有理数大小比较的方法

#### 方法1：利用法则直接比较

**适用情况**：正数与负数比较，0与负数比较

**例24**：比较大小
（1）3 和 -5 → 3 > -5（正数大于负数）
（2）0 和 -100 → 0 > -100（0大于负数）

#### 方法2：比较绝对值

**适用情况**：两个负数比较

**步骤**：
1. 分别求出两个负数的绝对值
2. 比较绝对值的大小
3. 绝对值大的反而小

**例25**：比较 -8 和 -6
|-8| = 8，|-6| = 6
因为 8 > 6，所以 -8 < -6

#### 方法3：数轴比较法

**步骤**：
1. 画出数轴
2. 在数轴上标出各数
3. 右边的数总比左边的大

**例26**：比较 -2、0、-1.5、1 的大小

**步骤**：画数轴
```
      -2    -1.5   0    1
  ←━━━┯━━━┯━━━┯━━━┯→
```
从左到右的顺序：-2 < -1.5 < 0 < 1

**答案**：-2 < -1.5 < 0 < 1

#### 方法4：作差法

**法则**：若 a - b > 0，则 a > b；若 a - b < 0，则 a < b

**例27**：比较 -3和 -1
-3 - (-1) = -3 + 1 = -2 < 0
所以 -3 < -1

### 1.6.3 多个数的大小比较

**步骤**：
1. 先分类：正数、0、负数
2. 负数之间比较（用绝对值）
3. 综合排序

**例28**：把下列各数按从小到大的顺序排列
-3，0，2，-5，1，-1

**步骤1**：分类
- 正数：2，1
- 0：0
- 负数：-3，-5，-1

**步骤2**：负数之间比较
|-5| = 5，|-3| = 3，|-1| = 1
因为 5 > 3 > 1，所以 -5 < -3 < -1

**步骤3**：综合排序
-5 < -3 < -1 < 0 < 1 < 2

**答案**：-5 < -3 < -1 < 0 < 1 < 2

### 1.6.4 有理数大小比较的注意事项

⚠️ **注意1**：不要只看符号
- 错误：认为-5 > -3（因为-5的符号更"强"）
- 正确：-5 < -3（比较绝对值）

⚠️ **注意2**：比较分数大小时要通分
- 例：比较 -½ 和 -⅓
- |-½| = ½ = 3/6，|-⅓| = ⅓ = 2/6
- 因为 3/6 > 2/6，所以 -½ < -⅓

⚠️ **注意3**：比较小数大小时要注意位数
- 例：比较 -0.9 和 -0.89
- |-0.9| = 0.90，|-0.89| = 0.89
- 因为 0.90 > 0.89，所以 -0.9 < -0.89
