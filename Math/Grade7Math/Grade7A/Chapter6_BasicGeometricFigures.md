# 第六章 几何图形初步

## 📚 知识要点

### 6.1 几何图形

#### 6.1.1 立体图形与平面图形

**立体图形**：图形的各部分不都在同一平面内
- 例如：长方体、正方体、圆柱、圆锥、球、棱柱、棱锥等

**平面图形**：图形的各部分都在同一平面内
- 例如：线段、角、三角形、长方形、圆等

#### 6.1.2 从不同方向看立体图形

**三视图**：从正面、上面、左面三个方向看立体图形，得到的平面图形

- **主视图**（正视图）：从正面看
- **俯视图**：从上面看
- **左视图**：从左面看

**例子**：
- 正方体的三视图都是正方形
- 圆柱的三视图：主视图和左视图是长方形，俯视图是圆

#### 6.1.3 立体图形的展开图

**定义**：将立体图形的表面展开成平面图形，叫做展开图

**常见立体图形的展开图**：

1. **正方体的展开图**：有11种不同的展开方式
   - "一四一"型：4个正方形排成一排，两端各1个
   - "二三一"型：3个正方形排成一排，一端2个，另一端1个
   - "二二二"型：2个正方形一排，共三排
   - "三三"型：3个正方形一排，共两排

2. **长方体的展开图**：与正方体类似，只是面是长方形

3. **圆柱的展开图**：两个圆（底面）和一个长方形（侧面）

4. **圆锥的展开图**：一个扇形（侧面）和一个圆（底面）

### 6.2 直线、射线、线段

#### 6.2.1 直线

**定义**：向两方无限延伸的直的线

**表示方法**：
1. 用一个小写字母表示：直线 l
2. 用两个大写字母表示：直线 AB（或直线 BA）

**性质**：
1. 经过两点有一条直线，并且只有一条直线（两点确定一条直线）
2. 两条直线相交，有且只有一个公共点（交点）

#### 6.2.2 射线

**定义**：直线上的一点和它一旁的部分叫做射线

**表示方法**：
1. 用一个小写字母表示：射线 l
2. 用两个大写字母表示：射线 OA（端点字母在前）

**注意**：
- 射线只有一个端点
- 射线向一方无限延伸
- 射线 OA 和射线 AO 是不同的射线

#### 6.2.3 线段

**定义**：直线上两个点和它们之间的部分叫做线段

**表示方法**：
1. 用一个小写字母表示：线段 l
2. 用两个大写字母表示：线段 AB（或线段 BA）

**性质**：
1. 两点之间，线段最短
2. 连接两点的线段的长度，叫做这两点的距离

**线段的中点**：
- 定义：把一条线段分成两条相等线段的点
- 公式：若点 M 是线段 AB 的中点，则 AM = MB = ½AB

#### 6.2.4 直线、射线、线段的区别与联系

| | 直线 | 射线 | 线段 |
|---|---|---|---|
| 端点个数 | 0个 | 1个 | 2个 |
| 延伸情况 | 向两方无限延伸 | 向一方无限延伸 | 不延伸 |
| 可否度量 | 不可 | 不可 | 可以 |
| 表示 | 直线AB或直线BA | 射线OA（端点在前） | 线段AB或线段BA |

### 6.3 角

#### 6.3.1 角的概念

**定义**：有公共端点的两条射线组成的图形叫做角

**组成**：
- **顶点**：公共端点
- **边**：两条射线

**表示方法**：
1. 用三个大写字母表示：∠AOB（顶点字母O在中间）
2. 用一个大写字母表示：∠O（当顶点处只有一个角时）
3. 用数字或希腊字母表示：∠1，∠α

#### 6.3.2 角的度量

**单位**：度、分、秒
- 1° = 60'
- 1' = 60"
- 1周角 = 360°
- 1平角 = 180°
- 1直角 = 90°

**角的分类**：
- **锐角**：大于0°小于90°的角
- **直角**：等于90°的角
- **钝角**：大于90°小于180°的角
- **平角**：等于180°的角
- **周角**：等于360°的角

#### 6.3.3 角的比较与运算

**角的比较**：
- 叠合法：将两个角的顶点和一边重合，比较另一边的位置
- 度量法：用量角器度量角的度数

**角的和差倍分**：
- ∠AOB + ∠BOC = ∠AOC（当OB在∠AOC内部时）
- ∠AOC - ∠BOC = ∠AOB（当OB在∠AOC内部时）

**角的平分线**：
- 定义：从一个角的顶点出发，把这个角分成两个相等的角的射线
- 公式：若射线 OC 是∠AOB的平分线，则∠AOC = ∠BOC = ½∠AOB

#### 6.3.4 余角和补角

**余角**：
- 定义：如果两个角的和等于90°，就说这两个角互为余角
- 性质：同角（或等角）的余角相等

**补角**：
- 定义：如果两个角的和等于180°，就说这两个角互为补角
- 性质：同角（或等角）的补角相等

**例子**：
- 30°的余角是60°（因为30° + 60° = 90°）
- 30°的补角是150°（因为30° + 150° = 180°）
- 若∠1 + ∠2 = 90°，∠2 + ∠3 = 90°，则∠1 = ∠3（同角的余角相等）

### 例题2：正方体的展开图

**题目**：下列图形中，是正方体展开图的是（  ）
A. 四个正方形排成一排，两端各1个正方形
B. 五个正方形排成一排，一端1个正方形
C. 两个正方形一排，共两排
D. 三个正方形排成一排，一端1个正方形，另一端2个正方形

**解答**：
**答案：A、D**

解析：
- A是"一四一"型，是正方体的展开图
- D是"二三一"型，是正方体的展开图
- B有6个正方形，但排列方式不对
- C只有4个正方形，不是正方体的展开图

### 例题4：线段的计算

**题目**：如图，点C是线段AB的中点，点D是线段AC的中点，若AB = 8cm，求CD的长度。

**解答**：
因为点C是线段AB的中点，
所以 AC = ½AB = ½ × 8 = 4（cm）

又因为点D是线段AC的中点，
所以 CD = ½AC = ½ × 4 = 2（cm）

答：CD的长度是2cm。

---

### 例题5：两点之间的距离

**题目**：如图，点B在线段AC上，AB = 5cm，BC = 3cm，求AC的长度。

**解答**：
AC = AB + BC = 5 + 3 = 8（cm）

答：AC的长度是8cm。

---

### 例题6：角的表示

**题目**：如图，图中以O为顶点的角有哪些？请把它们表示出来。

**解答**：
以O为顶点的角有：
- ∠AOB（或∠1）
- ∠BOC（或∠2）
- ∠AOC（或∠AOB + ∠BOC）

### 例题7：角的度数计算

**题目**：
（1）把30°15′化成度
（2）把15.4°化成度分秒

**解答**：
（1）30°15′ = 30 + 15/60 = 30 + 0.25 = **30.25°**

（2）15.4° = 15° + 0.4° = 15° + 0.4 × 60′ = 15° + 24′ = **15°24′**

---

### 例题8：角的和差计算

**题目**：如图，∠AOB = 60°，∠BOC = 40°，求∠AOC的度数。

**解答**：
∠AOC = ∠AOB + ∠BOC = 60° + 40° = **100°**

---

### 例题9：角的平分线

**题目**：如图，OC是∠AOB的平分线，∠AOB = 80°，求∠AOC和∠BOC的度数。

**解答**：
因为OC是∠AOB的平分线，
所以 ∠AOC = ∠BOC = ½∠AOB = ½ × 80° = **40°**

答：∠AOC = 40°，∠BOC = 40°。

---

### 例题10：余角和补角

**题目**：
（1）一个角的余角是35°，求这个角的度数。
（2）一个角的补角是120°，求这个角的度数。

**解答**：
（1）设这个角为 x°
    根据题意：x + 35 = 90
    解得：x = 55

    答：这个角是55°。

（2）设这个角为 x°
    根据题意：x + 120 = 180
    解得：x = 60

    答：这个角是60°。

---

### 例题11：余角和补角的性质

**题目**：已知∠1与∠2互余，∠2与∠3互余，求证：∠1 = ∠3。

**证明**：
因为∠1与∠2互余，所以 ∠1 + ∠2 = 90°
又因为∠2与∠3互余，所以 ∠2 + ∠3 = 90°
所以 ∠1 + ∠2 = ∠2 + ∠3
所以 ∠1 = ∠3

---

### 例题12：时钟问题

**题目**：
（1）时钟3点整时，时针和分针的夹角是多少度？
（2）从3点整到3点30分，分针转了多少度？

**解答**：
（1）钟面上有12个大格，每格30°
    3点整时，时针指向3，分针指向12
    时针和分针之间有3格
    夹角 = 3 × 30° = **90°**

（2）分针每分钟转6°（360° ÷ 60 = 6°）
    30分钟分针转：30 × 6° = **180°**

## 💡 知识结构图

```
几何图形初步
├── 几何图形
│   ├── 立体图形：长方体、圆柱、圆锥、球
│   └── 平面图形：线段、角、三角形、长方形

├── 直线、射线、线段
│   ├── 直线：无端点，向两方无限延伸
│   ├── 射线：一个端点，向一方无限延伸
│   └── 线段：两个端点，可度量
│
└── 角
    ├── 概念：两条有公共端点的射线
    ├── 度量：度、分、秒
    ├── 分类：锐角、直角、钝角、平角、周角
    └── 关系：余角、补角、角平分线
```

3. **区分概念**：注意直线、射线、线段的区别

5. **角的度量**：熟练掌握量角器的使用
