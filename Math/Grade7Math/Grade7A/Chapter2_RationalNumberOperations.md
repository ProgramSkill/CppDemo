## 📖 引言：从小学运算到有理数运算

### 2.1 回顾小学的运算

在小学里，我们已经学习了正数的加减乘除运算：

**加法**：5 + 3 = 8（表示5和3合并）
**减法**：8 - 5 = 3（表示从8中去掉5）
**乘法**：5 × 3 = 15（表示3个5相加）
**除法**：15 ÷ 5 = 3（表示15平均分成5份）

这些运算在日常生活中经常使用：
- 计算：苹果有5个，又买了3个，一共有多少个？
- 测量：一根绳子长8米，剪去5米，还剩多少米？
- 购物：一本笔记本5元，买3本需要多少钱？

### 2.2 新的问题：引入负数后

现在我们认识了负数，新的问题出现了：

**问题1**：温度从-3℃上升了5℃，现在的温度是多少？
- 在小学里，我们只会算 3 + 5 = 8
- 但现在是 (-3) + 5，该怎么算？

**问题2**：潜水艇在水下50米处，又下潜了20米，现在有多深？
- 如果向下为负，这是 (-50) + (-20)，结果是多少？

**问题3**：小明有5元，花了8元，还剩多少？
- 这是 5 - 8，但5比8小，小学里说"不够减"
- 现在有了负数，这个减法可以做吗？

**问题4**：(-3) × 5 表示什么？结果是多少？
**问题5**：15 ÷ (-3) 表示什么？结果是多少？

为了解决这些问题，我们需要建立有理数的运算法则。

### 2.3 本章的学习路径

1. **加法**：从最简单的同号相加开始，逐步过渡到异号相加
2. **减法**：将减法转化为加法

## 2.1 有理数的加法

### 2.1.1 同号两数相加

#### 正数 + 正数

**这是小学已经学过的内容**

**例子1**：5 + 3 = ?

**分析**：
- 5表示有5个单位
- 3表示有3个单位
- 合起来是5 + 3 = 8个单位

**结果**：5 + 3 = 8

**图示**：
```
★★★★★ + ★★★ = ★★★★★★★★
  5个      3个       8个
```

#### 负数 + 负数

**这是新内容！**

**例子2**：(-3) + (-5) = ?

**思考**：
- (-3)表示欠3元
- (-5)表示欠5元
- 一共欠多少元？

**直观理解**：欠3元，又欠5元，一共欠8元
所以：(-3) + (-5) = -8

**图示**（用"▼"表示欠债）：
```
▼▼▼ + ▼▼▼▼▼ = ▼▼▼▼▼▼▼▼
 -3      -5       -8
```

**总结法则**：
**同号两数相加，取相同的符号，并把绝对值相加**

**详细步骤**：
1. 确定符号：两数同号，结果取这个符号
2. 计算绝对值：将两个数的绝对值相加
3. 写出结果：符号 + 绝对值

**例3**：计算 (-8) + (-12)

**步骤1**：确定符号
- 两个数都是负数
- 结果取负号

**步骤2**：计算绝对值
- |-8| = 8
- |-12| = 12
- 8 + 12 = 20

**步骤3**：写出结果
- (-8) + (-12) = -20

### 2.1.2 异号两数相加

#### 正数 + 负数

**这是难点！需要仔细理解**

**例子4**：5 + (-3) = ?

**生活情境**：
- 早上有5元（+5）
- 花费了3元（-3）
- 还剩多少元？

**直观理解**：
- 有5元，花掉3元，还剩2元
- 所以：5 + (-3) = 2

**图示**：
```
收入：★★★★★（5个）
支出：▼▼▼（3个）
结果：★★（2个）
```

**另一种理解**：
- 5 + (-3) 可以看作 5 - 3
- 5 - 3 = 2
- 所以：5 + (-3) = 2

**例子5**：(-5) + 3 = ?

**生活情境**：
- 早上欠5元（-5）
- 赚了3元（+3）
- 现在的财务状况如何？

**直观理解**：
- 欠5元，还了3元，还欠2元
- 所以：(-5) + 3 = -2

**图示**：
```
欠款：▼▼▼▼▼（5个）
还款：★★★（3个）
结果：▼▼（2个，还欠2元）
```

**总结法则**：
**异号两数相加，取绝对值较大的加数的符号，并用较大的绝对值减去较小的绝对值**

**详细步骤**：
1. 比较绝对值：|5| = 5，|(-3)| = 3
2. 确定符号：绝对值大的（5）对应的数是正数，结果取正号
3. 计算差值：5 - 3 = 2
4. 写出结果：5 + (-3) = 2

**例6**：计算 (-8) + 5

**步骤1**：比较绝对值
- |-8| = 8
- |5| = 5
- 8 > 5

**步骤2**：确定符号
- 绝对值大的是|-8| = 8，对应的数是-8（负数）
- 结果取负号

**步骤3**：计算差值
- 8 - 5 = 3

**步骤4**：写出结果
- (-8) + 5 = -3

#### 互为相反数的两个数相加

**特殊情况**

**例子7**：5 + (-5) = ?

**生活情境**：
- 有5元（+5）
- 欠5元（-5）
- 财务状况如何？

**直观理解**：
- 有5元，欠5元，刚好抵消
- 结果是0，不欠不赚
- 所以：5 + (-5) = 0

**图示**：
```
有：★★★★★（5个）
欠：▼▼▼▼▼（5个）
结果：0（刚好抵消）
```

**总结法则**：
**互为相反数的两个数相加得0**

**数学表达**：a + (-a) = 0

### 2.1.3 任何数与0相加

**法则**：**一个数同0相加，仍得这个数**

**例子8**：
- 5 + 0 = 5（有5元，没变化，还是5元）
- (-5) + 0 = -5（欠5元，没变化，还是欠5元）
- 0 + 0 = 0

**数学表达**：a + 0 = a

### 2.1.4 加法法则的总结

让我们把所有情况总结一下：

**有理数加法法则**：

1. **同号两数相加**
   - 取相同的符号
   - 并把绝对值相加

   例如：
   - 5 + 3 = 8
   - (-5) + (-3) = -8

2. **异号两数相加**
   - 取绝对值较大的加数的符号
   - 并用较大的绝对值减去较小的绝对值

   例如：
   - 5 + (-3) = 2
   - (-5) + 3 = -2

3. **互为相反数的两个数相加得0**
   - a + (-a) = 0

4. **一个数同0相加，仍得这个数**
   - a + 0 = a

### 2.1.5 加法运算律

#### 交换律

**回顾小学**：3 + 5 = 5 + 3 = 8

**有理数也满足交换律**：
a + b = b + a

**验证**：
- (-5) + 3 = -2
- 3 + (-5) = -2
- 所以：(-5) + 3 = 3 + (-5)

**意义**：加数的位置可以交换

#### 结合律

**回顾小学**：(2 + 3) + 4 = 2 + (3 + 4) = 9

**有理数也满足结合律**：
(a + b) + c = a + (b + c)

**验证**：
- [(-8) + 5] + (-3) = (-3) + (-3) = -6
- (-8) + [5 + (-3)] = (-8) + 2 = -6
- 所以：[(-8) + 5] + (-3) = (-8) + [5 + (-3)]

**意义**：加数可以任意结合

#### 运算律的应用

**应用1：简化计算**

**例9**：计算 (-8) + (-5) + 8 + (-6)

**方法1**：从左到右
(-8) + (-5) + 8 + (-6)
= (-13) + 8 + (-6)
= (-5) + (-6)
= -11

**方法2**：运用交换律和结合律（更简便）
(-8) + (-5) + 8 + (-6)
= [(-8) + 8] + [(-5) + (-6)]
= 0 + (-11)
= -11

**技巧**：
- 把互为相反数的数先相加
- 把同号的数先相加

**应用2：凑整法**

**例10**：计算 (-32) + 18 + (-68) + 52

**分析**：
- -32和-68可以凑成-100
- 18和52可以凑成70

**解法**：
(-32) + 18 + (-68) + 52
= [(-32) + (-68)] + (18 + 52)
= (-100) + 70
= -30

### 2.1.6 加法的常见错误

❌ **错误1**：异号相加时，符号确定错误
- 错误：(-5) + 3 = 2（符号错了）
- 正确：(-5) + 3 = -2（绝对值大的是|-5|=5，取负号）

❌ **错误2**：忘记比较绝对值
- 错误：5 + (-3) = -8（直接把绝对值相加了）
- 正确：5 + (-3) = 2（应该用大的绝对值减去小的）

❌ **错误3**：0的加法
- 错误：0 + (-5) = 0
- 正确：0 + (-5) = -5

✅ **记忆口诀**：
同号相加符号不变，绝对值相加
异号相加大数定号，大减小数来计算

## 2.2 有理数的减法

### 2.2.1 减法的困惑

**小学里的减法**：
- 被减数 ≥ 减数
- 例如：8 - 5 = 3，10 - 7 = 3

**问题**：5 - 8 = ?
- 小学说"不够减"
- 但生活中常有这种情况：
  - 有5元，花了8元，结果是什么？
  - 应该是欠3元，即-3

为了解决这个问题，我们需要建立有理数的减法法则。

### 2.2.2 减法法则

**核心思想**：**减去一个数，等于加上这个数的相反数**

**数学表达**：a - b = a + (-b)

**理解**：
- 减法转化为加法
- 减数变成它的相反数
- 然后按加法法则计算

### 2.2.3 减法的计算步骤

**步骤1**：将减法转化为加法
- 减号变加号
- 减数变为其相反数

**步骤2**：按加法法则计算

### 2.2.4 减法的各种情况

#### 情况1：正数减正数

**例11**：5 - 3 = ?

**方法**：转化为加法
5 - 3 = 5 + (-3)

**计算**：
5 + (-3) = 2（异号，大数定号，大减小）

**结果**：5 - 3 = 2

这与小学一致 ✓

#### 情况2：正数减较大的正数

**例12**：3 - 5 = ?

**方法**：转化为加法
3 - 5 = 3 + (-5)

**计算**：
|3| = 3，|(-5)| = 5
3 < 5，取负号
5 - 3 = 2
所以：3 + (-5) = -2

**结果**：3 - 5 = -2

**生活意义**：
- 有3元，花了5元
- 欠2元
- 所以是-2

#### 情况3：正数减负数

**例13**：5 - (-3) = ?

**方法**：转化为加法
5 - (-3) = 5 + 3

**计算**：
5 + 3 = 8

**结果**：5 - (-3) = 8

**生活意义**：
- 有5元
- 免除了3元的债务（-3的相反是+3）
- 相当于又多了3元
- 所以是8元

**记忆**：减去负数等于加上正数（负负得正）

#### 情况4：负数减正数

**例14**：(-5) - 3 = ?

**方法**：转化为加法
(-5) - 3 = (-5) + (-3)

**计算**：
(-5) + (-3) = -8（同号相加）

**结果**：(-5) - 3 = -8

**生活意义**：
- 欠5元
- 又欠3元
- 一共欠8元

#### 情况5：负数减负数

**例15**：(-5) - (-3) = ?

**方法**：转化为加法
(-5) - (-3) = (-5) + 3

**计算**：
(-5) + 3 = -2（异号）

**结果**：(-5) - (-3) = -2

**生活意义**：
- 欠5元
- 免除了3元的债务
- 还欠2元

#### 情况6：一个数减0

**法则**：a - 0 = a

**例子**：
- 5 - 0 = 5
- (-5) - 0 = -5

#### 情况7：0减去一个数

**法则**：0 - a = -a（0减去任何数等于这个数的相反数）

**例子**：
- 0 - 5 = -5
- 0 - (-5) = 5

**记忆**：0减去正数得负数，0减去负数得正数

### 2.2.5 减法法则总结

**有理数减法法则**：
**减去一个数，等于加上这个数的相反数**

**数学表达**：a - b = a + (-b)

**各种情况的汇总**：

| 减法 | 转化为加法 | 结果 |
|------|-----------|------|
| 5 - 3 | 5 + (-3) | 2 |
| 3 - 5 | 3 + (-5) | -2 |
| 5 - (-3) | 5 + 3 | 8 |
| (-5) - 3 | (-5) + (-3) | -8 |
| (-5) - (-3) | (-5) + 3 | -2 |
| 5 - 0 | 5 + 0 | 5 |
| 0 - 5 | 0 + (-5) | -5 |

### 2.2.6 加减法混合运算

**策略**：统一为加法

**步骤**：
1. 将所有减法转化为加法
2. 应用加法交换律和结合律
3. 按加法法则计算

**例16**：计算 (-8) - (-5) + (-3) - 6

**步骤1**：统一为加法
= (-8) + 5 + (-3) + (-6)

**步骤2**：运用交换律和结合律
= [(-8) + (-3) + (-6)] + 5
= (-17) + 5

**步骤3**：计算
= -12

**例17**：计算 |(-5)| - |(-3)| + |2| - |(-4)|

**步骤1**：先算绝对值
= 5 - 3 + 2 - 4

**步骤2**：统一为加法
= 5 + (-3) + 2 + (-4)

**步骤3**：运用交换律和结合律
= (5 + 2) + [(-3) + (-4)]
= 7 + (-7)
= 0

### 2.2.7 减法的常见错误

❌ **错误1**：忘记把减号变成加号
- 错误：5 - (-3) = 5 - 3 = 2
- 正确：5 - (-3) = 5 + 3 = 8

❌ **错误2**：忘记把减数变成相反数
- 错误：5 - (-3) = 5 + (-3) = 2
- 正确：5 - (-3) = 5 + 3 = 8

❌ **错误3**：符号混乱
- 错误：(-5) - (-3) = (-5) - 3 = -8
- 正确：(-5) - (-3) = (-5) + 3 = -2

✅ **记忆口诀**：
减变加，后面变相反
符号要变两处
减号变加号
减数变相反

## 2.3 有理数的乘法

### 2.3.1 乘法的引入

**回顾小学**：乘法是"几个相同加数的和的简便运算"

例如：5 × 3 = 5 + 5 + 5 = 15

**问题**：(-5) × 3 = ? 表示什么？

**思考**：
- 5 × 3 表示3个5相加
- (-5) × 3 应该表示3个(-5)相加

**计算**：
(-5) × 3 = (-5) + (-5) + (-5) = -15

所以：**负数 × 正数 = 负数**

### 2.3.2 乘法法则

**法则**：**两数相乘，同号得正，异号得负，并把绝对值相乘**

#### 正数 × 正数

**例18**：5 × 3 = 15

- 同号（都是正号）→ 得正
- 绝对值相乘：5 × 3 = 15
- 结果：15

#### 负数 × 负数

**例19**：(-5) × (-3) = ?

**推理**：
- 我们知道：(-5) × 3 = -15（异号得负）
- 如果我们把3换成(-3)，符号要改变
- 所以：(-5) × (-3) = 15（同号得正）

**结果**：(-5) × (-3) = 15

#### 正数 × 负数

**例20**：5 × (-3) = ?

**方法1**：用乘法交换律
5 × (-3) = (-3) × 5 = -15

**方法2**：直接用法则
- 异号 → 得负
- 绝对值相乘：5 × 3 = 15
- 结果：-15

**结果**：5 × (-3) = -15

#### 负数 × 正数

**例21**：(-5) × 3 = ?

**计算**：
- 异号 → 得负
- 绝对值相乘：5 × 3 = 15
- 结果：-15

**验证**：
(-5) × 3 = (-5) + (-5) + (-5) = -15 ✓

#### 与0相乘

**法则**：**任何数同0相乘，都得0**

**例子**：
- 5 × 0 = 0
- (-5) × 0 = 0
- 0 × 0 = 0

### 2.3.3 多个有理数相乘

**符号规则**：
- **负因数有偶数个时，积为正**
- **负因数有奇数个时，积为负**

**例22**：判断下列各式的符号
（1）(-2) × (-3) × 4 → 2个负因数（偶数）→ 积为正
（2）(-2) × 3 × 4 → 1个负因数（奇数）→ 积为负
（3）(-2) × (-3) × (-4) → 3个负因数（奇数）→ 积为负
（4）(-2) × (-3) × (-4) × (-5) → 4个负因数（偶数）→ 积为正

**计算**：
（1）(-2) × (-3) × 4 = 24
（2）(-2) × 3 × 4 = -24
（3）(-2) × (-3) × (-4) = -24
（4）(-2) × (-3) × (-4) × (-5) = 120

**记忆口诀**：
偶数个负，积为正
奇数个负，积为负
有个因子等于0，积一定是0

### 2.3.4 乘法运算律

#### 交换律

**法则**：a × b = b × a

**验证**：
(-5) × 3 = -15
3 × (-5) = -15
所以：(-5) × 3 = 3 × (-5)

#### 结合律

**法则**：(a × b) × c = a × (b × c)

**验证**：
[(-5) × 3] × (-2) = (-15) × (-2) = 30
(-5) × [3 × (-2)] = (-5) × (-6) = 30
所以：[(-5) × 3] × (-2) = (-5) × [3 × (-2)]

#### 分配律

**法则**：a × (b + c) = a × b + a × c

**验证**：
5 × [3 + (-2)] = 5 × 1 = 5
5 × 3 + 5 × (-2) = 15 + (-10) = 5
所以：5 × [3 + (-2)] = 5 × 3 + 5 × (-2)

**应用**：
**例23**：计算 (-5) × 6 + (-5) × 4

**方法1**：直接计算
= (-30) + (-20) = -50

**方法2**：运用分配律（逆向）
= (-5) × (6 + 4)
= (-5) × 10
= -50

### 2.3.5 倒数

**定义**：**乘积是1的两个数互为倒数**

**例子**：
- 5的倒数是⅕（因为 5 × ⅕ = 1）
- -5的倒数是-⅕（因为 (-5) × (-⅕) = 1）
- ⅔的倒数是³⁄₂
- ½的倒数是2
- 1的倒数是1（因为 1 × 1 = 1）
- -1的倒数是-1（因为 (-1) × (-1) = 1）
- 0没有倒数（0不能作除数）

**注意**：
- 正数的倒数是正数
- 负数的倒数是负数
- 求倒数的方法：分子分母颠倒位置

### 2.3.6 乘法的常见错误

❌ **错误1**：符号判断错误
- 错误：(-5) × (-3) = -15
- 正确：(-5) × (-3) = 15（同号得正）

❌ **错误2**：多个负数相乘时符号错误
- 错误：(-2) × (-3) × (-4) = 24
- 正确：(-2) × (-3) × (-4) = -24（3个负数，奇数个，积为负）

❌ **错误3**：与0相乘
- 错误：(-5) × 0 = -5
- 正确：(-5) × 0 = 0

✅ **记忆口诀**：
同号得正异号负
绝对值相乘别马虎
偶负正奇负记住
遇零得零不犯愁

## 2.4 有理数的除法

### 2.4.1 除法的引入

**回顾小学**：除法是"已知积和其中一个因数，求另一个因数的运算"

例如：15 ÷ 5 = 3（因为 5 × 3 = 15）

**问题**：(-15) ÷ 5 = ? 15 ÷ (-5) = ? (-15) ÷ (-5) = ?

**思考**：
- (-15) ÷ 5 = ? 因为 5 × (-3) = -15，所以 (-15) ÷ 5 = -3
- 15 ÷ (-5) = ? 因为 (-5) × (-3) = 15，所以 15 ÷ (-5) = -3
- (-15) ÷ (-5) = ? 因为 (-5) × 3 = -15，所以 (-15) ÷ (-5) = 3

### 2.4.2 除法法则

**法则一**：**除以一个不等于0的数，等于乘这个数的倒数**

**数学表达**：a ÷ b = a × (1/b)  (b ≠ 0)

**法则二**：**两数相除，同号得正，异号得负，并把绝对值相除**

**0除以任何一个不等于0的数，都得0**

**注意**：**0不能作除数，0没有倒数**

### 2.4.3 除法的各种情况

#### 正数 ÷ 正数

**例24**：15 ÷ 5 = 3

- 同号 → 得正
- 绝对值相除：15 ÷ 5 = 3
- 结果：3

#### 负数 ÷ 正数

**例25**：(-15) ÷ 5 = ?

**方法1**：转化为乘法
= (-15) × (⅕) = -3

**方法2**：直接用法则
- 异号 → 得负
- 绝对值相除：15 ÷ 5 = 3
- 结果：-3

#### 正数 ÷ 负数

**例26**：15 ÷ (-5) = ?

**方法**：转化为乘法
= 15 × (-⅕) = -3

#### 负数 ÷ 负数

**例27**：(-15) ÷ (-5) = ?

**方法**：转化为乘法
= (-15) × (-⅕) = 3

#### 0除以任何数

**例28**：
- 0 ÷ 5 = 0
- 0 ÷ (-5) = 0

### 2.4.4 乘除混合运算

**策略**：将除法转化为乘法

**步骤**：
1. 将除法转化为乘法（除以一个数 = 乘这个数的倒数）
2. 确定积的符号
3. 计算绝对值的乘积

**例29**：计算 (-2) × (-3) ÷ (-6) × 5

**步骤1**：转化为乘法
= (-2) × (-3) × (-⅙) × 5

**步骤2**：确定符号
有3个负因数（奇数个）→ 积为负

**步骤3**：计算
= -[2 × 3 × ⅙ × 5]
= -[1 × 5]
= -5

### 2.4.5 除法的常见错误

❌ **错误1**：符号判断错误
- 错误：(-15) ÷ (-5) = -3
- 正确：(-15) ÷ (-5) = 3（同号得正）

❌ **错误2**：忘记倒数
- 错误：(-15) ÷ (-5) = (-15) × (-5) = 75
- 正确：(-15) ÷ (-5) = (-15) × (-⅕) = 3

❌ **错误3**：0作除数
- 错误：5 ÷ 0 = 0
- 正确：0不能作除数，5 ÷ 0 无意义

✅ **记忆口诀**：
除法变乘法
除数变倒数
符号看原来
计算别忘掉

## 2.5 有理数的乘方

### 2.5.1 乘方的引入

**问题情境**：

边长为2cm的正方形，面积是多少？
- 面积 = 2 × 2 = 4 cm²

边长为2cm的立方体，体积是多少？
- 体积 = 2 × 2 × 2 = 8 cm³

**问题**：如果有10个2相乘，该怎么写？

2 × 2 × 2 × 2 × 2 × 2 × 2 × 2 × 2 × 2

这样写太长了！数学家们发明了**乘方**来简化这种表示。

### 2.5.2 乘方的概念

**定义**：**求n个相同因数的积的运算，叫做乘方**

**一般形式**：
```
a × a × a × ... × a = aⁿ
           └── n个 ──┘
```

**各部分名称**：
- **a** 叫做**底数**（表示要相乘的因数）
- **n** 叫做**指数**（表示因数的个数）
- **aⁿ** 叫做**幂**（运算的结果）

**读法**：
- aⁿ 读作 "a的n次方" 或 "a的n次幂"

**例子**：
- 2³ = 2 × 2 × 2 = 8（读作：2的3次方）
- 5² = 5 × 5 = 25（读作：5的2次方，也叫5的平方）
- 2⁴ = 2 × 2 × 2 × 2 = 16（读作：2的4次方）

### 2.5.3 乘方的符号法则

**规律**：

1. **正数的任何次幂都是正数**
   - 2³ = 8
   - 2¹⁰⁰ = 一个很大的正数

2. **负数的奇次幂是负数**
   - (-2)³ = -8
   - (-2)⁵ = -32
   - **奇数个负数相乘，积为负**

3. **负数的偶次幂是正数**
   - (-2)² = 4
   - (-2)⁴ = 16
   - **偶数个负数相乘，积为正**

**记忆口诀**：
正数乘方总是正
负数奇次幂为负
负数偶次幂为正

### 2.5.4 乘方中的括号问题（重要！）

**这是最易错的地方！**

**对比**：
- (-2)² = (-2) × (-2) = 4 ✓
- -2² = -(2²) = -(2 × 2) = -4 ✓

**区别**：
- (-2)²：底数是-2，指数是2
- -2²：底数是2，指数是2，前面有个负号

**例子**：
- (-3)² = 9（底数是-3）
- -3² = -9（底数是3）
- (-3)³ = -27（底数是-3）
- -3³ = -27（底数是3）

**注意**：
- (-3)² = 9 ≠ -3² = -9
- 但 (-3)³ = -27 = -3³（这个巧合）

**例30**：计算
（1）(-4)³ = -64
（2）-4³ = -64
（3）(-4)² = 16
（4）-4² = -16

### 2.5.5 科学记数法

**引入**：

地球的半径约是6400千米
该怎么写成"一个数乘以10的幂"的形式？

**定义**：**把一个大于10的数表示成 a × 10ⁿ 的形式**（其中1 ≤ |a| < 10，n是正整数）

**确定n的方法**：
- n等于原数的整数位数减1

**例子**：
- 300 000 000 = 3 × 10⁸（整数位数是9，n = 9 - 1 = 8）
- 12 500 = 1.25 × 10⁴（整数位数是5，n = 5 - 1 = 4）
- 6400 = 6.4 × 10³（整数位数是4，n = 4 - 1 = 3）

**应用**：
- 光速约是300 000 000米/秒 = 3 × 10⁸米/秒
- 地球到太阳的距离约是150 000 000千米 = 1.5 × 10⁸千米

## 2.6 有理数的混合运算

### 2.6.1 运算顺序

**原则**：**先乘方，再乘除，最后加减；如有括号，先做括号内的运算**

**详细的运算顺序**：

1. **括号**（从内到外）
   - 小括号()
   - 中括号[]
   - 大括号{}

2. **乘方**（最高级运算）

3. **乘除**（二级运算，从左到右）

4. **加减**（一级运算，从左到右）

### 2.6.2 混合运算的例子

**例31**：计算 (-2)² + 4 × (-3) - (-6) ÷ 2

**步骤1**：乘方
= 4 + 4 × (-3) - (-6) ÷ 2

**步骤2**：乘除
= 4 + (-12) - (-3)

**步骤3**：统一为加法
= 4 + (-12) + 3

**步骤4**：加减
= (-8) + 3
= -5

**例32**：计算 -1⁴ - (1 - 0.5) × ⅓ × [2 - (-3)²]

**步骤1**：小括号内
= -1⁴ - 0.5 × ⅓ × [2 - (-3)²]

**步骤2**：乘方
= -1 - 0.5 × ⅓ × [2 - 9]

**步骤3**：中括号内
= -1 - 0.5 × ⅓ × (-7)

**步骤4**：乘除
= -1 - (-0.5 × ⅓ × 7)
= -1 - (-7/6)
= -1 + 7/6
= 1/6

   - 减法变加法
   - 除法变乘法
