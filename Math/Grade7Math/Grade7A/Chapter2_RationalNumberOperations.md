# 第二章 有理数的运算

## 📚 知识要点

### 2.1 有理数的加法

#### 2.1.1 加法法则

**法则一**：同号两数相加，取相同的符号，并把绝对值相加

- 正数 + 正数 = 正数
  - 例如：5 + 3 = 8

- 负数 + 负数 = 负数
  - 例如：(-5) + (-3) = -8
  - 解：|(-5)| = 5，|(-3)| = 3
        5 + 3 = 8，取负号
        所以 (-5) + (-3) = -8

**法则二**：绝对值不相等的异号两数相加，取绝对值较大的加数的符号，并用较大的绝对值减去较小的绝对值

- 例如：5 + (-3) = 2
  - 解：|5| = 5，|(-3)| = 3
        5 > 3，取正号
        5 - 3 = 2
        所以 5 + (-3) = 2

- 例如：(-5) + 3 = -2
  - 解：|(-5)| = 5，|3| = 3
        5 > 3，取负号
        5 - 3 = 2
        所以 (-5) + 3 = -2

**法则三**：互为相反数的两个数相加得0

- 例如：5 + (-5) = 0
- 例如：(-8) + 8 = 0

**法则四**：一个数同0相加，仍得这个数

- 例如：5 + 0 = 5
- 例如：(-5) + 0 = -5

#### 2.1.2 加法运算律

**交换律**：两个数相加，交换加数的位置，和不变
```
a + b = b + a
```
例如：(-5) + 3 = 3 + (-5) = -2

**结合律**：三个数相加，先把前两个数相加，或者先把后两个数相加，和不变
```
(a + b) + c = a + (b + c)
```
例如：[(-8) + 5] + (-3) = (-8) + [5 + (-3)] = -6

**应用**：
- 简化计算
- 把同号数先相加
- 把互为相反数的数先相加
- 把同分母的数先相加

### 2.2 有理数的减法

#### 2.2.1 减法法则

**法则**：减去一个数，等于加上这个数的相反数
```
a - b = a + (-b)
```

**说明**：
- 有理数的减法可以转化为加法
- 0减去一个数，等于这个数的相反数
- 减法没有交换律和结合律

**计算步骤**：
1. 将减法转化为加法（减号变加号，减数变为其相反数）
2. 按加法法则计算

**例子**：
- 5 - 3 = 5 + (-3) = 2
- 5 - (-3) = 5 + 3 = 8
- (-5) - 3 = (-5) + (-3) = -8
- (-5) - (-3) = (-5) + 3 = -2
- 0 - 5 = 0 + (-5) = -5

#### 2.2.2 加减法混合运算

**步骤**：
1. 将减法统一为加法
2. 应用加法交换律和结合律简化计算
3. 按加法法则计算

**例子**：
(-8) - (-5) + (-3) - 6
= (-8) + 5 + (-3) + (-6)
= [(-8) + (-3) + (-6)] + 5
= (-17) + 5
= -12

### 2.3 有理数的乘法

#### 2.3.1 乘法法则

**法则一**：两数相乘，同号得正，异号得负，并把绝对值相乘

- 正数 × 正数 = 正数
  - 例如：5 × 3 = 15

- 负数 × 负数 = 正数
  - 例如：(-5) × (-3) = 15

- 正数 × 负数 = 负数
  - 例如：5 × (-3) = -15

- 负数 × 正数 = 负数
  - 例如：(-5) × 3 = -15

**法则二**：任何数同0相乘，都得0

- 例如：5 × 0 = 0
- 例如：(-5) × 0 = 0

**注意**：几个不等于0的数相乘，积的符号由负因数的个数决定
- 负因数有**偶数**个时，积为**正**
- 负因数有**奇数**个时，积为**负**

例如：
- (-2) × (-3) × 4 = 24（2个负因数，积为正）
- (-2) × 3 × 4 = -24（1个负因数，积为负）
- (-2) × (-3) × (-4) = -24（3个负因数，积为负）

#### 2.3.2 乘法运算律

**交换律**：两个数相乘，交换因数的位置，积不变
```
a × b = b × a
```
例如：(-5) × 3 = 3 × (-5) = -15

**结合律**：三个数相乘，先把前两个数相乘，或者先把后两个数相乘，积不变
```
(a × b) × c = a × (b × c)
```
例如：[(-5) × 3] × (-2) = (-5) × [3 × (-2)] = 30

**分配律**：一个数同两个数的和相乘，等于把这个数分别同这两个数相乘，再把积相加
```
a × (b + c) = a × b + a × c
```
例如：5 × [3 + (-2)] = 5 × 3 + 5 × (-2) = 15 + (-10) = 5

#### 2.3.3 倒数

**定义**：乘积是1的两个数互为倒数

**例子**：
- 5的倒数是 ⅕（因为 5 × ⅕ = 1）
- -5的倒数是 -⅕（因为 (-5) × (-⅕) = 1）
- ⅔的倒数是 ³⁄₂
- 0没有倒数

**注意**：
- 正数的倒数是正数
- 负数的倒数是负数
- 1的倒数是1，-1的倒数是-1

### 2.4 有理数的除法

#### 2.4.1 除法法则

**法则一**：除以一个不等于0的数，等于乘这个数的倒数
```
a ÷ b = a × (1/b)  (b ≠ 0)
```

**法则二**：两数相除，同号得正，异号得负，并把绝对值相除
- 0除以任何一个不等于0的数，都得0

**注意**：0不能作除数，0没有倒数

**例子**：
- 6 ÷ (-3) = 6 × (-⅓) = -2
- (-6) ÷ (-3) = (-6) × (-⅓) = 2
- (-6) ÷ 3 = (-6) × ⅓ = -2
- 0 ÷ (-5) = 0

#### 2.4.2 有理数的乘除混合运算

**步骤**：
1. 将除法转化为乘法
2. 确定积的符号（负因数的个数）
3. 计算绝对值的乘积

**例子**：
(-2) × (-3) ÷ (-6) × 5
= (-2) × (-3) × (-⅙) × 5
= -[2 × 3 × ⅙ × 5]
= -5

### 2.5 有理数的乘方

#### 2.5.1 乘方的概念

**定义**：求n个相同因数的积的运算，叫做乘方

**表示**：
```
a × a × a × ... × a = aⁿ
           └── n个 ──┘
```

其中：
- **a** 叫做底数
- **n** 叫做指数
- **aⁿ** 叫做幂

**读法**：aⁿ 读作 "a的n次方" 或 "a的n次幂"

**例如**：
- 2³ = 2 × 2 × 2 = 8（读作：2的3次方）
- (-2)³ = (-2) × (-2) × (-2) = -8
- (-2)⁴ = (-2) × (-2) × (-2) × (-2) = 16

#### 2.5.2 乘方的符号法则

**规律**：
- 正数的任何次幂都是正数
- 负数的**奇次幂**是**负数**
- 负数的**偶次幂**是**正数**

**例如**：
- 2³ = 8（正数）
- (-2)³ = -8（奇次幂，负数）
- (-2)⁴ = 16（偶次幂，正数）

**注意**：
- (-2)³ = -8（底数是-2，指数是3）
- -2³ = -(2³) = -8（底数是2，指数是3，最后加负号）
- (-2)² = 4（底数是-2，指数是2）
- -2² = -(2²) = -4（底数是2，指数是2，最后加负号）

#### 2.5.3 科学记数法

**定义**：把一个大于10的数表示成 a × 10ⁿ 的形式（其中1 ≤ a < 10，n是正整数），这种记数方法叫做科学记数法

**确定n的方法**：
- n等于原数的整数位数减1

**例如**：
- 300 000 000 = 3 × 10⁸
- 12 500 = 1.25 × 10⁴
- 3.05 × 10⁶ = 3 050 000

### 2.6 有理数的混合运算

#### 2.6.1 运算顺序

**原则**：先乘方，再乘除，最后加减；如有括号，先做括号内的运算

**顺序**：
1. **乘方**（最高级运算）
2. **乘除**（二级运算）
3. **加减**（一级运算）
4. **括号**：从内到外

**例子**：
(-2)² + 4 × (-3) - (-6) ÷ 2
= 4 + (-12) - (-3)
= 4 - 12 + 3
= -8 + 3
= -5

#### 2.6.2 运算技巧

1. **灵活运用运算律**
2. **注意符号变化**
3. **先化简再计算**
4. **巧用分配律**

## 📖 典型例题

### 例题1：有理数的加法

**题目**：计算下列各式
（1）(-8) + (-5)
（2）8 + (-5)
（3）(-8) + 5
（4）8 + (-8)

**解答**：
（1）(-8) + (-5)
     同号两数相加，取负号，绝对值相加
     = -(8 + 5)
     = **-13**

（2）8 + (-5)
     异号两数相加，|8| = 8，|(-5)| = 5，8 > 5，取正号
     = +(8 - 5)
     = **3**

（3）(-8) + 5
     异号两数相加，|(-8)| = 8，|5| = 5，8 > 5，取负号
     = -(8 - 5)
     = **-3**

（4）8 + (-8)
     互为相反数相加
     = **0**

### 例题2：运用运算律简化计算

**题目**：计算
(-18) + 25 + (-32) + 75

**解答**：
(-18) + 25 + (-32) + 75
= [(-18) + (-32)] + (25 + 75)  ← 同号结合
= (-50) + 100
= **50**

### 例题3：有理数的减法

**题目**：计算下列各式
（1）5 - 8
（2）5 - (-8)
（3）(-5) - 8
（4）(-5) - (-8)

**解答**：
（1）5 - 8 = 5 + (-8) = **-3**

（2）5 - (-8) = 5 + 8 = **13**

（3）(-5) - 8 = (-5) + (-8) = **-13**

（4）(-5) - (-8) = (-5) + 8 = **3**

### 例题4：加减混合运算

**题目**：计算
(-12) - (-8) + (-7) - 5

**解答**：
(-12) - (-8) + (-7) - 5
= (-12) + 8 + (-7) + (-5)  ← 减法变加法
= (-12) + (-7) + (-5) + 8  ← 同号结合
= (-24) + 8
= **-16**

### 例题5：有理数的乘法

**题目**：计算下列各式
（1）(-6) × 4
（2）(-6) × (-4)
（3）(-2) × (-3) × (-4)
（4）(-2) × 3 × (-4) × 5

**解答**：
（1）(-6) × 4
     异号得负，绝对值相乘
     = -(6 × 4)
     = **-24**

（2）(-6) × (-4)
     同号得正，绝对值相乘
     = +(6 × 4)
     = **24**

（3）(-2) × (-3) × (-4)
     3个负因数（奇数），积为负
     = -(2 × 3 × 4)
     = **-24**

（4）(-2) × 3 × (-4) × 5
     2个负因数（偶数），积为正
     = +(2 × 3 × 4 × 5)
     = **120**

### 例题6：运用分配律

**题目**：计算
（1）(-5) × [3 + (-7)]
（2）18 × (-4) + (-12) × (-4)

**解答**：
（1）(-5) × [3 + (-7)]
     = (-5) × (-4)
     = **20**

     或者用分配律：
     = (-5) × 3 + (-5) × (-7)
     = (-15) + 35
     = **20**

（2）18 × (-4) + (-12) × (-4)
    逆用分配律：
    = [18 + (-12)] × (-4)
    = 6 × (-4)
    = **-24**

### 例题7：有理数的除法

**题目**：计算下列各式
（1）(-36) ÷ 9
（2）(-36) ÷ (-9)
（3）0 ÷ (-8)
（4）(-1) ÷ (-⅓)

**解答**：
（1）(-36) ÷ 9
     异号得负，绝对值相除
     = -(36 ÷ 9)
     = **-4**

（2）(-36) ÷ (-9)
     同号得正，绝对值相除
     = +(36 ÷ 9)
     = **4**

（3）0 ÷ (-8) = **0**

（4）(-1) ÷ (-⅓)
     = (-1) × (-3)
     = **3**

### 例题8：有理数的乘方

**题目**：计算下列各式
（1）(-3)²
（2）-3²
（3）（-2）³
（4）-2³

**解答**：
（1）(-3)² = (-3) × (-3) = **9**
（底数是-3，指数是2）

（2）-3² = -(3²) = -(3 × 3) = **-9**
（底数是3，指数是2，最后加负号）

（3）（-2）³ = (-2) × (-2) × (-2) = **-8**
（底数是-2，指数是3）

（4）-2³ = -(2³) = -(2 × 2 × 2) = **-8**
（底数是2，指数是3，最后加负号）

### 例题10：混合运算

**题目**：计算
（1）-2² + 5 × (-3) - (-6)
（2）(-3)² × 2 - (-4) × (-2) + 10

**解答**：
（1）-2² + 5 × (-3) - (-6）
     = -4 + (-15) - (-6)
     = -4 - 15 + 6
     = -19 + 6
     = **-13**

（2）(-3)² × 2 - (-4) × (-2) + 10
     = 9 × 2 - 8 + 10
     = 18 - 8 + 10
     = 10 + 10
     = **20**

### 例题11：综合应用

**题目**：
某食品厂从生产的食品罐头中，抽出20听检查质量，将超过标准质量的用正数表示，不足标准质量的用负数表示，结果记录如下：
+10，-5，+8，-3，-8，+12，+5，-7，+9，-10，
-6，+4，-9，+7，+3，-4，-12，+6，-11，+2

（1）这20听罐头的总质量是多少？
（2）平均每听罐头的质量是多少？

**解答**：
（1）正数之和：10 + 8 + 12 + 5 + 9 + 4 + 7 + 3 + 6 + 2 = 66(g)
    负数之和：(-5) + (-3) + (-8) + (-7) + (-10) + (-6) + (-9) + (-4) + (-12) + (-11) = -75(g)

    总质量 = 66 + (-75) = **-9(g)**

（2）平均质量 = (-9) ÷ 20 = **-0.45(g)**

    答：这20听罐头的总质量比标准少9g，平均每听比标准少0.45g。

## 📚 学习建议

1. **掌握运算法则**：理解记忆各类运算的法则，特别是符号规则
2. **注意符号变化**：减法变加法时，减数要变号；除法变乘法时，除数变倒数
3. **区分乘方**：注意 (-a)² 与 -a² 的区别
4. **运用运算律**：合理运用交换律、结合律、分配律可以简化计算
5. **细心计算**：混合运算步骤多，容易出错，要耐心细致
