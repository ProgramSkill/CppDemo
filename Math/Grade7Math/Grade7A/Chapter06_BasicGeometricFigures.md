## 6.1 几何图形

### 6.1.1 什么叫做几何图形？

**定义**：**对于各种各样的物体，如果不管它们的颜色、材料、质量等，而只注意它们的形状、大小、位置，就得到几何图形。**

**理解**：
- 几何图形是对实际物体的抽象
- 只保留形状、大小、位置这些几何属性
- 忽略颜色、材料等非几何属性

**例子**：
- 篮球 → 球体（只看形状）
- 粉笔盒 → 长方体（只看形状）
- 一张纸 → 长方形（忽略厚度）

### 6.1.2 立体图形与平面图形

**立体图形**：**图形的各部分不都在同一平面内**

**常见的立体图形**：

**1. 棱柱**
- **特征**：有两个面互相平行，其余各面都是四边形
- **例子**：三棱柱、四棱柱（长方体）、五棱柱、六棱柱

**2. 棱锥**
- **特征**：一个面是多边形，其余各面是有一个公共顶点的三角形
- **例子**：三棱锥（四面体）、四棱锥（金字塔）

**3. 圆柱**
- **特征**：两个底面是圆，侧面是曲面
- **例子**：水杯、罐头、柱子

**4. 圆锥**
- **特征**：底面是圆，顶点是一个点，侧面是曲面
- **例子**：冰淇淋筒、漏斗

**5. 球**
- **特征**：所有点到球心的距离相等
- **例子**：篮球、足球、地球

**平面图形**：**图形的各部分都在同一平面内**

**常见的平面图形**：
- **线**：线段、射线、直线
- **角**：锐角、直角、钝角
- **多边形**：三角形、四边形、五边形、六边形...
- **圆**：圆、扇形、弧

**例1**：判断下列图形是立体图形还是平面图形

（1）长方体 → **立体图形**（各部分不在同一平面）
（2）三角形 → **平面图形**（在同一平面）
（3）球 → **立体图形**
（4）圆 → **平面图形**
（5）棱锥 → **立体图形**
（6）角 → **平面图形**

**练习**：将下列图形分类
① 长方形 ② 圆柱 ③ 圆 ④ 正方体 ⑤ 线段 ⑥ 圆锥

**平面图形**：① 长方形、③ 圆、⑤ 线段
**立体图形**：② 圆柱、④ 正方体、⑥ 圆锥

### 6.1.3 从不同方向看立体图形

**为什么要从不同方向看？**

**思考**：
- 从正面看圆柱，看到什么？→ 长方形
- 从上面看圆柱，看到什么？→ 圆
- 从侧面看圆柱，看到什么？→ 长方形

**结论**：同一个立体图形，从不同方向看，看到的平面图形可能不同！

**三视图**：**从正面、上面、左面三个方向看立体图形，得到的平面图形叫做三视图。**

**三个视图**：
1. **主视图（正视图）**：从正面看
2. **俯视图**：从上面看
3. **左视图**：从左面看

## 6.2 直线、射线、线段

### 6.2.1 直线

**什么叫做直线？**

**定义**：**向两方无限延伸的直的线叫做直线。**

**理解**：
- "直的线"：不是弯曲的
- "向两方无限延伸"：没有端点，可以无限延伸
- 在实际中，我们只能画出直线的一部分

**直线的表示方法**：

**方法1：用一个小写字母表示**
- 例如：直线 l、直线 m
- 读作："直线 el" 或 "直线 em"

**方法2：用两个大写字母表示**
- 例如：直线 AB、直线 BA
- 注意：这两个字母表示直线上的任意两点
- 读作："直线 AB" 或 "直线 BA"

**注意**：
- 直线 AB 和直线 BA 是同一条直线！
- 表示方法可以交换字母顺序

**图示**：
```
A ←━━━━━━━━━━━━ B
```
这是直线 AB（或直线 BA），两端可以无限延伸

**直线的性质**：

**性质1：两点确定一条直线**

**表述**：**经过平面内两点有一条直线，并且只有一条直线。**

**理解**：
- "有一条"：存在性（一定能画出）
- "只有一条"：唯一性（只能画一条）

**应用**：
- 木匠弹墨线：两点确定一条直线
- 射击瞄准：枪管、目标点、眼睛三点一线
- 植树：两点确定树坑的直线

**性质2：两条直线相交，有且只有一个交点**

**表述**：**两条不同的直线，如果它们相交，那么有且只有一个公共点（交点）。**

**理解**：
- 两条直线最多一个交点
- 如果有两个交点，就是同一条直线了

**例6**：回答下列问题

（1）经过一点A可以画多少条直线？
**答**：无数条
**图示**：
```
    A
  ↙ ↖ ↶
  ╱  ╲   （从A出发可以向各个方向画直线）
```

（2）经过两点A、B可以画多少条直线？
**答**：只有一条
**图示**：
```
A ←━━━━━━ B
```

（3）三条直线两两相交，最多有几个交点？
**答**：3个
**图示**：
```
    ╲
     ╲
      ╲
   ━━━━╱━━━
     ╱
    ╱
```

### 6.2.2 射线

**什么叫做射线？**

**定义**：**直线上的一点和它一旁的部分叫做射线。**

**理解**：
- 射线是直线的一部分
- 有一个端点
- 向一方无限延伸
- 像手电筒的光线

**射线的表示方法**：

**方法1：用一个小写字母表示**
- 例如：射线 l
- 读作："射线 el"

**方法2：用两个大写字母表示**
- 例如：射线 OA
- **注意**：端点字母写在前面！
- 读作："射线 OA"

**重要**：
- 射线 OA ≠ 射线 AO
- 射线 OA：从O出发，经过A，向A方向无限延伸
- 射线 AO：从A出发，经过O，向O方向无限延伸

**图示**：
```
O ━━━━━━━━━━→ A
```
这是射线 OA（端点是O，向A方向延伸）

**射线的性质**：
- 只有一个端点
- 向一方无限延伸
- 不能度量

**例7**：判断下列说法是否正确

（1）射线可以延长
**答**：错误 ✗
**理由**：射线本身已经向一方无限延伸，不需要再延长

（2）射线 AB 和射线 BA 是同一条射线
**答**：错误 ✗
**理由**：射线 AB 的端点是A，射线 BA 的端点是B，不是同一条射线

（3）手电筒的光线类似于射线
**答**：正确 ✓

### 6.2.3 线段

**什么叫做线段？**

**定义**：**直线上两个点和它们之间的部分叫做线段。**

**理解**：
- 线段是直线的一部分
- 有两个端点
- 不能延伸
- 可以度量长度

**线段的表示方法**：

**方法1：用一个小写字母表示**
- 例如：线段 l
- 读作："线段 el"

**方法2：用两个大写字母表示**
- 例如：线段 AB、线段 BA
- 读作："线段 AB" 或 "线段 BA"

**注意**：
- 线段 AB = 线段 BA
- 表示方法可以交换字母顺序

**图示**：
```
A ←━━━━━━→ B
```
这是线段 AB（或线段 BA）

**线段的性质**：

**性质1：两点之间，线段最短**

**表述**：**两点之间的所有连线中，线段最短。**

**理解**：
- 两点之间可以画很多连线（曲线、折线）
- 但线段是最短的
- 简单说：两点之间直线最短

**应用**：
- 走路：尽量走直线，省时间
- 尺子：测量两点之间的距离

**性质2：两点之间的距离**

**定义**：**连接两点的线段的长度，叫做这两点的距离。**

**理解**：
- 距离是一个数量（长度）
- 距离可以用尺子度量
- 距离的单位：厘米(cm)、米(m)、千米(km)等

**例8**：如图，A、B、C三点在一条直线上
```
A ←━ B ━━→ C
```
（1）图中有几条线段？
**答**：3条
- 线段 AB
- 线段 BC
- 线段 AC

（2）哪条线段最短？
**答**：无法判断（需要知道具体长度）

（3）如果 AB = 3cm，BC = 5cm，求 AC
**答**：AC = AB + BC = 3 + 5 = 8cm

**线段的比较**：

**方法1：度量法**
- 用尺子量出长度
- 比较数值大小

**方法2：叠合法**
- 将一条线段移到另一条上
- 比较端点的位置

**例9**：比较下列线段的大小

（1）
```
A ←━━━━→ B
C ←━━━→ D
```
**方法1（度量）**：AB = 4cm，CD = 3cm，所以 AB > CD
**方法2（叠合）**：将CD移到AB上，D点在AB内部，所以 AB > CD

**线段的中点**：

**定义**：**把一条线段分成两条相等线段的点，叫做线段的中点。**

**理解**：
- 中点把线段二等分
- 中点到两个端点的距离相等

**数学表达**：
如果点M是线段AB的中点，那么：
- AM = MB = ½AB
- AB = 2AM = 2MB

**图示**：
```
A ←━ M ━→ B
```
点M是线段AB的中点

**例10**：如图，点C是线段AB的中点，AB = 10cm，求AC和CB的长度。

**解**：
因为点C是线段AB的中点，
所以 AC = CB = ½AB = ½ × 10 = 5（cm）

**答**：AC = 5cm，CB = 5cm。

**例11**：如图，点B在线段AC上，AB = 6cm，BC = 4cm，点D是AB的中点，点E是BC的中点，求DE的长度。

**分析**：
```
A ←━ D ━→ B ←━ E ━→ C
```

**解**：
因为点D是AB的中点，
所以 AD = DB = ½AB = ½ × 6 = 3（cm）

因为点E是BC的中点，
所以 BE = EC = ½BC = ½ × 4 = 2（cm）

所以 DE = DB + BE = 3 + 2 = 5（cm）

**答**：DE的长度是5cm。

### 6.2.4 直线、射线、线段的区别与联系

**联系**：
- 射线和线段都是直线的一部分
- 把线段向一方无限延伸得到射线
- 把线段向两方无限延伸得到直线

**区别对比表**：

| | 直线 | 射线 | 线段 |
|---|---|---|---|
| **端点个数** | 0个 | 1个 | 2个 |
| **延伸情况** | 向两方无限延伸 | 向一方无限延伸 | 不延伸 |
| **可否度量** | 不可 | 不可 | 可以 |
| **表示方法** | 直线AB或直线BA | 射线OA（端点在前） | 线段AB或线段BA |
| **图示** | ←━━━→ | ←━━━→ | ←━━→ |

**记忆口诀**：
直线无端点，两头无限延
射线一端点，一头无限延
线段两端点，不延能量度

**例12**：填空题

（1）如图，A、B、C三点在同一条直线上，图中有____条线段，____条射线，____条直线。

```
A ←━━ B ━━→ C
```

**分析**：
- 线段：AB、BC、AC，共3条
- 射线：以A为端点的射线有2条（左、右），以B为端点的射线有2条，以C为端点的射线有2条，共6条
- 直线：只有1条（ABC所在的直线）

**答案**：3条线段，6条射线，1条直线

（2）经过平面内三点可以画____条直线。

**分析**：
- 如果三点在同一条直线上，可以画1条直线
- 如果三点不在同一条直线上，每两点确定一条直线，共3条直线

**答案**：1条或3条

## 6.3 角

### 6.3.1 角的概念

**什么叫做角？**

**定义1（静态定义）**：**有公共端点的两条射线组成的图形叫做角。**

**理解**：
- 角由两条射线组成
- 这两条射线有公共端点
- 这两条射线叫做角的边
- 公共端点叫做角的顶点

**定义2（动态定义）**：**一条射线绕着它的端点旋转得到的图形叫做角。**

**理解**：
- 射线旋转时，起始位置和终止位置形成一个角
- 旋转过的部分叫做角的内部
- 起始位置的射线叫始边
- 终止位置的射线叫终边

**角的表示方法**：

**方法1：用三个大写字母表示**
- 例如：∠AOB
- **注意**：顶点字母O必须在中间！
- 读作："角 AOH" 或 "∠AOB"

### 6.3.2 角的度量

**角的单位**：度、分、秒

**六十进制**：
- 1° = 60′（1度 = 60分）
- 1′ = 60″（1分 = 60秒）
- 1° = 3600″（1度 = 3600秒）

**理解**：
- 这也是古代巴比伦人发明的
- 时间也是60进制：1小时 = 60分，1分 = 60秒

**角的分类**：

**1. 锐角**
**定义**：**大于0°且小于90°的角叫做锐角。**
**范围**：0° < 锐角 < 90°
**例子**：30°、45°、60°、89°都是锐角

**2. 直角**
**定义**：**等于90°的角叫做直角。**
**符号**：rt∠ 或 在角上画一个小正方形□
**图示**：
```
   ┃
   ┃（直角符号）
━━━�━━━━
```

**3. 钝角**
**定义**：**大于90°且小于180°的角叫做钝角。**
**范围**：90° < 钝角 < 180°
**例子**：100°、120°、150°、179°都是钝角

**4. 平角**
**定义**：**等于180°的角叫做平角。**
**图示**：
```
←━━━━━━━→（一条直线）
```
**理解**：平角的两边在一条直线上，方向相反

**5. 周角**
**定义**：**等于360°的角叫做周角。**
**图示**：
```
      ↑
      │
←─────┼─────→
      │
      ↓
```
**理解**：周角的两边重合，旋转了一整圈

**分类总结**：
- 按大小分类：锐角、直角、钝角、平角、周角
- 按范围：0° < 锐角 < 90° < 钝角 < 180° < 周角 = 360°

**例15**：判断下列角是哪类角

（1）30° → **锐角**（0° < 30° < 90°）
（2）90° → **直角**
（3）120° → **钝角**（90° < 120° < 180°）
（4）180° → **平角**
（5）360° → **周角**
（6）89° → **锐角**
（7）91° → **钝角**
（8）0° → 不是角（角的边重合，但没有旋转）
（9）-30° → 不是角（角度不能为负）

**角的度量的换算**：

**度化度分秒**：
**方法**：小数部分 × 60 得分，分的剩余小数部分 × 60 得秒

**例16**：把下列角度化成度分秒的形式

（1）30.5°
**解**：0.5° = 0.5 × 60′ = 30′
所以 30.5° = **30°30′**

（2）45.25°
**解**：0.25° = 0.25 × 60′ = 15′
所以 45.25° = **45°15′**

（3）60.45°
**解**：0.45° = 0.45 × 60′ = 27′
所以 60.45° = **60°27′**

（4）15.4°
**解**：0.4° = 0.4 × 60′ = 24′
所以 15.4° = **15°24′**

（5）12.345°
**解**：
0.345° = 0.345 × 60′ = 20.7′
0.7′ = 0.7 × 60″ = 42″
所以 12.345° = **12°20′42″**

**度分秒化为度**：
**方法**：秒÷60加分，分÷60加度

**例17**：把下列角度化成度

（1）30°30′
**解**：30′ = 30 ÷ 60 = 0.5°
所以 30°30′ = **30.5°**

（2）45°15′
**解**：15′ = 15 ÷ 60 = 0.25°
所以 45°15′ = **45.25°**

（3）2°36′
**解**：36′ = 36 ÷ 60 = 0.6°
所以 2°36′ = **2.6°**

（4）1°12′36″
**解**：
36″ = 36 ÷ 60 = 0.6′
12.6′ = 12.6 ÷ 60 = 0.21°
所以 1°12′36″ = **1.21°**

### 6.3.3 角的比较与运算

**角的比较方法**：

**方法1：度量法**
- 用量角器量出角的度数
- 比较数值大小

**方法2：叠合法**
- 将一个角移到另一个角上
- 使顶点重合，一边重合
- 比较另一边的位置

**例18**：比较下列各组角的大小

（1）∠1 = 30°，∠2 = 45°
**答**：∠1 < ∠2（30° < 45°）

（2）∠AOB是锐角，∠COD是钝角
**答**：∠AOB < ∠COD（锐角 < 钝角）

**角的和差倍分**：

**理解**：
角也可以像数一样进行加减乘除运算

**角平分线**：

**定义**：**从一个角的顶点出发，把这个角分成两个相等的角的射线，叫做这个角的平分线。**

**理解**：

- 角平分线在角的内部

### 6.3.4 余角和补角

**余角**：

**定义**：**如果两个角的和等于90°，就说这两个角互为余角。**

**理解**：
- "互为"的意思是相互的
- 如果∠1是∠2的余角，那么∠2也是∠1的余角
- 锐角才有余角

**例子**：
- 30°的余角是60°（因为30° + 60° = 90°）
- 60°的余角是30°（因为60° + 30° = 90°）
- 45°的余角是45°（因为45° + 45° = 90°）
- 0°的余角是90°
- 90°的余角是0°

**数学表达**：
如果∠1 + ∠2 = 90°，那么∠1和∠2互为余角。

**例22**：填空

（1）30°的余角是______度。
**答**：60（因为90° - 30° = 60°）

（2）如果∠A = 35°，那么∠A的余角是______度。
**答**：55（因为90° - 35° = 55°）

（3）如果一个角的余角是它本身，那么这个角是______度。
**答**：45（设这个角为x，则x + x = 90°，2x = 90°，x = 45°）

**补角**：

**定义**：**如果两个角的和等于180°，就说这两个角互为补角。**

**理解**：
- "互为"的意思是相互的
- 如果∠1是∠2的补角，那么∠2也是∠1的补角
- 锐角、直角、钝角都有补角

**例子**：
- 30°的补角是150°（因为30° + 150° = 180°）
- 60°的补角是120°（因为60° + 120° = 180°）
- 90°的补角是90°（因为90° + 90° = 180°）
- 120°的补角是60°（因为120° + 60° = 180°）

**数学表达**：
如果∠1 + ∠2 = 180°，那么∠1和∠2互为补角。

**例23**：填空

（1）50°的补角是______度。
**答**：130（因为180° - 50° = 130°）

（2）如果∠B = 120°，那么∠B的补角是______度。
**答**：60（因为180° - 120° = 60°）

（3）如果一个角的补角是它本身，那么这个角是______度。
**答**：90（设这个角为x，则x + x = 180°，2x = 180°，x = 90°）

**余角和补角的性质**：

**性质1：同角（或等角）的余角相等**

**表述**：**如果两个角都等于第三个角的余角，那么这两个角相等。**

**数学表达**：
如果∠1 + ∠3 = 90°，∠2 + ∠3 = 90°，
那么 ∠1 = ∠2

**证明**：
因为∠1 + ∠3 = 90°
又因为∠2 + ∠3 = 90°
所以 ∠1 + ∠3 = ∠2 + ∠3
所以 ∠1 = ∠2

**例24**：已知∠1与∠2互余，∠2与∠3互余，求证：∠1 = ∠3。

**证明**：
因为∠1与∠2互余，所以 ∠1 + ∠2 = 90°
又因为∠2与∠3互余，所以 ∠2 + ∠3 = 90°
所以 ∠1 + ∠2 = ∠2 + ∠3
所以 ∠1 = ∠3

**性质2：同角（或等角）的补角相等**

**表述**：**如果两个角都等于第三个角的补角，那么这两个角相等。**

**数学表达**：
如果∠1 + ∠3 = 180°，∠2 + ∠3 = 180°，
那么 ∠1 = ∠2

**证明**：与余角的证明类似

**例25**：已知∠1与∠2互补，∠2与∠3互补，求证：∠1 = ∠3。

**证明**：
因为∠1与∠2互补，所以 ∠1 + ∠2 = 180°
又因为∠2与∠3互补，所以 ∠2 + ∠3 = 180°
所以 ∠1 + ∠2 = ∠2 + ∠3
所以 ∠1 = ∠3

**例26**：已知一个角的补角是它的余角的3倍，求这个角的度数。

**分析**：
- 设这个角为x°
- 它的余角：90° - x
- 它的补角：180° - x
- 等量关系：补角 = 3 × 余角

**解**：设这个角为x°
根据题意：180° - x = 3(90° - x)
180° - x = 270° - 3x
-x + 3x = 270° - 180°
2x = 90°
x = 45°

**答**：这个角是45°。

**检验**：余角：90° - 45° = 45°
补角：180° - 45° = 135°
135° = 3 × 45° ✓

**时钟问题**：

**钟面上有12个大格，每个大格30°**
- 360° ÷ 12 = 30°

**时针每小时走30°**
- 每分钟走0.5°（30° ÷ 60 = 0.5°）

**分针每分钟走6°**
- 360° ÷ 60 = 6°

## 📚 学习建议

### 🎯 学习方法

1. **培养空间观念**
   - 多观察生活中的立体图形
   - 动手制作模型
   - 画三视图和展开图

2. **理解几何概念**
   - 从具体到抽象
   - 结合图形理解
   - 注意概念的区别

3. **掌握画图技能**
   - 用尺规作图
   - 标注字母和符号
   - 图形要规范

4. **注意单位换算**
   - 时间单位（60进制）
   - 角度单位（60进制）
   - 不要按100进制算！

5. **多做动手操作**
   - 制作正方体展开图
   - 测量角度
   - 画几何图形

### ⚠️ 常见错误提醒

1. ❌ 混淆直线、射线、线段
   ✅ 记住端点个数和延伸情况

2. ❌ 60进制算成100进制
   ✅ 1° = 60′，1′ = 60″

3. ❌ 忘记余角和补角的定义
   ✅ 余角和为90°，补角和为180°

4. ❌ 角平分线画成直线
   ✅ 角平分线是射线

5. ❌ 三视图方向搞错
   ✅ 主视图（正）、俯视图（上）、左视图（左）
