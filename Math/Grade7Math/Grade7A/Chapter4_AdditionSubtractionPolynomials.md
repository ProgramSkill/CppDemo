## 4.1 整式

### 4.1.1 单项式

**定义**：**由数或字母的积组成的代数式叫做单项式**

**理解**：
- 单项式中只含有乘法和乘方运算
- 单项式不含有加减运算
- 单项式不含有除法运算（除以字母的情况）

**例子**：

**是单项式的**：
- 2x（数与字母的积）
- -3xy（数与字母的积）
- ⅓x²y（分数与字母的积）
- 5（单独的数）
- x（单独的字母）
- -7（单独的数）
- x²（字母的乘方）

**不是单项式的**：
- x + 1（含有加法）
- x - y（含有减法）
- x/y（含有除法）
- (x + y)（是多项式）

#### 单项式的系数

**定义**：**单项式中的数字因数叫做这个单项式的系数**

**理解**：系数就是单项式前面的数字部分

**例子**：
- 2x 的系数是 **2**
- -3x 的系数是 **-3**
- 5xy 的系数是 **5**
- -½ab 的系数是 **-½**
- xy 的系数是 **1**（1省略不写）
- -x 的系数是 **-1**（-1省略不写）
- x² 的系数是 **1**
- 5 的系数是 **5**（单独的数，系数就是它本身）

**特别注意**：
- 系数包括符号！
- -3x的系数是-3，不是3
- x的系数是1，不是0
- -x的系数是-1

**练习**：
说出下列单项式的系数：
1. 3x² → **3**
2. -5xy → **-5**
3. xy → **1**
4. -x → **-1**
5. 7 → **7**

#### 单项式的次数

**定义**：**一个单项式中，所有字母的指数的和叫做这个单项式的次数**

**理解**：
- 次数是所有字母指数相加的和
- 与系数无关
- 单独的数的次数是0

**例子**：
- 2x 的次数是 **1**（x的指数是1）
- -3x²y 的次数是 **3**（x的指数是2，y的指数是1，2 + 1 = 3）
- 5xy²z 的次数是 **4**（x、y²、z，1 + 2 + 1 = 4）
- 5 的次数是 **0**（单独的数，没有字母）

**记忆口诀**：
系数看数字
次数看指数
各项指数要相加

**练习**：
说出下列单项式的次数：
1. 3x² → **2**
2. -5xy → **2**
3. x³y → **4**
4. 5 → **0**
5. -x²y³z → **6**

**单项式的命名**：
- 一次单项式：如 2x，-3y
- 二次单项式：如 3x²，-5xy
- 三次单项式：如 2x³，-3x²y

### 4.1.2 多项式

**定义**：**几个单项式的和叫做多项式**

**理解**：
- 多项式是单项式的和
- 多项式中含有加减运算
- 每个单项式叫做多项式的"项"

**例子**：
- x + 1（两个单项式的和）
- 3x² - 2x + 1（三个单项式的和）
- a² + 2ab + b²（三个单项式的和）

#### 多项式的项

**定义**：多项式中的每个单项式叫做多项式的项

**注意**：
- 项包括前面的符号
- 不含字母的项叫做常数项

**例子**：
多项式 3x² - 2x + 1 有三项：
- 第一项：3x²
- 第二项：-2x（注意包括负号）
- 第三项：1（常数项）

**练习**：
说出下列多项式的项：
1. 2x + 3 → **2x，3**
2. x² - 2x + 3 → **x²，-2x，3**
3. -a + b - 1 → **-a，b，-1**

#### 多项式的次数

**定义**：**多项式中次数最高的项的次数，就是这个多项式的次数**

**理解**：
- 先找出每一项的次数
- 其中最大的就是多项式的次数

**例子**：
- x + 1 是**一次多项式**（x的次数是1，1的次数是0）
- 3x² - 2x + 1 是**二次多项式**（3x²的次数是2，最高）
- x³ - 2x² + 3x - 1 是**三次多项式**（x³的次数是3，最高）

**练习**：
说出下列多项式的次数：
1. 2x + 3 → **一次**
2. x² - 2x + 1 → **二次**
3. 2x³ - x² + 3x - 1 → **三次**

#### 多项式的命名

**按项数命名**：
- 二项式：两项，如 x + 1，2x - 3y
- 三项式：三项，如 x² + 2x + 1，a + b + c
- 四项式：四项，如 x³ - 2x² + 3x - 1

**按次数命名**：
- 一次多项式：如 x + 1，2x - 3
- 二次多项式：如 x² + 2x + 1，a² - b²
- 三次多项式：如 x³ - 1

**综合命名**：
- 一次二项式：如 x + 1
- 二次三项式：如 x² + 2x + 1
- 三次四项式：如 x³ - 2x² + 3x - 1

**练习**：
给下列多项式命名：
1. 2x + 3 → **一次二项式**
2. x² - 2x + 1 → **二次三项式**
3. x³ - 1 → **一次二项式**（注意：只有两项！）

#### 升幂排列与降幂排列

**降幂排列**：按某个字母的指数从大到小排列

**例子**：
- 3x² - 2x + 1 已经是按x降幂排列
- 2x - 3x² + 1 改写成降幂排列：**-3x² + 2x + 1**

**升幂排列**：按某个字母的指数从小到大排列

**例子**：
- 3x² - 2x + 1 改写成升幂排列：**1 - 2x + 3x²**
- -3x² + 2x - 1 改写成升幂排列：**-1 + 2x - 3x²**

**注意**：
- 重新排列时，要连同符号一起移动
- 第一项系数为负时，通常保持负号在前面

### 4.1.3 整式

**定义**：**单项式和多项式统称为整式**

**分类**：
```
整式
├── 单项式
│   └── 例如：2x，-3xy，5，x
└── 多项式
    └── 例如：x + 1，3x² - 2x + 1，a² + 2ab + b²
```

**判断**：
- 是单项式或多项式 → 是整式 ✓
- 含有除以字母的 → 不是整式 ✗

**例子**：
- 是整式的：2x，x + 1，3x² - 2x + 1
- 不是整式的：x/y，1/(x + 1)，(x + y)/(x - y)

## 4.2 同类项

### 4.2.1 同类项的概念

**定义**：**所含字母相同，并且相同字母的指数也相同的项叫做同类项**

**三个条件**：
1. **所含字母相同**
2. **相同字母的指数相同**
3. **与系数无关**（系数可以不同）
4. **与字母顺序无关**（xy和yx是同类项）

**例子**：

**是同类项的**：
- 3x² 和 -2x²（都含x²）
- 5xy 和 -3xy（都含xy）
- 3x²y 和 5x²y（都含x²y）
- 2 和 -3（所有常数项都是同类项）
- -x²y 和 3yx²（都含x²y，顺序不重要）

**不是同类项的**：
- 3x² 和 2x³（x的指数不同）
- 3xy 和 3x²（字母不同）
- 3x²y 和 3xy²（指数不同）

**记忆口诀**：
两相同，一无关
字母相同指数同
系数顺序没关系

### 4.2.2 判断同类项的步骤

**步骤1**：看所含字母是否相同
**步骤2**：看相同字母的指数是否相同
**步骤3**：确认是否满足两个条件

**例子**：

**例1**：判断 3x²y 和 -5x²y 是否是同类项

**分析**：
- 所含字母：都有 x 和 y ✓
- x的指数：都是 2 ✓
- y的指数：都是 1 ✓

**答案**：**是同类项**

---

**例2**：判断 3x² 和 2x³ 是否是同类项

**分析**：
- 所含字母：都有 x ✓
- x的指数：一个是2，一个是3 ✗

**答案**：**不是同类项**

---

**例3**：判断 3ab 和 5ba 是否是同类项

**分析**：
- 所含字母：都有 a 和 b ✓
- a的指数：都是 1 ✓
- b的指数：都是 1 ✓

**答案**：**是同类项**（字母顺序不影响）

### 4.2.3 合并同类项

**定义**：**把多项式中的同类项合并成一项，叫做合并同类项**

**法则**：
1. **系数相加**：同类项的系数相加，所得结果作为新的系数
2. **字母和指数不变**：字母和字母的指数保持不变

**数学表达**：
ax + bx = (a + b)x

**例子**：

**例1**：合并同类项 3x² + 2x²

**分析**：
- 3x² 和 2x² 是同类项（都含x²）
- 系数：3 和 2

**计算**：
3x² + 2x² = (3 + 2)x² = 5x²

---

**例2**：合并同类项 5xy - 3xy

**分析**：
- 5xy 和 -3xy 是同类项（都含xy）
- 系数：5 和 -3

**计算**：
5xy - 3xy = (5 - 3)xy = 2xy

---

**例3**：合并同类项 2a + 3b - a + 4b

**分析**：
- 2a 和 -a 是同类项（都含a）
- 3b 和 4b 是同类项（都含b）

**计算**：
= (2a - a) + (3b + 4b)
= a + 7b

---

**例4**：合并同类项 3x² - 2x + 1 - x² + 3x - 2

**分析**：
- 3x² 和 -x² 是同类项
- -2x 和 3x 是同类项
- 1 和 -2 是同类项

**计算**：
= (3x² - x²) + (-2x + 3x) + (1 - 2)
= 2x² + x - 1

### 4.2.4 合并同类项的步骤

**步骤1**：找出同类项
- 画线标记相同的项
- 注意连同符号一起标记

**步骤2**：合并同类项
- 系数相加（连同符号）
- 字母和指数不变

**步骤3**：写出结果
- 按降幂排列（通常）
- 去掉系数为1时的"1"

**注意事项**：
1. **常数项也是同类项**
   - 3 和 -5 可以合并
   - 0 可以省略不写

2. **不是同类项不能合并**
   - 3x² 和 2x 不能合并
   - 3x 和 3y 不能合并

3. **合并时要连同符号**
   - 3x² - 2x² = (3 - 2)x² = x²
   - 不是：3x² - 2x² = 1x²

4. **系数为1或-1时的写法**
   - 1x 写成 x（省略1）
   - -1x 写成 -x

**练习**：
合并下列同类项：
1. 3a + 2a = **5a**
2. 5x - 3x = **2x**
3. 2x² - 3x² = **-x²**
4. 3xy + 2xy = **5xy**

## 4.3 去括号与添括号

### 4.3.1 去括号法则

**为什么要学习去括号？**

**问题**：化简 (3x + 2) + (2x - 1)

**思路**：
- 先去括号，再合并同类项

**法则一**：括号前面是"+"号

**法则**：**括号前面是"+"号，把括号和它前面的"+"号去掉，括号里各项都不变符号**

**例子**：
- a + (b + c) = a + b + c（各项都不变号）
- a + (b - c) = a + b - c（各项都不变号）
- a + (-b + c) = a - b + c（各项都不变号）

**法则二**：括号前面是"-"号

**法则**：**括号前面是"-"号，把括号和它前面的"-"号去掉，括号里各项都改变符号**

**例子**：
- a - (b + c) = a - b - c（都变号）
- a - (b - c) = a - b + c（都变号）
- a - (-b + c) = a + b - c（都变号）

**记忆口诀**：
去括号，看符号
正号不变号
负号都变号

**例1**：去括号
（1）a + (b + c) = a + b + c
（2）a - (b + c) = a - b - c
（3）a - (b - c) = a - b + c
（4）a + (-b + c) = a - b + c

---

**例2**：去括号
（1）3(x + 2) = 3x + 6
（2）-2(x - 1) = -2x + 2
（3）-3(2x - 1) = -6x + 3
（4）½(4x - 2) = 2x - 1

**注意**：
- 括号前面有系数时，要用分配律
- 系数与括号内每一项相乘
- 不要漏项！

### 4.3.2 添括号法则

**为什么要学习添括号？**

**问题**：把 a - b + c 添上括号，使第一个括号内含有a和b

**思路**：
- 需要把 a - b 用括号括起来
- 括号前用"+"号，括号内不变号
- 括号前用"-"号，括号内都变号

**法则一**：所添括号前面是"+"号

**法则**：**所添括号前面是"+"号，括到括号里的各项都不变符号**

**例子**：
- a + b - c = a + (b - c)（括b和c，不变号）
- a - b + c = a + (-b + c)（括-b和c，不变号）
- a + b + c = a + (b + c)（括b和c，不变号）

**法则二**：所添括号前面是"-"号

**法则**：**所添括号前面是"-"号，括到括号里的各项都改变符号**

**例子**：
- a - b - c = a - (b + c)（括b和c，都变号）
- a - b + c = a - (b - c)（括b和-c，都变号变成-b + c）
- a + b + c = a - (-b - c)（括b和c，都变号变成-b - c）

**例3**：添括号
（1）a + b - c = a + (b - c)
（2）a - b + c = a - (b - c)
（3）a - b - c = a - (b + c)

**注意**：
- 添括号是去括号的逆运算
- 检验：添括号后再去括号，应该得到原式

### 4.3.3 多重括号的处理

**方法**：从内到外，逐层去括号

**例4**：化简 a - [b - (a - c)]

**步骤1**：先去小括号
= a - [b - a + c]

**步骤2**：再去中括号（注意符号）
= a - b + a - c

**步骤3**：合并同类项
= 2a - b - c

**例5**：化简 2x - [3x - (x - 2) + 1]

**步骤1**：先去小括号
= 2x - [3x - x + 2 + 1]

**步骤2**：合并中括号内的同类项
= 2x - [2x + 3]

**步骤3**：去中括号
= 2x - 2x - 3
= -3

## 4.4 整式的加减

### 4.4.1 整式加减运算的步骤

**步骤1**：去括号
- 如果有括号，先运用去括号法则

**步骤2**：合并同类项
- 找出同类项
- 系数相加，字母和指数不变

**步骤3**：写出结果
- 按降幂排列
- 省略系数为1时的"1"

### 4.4.2 整式加减的几种类型

#### 类型一：单项式与单项式的加减

**例1**：计算 3x + 2x

**解**：
3x + 2x = (3 + 2)x = 5x

---

**例2**：计算 5xy - 2xy

**解**：
5xy - 2xy = (5 - 2)xy = 3xy

---

**例3**：计算 -2a + 3a - a

**解**：
-2a + 3a - a = (-2 + 3 - 1)a = 0a = 0

---

#### 类型二：单项式与多项式的加减

**例4**：计算 3x + (2x - 1)

**解**：
3x + (2x - 1)
= 3x + 2x - 1（去括号，不变号）
= 5x - 1（合并同类项）

---

**例5**：计算 2a - (3a - 1)

**解**：
2a - (3a - 1)
= 2a - 3a + 1（去括号，都变号）
= -a + 1（合并同类项）

---

**例6**：计算 -3x + (2x - 5) - (x - 3)

**解**：
-3x + (2x - 5) - (x - 3)
= -3x + 2x - 5 - x + 3（去括号）
= (-3x + 2x - x) + (-5 + 3)（分组）
= -2x - 2（合并同类项）

---

#### 类型三：多项式与多项式的加减

**例7**：计算 (3x + 2) + (2x - 1)

**解法1**：分别去括号
= 3x + 2 + 2x - 1
= 5x + 1

**解法2**：利用分配律
= (3x + 2x) + (2 - 1)
= 5x + 1

---

**例8**：计算 (2x - 3) - (x - 2)

**解**：
(2x - 3) - (x - 2)
= 2x - 3 - x + 2（去括号）
= (2x - x) + (-3 + 2)
= x - 1

---

**例9**：计算 (a² + 2ab + b²) - (a² - 2ab + b²)

**解**：
= a² + 2ab + b² - a² + 2ab - b²（去括号）
= (a² - a²) + (2ab + 2ab) + (b² - b²)
= 0 + 4ab + 0
= 4ab

---

**例10**：化简 2(3x - 1) - 3(2x + 3)

**解**：
= 6x - 2 - 6x - 9（利用分配律）
= (6x - 6x) + (-2 - 9)
= 0 - 11
= -11

---

### 4.4.3 化简多项式

**例11**：化简 (2x² - 3x + 1) - (x² - 2x + 3)

**解**：
= 2x² - 3x + 1 - x² + 2x - 3
= (2x² - x²) + (-3x + 2x) + (1 - 3)
= x² - x - 2

---

**例12**：化简 3(x² - 2x) - 2(x² - 3x + 1)

**解**：
= 3x² - 6x - 2x² + 6x - 2
= (3x² - 2x²) + (-6x + 6x) - 2
= x² + 0x - 2
= x² - 2

   - 去括号：看符号，正不变负都变
   - 合并同类项：系数相加，字母指数不变
   - 化简：去括号→合并同类项→整理

### ⚠️ 常见错误提醒

1. ❌ 判断同类项时只看字母
   ✅ 字母相同，指数也要相同

2. ❌ 去括号时符号错误
   ✅ 括号前是负号，里面都变号

3. ❌ 合并同类项时遗漏项
   ✅ 一项一项地合并，不要遗漏

4. ❌ 括号前有系数时漏乘
   ✅ 用分配律，每一项都要乘

5. ❌ 去括号时忘记变号
   ✅ 负号去括号，里面每一项都变号
