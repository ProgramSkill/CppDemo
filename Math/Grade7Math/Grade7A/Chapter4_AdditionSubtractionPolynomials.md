# 第四章 整式的加减

## 📚 知识要点

### 4.1 整式

#### 4.1.1 单项式

**定义**：由数或字母的积组成的代数式叫做单项式

**注意**：
- 单独一个数或一个字母也是单项式
  - 例如：5，x，-3都是单项式

- 单项式中只含有乘法和乘方运算
  - 例如：2x²，-3ab，⅓xy都是单项式
  - 例如：x + 1，a/b不是单项式

**单项式的系数**：
- 定义：单项式中的数字因数叫做这个单项式的系数
- 例如：2x的系数是2
- 例如：-3xy的系数是-3
- 例如：xy的系数是1（1省略不写）
- 例如：-x的系数是-1

**单项式的次数**：
- 定义：一个单项式中，所有字母的指数的和叫做这个单项式的次数
- 例如：2x的次数是1（一次单项式）
- 例如：-3x²y的次数是3（2 + 1 = 3，三次单项式）
- 例如：5的次数是0（零次单项式，单独的数）

#### 4.1.2 多项式

**定义**：几个单项式的和叫做多项式

**多项式的项**：
- 多项式中的每个单项式叫做多项式的项
- 不含字母的项叫做常数项

**例如**：多项式 3x² - 2x + 1
- 3x²，-2x，1 都是它的项
- 3x² 是二次项，-2x 是一次项，1 是常数项

**多项式的次数**：
- 定义：多项式中次数最高的项的次数，就是这个多项式的次数
- 例如：3x² - 2x + 1 是二次多项式（最高次项是3x²，次数是2）
- 例如：x³ - 2x² + 3x - 1 是三次多项式

**多项式的命名**：
- 按项数：二项式、三项式等
- 按次数：一次多项式、二次多项式等
- 例如：x + 1 是一次二项式
- 例如：3x² - 2x + 1 是二次三项式

**升幂排列与降幂排列**：
- 降幂排列：按某个字母的指数从大到小排列
  - 例如：3x² - 2x + 1（按x降幂排列）

- 升幂排列：按某个字母的指数从小到大排列
  - 例如：1 - 2x + 3x²（按x升幂排列）

#### 4.1.3 整式

**定义**：单项式和多项式统称为整式

**分类**：
```
整式
├── 单项式
│   └── 例如：2x，-3xy，5
└── 多项式
    └── 例如：x + 1，3x² - 2x + 1
```

### 4.2 同类项

#### 4.2.1 同类项的概念

**定义**：所含字母相同，并且相同字母的指数也相同的项叫做同类项

**要点**：
1. 所含字母相同
2. 相同字母的指数相同
3. 与系数无关
4. 与字母顺序无关

**例子**：
- 3x² 与 -2x² 是同类项（都含x²）
- 5xy 与 -3xy 是同类项（都含xy）
- 3x²y 与 5x²y 是同类项（都含x²y）
- 3x² 与 3x³ 不是同类项（x的指数不同）

**注意**：
- 所有常数项都是同类项
  - 例如：3 与 -5 是同类项

#### 4.2.2 合并同类项

**定义**：把多项式中的同类项合并成一项，叫做合并同类项

**法则**：
- 同类项的系数相加，所得结果作为系数
- 字母和字母的指数保持不变

**例子**：
3x² + 2x² = (3 + 2)x² = 5x²

5xy - 3xy = (5 - 3)xy = 2xy

**步骤**：
1. 找出同类项
2. 合并同类项（系数相加，字母和指数不变）
3. 写出结果

**注意**：
- 不是同类项不能合并
- 合并同类项时，要连同符号一起移动
- 合并后，系数为0时，该项为0，可以省略不写

### 4.3 去括号与添括号

#### 4.3.1 去括号法则

**法则一**：括号前面是"+"号，把括号和它前面的"+"号去掉，括号里各项都不变符号

**例子**：
a + (b + c) = a + b + c
a + (b - c) = a + b - c

**法则二**：括号前面是"-"号，把括号和它前面的"-"号去掉，括号里各项都改变符号

**例子**：
a - (b + c) = a - b - c
a - (b - c) = a - b + c

**注意**：
- 括号前面有系数时，先用分配律，再去括号
- 例如：3(a + 2b) = 3a + 6b
- 例如：-2(3x - 1) = -6x + 2

#### 4.3.2 添括号法则

**法则一**：所添括号前面是"+"号，括到括号里的各项都不变符号

**例子**：
a + b - c = a + (b - c)
a - b + c = a + (-b + c)

**法则二**：所添括号前面是"-"号，括到括号里的各项都改变符号

**例子**：
a - b - c = a - (b + c)
a - b + c = a - (b - c)

### 4.4 整式的加减

#### 4.4.1 整式加减运算的步骤

**步骤**：
1. **去括号**：运用去括号法则
2. **合并同类项**：运用合并同类项法则
3. **写出结果**

#### 4.4.2 整式加减的几种类型

**类型一**：单项式与单项式的加减
- 例如：3x + 2x = 5x

**类型二**：单项式与多项式的加减
- 例如：3x + (2x - 1) = 3x + 2x - 1 = 5x - 1

**类型三**：多项式与多项式的加减
- 例如：(3x + 2) + (2x - 1) = 3x + 2 + 2x - 1 = 5x + 1

## 📖 典型例题

### 例题1：判断单项式

**题目**：下列各式中，哪些是单项式？
（1）2x
（2）x + 1
（3）-3xy
（4）a/b
（5）5
（6）x²

**解答**：
单项式有：**（1）2x，（3）-3xy，（5）5，（6）x²**

不是单项式的是：
- （2）x + 1（是多项式）
- （4）a/b（是分式，不是整式）

---

### 例题2：单项式的系数和次数

**题目**：说出下列单项式的系数和次数
（1）3x²
（2）-2xy
（3）x
（4）-5
（5）⅓ab²

**解答**：
（1）3x²
    系数：**3**
    次数：**2**（x的指数是2）

（2）-2xy
    系数：**-2**
    次数：**2**（x的指数1 + y的指数1 = 2）

（3）x
    系数：**1**
    次数：**1**

（4）-5
    系数：**-5**
    次数：**0**

（5）⅓ab²
    系数：**⅓**
    次数：**3**（a的指数1 + b的指数2 = 3）

---

### 例题3：多项式的项和次数

**题目**：
指出多项式 3x²y - 4xy² + x - 5 是几次几项式，并说明各项的次数。

**解答**：
- 多项式 3x²y - 4xy² + x - 5 共有4项
- 各项分别为：3x²y，-4xy²，x，-5
- 各项的次数：
  - 3x²y：2 + 1 = 3（三次项）
  - -4xy²：1 + 2 = 3（三次项）
  - x：1（一次项）
  - -5：0（常数项）
- 最高次项的次数是3
- 所以这是**三次四项式**

### 例题4：同类项的判断

**题目**：下列各组中的两项是不是同类项？
（1）2x 与 3x
（2）2x² 与 2x³
（3）3xy 与 -3yx
（4）5 与 -5

**解答**：
（1）2x 与 3x 是同类项 ✓（都含x，指数相同）

（2）2x² 与 2x³ 不是同类项 ✗（x的指数不同）

（3）3xy 与 -3yx 是同类项 ✓（都含x和y，指数相同）

（4）5 与 -5 是同类项 ✓（所有常数项都是同类项）

### 例题5：合并同类项

**题目**：合并下列各式中的同类项
（1）3x + 2x
（2）3x² - 2x²
（3）5xy - 3xy + xy
（4）3a + 2b - a + 3b

**解答**：
（1）3x + 2x = (3 + 2)x = **5x**

（2）3x² - 2x² = (3 - 2)x² = **x²**

（3）5xy - 3xy + xy
     = (5 - 3 + 1)xy
     = **3xy**

（4）3a + 2b - a + 3b
     = (3 - 1)a + (2 + 3)b
     = **2a + 5b**

---

### 例题6：去括号

**题目**：去括号
（1）a + (b + c)
（2）a - (b + c)
（3）3(x + 2)
（4）-2(x - 3)

**解答**：
（1）a + (b + c) = **a + b + c**

（2）a - (b + c) = **a - b - c**

（3）3(x + 2) = **3x + 6**

（4）-2(x - 3) = **-2x + 6**

---

### 例题7：整式的加减（基础）

**题目**：计算
（1）3x + 2x
（2）（3x + 2） + （2x - 1）
（3）（3x + 2） - （2x - 1）

**解答**：
（1）3x + 2x = **5x**

（2）（3x + 2） + （2x - 1）
     = 3x + 2 + 2x - 1
     = (3 + 2)x + (2 - 1)
     = **5x + 1**

（3）（3x + 2） - （2x - 1）
     = 3x + 2 - 2x + 1
     = (3 - 2)x + (2 + 1)
     = **x + 3**

---

### 例题8：整式的加减（进阶）

**题目**：计算
（1）2(3x - 1) + 3(x + 2)
（2）3(x² - 2x) - 2(x² - 3)

**解答**：
（1）2(3x - 1) + 3(x + 2)
     = 6x - 2 + 3x + 6
     = (6 + 3)x + (-2 + 6)
     = **9x + 4**

（2）3(x² - 2x) - 2(x² - 3)
     = 3x² - 6x - 2x² + 6
     = (3 - 2)x² - 6x + 6
     = **x² - 6x + 6**

---

### 例题9：化简求值

**题目**：
（1）化简：2(3x - 1) - 3(2x + 1)
（2）当 x = -2 时，求上式的值

**解答**：
（1）2(3x - 1) - 3(2x + 1)
     = 6x - 2 - 6x - 3
     = (6 - 6)x + (-2 - 3)
     = **-5**

（2）当 x = -2 时，
     原式 = **-5**
     （注意：化简后结果不含x，所以x取任意值结果都是-5）

---

### 例题10：多重括号

**题目**：计算
3x - 2[3x - 2(x - 1)]

**解答**：
3x - 2[3x - 2(x - 1)]
= 3x - 2[3x - 2x + 2]  ← 先去小括号
= 3x - 2[x + 2]         ← 合并同类项
= 3x - 2x - 4           ← 再去中括号
= x - 4                 ← 合并同类项

---

### 例题11：实际应用

**题目**：
某商店第一种图书的单价是x元，第二种图书的单价比第一种多3元，
（1）买3本第一种图书和2本第二种图书，共需多少元？
（2）当 x = 15 时，共需多少元？

**解答**：
（1）第二种图书单价：(x + 3)元
    总费用：3x + 2(x + 3)
          = 3x + 2x + 6
          = **(5x + 6)元**

（2）当 x = 15 时：
    5x + 6 = 5 × 15 + 6 = 75 + 6 = 81（元）

    答：共需81元。

## 📚 学习建议

1. **准确掌握概念**：单项式、多项式、同类项等概念要理解清楚

4. **化简后求值**：养成先化简再代入求值的习惯
5. **多重括号**：从内到外，逐层去括号
