# 第三章 代数式

## 📚 知识要点

### 3.1 用字母表示数

#### 3.1.1 字母表示数的意义

**基本概念**：
- 用字母可以表示我们已经学过的和将来要学到的任何一个数
- 用字母表示数，可以把数量关系简明地表达出来

**例子**：
- 加法交换律：a + b = b + a
- 乘法交换律：ab = ba
- 乘法分配律：a(b + c) = ab + ac
- 正方形面积公式：S = a²（a表示边长）
- 长方形面积公式：S = ab（a表示长，b表示宽）

#### 3.1.2 字母表示数的注意事项

1. **字母与数相乘**：省略乘号，数字写在字母前面
   - 2 × a 写作 **2a**
   - a × 3 写作 **3a**（不是a3）

2. **字母与字母相乘**：省略乘号
   - a × b 写作 **ab**
   - a × a 写作 **a²**

3. **1与字母相乘**：1省略不写
   - 1 × a 写作 **a**（不是1a）

4. **带分数与字母相乘**：把带分数化成假分数
   - 1½ × a 写作 **³⁄₂a**

5. **除法**：除法写成分数的形式
   - a ÷ b 写作 **a/b**

6. **括号的使用**：表示含字母的式子时，后面带单位要加括号
   - (a + b)米（不是 a + b 米）

### 3.2 代数式

#### 3.2.1 代数式的概念

**定义**：用基本运算符号（加、减、乘、除、乘方）把数和表示数的字母连接而成的式子叫做代数式

**注意**：
- 单独一个数或一个字母也是代数式
  - 例如：5，x，-3，a都是代数式

- 代数式中不含等号或不等号
  - 例如：a + b，2x，x² - 1 是代数式
  - 例如：a + b = 5，x > 3 不是代数式（是等式或不等式）

#### 3.2.2 代数式的书写规则

1. **先乘方，再乘除，最后加减**
   - 例如：a × a × b 写作 **a²b**

2. **数字因数写在字母因数前面**
   - 例如：a × 2 × b 写作 **2ab**

3. **带分数要化成假分数**
   - 例如：1½a 写作 **³⁄₂a**

4. **除法要写成分数形式**
   - 例如：a ÷ b 写作 **a/b**

5. **单位问题**：
   - 如果是积或商的形式，后面直接写单位
     - 例如：3ab米，5x²kg
   - 如果是和或差的形式，要先加括号，再写单位
     - 例如：(a + b)米，(2x - 3)元

### 3.3 列代数式

#### 3.3.1 列代数式的方法

**步骤**：
1. **认真审题**：弄清题目中的数量关系
2. **分析数量关系**：找出运算顺序
3. **用字母表示数**：根据题目要求选择合适的字母
4. **列出代数式**：按运算顺序写出代数式

**常见数量关系**：

1. **和差问题**：
   - a与b的和：a + b
   - a与b的差：a - b
   - a比b大多少：a - b

2. **倍数问题**：
   - a的3倍：3a
   - a的½：½a 或 a/2
   - a比b多2倍：a - 2b

3. **积商问题**：
   - a与b的积：ab
   - a除以b的商：a/b
   - a与b的积的一半：ab/2

4. **连续整数**：
   - 三个连续整数：n，n+1，n+2
   - 三个连续偶数：2n，2n+2，2n+4
   - 三个连续奇数：2n-1，2n+1，2n+3

#### 3.3.2 列代数式的注意事项

1. **注意运算顺序**：
   - a与b的和的3倍：3(a + b)
   - a的3倍与b的和：3a + b

2. **注意关键词**：
   - "的"表示前面的数运算后再与后面的数运算
   - "与"表示两个数并列
   - "大"、"小"、"多"、"少"、"增加"、"减少"等词的含义

3. **区分不同表述**：
   - a、b两数的平方和：a² + b²
   - a、b两数和的平方：(a + b)²
   - a、b两数的平方差：a² - b²
   - a、b两数差的平方：(a - b)²

### 3.4 代数式的值

#### 3.4.1 代数式的值的概念

**定义**：用数值代替代数式里的字母，按照代数式中的运算关系计算得出的结果，叫做代数式的值

**注意**：
- 一个代数式的值由代数式中的字母所取的值确定
- 同一个代数式，当字母取不同数值时，代数式的值可能不同

#### 3.4.2 求代数式的值的步骤

**步骤**：
1. **代入**：用数值代替代数式中的字母
2. **计算**：按照运算顺序计算

**注意事项**：
1. 代入时，如果字母取值是负数或分数，要加括号
2. 代入时，省略了乘号的要恢复乘号
3. 计算时，要按照运算顺序进行

**例子**：
当 a = 2，b = -3 时，求 2a² - 3b 的值

**解**：
2a² - 3b
= 2 × 2² - 3 × (-3)
= 2 × 4 - (-9)
= 8 + 9
= 17

## 📖 典型例题

### 例题1：用字母表示运算律

**题目**：用字母表示下列运算律
（1）加法结合律
（2）乘法分配律
（3）乘法结合律

**解答**：
（1）加法结合律：(a + b) + c = a + (b + c)

（2）乘法分配律：a(b + c) = ab + ac

（3）乘法结合律：(ab)c = a(bc)

---

### 例题2：用字母表示公式

**题目**：
（1）三角形面积公式（S表示面积，a表示底，h表示高）
（2）圆的周长公式（C表示周长，r表示半径，π ≈ 3.14）
（3）梯形面积公式（S表示面积，a表示上底，b表示下底，h表示高）

**解答**：
（1）三角形面积公式：**S = ah/2**

（2）圆的周长公式：**C = 2πr**

（3）梯形面积公式：**S = (a + b)h ÷ 2** 或 **S = (a + b)h/2**

### 例题3：改正错误的代数式

**题目**：下列各式的书写是否正确？如果不正确，请改正。
（1）a × 5
（2）1½x
（3）a ÷ b
（4）(a + b)米

**解答**：
（1）a × 5 写法不正确，应写作：**5a**

（2）1½x 写法不正确，应写作：**³⁄₂x**

（3）a ÷ b 写法不正确，应写作：**a/b**

（4）(a + b)米 写法**正确**

**题目1**：

### 例题9：求代数式的值（基础）

**题目**：当 a = 3，b = -2 时，求下列代数式的值
（1）a + b
（2）a - b
（3）3a - 2b
（4）ab

**解答**：
（1）当 a = 3，b = -2 时：
    a + b = 3 + (-2) = **1**

（2）当 a = 3，b = -2 时：
    a - b = 3 - (-2) = 3 + 2 = **5**

（3）当 a = 3，b = -2 时：
    3a - 2b = 3 × 3 - 2 × (-2) = 9 + 4 = **13**

（4）当 a = 3，b = -2 时：
    ab = 3 × (-2) = **-6**

---

### 例题10：求代数式的值（含乘方）

**题目**：当 x = -2 时，求下列代数式的值
（1）x²
（2）-x²
（3）2x² - 3x + 1
（4）(x + 1)²

**解答**：
（1）当 x = -2 时：
    x² = (-2)² = **4**

（2）当 x = -2 时：
    -x² = -(-2)² = -(4) = **-4**

（3）当 x = -2 时：
    2x² - 3x + 1
    = 2 × (-2)² - 3 × (-2) + 1
    = 2 × 4 + 6 + 1
    = 8 + 6 + 1
    = **15**

（4）当 x = -2 时：
    (x + 1)²
    = (-2 + 1)²
    = (-1)²
    = **1**

### 例题11：代数式的实际应用

**题目**：
某商店进了一批玩具，每个进价a元，售价每个a + 5元，
（1）卖出20个，可获利润多少元？
（2）若a = 15元，利润是多少元？

**解答**：
（1）每个玩具的利润：(a + 5) - a = 5（元）
    卖出20个的利润：**5 × 20 = 100（元）**

    或者：
    总售价：20(a + 5)元
    总成本：20a元
    利润：20(a + 5) - 20a = 20a + 100 - 20a = 100（元）

（2）当 a = 15 时：
    利润 = **100元**

1. **理解字母表示数的意义**：字母可以代表任意数，具有一般性
