# 第一章 有理数

## 📚 知识要点

### 1.1 正数和负数

#### 正数的概念
- **定义**：大于0的数叫做正数
- **表示**：正数前面可以放上"+"（正）号，也可以省略不写
- **例子**：+5、10、0.3、½ 都是正数

#### 负数的概念
- **定义**：小于0的数叫做负数（在正数前面放上"-"（负）号）
- **表示**：负数前面必须放上"-"号
- **例子**：-5、-10、-0.3、-½ 都是负数

#### 0的意义
- 0既不是正数，也不是负数
- 0是正数与负数的分界点
- 0在实际生活中具有特定意义（如温度0℃）

#### 正负数的实际应用
- **温度**：零上为正，零下为负
- **海拔**：海平面以上为正，海平面以下为负
- **财务**：收入为正，支出为负
- **方向**：向东为正，向西为负

### 1.2 有理数

#### 1.2.1 有理数的概念

**定义**：整数和分数统称为有理数

**分类（一）**：
```
有理数
├── 整数
│   ├── 正整数（如：1、2、3...）
│   ├── 0
│   └── 负整数（如：-1、-2、-3...）
└── 分数
    ├── 正分数（如：½、0.3、5⅓...）
    └── 负分数（如：-½、-0.3、-5⅓...）
```

**分类（二）**：
```
有理数
├── 正有理数
│   ├── 正整数
│   └── 正分数
├── 0
└── 负有理数
    ├── 负整数
    └── 负分数
```

**注意**：
- 任何一个有理数都可以写成形式为 p/q（p、q为整数，q≠0）
- 有限小数和无限循环小数都是分数，因此都是有理数
- 无限不循环小数不是有理数

#### 1.2.2 数轴

**定义**：规定了原点、正方向和单位长度的直线叫做数轴

**三要素**：
1. **原点**：在数轴上取一点表示0，这个点叫做原点
2. **正方向**：一般规定箭头指向的方向（通常向右）为正方向
3. **单位长度**：选取适当的长度作为单位长度

**数轴的画法**：
1. 画一条水平直线
2. 在直线上取一点作为原点
3. 确定正方向（通常向右）
4. 选取单位长度，标出刻度

**数轴上的点与有理数的关系**：
- 所有的有理数都可以用数轴上的点表示
- 数轴上原点左边的点表示负数
- 数轴上原点右边的点表示正数
- 原点表示0

**注意**：虽然有理数都可以用数轴上的点表示，但数轴上的点不全是有理数（还有无理数）

#### 1.2.3 相反数

**定义**：只有符号不同的两个数叫做互为相反数

**性质**：
1. 0的相反数是0
2. 互为相反数的两个数在数轴上位于原点的两侧，且到原点的距离相等
3. a的相反数是-a，即 a + (-a) = 0
4. 相反数的相反数等于它本身，即 -(-a) = a

**表示方法**：
- 数a的相反数记作：-a
- 例如：5的相反数是-5，-5的相反数是5

**注意**：
- 相反数是成对出现的，不能单独存在
- 正数的相反数是负数，负数的相反数是正数
- 0的相反数是它本身

#### 1.2.4 绝对值

**定义**：
- 一个正数的绝对值是它本身
- 一个负数的绝对值是它的相反数
- 0的绝对值是0

**数学表达式**：
```
|a| = a      （当 a > 0 时）
|a| = 0      （当 a = 0 时）
|a| = -a     （当 a < 0 时）
```

**几何意义**：
- 在数轴上，一个数的绝对值就是表示这个数的点到原点的距离
- 距离是非负数，所以绝对值是非负数

**性质**：
1. 任何有理数的绝对值都是非负数，即 |a| ≥ 0
2. 互为相反数的两个数的绝对值相等，即 |a| = |-a|
3. 若 |a| = |b|，则 a = b 或 a = -b

**注意**：
- 绝对值最小的数是0
- 绝对值为同一个正数的有理数有两个，它们互为相反数

#### 1.2.5 有理数的大小比较

**比较法则**：
1. **正数大于0**，0大于负数，正数大于负数
2. **两个负数比较**，绝对值大的反而小

**数轴比较法**：
- 在数轴上，右边的数总比左边的数大

**步骤**：
1. 先比较符号：正数 > 0 > 负数
2. 两个正数比较：绝对值大的大
3. 两个负数比较：绝对值大的反而小

**常用结论**：
- 两个负数比较，先看它们的绝对值，绝对值大的反而小
- 例如：|-8| > |-5|，但 -8 < -5

## 📖 典型例题

### 例题1：正数和负数的认识

**题目**：
（1）如果收入50元记作+50元，那么支出30元记作什么？
（2）如果向东走5m记作+5m，那么向西走3m记作什么？
（3）在4个不同时刻，对同一水库的水位进行监测，记录如下：上升3cm、下降2cm、上升1cm、下降4cm。如果上升记为正，请用正负数表示这4次的水位变化。

**解答**：
（1）支出30元记作 **-30元**

（2）向西走3m记作 **-3m**

（3）4次水位变化分别为：**+3cm、-2cm、+1cm、-4cm**

（2）|-8| = ?

**解答**：

### 例题6：有理数的大小比较

**题目**：
比较下列各组数的大小：
（1）-3 和 -5
（2）-0.9 和 -1.1
（3）0 和 -100
（4）-½ 和 -⅓

**解答**：
（1）|-3| = 3，|-5| = 5
    因为 3 < 5，所以 **-3 > -5**

（2）|-0.9| = 0.9，|-1.1| = 1.1
    因为 0.9 < 1.1，所以 **-0.9 > -1.1**

（3）因为正数和0大于一切负数，所以 **0 > -100**

（4）|-½| = ½ = 3/6，|-⅓| = ⅓ = 2/6
    因为 3/6 > 2/6，所以 **-½ < -⅓**

---

### 例题7：综合应用

**题目**：
已知 |a| = 3，|b| = 2，且 a < b，求 a、b 的值。

**解答**：
因为 |a| = 3，所以 a = 3 或 a = -3
因为 |b| = 2，所以 b = 2 或 b = -2

又因为 a < b，我们需要讨论：
- 当 a = 3 时：b > 3，但 b 只能是 2 或 -2，**不满足**
- 当 a = -3 时：
  - 若 b = 2，则 -3 < 2，**满足**
  - 若 b = -2，则 -3 < -2，**满足**

所以，**a = -3，b = 2 或 b = -2**

## 📚 学习建议

1. **理解负数的意义**：通过实际例子（温度、海拔等）理解负数的概念
2. **掌握数轴**：数轴是学习有理数的重要工具，要多练习画数轴和用数轴表示数
3. **注意绝对值**：绝对值是非负数，这是易错点
4. **比较负数大小**：记住"两个负数比较，绝对值大的反而小"
5. **数形结合**：善于用数轴来理解相反数、绝对值等概念
