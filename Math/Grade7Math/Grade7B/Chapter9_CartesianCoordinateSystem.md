## 🌟 章节引入：如何用数字描述位置？

### 生活中的"定位"问题

**【场景1：电影院找座位】**

```
电影票上写着：5排13号

你应该怎么走？
→ 先找到第5排（从前向后数）
→ 再在第5排找到13号座（从左向右数）

这就是用"两个数字"确定一个位置！
```

**【场景2：地球上的位置】**

```
北京的位置：北纬39.9°，东经116.4°

这也需要"两个数字"：
→ 纬度（南北方向）
→ 经度（东西方向）
```

**【场景3：教室里的座位】**

```
老师说："第3行第4列的同学请回答问题"

谁应该站起来？
→ 第3行从前向后数的第3行
→ 第4列从左向右数的第4列
→ 交点处的那个同学
```

### 从生活到数学：平面直角坐标系的诞生

**【问题引入】**

如何用数学的方法精确描述平面上任意一点的位置？

**【历史上的天才】**

**勒内·笛卡尔 (René Descartes, 1596-1650)**

传说，笛卡尔躺在床上观察天花板上的苍蝇，他思考：
> 如何用数字准确描述苍蝇在天花板上的位置？

这个问题启发了他，于是他发明了**平面直角坐标系**！

**【笛卡尔的伟大贡献】**

```
在笛卡尔之前：
→ 几何是图形
→ 代数是数字
→ 两者分离

笛卡尔坐标系：
→ 用数字（坐标）表示图形
→ 图形问题转化为代数问题
→ 创立了解析几何！
```

## 📚 9.1 平面直角坐标系

### 9.1.1 平面直角坐标系的组成

#### 📖 从数轴到坐标平面

**【回顾：数轴】**

在之前的学习中，我们认识了一维的数轴：

```
        ────┬────┬────┬────┬────→
          -2   -1    0    1    2

数轴上的每个点都对应一个数！
```

**【问题：如何描述平面上的点？**

数轴只能描述直线上点的位置（一维）
如何描述平面上点的位置（二维）？

**【笛卡尔的解决方案】**

用**两条数轴**！

```
      y
      ↑
      │
      │
      │
──────┼──────→ x
      │
      │
      │
      ↓
```

**【定义的形成】**

> 在平面内画两条互相垂直且有公共原点的数轴，就组成了平面直角坐标系。

#### 🔍 深入理解：坐标系的组成要素

**【要素1：横轴（x轴）**

```
水平方向的数轴
→ 向右为正方向
→ 通常画成水平线
```

**【要素2：纵轴（y轴）**

```
竖直方向的数轴
→ 向上为正方向
→ 通常画成竖直线
```

**【要素3：原点 (O)**

```
两坐标轴的交点
→ 记作点 O
→ 坐标是 (0, 0)
→ 是整个坐标系的"中心"
```

**【要素4：单位长度**

```
x轴和y轴通常取相同的单位长度
→ 方便比较和计算
```

#### 📊 坐标平面的象限划分

**【为什么要划分象限？**

象限帮助我们快速判断点的位置和坐标符号！

**【四个象限】**

```
      y
      ↑
 第二 │ 第一
 象限 │ 象限
──────┼──────→ x
 第三 │ 第四
 象限 │ 象限
      ↓
```

**【详细说明】**

| 象限 | 位置特征 | 横坐标x | 纵坐标y | 记忆口诀 |
|------|---------|---------|---------|---------|
| **第一象限** | 右上 | +（正） | +（正） | 一正正 |
| **第二象限** | 左上 | -（负） | +（正） | 二负正 |
| **第三象限** | 左下 | -（负） | -（负） | 三负负 |
| **第四象限** | 右下 | +（正） | -（负） | 四正负 |

**【重要提醒】**

> ⚠️ 坐标轴上的点**不属于任何象限**！

```
x轴上的点：(a, 0) → 不在任何象限
y轴上的点：(0, b) → 不在任何象限
原点：(0, 0) → 不在任何象限
```

#### 💡 象限的记忆技巧

**【方法1：按逆时针顺序记忆】**

```
从右上开始，逆时针转：
→ 第一象限（右上）
→ 第二象限（左上）
→ 第三象限（左下）
→ 第四象限（右下）
```

**【方法2：用钟表类比】**

```
第一象限：1-3点方向（右上）
第二象限：10-12点方向（左上）
第三象限：7-9点方向（左下）
第四象限：4-6点方向（右下）
```

**【方法3：符号记忆表】**

```
象限  |  x符号  |  y符号
------|---------|----------
  一  |    +    |    +
  二  |    -    |    +
  三  |    -    |    -
  四  |    +    |    -
```

### 9.1.2 点的坐标

#### 📖 如何确定点的坐标？

**【基本方法】**

```
已知：平面内一点P

求：点P的坐标

步骤：
1. 过点P向x轴作垂线
   → 垂足对应的数就是横坐标

2. 过点P向y轴作垂线
   → 垂足对应的数就是纵坐标

3. 写成：P(a, b)
   → a是横坐标
   → b是纵坐标
```

**【图示说明】**

```
      y
      ↑
      │    P(3, 2)
      │     *────────┐
      │     │        │
      │     │        │ 纵坐标=2
      │     └────────│
──────┼──────┬───────┘
      │     │
      │     └── 横坐标=3
      │
      ↓

点P的坐标：(3, 2)
```

#### 🔍 深入理解：坐标的含义

**【坐标是一对有序数对】**

```
P(3, 2) ≠ P(2, 3)

(3, 2)：先向右3个单位，再向上2个单位
(2, 3)：先向右2个单位，再向上3个单位

顺序很重要！
```

**【坐标的几何意义】**

```
横坐标：点到y轴的水平距离
→ 向右为正，向左为负

纵坐标：点到x轴的垂直距离
→ 向上为正，向下为负
```

#### 📊 特殊位置的点的坐标特征

**【类型1：坐标轴上的点】**

| 位置 | 坐标特征 | 说明 |
|------|---------|------|
| **原点** | (0, 0) | 两坐标轴的交点 |
| **x轴上的点** | (a, 0) | 纵坐标为0 |
| **y轴上的点** | (0, b) | 横坐标为0 |
| **正半x轴** | (a, 0), a>0 | 横坐标为正 |
| **负半x轴** | (a, 0), a<0 | 横坐标为负 |
| **正半y轴** | (0, b), b>0 | 纵坐标为正 |
| **负半y轴** | (0, b), b<0 | 纵坐标为负 |

**【类型2：象限角平分线上的点】**

**第一、三象限角平分线（y = x）**：
```
特征：横坐标等于纵坐标

例子：
(1, 1)，(2, 2)，(3, 3)...
(-1, -1)，(-2, -2)，(-3, -3)...

一般形式：(a, a)
```

**第二、四象限角平分线（y = -x）**：
```
特征：横坐标与纵坐标互为相反数

例子：
(1, -1)，(2, -2)，(3, -3)...
(-1, 1)，(-2, 2)，(-3, 3)...

一般形式：(a, -a)
```

**【类型3：平行于坐标轴的直线上的点】**

**平行于x轴的直线**：
```
特征：纵坐标相同

例子：
y = 3 直线上的点：(1, 3), (2, 3), (3, 3)...
```

**平行于y轴的直线**：
```
特征：横坐标相同

例子：
x = 2 直线上的点：(2, 1), (2, 2), (2, 3)...
```

#### ⚠️ 易错点分析

**【易错点1：坐标顺序错误】**

❌ 错误：把横坐标和纵坐标写反

✓ 正确：
```
P(a, b) → a是横坐标，b是纵坐标
```

**【易错点2：象限判断错误】**

常见错误：只看横坐标或纵坐标的符号

✓ 正确方法：
```
步骤1：看横坐标的符号
步骤2：看纵坐标的符号
步骤3：两者结合判断象限
```

**【易错点3：忘记坐标轴上的点不属于任何象限】**

❌ 错误：(2, 0)在第四象限

✓ 正确：(2, 0)在x轴上，不属于任何象限

**【易错点4：混淆距离和坐标】**

❌ 错误：点P(3, -2)到x轴的距离是-2

✓ 正确：点P(3, -2)到x轴的距离是|-2| = 2

**【易错点5：平移时坐标变化搞错】**

记忆口诀：
```
左右平移：横坐标变（右加左减）
上下平移：纵坐标变（上加下减）
```

**练习3**：

<summary>💡 点击查看详细解答</summary>

**解答**：

**（1）在x轴上，到原点距离为3的点**
```
分析：
在x轴上 → 纵坐标为0
设点坐标为(x, 0)

到原点距离 = |x|
|x| = 3
∴ x = ±3

答案：(3, 0)和(-3, 0)
```

**（2）在y轴上，到原点距离为4的点**
```
分析：
在y轴上 → 横坐标为0
设点坐标为(0, y)

到原点距离 = |y|
|y| = 4
∴ y = ±4

答案：(0, 4)和(0, -4)
```

```
分析：
在第一、三象限角平分线上 → 横坐标等于纵坐标
设点坐标为(a, a)

到原点距离 = √(a² + a²) = √(2a²)
√(2a²) = √2
2a² = 2
a² = 1
a = ±1

答案：(1, 1)和(-1, -1)
```

## 📚 9.2 坐标方法的简单应用

### 9.2.1 用坐标表示平移

#### 📖 从生活中的移动说起

**【游戏中的角色移动】**

```
游戏角色在地图上的位置是 (2, 3)

玩家按下"右"键，角色向右移动2格
→ 新位置：(4, 3)

玩家再按下"上"键，角色向上移动1格
→ 新位置：(4, 4)
```

**【问题】**
移动前后，坐标有什么变化规律？

#### 🔍 深入理解：平移的坐标变化规律

**【左右平移：横坐标变化】**

```
向右平移a个单位：
(x, y) → (x + a, y)

为什么？
→ 向右移动，横坐标增大
→ 纵坐标不变（不上下移动）

例子：
点 A(2, 3) 向右平移2个单位
→ A'(2 + 2, 3) = (4, 3)
```

```
向左平移a个单位：
(x, y) → (x - a, y)

为什么？
→ 向左移动，横坐标减小
→ 纵坐标不变

例子：
点 A(2, 3) 向左平移3个单位
→ A'(2 - 3, 3) = (-1, 3)
```

**【上下平移：纵坐标变化】**

```
向上平移b个单位：
(x, y) → (x, y + b)

为什么？
→ 向上移动，纵坐标增大
→ 横坐标不变（不左右移动）

例子：
点 A(2, 3) 向上平移1个单位
→ A'(2, 3 + 1) = (2, 4)
```

```
向下平移b个单位：
(x, y) → (x, y - b)

为什么？
→ 向下移动，纵坐标减小
→ 横坐标不变

例子：
点 A(2, 3) 向下平移2个单位
→ A'(2, 3 - 2) = (2, 1)
```

#### 📊 平移规律总结表

| 平移方向 | 坐标变化 | 记忆口诀 |
|---------|---------|---------|
| **向右** | 横坐标+a | 右加 |
| **向左** | 横坐标-a | 左减 |
| **向上** | 纵坐标+b | 上加 |
| **向下** | 纵坐标-b | 下减 |

**【复合平移】**

```
先向右平移a个单位，再向上平移b个单位：
(x, y) → (x + a, y + b)

例子：
点 A(2, 3)
→ 向右平移2个单位：(4, 3)
→ 再向上平移1个单位：(4, 4)

最终坐标：(4, 4)
```

#### ⚠️ 易错点分析

**【易错点1：方向搞反】**

❌ 错误：向右平移时横坐标减

✓ 记忆：右是正方向，所以加

**【易错点2：横纵坐标混淆】**

❌ 错误：左右平移改变纵坐标

✓ 记忆：左右是水平方向，只改变横坐标

**【易错点3：连续平移计算错误】**

正确方法：
```
按顺序逐步计算
不要一次性加多个值
```

### 9.2.2 用坐标表示对称

#### 📖 镜子中的对称

**【照镜子的启示】**

```
站在镜子前，举起右手
镜子里的人举起左手

这就是关于镜面的对称！
```

**【平面图形的对称】**

在平面直角坐标系中：
- **x轴** 就像一面水平放置的镜子
- **y轴** 就像一面竖直放置的镜子
- **原点** 是中心对称的中心

#### 🔍 深入理解：对称的坐标变化规律

**【关于x轴对称】**

```
定义：沿x轴对折，点P与点P'重合

坐标变化：(a, b) → (a, -b)

为什么？
→ 横坐标不变（都在x轴同侧）
→ 纵坐标变为相反数（上下互换）

图示：
      y
      ↑
  P(a,b)│
     *  │
      │  │
──────┼──┼──────→ x
      │  │
     *  │
  P'(a,-b)│
      ↓

例子：
点 A(3, 2) 关于x轴的对称点
→ A'(3, -2)
```

**【关于y轴对称】**

```
定义：沿y轴对折，点P与点P'重合

坐标变化：(a, b) → (-a, b)

为什么？
→ 横坐标变为相反数（左右互换）
→ 纵坐标不变（都在y轴同侧）

图示：
      y
      ↑
      │  * P'(-a,b)
      │  │
──────┼──┼──────→ x
      │  │
      │  * P(a,b)
      │

例子：
点 A(3, 2) 关于y轴的对称点
→ A'(-3, 2)
```

**【关于原点对称】**

```
定义：绕原点旋转180°，点P与点P'重合

坐标变化：(a, b) → (-a, -b)

为什么？
→ 横坐标变为相反数
→ 纵坐标变为相反数

图示：
      y
      │
  P'(-a,b)  P(a,b)
      *      *
      │      │
──────┼──────┼────→ x
      │      │
      *      *
  P''(-a,-b) P'(a,-b)
      │

例子：
点 A(3, 2) 关于原点的对称点
→ A'(-3, -2)
```

#### 📊 对称规律总结表

| 对称类型 | 坐标变化 | 记忆口诀 |
|---------|---------|---------|
| **关于x轴** | (a, b) → (a, -b) | 横不变，纵相反 |
| **关于y轴** | (a, b) → (-a, b) | 横相反，纵不变 |
| **关于原点** | (a, b) → (-a, -b) | 横纵都相反 |

#### 💡 对称的快速判断技巧

**【技巧1：看轴记忆】**

```
关于哪个轴对称，哪个坐标就不变

关于x轴对称 → x坐标不变
关于y轴对称 → y坐标不变
```

**【技巧2：相反数的应用】**

```
关于原点对称 → 两个坐标都取相反数

可以理解为：
先关于x轴对称，再关于y轴对称
或
先关于y轴对称，再关于x轴对称
```

### 9.2.3 用坐标表示距离

#### 📖 点到坐标轴的距离

**【直观理解】**

```
      y
      ↑
      │
      │    P(3, 2)
      │     *
      │     │
      │     │ 距离=2
──────┼─────*──────→ x
      │     │
      │     │ 距离=3
      │     *
      │

点P到x轴的距离：2（纵坐标的绝对值）
点P到y轴的距离：3（横坐标的绝对值）
```

#### 🔍 深入理解：距离的计算公式

**【点到x轴的距离】**

```
定义：点到x轴的垂直距离

公式：点P(a, b)到x轴的距离 = |b|

为什么是绝对值？
→ 距离永远是正数
→ 无论在x轴上方还是下方

例子：
P(3, 2) 到x轴的距离 = |2| = 2
P(3, -2) 到x轴的距离 = |-2| = 2
```

**【点到y轴的距离】**

```
定义：点到y轴的水平距离

公式：点P(a, b)到y轴的距离 = |a|

为什么是绝对值？
→ 距离永远是正数
→ 无论在y轴左侧还是右侧

例子：
P(3, 2) 到y轴的距离 = |3| = 3
P(-3, 2) 到y轴的距离 = |-3| = 3
```

**【点到原点的距离】**

```
定义：点和原点之间的线段长度

公式：点P(a, b)到原点的距离 = √(a² + b²)

为什么？（勾股定理）

      y
      ↑
      │    P(a, b)
      │     *
      │     │
      │     │ |b|
──────┼─────*──────→ x
      │     │ |a|
      │     *
      │

直角三角形：
直角边1：|a|（水平距离）
直角边2：|b|（垂直距离）
斜边：原点到P的距离 = √(a² + b²)

例子：
P(3, 4) 到原点的距离 = √(3² + 4²) = √25 = 5
```

#### ⚠️ 易错点分析

**【易错点1：忘记取绝对值】**

❌ 错误：P(3, -2)到x轴的距离是-2

✓ 正确：P(3, -2)到x轴的距离是|-2| = 2

**【易错点2：混淆横纵坐标】**

记忆：
```
到x轴的距离：看纵坐标（竖直方向）
到y轴的距离：看横坐标（水平方向）
```

**【易错点3：原点距离公式记错】**

✓ 正确：√(a² + b²)

❌ 错误：|a| + |b| 或 |a + b|
1. **左右平移**：横坐标变化，纵坐标不变
   - 向右平移a个单位：(x, y) → (x + a, y)
   - 向左平移a个单位：(x, y) → (x - a, y)

2. **上下平移**：横坐标不变，纵坐标变化
   - 向上平移b个单位：(x, y) → (x, y + b)
   - 向下平移b个单位：(x, y) → (x, y - b)

**例子**：
- 点A(2, 3)向右平移2个单位，得到点A'(4, 3)
- 点A(2, 3)向上平移1个单位，得到点A''(2, 4)

#### 9.2.2 用坐标表示对称

**规律**：
1. **关于x轴对称**：横坐标不变，纵坐标变为相反数
   - 点P(a, b)关于x轴的对称点是P'(a, -b)

2. **关于y轴对称**：横坐标变为相反数，纵坐标不变
   - 点P(a, b)关于y轴的对称点是P'(-a, b)

3. **关于原点对称**：横坐标和纵坐标都变为相反数
   - 点P(a, b)关于原点的对称点是P'(-a, -b)

**例子**：
- 点A(2, 3)关于x轴的对称点是A'(2, -3)
- 点A(2, 3)关于y轴的对称点是A''(-2, 3)
- 点A(2, 3)关于原点的对称点是A'''(-2, -3)

#### 9.2.3 用坐标表示距离

**点到x轴的距离**：|纵坐标|
- 点P(a, b)到x轴的距离是 |b|

**点到y轴的距离**：|横坐标|
- 点P(a, b)到y轴的距离是 |a|

**点到原点的距离**：√(横坐标² + 纵坐标²)
- 点P(a, b)到原点的距离是 √(a² + b²)

### 例题6：点到坐标轴的距离

**题目**：

（2）点A(3, -2)到y轴的距离是多少？
（3）点A(3, -2)到原点的距离是多少？

**解答**：
（1）点A到x轴的距离 = |-2| = **2**

（2）点A到y轴的距离 = |3| = **3**

（3）点A到原点的距离 = √(3² + (-2)²) = √(9 + 4) = **√13**

### 例题7：特殊位置的点

**题目**：
（1）在x轴上，到原点距离为3的点有哪些？
（2）在y轴上，到原点距离为4的点有哪些？
（3）在第一、三象限角平分线上，到原点距离为√2的点有哪些？

**解答**：
（1）在x轴上，点的纵坐标为0
    设点坐标为(x, 0)，则 |x| = 3
    所以 x = ±3
    点有：**(3, 0)和(-3, 0)**

（2）在y轴上，点的横坐标为0
    设点坐标为(0, y)，则 |y| = 4
    所以 y = ±4
    点有：**(0, 4)和(0, -4)**

（3）在第一、三象限角平分线上，横坐标等于纵坐标
    设点坐标为(a, a)，则距离 = √(a² + a²) = √(2a²) = √2
    所以 a = ±1
    点有：**(1, 1)和(-1, -1)**

**题目**：如图，在学校平面图中，教学楼在校门正北方向100m处，图书馆在校门正东方向50m处，实验室在校门北偏东60°方向80m处。请建立适当的平面直角坐标系，表示各建筑的坐标。

**解答**：
**建立坐标系**：
- 以校门为原点
- 以正东方向为x轴正方向
- 以正北方向为y轴正方向
- 单位长度：10m

**各建筑的坐标**：
- 校门：**(0, 0)**
- 教学楼：**(0, 10)**（正北100m，即y轴正方向10个单位）

### 例题9：图形的平移

**题目**：如图，△ABC的三个顶点坐标分别为A(1, 2)，B(3, 1)，C(2, 3)，将△ABC向右平移2个单位，再向下平移1个单位，得到△A'B'C'，求△A'B'C'的三个顶点坐标。

**解答**：
向右平移2个单位，再向下平移1个单位：
- 横坐标加2，纵坐标减1

- A'(1 + 2, 2 - 1) = **(3, 1)**
- B'(3 + 2, 1 - 1) = **(5, 0)**
- C'(2 + 2, 3 - 1) = **(4, 2)**

### 例题10：综合应用

**题目**：在平面直角坐标系中，已知点A(2, 3)，点B在点A的左侧，且AB = 3，点B的纵坐标与点A相同，求点B的坐标。

**解答**：
点B在点A的左侧，且AB = 3
点B的纵坐标与点A相同，都是3
点B的横坐标 = 2 - 3 = -1
所以点B的坐标是 **(-1, 3)**
