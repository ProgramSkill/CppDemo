# 第九章 平面直角坐标系

## 📚 知识要点

### 9.1 平面直角坐标系

#### 9.1.1 平面直角坐标系的组成

**定义**：在平面内画两条互相垂直且有公共原点的数轴，就组成了平面直角坐标系

**组成要素**：
1. **横轴（x轴）**：水平的数轴，向右为正方向
2. **纵轴（y轴）**：竖直的数轴，向上为正方向
3. **原点**：两坐标轴的交点，记作点O
4. **坐标平面**：两条坐标轴把平面分成的四个部分

**坐标平面的象限**：
```
第二象限  |  第一象限
────────┼────────
第三象限  |  第四象限
```

- **第一象限**：+x，+y（x > 0，y > 0）
- **第二象限**：-x，+y（x < 0，y > 0）
- **第三象限**：-x，-y（x < 0，y < 0）
- **第四象限**：+x，-y（x > 0，y < 0）

**注意**：坐标轴上的点不属于任何象限

#### 9.1.2 点的坐标

**表示方法**：
- 平面内一点P的坐标记作：P(a, b)
- a叫做横坐标（x坐标）
- b叫做纵坐标（y坐标）

**确定点的坐标**：
- 过点P向x轴作垂线，垂足对应的数是横坐标
- 过点P向y轴作垂线，垂足对应的数是纵坐标

**特殊位置的点的坐标特征**：
1. **原点**：O(0, 0)
2. **x轴上的点**：(a, 0)
3. **y轴上的点**：(0, b)
4. **第一、三象限角平分线上的点**：横坐标等于纵坐标，即 (a, a)
5. **第二、四象限角平分线上的点**：横坐标与纵坐标互为相反数，即 (a, -a)
6. **平行于x轴的直线上的点**：纵坐标相同
7. **平行于y轴的直线上的点**：横坐标相同

#### 9.1.3 用坐标表示地理位置

**方法**：
1. 建立平面直角坐标系，选择适当的原点、坐标轴和单位长度
2. 确定各地点的坐标
3. 用坐标表示地理位置

**注意**：
- 原点的选择要适当，使各地点的坐标尽可能简单
- 坐标轴的正方向和单位长度要合理

### 9.2 坐标方法的简单应用

#### 9.2.1 用坐标表示平移

**规律**：
1. **左右平移**：横坐标变化，纵坐标不变
   - 向右平移a个单位：(x, y) → (x + a, y)
   - 向左平移a个单位：(x, y) → (x - a, y)

2. **上下平移**：横坐标不变，纵坐标变化
   - 向上平移b个单位：(x, y) → (x, y + b)
   - 向下平移b个单位：(x, y) → (x, y - b)

**例子**：
- 点A(2, 3)向右平移2个单位，得到点A'(4, 3)
- 点A(2, 3)向上平移1个单位，得到点A''(2, 4)

#### 9.2.2 用坐标表示对称

**规律**：
1. **关于x轴对称**：横坐标不变，纵坐标变为相反数
   - 点P(a, b)关于x轴的对称点是P'(a, -b)

2. **关于y轴对称**：横坐标变为相反数，纵坐标不变
   - 点P(a, b)关于y轴的对称点是P'(-a, b)

3. **关于原点对称**：横坐标和纵坐标都变为相反数
   - 点P(a, b)关于原点的对称点是P'(-a, -b)

**例子**：
- 点A(2, 3)关于x轴的对称点是A'(2, -3)
- 点A(2, 3)关于y轴的对称点是A''(-2, 3)
- 点A(2, 3)关于原点的对称点是A'''(-2, -3)

#### 9.2.3 用坐标表示距离

**点到x轴的距离**：|纵坐标|
- 点P(a, b)到x轴的距离是 |b|

**点到y轴的距离**：|横坐标|
- 点P(a, b)到y轴的距离是 |a|

**点到原点的距离**：√(横坐标² + 纵坐标²)
- 点P(a, b)到原点的距离是 √(a² + b²)

### 例题2：由坐标描点

**题目**：在平面直角坐标系中描出下列各点：
A(2, 3)，B(-3, 2)，C(-2, -3)，D(3, -2)

**解答**：
1. 点A(2, 3)：在第一象限，先向右2个单位，再向上3个单位
2. 点B(-3, 2)：在第二象限，先向左3个单位，再向上2个单位
3. 点C(-2, -3)：在第三象限，先向左2个单位，再向下3个单位
4. 点D(3, -2)：在第四象限，先向右3个单位，再向下2个单位

---

### 例题3：象限的判断

**题目**：点P(-3, 2)在第几象限？点Q(3, -2)在第几象限？

**解答**：
- 点P(-3, 2)：横坐标为负，纵坐标为正，在**第二象限**
- 点Q(3, -2)：横坐标为正，纵坐标为负，在**第四象限**

### 例题4：平移问题

**题目**：
（1）点A(2, 3)向右平移2个单位，得到点A'，求点A'的坐标。
（2）点B(-1, 2)向下平移3个单位，得到点B'，求点B'的坐标。
（3）点C(1, -2)向左平移2个单位，再向上平移4个单位，得到点C'，求点C'的坐标。

**解答**：
（1）点A'(2 + 2, 3) = **(4, 3)**

（2）点B'(-1, 2 - 3) = **(-1, -1)**

（3）点C'(1 - 2, -2 + 4) = **(-1, 2)**

---

### 例题5：对称问题

**题目**：
（1）点A(3, 2)关于x轴的对称点是A'，求A'的坐标。
（2）点B(-2, 3)关于y轴的对称点是B'，求B'的坐标。
（3）点C(1, -2)关于原点的对称点是C'，求C'的坐标。

**解答**：
（1）A'(3, -2)

（2）B'(2, 3)

（3）C'(-1, 2)

---

### 例题6：点到坐标轴的距离

**题目**：
（1）点A(3, -2)到x轴的距离是多少？
（2）点A(3, -2)到y轴的距离是多少？
（3）点A(3, -2)到原点的距离是多少？

**解答**：
（1）点A到x轴的距离 = |-2| = **2**

（2）点A到y轴的距离 = |3| = **3**

（3）点A到原点的距离 = √(3² + (-2)²) = √(9 + 4) = **√13**

---

### 例题7：特殊位置的点

**题目**：
（1）在x轴上，到原点距离为3的点有哪些？
（2）在y轴上，到原点距离为4的点有哪些？
（3）在第一、三象限角平分线上，到原点距离为√2的点有哪些？

**解答**：
（1）在x轴上，点的纵坐标为0
    设点坐标为(x, 0)，则 |x| = 3
    所以 x = ±3
    点有：**(3, 0)和(-3, 0)**

（2）在y轴上，点的横坐标为0
    设点坐标为(0, y)，则 |y| = 4
    所以 y = ±4
    点有：**(0, 4)和(0, -4)**

（3）在第一、三象限角平分线上，横坐标等于纵坐标
    设点坐标为(a, a)，则距离 = √(a² + a²) = √(2a²) = √2
    所以 a = ±1
    点有：**(1, 1)和(-1, -1)**

### 例题8：建立坐标系表示地理位置

**题目**：如图，在学校平面图中，教学楼在校门正北方向100m处，图书馆在校门正东方向50m处，实验室在校门北偏东60°方向80m处。请建立适当的平面直角坐标系，表示各建筑的坐标。

**解答**：
**建立坐标系**：
- 以校门为原点
- 以正东方向为x轴正方向
- 以正北方向为y轴正方向
- 单位长度：10m

**各建筑的坐标**：
- 校门：**(0, 0)**
- 教学楼：**(0, 10)**（正北100m，即y轴正方向10个单位）
- 图书馆：**(5, 0)**（正东50m，即x轴正方向5个单位）
- 实验室：**(40√3, 40)**（北偏东60°，横坐标 = 80sin60° = 40√3，纵坐标 = 80cos60° = 40）

### 例题9：图形的平移

**题目**：如图，△ABC的三个顶点坐标分别为A(1, 2)，B(3, 1)，C(2, 3)，将△ABC向右平移2个单位，再向下平移1个单位，得到△A'B'C'，求△A'B'C'的三个顶点坐标。

**解答**：
向右平移2个单位，再向下平移1个单位：
- 横坐标加2，纵坐标减1

- A'(1 + 2, 2 - 1) = **(3, 1)**
- B'(3 + 2, 1 - 1) = **(5, 0)**
- C'(2 + 2, 3 - 1) = **(4, 2)**

---

### 例题10：综合应用

**题目**：在平面直角坐标系中，已知点A(2, 3)，点B在点A的左侧，且AB = 3，点B的纵坐标与点A相同，求点B的坐标。

**解答**：
点B在点A的左侧，且AB = 3
点B的纵坐标与点A相同，都是3
点B的横坐标 = 2 - 3 = -1
所以点B的坐标是 **(-1, 3)**

1. **掌握坐标系结构**：理解平面直角坐标系的组成和象限的划分

3. **记忆平移规律**：左右平移横坐标变，上下平移纵坐标变
4. **记忆对称规律**：关于哪个轴对称，哪个坐标不变
5. **数形结合**：坐标系是数形结合的重要工具，要熟练运用
6. **建立坐标系**：根据实际问题建立适当的坐标系
