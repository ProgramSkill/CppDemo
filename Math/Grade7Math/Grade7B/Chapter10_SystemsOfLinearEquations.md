# 第十章 二元一次方程组

## 📚 知识要点

### 10.1 二元一次方程组

#### 10.1.1 二元一次方程的概念

**定义**：含有两个未知数，并且含有未知数的项的次数都是1的方程叫做二元一次方程

**标准形式**：ax + by = c（a、b不为0）

**例子**：
- 2x + 3y = 7（是二元一次方程）
- x - y = 5（是二元一次方程）
- x² + y = 3（不是，x的次数是2）
- xy + z = 5（不是，xy的次数是2）

#### 10.1.2 二元一次方程的解

**定义**：使二元一次方程两边的值相等的两个未知数的值，叫做二元一次方程的解

**注意**：
- 二元一次方程有无数个解
- 二元一次方程的解是一对数值

**例子**：x + y = 5的解有：
- x = 1, y = 4
- x = 2, y = 3
- x = 3, y = 2
- x = 4, y = 1
等等

#### 10.1.3 二元一次方程组的概念

**定义**：把具有相同未知数的两个二元一次方程合在一起，就组成了一个二元一次方程组

**标准形式**：
```
ax + by = c
dx + ey = f
```

**例子**：
```
x + y = 5
x - y = 1
```

#### 10.1.4 二元一次方程组的解

**定义**：二元一次方程组的两个方程的公共解，叫做二元一次方程组的解

**注意**：二元一次方程组通常只有一个解（特殊情况除外）

### 10.2 二元一次方程组的解法

#### 10.2.1 代入消元法

**基本思想**：将方程组中的一个未知数用含另一个未知数的代数式表示，代入另一个方程，实现消元

**步骤**：
1. 选取一个方程，将它变形为 y = ax + b 或 x = ay + b 的形式
2. 将这个式子代入另一个方程，消去一个未知数
3. 解出这个未知数
4. 将求得的值代回变形后的方程，求出另一个未知数
5. 写出方程组的解

**例子**：解方程组
```
x + y = 5  ①
x - y = 1  ②
```

**解**：
由①得：x = 5 - y ③
将③代入②，得：(5 - y) - y = 1
5 - 2y = 1
-2y = -4
y = 2

将 y = 2 代入③，得：x = 5 - 2 = 3

所以方程组的解是：**{x = 3, y = 2}**

#### 10.2.2 加减消元法

**基本思想**：通过两方程相加或相减，消去其中一个未知数

**步骤**：
1. 观察未知数的系数，选择要消去的未知数
2. 如果系数绝对值不相等，将方程变形，使要消去的未知数的系数绝对值相等
3. 将两方程相加或相减，消去一个未知数
4. 解出这个未知数
5. 将求得的值代入原方程组中的任意一个方程，求出另一个未知数
6. 写出方程组的解

**例子**：解方程组
```
3x + 2y = 13  ①
3x - 2y = 5   ②
```

**解**：
① + ②，得：6x = 18
x = 3

将 x = 3 代入①，得：3 × 3 + 2y = 13
9 + 2y = 13
2y = 4
y = 2

所以方程组的解是：**{x = 3, y = 2}**

### 10.3 实际问题与二元一次方程组

#### 10.3.1 列方程组解应用题的一般步骤

**步骤**：
1. **审题**：理解题意，明确已知量和未知量
2. **设未知数**：用字母表示未知数
3. **找等量关系**：找出题目中的两个等量关系
4. **列方程组**：根据等量关系列出二元一次方程组
5. **解方程组**：求出未知数的值
6. **检验**：检验求得的值是否符合题意
7. **作答**：写出答案

#### 10.3.2 常见应用题类型

**类型一：和差倍分问题**

**例子**：某班共有学生50人，其中男生比女生多10人，求男生和女生各有多少人？

**解答**：
设男生有x人，女生有y人
根据题意：
```
x + y = 50  ①
x - y = 10  ②
```

① + ②，得：2x = 60，x = 30
将 x = 30 代入①，得：y = 20

答：男生有30人，女生有20人。

---

**类型二：行程问题**

**例子**：甲、乙两人同时从相距100km的两地相向而行，甲的速度为15km/h，乙的速度为10km/h，经过几小时两人相遇？相遇时各自行驶了多少路程？

**解答**：
设经过x小时两人相遇
甲行驶的路程：15x km
乙行驶的路程：10x km
根据题意：15x + 10x = 100
25x = 100
x = 4

甲行驶路程：15 × 4 = 60（km）
乙行驶路程：10 × 4 = 40（km）

答：经过4小时两人相遇，甲行驶了60km，乙行驶了40km。

---

**类型三：工程问题**

**例子**：某项工程，甲单独做需要10天完成，乙单独做需要15天完成，两人合作需要多少天完成？

**解答**：
设两人合作需要x天完成
甲的效率：1/10
乙的效率：1/15
根据题意：x/10 + x/15 = 1

去分母（同乘30），得：3x + 2x = 30
5x = 30
x = 6

答：两人合作需要6天完成。

---

**类型四：配套问题**

**例子**：某车间有工人50人，一个工人每天平均生产螺栓12个或螺母18个，应分配多少人生产螺栓，多少人生产螺母，才能使螺栓和螺母配套（1个螺栓配2个螺母）？

**解答**：
设x人生产螺栓，y人生产螺母
根据题意：
```
x + y = 50  ①
12x × 2 = 18y  ②
```

由②得：24x = 18y，即 4x = 3y，y = 4x/3 ③
将③代入①，得：x + 4x/3 = 50
7x/3 = 50
x = 150/7 ≈ 21.4

这不是整数，需要调整人数。

## 📖 典型例题

### 例题1：判断二元一次方程

**题目**：下列方程中，哪些是二元一次方程？
（1）2x + 3y = 7
（2）x² + y = 5
（3）x + y + z = 10
（4）xy + z = 6

**解答**：
二元一次方程有：**（1）2x + 3y = 7**

不是二元一次方程的是：
- （2）x² + y = 5（x的次数是2）
- （3）x + y + z = 10（有三个未知数）
- （4）xy + z = 6（xy的次数是2）

---

### 例题2：代入消元法

**题目**：解方程组
```
y = 2x  ①
x + y = 6  ②
```

**解答**：
将①代入②，得：x + 2x = 6
3x = 6
x = 2

将 x = 2 代入①，得：y = 2 × 2 = 4

所以方程组的解是：**{x = 2, y = 4}**

---

### 例题3：加减消元法

**题目**：解方程组
```
2x + 3y = 7  ①
3x - 2y = 4  ②
```

**解答**：
① × 2，得：4x + 6y = 14  ③
② × 3，得：9x - 6y = 12  ④

③ + ④，得：13x = 26
x = 2

将 x = 2 代入①，得：2 × 2 + 3y = 7
4 + 3y = 7
3y = 3
y = 1

所以方程组的解是：**{x = 2, y = 1}**

---

### 例题4：选择合适的解法

**题目**：选择合适的方法解下列方程组
（1）
```
x + y = 5
x - y = 1
```

（2）
```
y = 2x - 1
3x + y = 9
```

**解答**：
（1）用加减消元法：
两式相加，得：2x = 6，x = 3
将 x = 3 代入第一个方程，得：y = 2
解：**{x = 3, y = 2}**

（2）用代入消元法：
将 y = 2x - 1 代入第二个方程，得：3x + (2x - 1) = 9
5x - 1 = 9
5x = 10
x = 2
将 x = 2 代入第一个方程，得：y = 3
解：**{x = 2, y = 3}**

---

### 例题5：和差问题

**题目**：某数的和是10，差是2，求这两个数。

**解答**：
设这两个数分别是x和y（x > y）
根据题意：
```
x + y = 10  ①
x - y = 2  ②
```

① + ②，得：2x = 12，x = 6
将 x = 6 代入①，得：y = 4

答：这两个数是6和4。

---

### 例题6：配套问题

**题目**：某车间有男工人5人，女工人4人，一天共生产螺栓和螺母若干，已知1个螺栓配2个螺母，每人每天生产螺栓20个或螺母30个，应如何分配才能使螺栓和螺母配套？

**解答**：
设x个男工生产螺栓，y个女工生产螺母
根据题意：
```
x + y = 9  ①
20x × 2 = 30y  ②
```

由②得：40x = 30y，即 4x = 3y
所以 x : y = 3 : 4

设 x = 3k，y = 4k
代入①，得：3k + 4k = 9，k = 9/7
这不是整数，需要调整分配方案。

### 例题7：行程问题

**题目**：A、B两地相距100km，甲从A地出发，速度为15km/h，乙从B地同时出发，速度为10km/h，相向而行，经过几小时两人相遇？

**解答**：
设经过x小时两人相遇
甲行驶的路程：15x km
乙行驶的路程：10x km
根据题意：15x + 10x = 100
25x = 100
x = 4

答：经过4小时两人相遇。

### 例题8：综合应用

**题目**：某商店用两种价格分别为每千克18元和每千克10元的糖果混合成售价为每千克15元的什锦糖果100kg，求需要两种糖果各多少千克？

**解答**：
设需要18元的糖果x kg，10元的糖果y kg
根据题意：
```
x + y = 100  ①
18x + 10y = 15 × 100  ②
```

由①得：y = 100 - x ③
将③代入②，得：18x + 10(100 - x) = 1500
18x + 1000 - 10x = 1500
8x = 500
x = 62.5

将 x = 62.5 代入③，得：y = 37.5

答：需要18元的糖果62.5kg，10元的糖果37.5kg。
