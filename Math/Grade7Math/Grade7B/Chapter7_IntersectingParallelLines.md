# 第七章 相交线与平行线

## 📚 知识要点

### 7.1 相交线

#### 7.1.1 邻补角和对顶角

**邻补角**：
- 定义：两条直线相交所构成的四个角中，有公共顶点且有一条公共边的两个角互为邻补角
- 特点：两个角之和等于180°

**对顶角**：
- 定义：两条直线相交所构成的四个角中，有公共顶点但没有公共边的两个角互为对顶角
- 性质：对顶角相等

**例子**：
直线AB和CD相交于点O，形成四个角：∠1、∠2、∠3、∠4
- ∠1和∠2是对顶角，∠1 = ∠2
- ∠3和∠4是对顶角，∠3 = ∠4
- ∠1和∠3是邻补角，∠1 + ∠3 = 180°

#### 7.1.2 垂线

**垂线**：
- 定义：当两条直线相交所成的四个角中，有一个角是直角时，就说这两条直线互相垂直，其中一条直线叫做另一条直线的垂线
- 表示：AB ⊥ CD（读作"AB垂直于CD"）
- 交点叫做垂足

**垂线的性质**：
1. 过一点有且只有一条直线与已知直线垂直
2. 连接直线外一点与直线上各点的所有线段中，垂线段最短

**点到直线的距离**：
- 定义：直线外一点到这条直线的垂线段的长度，叫做点到直线的距离

**注意**：垂线是直线，垂线段是线段，距离是数量

#### 7.1.3 同位角、内错角、同旁内角

**三线八角**：两条直线被第三条直线所截，形成的八个角

**同位角**：
- 定义：在两条直线的同侧，第三条直线的同旁的两个角
- 位置特征："同旁同侧"

**内错角**：
- 定义：在两条直线之间，第三条直线的两侧的两个角
- 位置特征："之间两侧"

**同旁内角**：
- 定义：在两条直线之间，第三条直线的同旁的两个角
- 位置特征："之间同旁"

### 7.2 平行线

#### 7.2.1 平行线的概念

**平行线**：
- 定义：在同一平面内，不相交的两条直线叫做平行线
- 表示：AB ∥ CD（读作"AB平行于CD"）

**平行公理**：
- 经过直线外一点，有且只有一条直线与这条直线平行

**平行公理的推论**：
- 如果两条直线都与第三条直线平行，那么这两条直线也互相平行
- 若 a ∥ c，b ∥ c，则 a ∥ b

#### 7.2.2 平行线的判定

**判定方法一**：同位角相等，两直线平行

**判定方法二**：内错角相等，两直线平行

**判定方法三**：同旁内角互补，两直线平行

**例子**：
- 若∠1 = ∠2（∠1和∠2是同位角），则 AB ∥ CD
- 若∠3 = ∠4（∠3和∠4是内错角），则 AB ∥ CD
- 若∠5 + ∠6 = 180°（∠5和∠6是同旁内角），则 AB ∥ CD

#### 7.2.3 平行线的性质

**性质一**：两直线平行，同位角相等

**性质二**：两直线平行，内错角相等

**性质三**：两直线平行，同旁内角互补

**注意**：平行线的判定与性质是互逆关系
- 判定：角的关系 → 直线平行
- 性质：直线平行 → 角的关系

### 7.3 平移

#### 7.3.1 平移的概念

**定义**：在平面内，将一个图形沿某个方向移动一定的距离，这样的图形运动称为平移

**平移的两个要素**：
1. 方向
2. 距离

#### 7.3.2 平移的性质

1. 平移不改变图形的形状和大小
2. 平移后，对应点所连的线段平行（或在同一直线上）且相等
3. 平移后，对应角相等
4. 平移后，对应线段平行（或在同一直线上）且相等

### 7.4 命题、定理、证明

#### 7.4.1 命题

**定义**：判断一件事情的语句叫做命题

**命题的结构**：
- 题设：已知事项
- 结论：由已知事项推出的事项

**命题的分类**：
- 真命题：正确的命题
- 假命题：错误的命题

**例子**：
- "如果两条直线都和第三条直线平行，那么这两条直线也互相平行"是真命题
- "相等的角是对顶角"是假命题（反例：等腰三角形底角）

#### 7.4.2 定理和证明

**定理**：用推理的方法判断为真命题的命题叫做定理

**证明**：除了公理外，其他真命题的正确性都需要通过推理的方法来证实，这样的过程叫做证明

**证明的步骤**：
1. 根据题意，画出图形
2. 根据题设、结论，结合图形，写出已知、求证
3. 经过分析，找出由已知推出求证的途径，写出证明过程

## 📖 典型例题

### 例题1：对顶角和邻补角

**题目**：如图，直线AB和CD相交于点O，∠AOC = 50°，求其他三个角的度数。

**解答**：
因为∠AOC和∠BOD是对顶角，
所以 ∠BOD = ∠AOC = **50°**

因为∠AOC和∠BOC是邻补角，
所以 ∠BOC = 180° - ∠AOC = 180° - 50° = **130°**

因为∠AOD和∠BOC是对顶角，
所以 ∠AOD = ∠BOC = **130°**

---

### 例题2：垂线

**题目**：如图，直线AB和CD相交于点O，OE ⊥ AB，∠BOD = 40°，求∠COE的度数。

**解答**：
因为OE ⊥ AB，
所以 ∠BOE = 90°

因为∠BOD = 40°，
所以 ∠DOE = ∠BOE - ∠BOD = 90° - 40° = 50°

因为∠COE和∠DOE是邻补角，
所以 ∠COE = 180° - ∠DOE = 180° - 50° = **130°**

---

### 例题3：识别同位角、内错角、同旁内角

**题目**：如图，直线AB、CD被EF所截，请指出：
（1）∠1和∠2是什么角？
（2）∠3和∠4是什么角？
（3）∠5和∠6是什么角？

**解答**：
（1）∠1和∠2是**同位角**（在两条直线的同侧，第三条直线的同旁）

（2）∠3和∠4是**内错角**（在两条直线之间，第三条直线的两侧）

（3）∠5和∠6是**同旁内角**（在两条直线之间，第三条直线的同旁）

---

### 例题4：平行线的判定

**题目**：如图，∠1 = 70°，∠2 = 70°，AB和CD平行吗？为什么？

**解答**：
因为 ∠1 = 70°，∠2 = 70°，
所以 ∠1 = ∠2

又因为∠1和∠2是同位角，
所以 **AB ∥ CD**（同位角相等，两直线平行）

---

### 例题5：平行线的性质

**题目**：如图，AB ∥ CD，∠1 = 60°，求∠2的度数。

**解答**：
因为 AB ∥ CD，
所以 ∠1和∠2是同位角，
所以 ∠2 = ∠1 = **60°**（两直线平行，同位角相等）

---

### 例题6：平行线的综合应用

**题目**：如图，AB ∥ CD，∠1 = 50°，∠2 = 60°，求∠3的度数。

**解答**：
因为 AB ∥ CD，
所以 ∠4 = ∠1 = 50°（两直线平行，同位角相等）

又因为 ∠2 + ∠3 + ∠4 = 180°（平角），
所以 ∠3 = 180° - ∠2 - ∠4 = 180° - 60° - 50° = **70°**

---

### 例题7：平移

**题目**：如图，将△ABC沿箭头方向平移2cm，得到△A'B'C'，请找出平移后的对应点。

**解答**：
对应点：
- 点A的对应点是点A'
- 点B的对应点是点B'
- 点C的对应点是点C'

平移的方向：箭头指向的方向
平移的距离：2cm

---

### 例题8：命题的题设和结论

**题目**：指出下列命题的题设和结论
（1）对顶角相等
（2）两条直线被第三条直线所截，同位角相等，两直线平行

**解答**：
（1）题设：两个角是对顶角
    结论：这两个角相等

（2）题设：两条直线被第三条直线所截，同位角相等
    结论：这两条直线平行

---

### 例题9：证明

**题目**：证明：对顶角相等。

**已知**：如图，直线AB和CD相交于点O
**求证**：∠AOC = ∠BOD

**证明**：
因为∠AOC和∠BOC是邻补角，
所以 ∠AOC + ∠BOC = 180°
即 ∠AOC = 180° - ∠BOC

又因为∠BOD和∠BOC是邻补角，
所以 ∠BOD + ∠BOC = 180°
即 ∠BOD = 180° - ∠BOC

所以 ∠AOC = ∠BOD

---

### 例题10：综合应用

**题目**：如图，AB ∥ CD，AD ∥ BC，∠A = 50°，求∠C的度数。

**解答**：
因为 AD ∥ BC，
所以 ∠A + ∠B = 180°（两直线平行，同旁内角互补）
所以 ∠B = 180° - ∠A = 180° - 50° = 130°

又因为 AB ∥ CD，
所以 ∠B + ∠C = 180°（两直线平行，同旁内角互补）
所以 ∠C = 180° - ∠B = 180° - 130° = **50°**

│
├── 平行线

│
