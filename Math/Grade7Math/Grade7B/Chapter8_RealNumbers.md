### 我们已经学过什么？

在之前的章节中，我们学习了：

- **有理数的运算**：加减乘除乘方

## 📚 8.1 平方根

### 8.1.1 从"平方"到"开平方"

#### 📖 问题的提出

**【情境问题】**

小明有一个正方形的手帕，它的面积是9平方分米。这个正方形的边长是多少分米？

**【思考过程】**

```
设正方形的边长为 x 分米

根据正方形面积公式：
边长 × 边长 = 面积
x · x = 9
x² = 9

现在的问题是：什么数乘以自己等于9？
```

**【尝试求解】**

让我们尝试一些数：
```
1 × 1 = 1  （太小了）
2 × 2 = 4  （还小）
3 × 3 = 9  （对了！）

所以 x = 3
```

但还有一个答案：
```
(-3) × (-3) = 9  （也对！）

所以 x = 3 或 x = -3
```

**【引入概念】**

这就是"平方根"的概念！
- 3 是 9 的平方根
- -3 也是 9 的平方根

#### 🔍 深入理解：平方根的概念

**【定义的形成】**

**平方运算**：
```
已知边长，求面积
边长 = 3 → 面积 = 3² = 9
```

**开平方运算**（平方的逆运算）：
```
已知面积，求边长
面积 = 9 → 边长 = ±√9 = ±3
```

**严格定义**：
> 如果一个数的平方等于a，那么这个数叫做a的平方根。

**数学表达**：
```
如果 x² = a，那么 x 叫做 a 的平方根
```

#### 📊 平方根的表示方法

**符号系统**：
```
a 的平方根记作：±√a

读法："正负根号a"

分解：
  √  → 根号（表示开方）
  a  → 被开方数
  ±  → 表示两个值（正的和负的）
```

**例子详解**：
```
例1：求9的平方根

解：设9的平方根为 x
    则 x² = 9

    ∵ 3² = 9，(-3)² = 9
    ∴ x = ±3

    即 ±√9 = ±3

说明：
- √9 = 3（9的算术平方根）
- -√9 = -3（9的负平方根）
- ±√9 = ±3（9的平方根）
```

#### 🎯 为什么一个正数有两个平方根？

**【直观理解】**

想象一个数轴：
```
      -3         0         +3
      ○───────────○

因为：(-3)² = 9，3² = 9
所以：-3 和 3 都是 9 的平方根
```

**【代数证明】**

```
已知：x² = 9（x ≠ 0）

求证：x 有两个解

证明：x² - 9 = 0
      (x + 3)(x - 3) = 0

∴ x + 3 = 0 或 x - 3 = 0
∴ x = -3 或 x = 3

结论：正数9有两个平方根，互为相反数
```

#### 📐 平方根的三大性质

**【性质1】正数的平方根**

> 一个正数有两个平方根，它们互为相反数。

**例子**：
```
16 的平方根：±4（因为 4² = 16，(-4)² = 16）
25 的平方根：±5（因为 5² = 25，(-5)² = 25）
0.01 的平方根：±0.1（因为 0.1² = 0.01，(-0.1)² = 0.01）
```

**【性质2】零的平方根**

> 0的平方根是0。

**理解**：
```
∵ 0² = 0
∴ 0 的平方根是 0
即 ±√0 = 0（也就是 √0 = 0）
```

**【性质3】负数没有平方根**

> 负数没有平方根。

**为什么？**

```
假设 -4 有平方根，设为 x
则 x² = -4

但我们知道：
- 正数的平方是正数（3² = 9）
- 负数的平方也是正数（(-3)² = 9）
- 0的平方是0（0² = 0）

结论：任何实数的平方都不可能是负数！

∴ 负数没有平方根
```

#### 🔑 重要概念：算术平方根

**【为什么需要这个概念？】**

在实际问题中，我们通常只需要正的那个平方根。

比如：
- 正方形的边长是正数（不能是负数）
- 距离、长度、面积等都是非负数

**【定义】**

> 正数a的正的平方根叫做a的算术平方根，记作√a。

**特别规定**：
```
0 的算术平方根是 0，即 √0 = 0
```

**符号区分**：
| 符号 | 含义 | 例子 | 结果 |
|------|------|------|------|
| √a | 算术平方根（非负） | √9 | 3 |
| ±√a | 平方根（两个值） | ±√9 | ±3 |
| -√a | 负平方根 | -√9 | -3 |

#### ⚠️ 易错点分析

**【易错点1】混淆平方根和算术平方根**

❌ 错误：√9 = ±3

✓ 正确：
```
√9 = 3（算术平方根，只有一个值）
±√9 = ±3（平方根，有两个值）
```

**【易错点2】认为负数有平方根**

❌ 错误：√(-4) = -2

✓ 正确：-4没有平方根（实数范围内）

**【易错点3】忘记0的算术平方根**

❌ 错误：认为 √0 不存在或没有意义

✓ 正确：√0 = 0

**【易错点4】√a² ≠ a**

❌ 错误：√(-3)² = -3

✓ 正确：√(-3)² = √9 = **3**

**重要公式**：
```
√a² = |a| = {
    a，当 a ≥ 0 时
    -a，当 a < 0 时
}
```

**练习3**：

**练习4**：已知 √(a - 1) + |b + 2| = 0，求 a 和 b 的值。

<details>


## 📚 8.2 立方根

### 8.2.1 从"立方"到"开立方"

#### 📖 问题的提出

**【情境问题】**

小明有一个正方体的包装盒，它的体积是27立方厘米。这个正方体的棱长是多少厘米？

**【思考过程】**

```
设正方体的棱长为 x 厘米

根据正方体体积公式：
棱长 × 棱长 × 棱长 = 体积
x · x · x = 27
x³ = 27

现在的问题是：什么数乘以自己三次等于27？
```

**【尝试求解】**

```
1 × 1 × 1 = 1   （太小了）
2 × 2 × 2 = 8   （还小）
3 × 3 × 3 = 27  （对了！）

所以 x = 3
```

**【引入概念】**

这就是"立方根"的概念！
- 3 是 27 的立方根

#### 🔍 深入理解：立方根的概念

**【立方运算】**

```
已知棱长，求体积
棱长 = 3 → 体积 = 3³ = 27
```

**【开立方运算】（立方的逆运算）**

```
已知体积，求棱长
体积 = 27 → 棱长 = ³√27 = 3
```

**严格定义**：
> 如果一个数的立方等于a，那么这个数叫做a的立方根。

**数学表达**：
```
如果 x³ = a，那么 x 叫做 a 的立方根
```

**表示方法**：
```
a 的立方根记作：³√a

读法："三次根号a"

分解：
  ³√  → 三次根号（表示开立方）
  a   → 被开方数
```

#### 📊 立方根的性质

**【性质1】正数的立方根**

> 正数的立方根是正数。

**例子**：
```
³√8 = 2     （因为 2³ = 8）
³√27 = 3    （因为 3³ = 27）
³√0.125 = 0.5  （因为 0.5³ = 0.125）
```

**【性质2】负数的立方根**

> 负数的立方根是负数。

**例子**：
```
³√(-8) = -2     （因为 (-2)³ = -8）
³√(-27) = -3    （因为 (-3)³ = -27）
³√(-0.125) = -0.5  （因为 (-0.5)³ = -0.125）
```

**【性质3】零的立方根**

> 0的立方根是0。

```
³√0 = 0  （因为 0³ = 0）
```

#### 🎯 平方根 vs 立方根：关键区别

| 特性 | 平方根 | 立方根 |
|------|--------|--------|
| **正数** | 有2个（±） | 只有1个（正） |
| **负数** | ❌ 没有 | ✓ 有（负数） |
| **0** | 有1个（0） | 有1个（0） |
| **符号** | ±√a | ³√a |

**【为什么负数有立方根但没有平方根？】**

```
平方的情况：
(-3)² = 9  （正数）
任何实数的平方都是非负数！
∴ 负数没有平方根

立方的情况：
(-3)³ = -27  （负数）
负数的立方仍然是负数！
∴ 负数有立方根
```

#### ⚠️ 易错点分析

**【易错点1】认为立方根有两个**

❌ 错误：³√8 = ±2

✓ 正确：³√8 = 2（立方根只有一个值）

**【易错点2】混淆符号**

```
√8  表示平方根（二次根号）
³√8 表示立方根（三次根号）
```

**【易错点3】负数开方的处理**

```
√(-8)  → 不存在（实数范围内）
³√(-8) → = -2（存在！）
```

#### 📐 循序渐进练习

**练习5**：求下列各数的立方根
（1）64
（2）-125
（3）0.001

<details>
<summary>💡 点击查看解答</summary>

**解**：

**（1）64**
```
设 x³ = 64
∵ 4³ = 64
∴ 64的立方根是 4
即 ³√64 = 4
```

**（2）-125**
```
设 x³ = -125
∵ (-5)³ = -125
∴ -125的立方根是 -5
即 ³√(-125) = -5
```

**（3）0.001**
```
设 x³ = 0.001
∵ 0.1³ = 0.001
∴ 0.001的立方根是 0.1
即 ³√0.001 = 0.1
```
</details>

## 📚 8.3 实数

### 8.3.1 数系的扩展：从有理数到实数

#### 📖 历史的启示

**【古希腊的发现】**

公元前500年，毕达哥拉斯学派发现了一个惊人的事实：

```
边长为1的正方形，对角线的长度是多少？

设对角线长度为 x
根据勾股定理：
1² + 1² = x²
2 = x²
x = √2

但是，√2 无法表示为两个整数的比！
```

**【第一次数学危机】**

这个发现震惊了当时的数学界：
- 毕达哥拉斯学派认为"万物皆数"（整数和分数）
- 但 √2 既不是整数，也不是分数
- 这打破了他们对数的认识

**【无理数的诞生】**

像 √2 这样的数，被称为**无理数**（ irrational number）：
- **"无理"** 不是"没有道理"，而是"无法表示为比率（分数）"
- 它们是无限不循环小数

#### 🔍 深入理解：无理数

**【定义】**

> 无限不循环小数叫做无理数。

**【关键特征】**
1. **无限**：小数位数无限多
2. **不循环**：没有循环节

**【常见的无理数类型】**

**类型1：开方开不尽的数**
```
√2 = 1.4142135623730950488...
√3 = 1.7320508075688772935...
√5 = 2.2360679774997896964...
```

这些数的小数部分无限不循环。

**类型2：圆周率 π**
```
π = 3.14159265358979323846...
```

**重要提醒**：
```
π ≠ 3.14
π ≠ 22/7
3.14 和 22/7 只是 π 的近似值
```

**类型3：特定结构的数**
```
0.101001000100001... （每两个1之间依次多一个0）
2.02002000200002... （每两个2之间依次多一个0）
```

#### ⚠️ 易错点：有理数 vs 无理数

**【有理数的判断】**

有理数 = 整数 + 分数 = 有限小数 + 无限循环小数

**【典型错误】**

| 数 | 类型 | 说明 |
|---|------|------|
| √4 | ✓ 有理数 | √4 = 2 |
| √2 | ✗ 无理数 | 开方开不尽 |
| π | ✗ 无理数 | 无限不循环 |
| 3.14 | ✓ 有理数 | 有限小数 |
| 0.333... | ✓ 有理数 | 无限循环小数 (= 1/3) |
| 0.1010010001... | ✗ 无理数 | 无限不循环 |

**【判断技巧】**

```
步骤1：能否化简？
  √4 = 2（有理数）

步骤2：是否循环？
  0.333... = 1/3（有理数）
  0.1010010001...（不循环，无理数）

步骤3：是否特殊数？
  π（无理数）
  e（无理数）
```

### 8.2.2 实数的概念

#### 🔍 深入理解

**【定义】**

> 有理数和无理数统称为实数。

**【分类体系】**

**方法一：按定义分类**

```
实数
├── 有理数
│   ├── 整数
│   │   ├── 正整数
│   │   ├── 0
│   │   └── 负整数
│   └── 分数
│       ├── 正分数
│       └── 负分数
└── 无理数
    ├── 正无理数（如 √2，π）
    └── 负无理数（如 -√2，-π）
```

**方法二：按大小分类**

```
实数
├── 正实数（正有理数 + 正无理数）
├── 0
└── 负实数（负有理数 + 负无理数）
```

#### 📊 实数与数轴

**【重要性质】**

> 实数与数轴上的点一一对应。

**【理解】**

```
数轴上的每一个点都表示一个实数
每一个实数都可以在数轴上找到一个对应的点

      ────┬────┬────┬────┬────┬───>
         -√3  -1   0   √2  2
         ↓    ↓   ↓   ↓   ↓
         ·    ·   ·   ·   ·
```

**【历史意义】**

这个性质填满了数轴上的"空隙"：
- 有理数在数轴上虽然是密密的，但还有空隙
- 实数填满了所有空隙，数轴真正"连续"了

#### 🎯 实数的运算

**【运算范围】**

在实数范围内，以下运算总可以进行：
- 加法 ✓
- 减法 ✓
- 乘法 ✓
- 除法 ✓（除数不为0）
- 乘方 ✓
- 开方 ✓（负数只能开奇次方）

**【运算法则】**

有理数的运算法则同样适用于实数：
```
交换律：a + b = b + a，ab = ba
结合律：(a + b) + c = a + (b + c)
分配律：a(b + c) = ab + ac
```

**【运算注意事项】**

1. **无理数的近似计算**
```
√2 ≈ 1.414
π ≈ 3.14 或 3.1416

计算时按要求取近似值
```

2. **结果可能是无理数**
```
√2 × √2 = 2（有理数）
√2 + √2 = 2√2（无理数）
```

#### ⚠️ 易错点分析

**【易错点1】认为所有根式都是无理数**

❌ 错误：√a 都是无理数

✓ 正确：要判断 a 是否能开尽
```
√4 = 2（有理数）
√2（无理数）
```

**【易错点2】混淆 π 和它的近似值**

❌ 错误：π = 3.14

✓ 正确：π ≈ 3.14
```
π 是精确值
3.14 是近似值
```

**【易错点3】实数运算的错误**

❌ 错误：√2 + √3 = √5

✓ 正确：√2 + √3 不能合并

```
√2 + √3 ≈ 1.414 + 1.732 = 3.146
```

#### 8.3.1 无理数

**定义**：无限不循环小数叫做无理数

**常见无理数**：
1. 开方开不尽的数：√2、√3、³√2等
2. 圆周率π
3. 一些特定结构的数：0.1010010001...（每两个1之间依次多一个0）

**注意**：
- 无限循环小数是有理数
- 无限不循环小数是无理数
- π ≠ 3.14，3.14是π的近似值

#### 8.3.2 实数的概念

**定义**：有理数和无理数统称为实数

**分类**：

**按定义分类**：
```
实数
├── 有理数
│   ├── 整数（正整数、0、负整数）
│   └── 分数（正分数、负分数）
└── 无理数（无限不循环小数）
```

**按大小分类**：
```
实数
├── 正实数
├── 0
└── 负实数
```

#### 8.3.3 实数的性质

1. **实数与数轴上的点一一对应**
2. **实数可以进行加减乘除乘方运算**
3. **实数的大小比较**：
   - 正数大于0，0大于负数
   - 两个负数比较，绝对值大的反而小

#### 8.3.4 实数的运算

**运算法则**：
- 有理数的运算法则同样适用于实数
- 在实数范围内，加、减、乘、除（除数不为0）、乘方运算总可以进行

**运算律**：
- 交换律：a + b = b + a，ab = ba
- 结合律：(a + b) + c = a + (b + c)，(ab)c = a(bc)
- 分配律：a(b + c) = ab + ac

**注意**：
- 涉及无理数的计算，常按要求取近似值
- 乘法公式的因式分解在实数范围内仍然成立

## 📖 典型例题

### 例题1：平方根的概念

**题目**：
（1）求25的平方根
（2）求36的算术平方根
（3）判断：-9有平方根吗？

**解答**：
（1）因为 5² = 25，(-5)² = 25，
    所以 25的平方根是 ±5，即 ±√25 = ±5

（2）因为 6² = 36，
    所以 36的算术平方根是 6，即 √36 = 6

（3）因为任何实数的平方都是非负数，
    所以 -9**没有平方根**

---

### 例题2：平方根的性质

**题目**：
（1）√16的值是多少？
（2）±√16的值是多少？
（3）-√16的值是多少？

**解答**：
（1）√16 = 4（16的算术平方根）

（2）±√16 = ±4（16的平方根）

（3）-√16 = -4（16的算术平方根的相反数）

**注意**：√16、±√16、-√16的含义不同！

### 例题3：立方根的概念

**题目**：
（1）求27的立方根
（2）求-8的立方根
（3）求0的立方根

**解答**：
（1）因为 3³ = 27，
    所以 27的立方根是 3，即 ³√27 = 3

（2）因为 (-2)³ = -8，
    所以 -8的立方根是 -2，即 ³√(-8) = -2

（3）0的立方根是 0，即 ³√0 = 0

---

### 例题4：平方根与立方根的区别

**题目**：
（1）-4有平方根吗？有立方根吗？
（2）比较√9和³√(-27)的大小

**解答**：
（1）-4**没有平方根**（负数没有平方根）
    -4**有立方根**（任何实数都有立方根）
    ³√(-4) = -³√4

（2）√9 = 3
    ³√(-27) = -3
    所以 √9 > ³√(-27)

**结论**：
- 负数没有平方根，但有立方根
- 平方根有两个（±），立方根只有一个

### 例题7：实数的大小比较

**题目**：比较下列各组数的大小
（1）√5 和 2.2
（2）-√3 和 -√2
（3）³√(-5) 和 ³√(-3)

**解答**：
（1）√5 ≈ 2.236，2.2 = 2.2
    因为 2.236 > 2.2，
    所以 √5 > 2.2

（2）√3 ≈ 1.732，√2 ≈ 1.414
    因为 1.732 > 1.414，
    所以 -√3 < -√2（两个负数比较，绝对值大的反而小）

（3）³√(-5) ≈ -1.71，³√(-3) ≈ -1.44
    因为 |-1.71| > |-1.44|，
    所以 ³√(-5) < ³√(-3)

### 例题8：实数的运算

**题目**：计算
（1）√9 + √16
（2）√12 × √3
（3）（1 + √2）（1 - √2）

**解答**：
（1）√9 + √16 = 3 + 4 = 7

（2）√12 × √3 = √(12 × 3) = √36 = 6

（3）（1 + √2）（1 - √2）
    = 1² - (√2)²
    = 1 - 2
    = -1

### 例题9：实数的化简

**题目**：
（1）√(2²) = ?
（2）√((-3)²) = ?
（3）√(a²) = ?

**解答**：
（1）√(2²) = √4 = 2

（2）√((-3)²) = √9 = 3

（3）√(a²) = |a|
    当 a ≥ 0 时，√(a²) = a
    当 a < 0 时，√(a²) = -a

### 例题10：求代数式的值

**题目**：已知 x = √5 + 1，y = √5 - 1，求 x² + y² 的值。

**解答**：
x² = (√5 + 1)² = 5 + 2√5 + 1 = 6 + 2√5
y² = (√5 - 1)² = 5 - 2√5 + 1 = 6 - 2√5

x² + y² = (6 + 2√5) + (6 - 2√5) = 12
