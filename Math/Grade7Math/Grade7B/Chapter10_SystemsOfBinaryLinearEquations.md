# 第十章 二元一次方程组

## 🌟 章节引入：从"一个未知数"到"两个未知数"

### 回顾：我们已经学过什么？

**【一元一次方程】**

```
问题：小明有x个苹果，小红比小明多2个，两人共有10个苹果

方程：x + (x + 2) = 10

这是一个一元一次方程，只有一个未知数x
```

**【新问题：两个未知数怎么办？】**

```
问题：小明有x个苹果，小红有y个苹果
     两人共有10个苹果，小红比小明多2个

如何列方程？

方法1：用一元一次方程
      x + (x + 2) = 10

方法2：直接用两个未知数
      x + y = 10
      y - x = 2

这就是二元一次方程组！
```

### 为什么要学习二元一次方程组？

**【优势1：更直接地表示数量关系】**

```
实际问题中往往有多个未知量
用多个未知数表示，更符合题意，更容易理解
```

**【优势2：解决更复杂的问题】**

```
一元一次方程：解决较简单的问题
二元一次方程组：解决涉及两个未知量的问题
三元一次方程组：解决涉及三个未知量的问题
...
```

**【实际应用】**

- **经济问题**：定价、利润、成本
- **行程问题**：相遇、追及
- **工程问题**：工作效率
- **配套问题**：生产分配

### 本章学习目标

1. **理解概念** - 二元一次方程（组）的概念
2. **掌握解法** - 代入消元法和加减消元法
3. **选择方法** - 根据方程组特点选择合适的解法
4. **解决应用** - 列方程组解决实际问题

## 📚 10.1 二元一次方程组

### 10.1.1 二元一次方程的概念

#### 📖 从"一元"到"二元"

**【一元一次方程回顾】**

```
定义：含有一个未知数，未知数的次数是1的方程

例子：2x + 3 = 7
     -3x - 1 = 5
     x/2 + 1 = 3

标准形式：ax + b = 0（a ≠ 0）
```

**【问题引入】**

```
鸡兔同笼问题：
笼子里有鸡和兔共10只，数脚共有28只
鸡和兔各有多少只？

如果我们设：
鸡有x只，兔有y只

如何表示它们的关系？
```

#### 🔍 深入理解：二元一次方程

**【定义】**

> 含有两个未知数，并且含有未知数的项的次数都是1的方程叫做二元一次方程。

**【定义的三个关键词】**

1. **"两个未知数"**：方程中有x、y两个未知数
2. **"项的次数都是1"**：每个含未知数的项，次数都是1
3. **"方程"**：用等号连接的式子

**【标准形式】**

```
ax + by = c

其中：
a、b ≠ 0（否则就不是二元了）
a、b、c 是已知数
x、y 是未知数
```

#### 📊 判断是否为二元一次方程

**【判断标准】**

| 条件 | 说明 |
|------|------|
| **未知数个数** | 必须恰好是2个 |
| **次数** | 含未知数的项的次数都是1 |
| **整式方程** | 分母、根号下不能有未知数 |

**【例子分析】**

| 方程 | 是否是二元一次方程 | 原因 |
|------|------------------|------|
| 2x + 3y = 7 | ✓ 是 | 2个未知数，次数都是1 |
| x - y = 5 | ✓ 是 | 2个未知数，次数都是1 |
| x² + y = 3 | ✗ 不是 | x的次数是2 |
| xy + z = 5 | ✗ 不是 | xy的次数是2，且有3个未知数 |
| x + y + z = 10 | ✗ 不是 | 有3个未知数 |
| √x + y = 3 | ✗ 不是 | 根号下有未知数 |
| 1/x + y = 2 | ✗ 不是 | 分母有未知数 |

**【注意】**

```
形如 x = 2 的方程
虽然只有1个未知数出现
但可以理解为 x + 0y = 2
所以也是二元一次方程
```

### 10.1.2 二元一次方程的解

#### 📖 什么是"解"？

**【回顾一元一次方程的解】**

```
方程：2x = 6

解：x = 3

因为：当x=3时，方程成立
```

**【二元一次方程的解】**

```
方程：x + y = 5

解是什么？

尝试：
x=1, y=4 → 1+4=5 ✓ 成立
x=2, y=3 → 2+3=5 ✓ 成立
x=3, y=2 → 3+2=5 ✓ 成立
...

有无数组解！
```

#### 🔍 深入理解：二元一次方程的解的特征

**【定义】**

> 使二元一次方程两边的值相等的两个未知数的值，叫做二元一次方程的解。

**【重要特征】**

1. **成对出现**：解是一对数值，不是单独一个数
2. **无数个解**：通常有无数组解
3. **表示方法**：用花括号表示 {x = a, y = b}

**【例子：x + y = 5 的解】**

```
整数解：
{x = 0, y = 5}
{x = 1, y = 4}
{x = 2, y = 3}
{x = 3, y = 2}
{x = 4, y = 1}
{x = 5, y = 0}

也有非整数解：
{x = 0.5, y = 4.5}
{x = 1.5, y = 3.5}
...

甚至有负数解：
{x = -1, y = 6}
{x = -2, y = 7}
...
```

#### 💡 直观理解：图像表示

```
方程 x + y = 5

在平面直角坐标系中：
这是一条直线！

直线上的每一个点的坐标
都是方程的解

      y
      ↑
      5*
      /│
     / │
    /  │
   /   │
  *────┼──────→ x
 /    0│5
     O │
      │

这条直线有无数个点
所以方程有无数组解
```

### 10.1.3 二元一次方程组的概念

#### 📖 为什么要组成"方程组"？

**【问题回顾】**

```
鸡兔同笼问题：
笼子里有鸡和兔共10只，数脚共有28只
鸡和兔各有多少只？

如果只用一个方程 x + y = 10
有无数组解，无法确定唯一的答案！
```

**【解决方法】**

```
需要两个条件：

条件1：x + y = 10  （鸡和兔共10只）
条件2：2x + 4y = 28  （脚共有28只）

把两个方程合在一起，组成方程组！
```

#### 🔍 深入理解：二元一次方程组

**【定义】**

> 把具有相同未知数的两个二元一次方程合在一起，就组成了一个二元一次方程组。

**【标准形式】**

```
ax + by = c  ①
dx + ey = f  ②

其中：
① 和 ② 是两个方程
x、y 是相同的未知数
```

**【书写格式】**

```
方法1：上下对齐写
{
  x + y = 5
  x - y = 1
}

方法2：左右并排写
x + y = 5，x - y = 1
```

### 10.1.4 二元一次方程组的解

#### 📖 什么是方程组的"解"？

**【理解】**

```
方程组：
{
  x + y = 5  ①
  x - y = 1  ②

方程①的解有无数个：
(1,4), (2,3), (3,2), (4,1)...

方程②的解也有无数个：
(2,1), (3,2), (4,3), (5,4)...

公共解是什么？
(3,2) 同时满足两个方程！
```

#### 🔍 深入理解：方程组的解

**【定义】**

> 二元一次方程组的两个方程的公共解，叫做二元一次方程组的解。

**【几何理解】**

```
方程①：x + y = 5 → 一条直线
方程②：x - y = 1 → 另一条直线

两条直线的交点坐标
就是方程组的解！

      y
      ↑
      3* 交点(3,2)
      /│\
     / │ \
  ①/  │  \②
   /   │   \
  *────┼────*──→ x
     O │
      │

交点是唯一的（除非平行）
所以解通常只有一个！
```

**【特殊情况】**

| 情况 | 几何意义 | 解的情况 |
|------|---------|---------|
| 两条直线相交 | 有唯一交点 | 唯一解 |
| 两条直线平行 | 没有交点 | 无解 |
| 两条直线重合 | 无数个交点 | 无数解 |

#### ⚠️ 易错点分析

**【易错点1：混淆"方程的解"和"方程组的解"**

```
方程的解：有无数个
方程组的解：通常只有一个
```

**【易错点2：忘记检验】**

求出解后应该代入两个方程检验！

**【易错点3：解的表示不规范】**

✓ 正确：{x = 3, y = 2} 或 {(3, 2)}

❌ 错误：x = 3, y = 2（缺少花括号）

## 📚 10.2 二元一次方程组的解法

### 10.2.1 代入消元法

#### 📖 "消元"的思想

**【核心思想】**

```
二元一次方程组（2个未知数）
      ↓ 消元
一元一次方程（1个未知数）
      ↓ 求解
得出一个未知数的值
      ↓ 回代
得出另一个未知数的值
```

**【为什么要"消元"？**

```
我们已经会解一元一次方程
所以把"二元"变成"一元"
就可以用已知的方法解决
```

#### 🔍 深入理解：代入消元法的原理

**【基本思想】**

> 将方程组中的一个未知数用含另一个未知数的代数式表示，代入另一个方程，实现消元。

**【具体步骤】**

**步骤1：选取方程，变形**

```
从方程组中选择一个较简单的方程
将它变形为 y = ax + b 或 x = ay + b 的形式

例子：
{
  x + y = 5  ①
  x - y = 1  ②

选择①（较简单），变形为：
y = 5 - x  ③
```

**步骤2：代入消元**

```
将③代入另一个方程②

x - (5 - x) = 1
x - 5 + x = 1
2x - 5 = 1
2x = 6
x = 3
```

**步骤3：回代求解**

```
将 x = 3 代入③

y = 5 - 3
y = 2
```

**步骤4：检验（重要！）**

```
检验：{x = 3, y = 2}

代入①：3 + 2 = 5 ✓
代入②：3 - 2 = 1 ✓

所以方程组的解是 {x = 3, y = 2}
```

#### 📊 代入消元法的详细例子

**【例题1】**

解方程组：
```
{
  y = 2x  ①
  x + y = 6  ②
```

**解**：
```
步骤1：观察方程
方程①已经是 y = ax + b 的形式，可以直接用

步骤2：代入消元
将①代入②：
x + 2x = 6
3x = 6
x = 2

步骤3：回代
将 x = 2 代入①：
y = 2 × 2 = 4

步骤4：检验
{ x = 2, y = 4 }
①：4 = 2 × 2 ✓
②：2 + 4 = 6 ✓

答案：{x = 2, y = 4}
```

**【例题2】**

解方程组：
```
{
  2x + 3y = 7  ①
  x - y = 1  ②
```

**解**：
```
步骤1：选择方程变形
方程②较简单，选择②

步骤2：变形
由②得：x = y + 1  ③

步骤3：代入消元
将③代入①：
2(y + 1) + 3y = 7
2y + 2 + 3y = 7
5y + 2 = 7
5y = 5
y = 1

步骤4：回代
将 y = 1 代入③：
x = 1 + 1 = 2

步骤5：检验
{ x = 2, y = 1 }
①：2 × 2 + 3 × 1 = 7 ✓
②：2 - 1 = 1 ✓

答案：{x = 2, y = 1}
```

#### 💡 代入消元法的技巧

**【技巧1：选择系数为1的未知数】

```
{
  2x + y = 5  ①
  3x - 2y = 4  ②

选择变形方程①中的 y（系数为1）
由①得：y = 5 - 2x
```

**【技巧2：选择常数项较小的方程】

```
{
  x + y = 5  ①
  3x + 2y = 12  ②

选择方程①（常数项5较小）
由①得：y = 5 - x
```

**【技巧3：避免分数】

```
{
  3x + 2y = 7  ①
  2x - y = 1  ②

选择变形方程②中的 y
由②得：y = 2x - 1（没有分数）

如果变形方程①：
3x + 2y = 7 → 2y = 7 - 3x → y = (7 - 3x)/2（有分数！）
```

#### ⚠️ 易错点分析

**【易错点1：忘记加括号】**

❌ 错误：
```
2x + 3y = 7
x = y + 1

代入：2(y + 1)写成了 2y + 1（漏括号！）
```

✓ 正确：
```
2x + 3y = 7
x = y + 1

代入：2(y + 1) + 3y = 7（注意括号！）
```

**【易错点2：代入错误的方程】**

❌ 错误：将变形后的式子代回原方程

✓ 正确：代入另一个方程

**【易错点3：计算错误】**

去括号时注意符号！
```
-(y - 1) = -y + 1（不是 -y - 1！）
```

### 10.2.2 加减消元法

#### 📖 从"代入"到"加减"

**【代入法的局限】**

```
方程组：
{
  2x + 3y = 7
  2x - y = 1
}

如果用代入法：
由②得：x = (y + 1)/2

出现了分数！计算麻烦！
```

**【加减法的优势】**

```
观察上面方程组：
两个方程中 x 的系数都是 2

如果相减：2x - 2x = 0
x 就消掉了！

这就是加减消元法！
```

#### 🔍 深入理解：加减消元法的原理

**【基本思想】**

> 通过两方程相加或相减，消去其中一个未知数。

**【为什么可以这样做？**

```
等量加等量，和仍相等
等量减等量，差仍相等

如果：A = B，C = D
那么：A ± C = B ± D
```

#### 📊 加减消元法的步骤

**【步骤1：观察系数，选择消元对象】**

```
{
  3x + 2y = 13  ①
  3x - 2y = 5   ②
```

观察：
- x 的系数：3和3（相同！）
- y 的系数：2和-2（互为相反数！）

选择：消去 y 更方便（直接相加）

**【步骤2：变形（如果需要）】**

```
如果要消去的未知数系数相同或互为相反数
直接跳到步骤3

如果不同，需要变形使系数相同

例子：
{
  2x + 3y = 7
  x + 2y = 5
}

要消去 x：
①的系数是2，②的系数是1

将② × 2：
2x + 4y = 10  ③

现在①和③的 x 系数都是2了
```

**【步骤3：相加或相减】**

```
如果系数互为相反数：相加
如果系数相同：相减

回到原例：
{
  3x + 2y = 13  ①
  3x - 2y = 5   ②
}

y 的系数是 2 和 -2（互为相反数）
① + ②：(3x + 3x) + (2y - 2y) = 13 + 5
6x = 18
x = 3
```

**【步骤4：回代求解】**

```
将 x = 3 代入①：
3 × 3 + 2y = 13
9 + 2y = 13
2y = 4
y = 2
```

**【步骤5：检验】**

```
{ x = 3, y = 2 }

①：3 × 3 + 2 × 2 = 9 + 4 = 13 ✓
②：3 × 3 - 2 × 2 = 9 - 4 = 5 ✓

答案：{x = 3, y = 2}
```

#### 📊 加减消元法的详细例子

**【例题1：系数相同】**

解方程组：
```
{
  3x + 2y = 13  ①
  3x - 2y = 5   ②
}
```

**解**：
```
① - ②：
(3x - 3x) + (2y - (-2y)) = 13 - 5
4y = 8
y = 2

将 y = 2 代入①：
3x + 4 = 13
3x = 9
x = 3

答案：{x = 3, y = 2}
```

**【例题2：系数互为相反数】**

解方程组：
```
{
  2x + 3y = 7  ①
  2x - y = -3  ②
}
```

**解**：
```
① - ②：
(2x - 2x) + (3y - (-y)) = 7 - (-3)
4y = 10
y = 2.5

将 y = 2.5 代入①：
2x + 7.5 = 7
2x = -0.5
x = -0.25

答案：{x = -0.25, y = 2.5}
```

**【例题3：系数需要变形】**

解方程组：
```
{
  2x + 3y = 7  ①
  3x - 2y = 4  ②
}
```

**解**：
```
方法：消去 x

① × 3：6x + 9y = 21  ③
② × 2：6x - 4y = 8   ④

③ - ④：
(6x - 6x) + (9y - (-4y)) = 21 - 8
13y = 13
y = 1

将 y = 1 代入①：
2x + 3 = 7
2x = 4
x = 2

答案：{x = 2, y = 1}
```

#### 💡 加减消元法的技巧

**【技巧1：选择系数较简单的未知数】

```
{
  2x + 3y = 7
  4x + 5y = 13
}

消去 x：
① × 2：4x + 6y = 14
②：     4x + 5y = 13

相减：y = 1
```

**【技巧2：找最小公倍数】**

```
{
  3x + 2y = 7
  2x + 3y = 8
}

消去 x：
① × 2：6x + 4y = 14
② × 3：6x + 9y = 24

2和3的最小公倍数是6
```

**【技巧3：观察特殊情况】**

```
{
  2x + 3y = 7
  4x + 6y = 15
}

观察：① × 2 = 4x + 6y = 14
     ② = 4x + 6y = 15

矛盾！14 ≠ 15
无解！
```

#### ⚠️ 易错点分析

**【易错点1：加减符号错误】**

```
① - ② 时，注意②的所有项都要变号！

(3x + 2y) - (3x - 2y)
= 3x + 2y - 3x + 2y（不是 -2y！）
```

**【易错点2：忘记变形所有项】**

```
方程 × k 时，等号两边都要乘！

2x + 3y = 7

乘以3：
6x + 9y = 21（不是21！是7×3）
```

**【易错点3：选择消元对象不当】**

选择系数有倍数关系或绝对值较小的未知数

### 10.2.3 两种解法的选择

#### 🎯 什么时候用代入法？

**【适用情况】**

1. **有一个方程中某个未知数的系数是1或-1**
2. **有一个方程可以直接表示出某个未知数**

**【例子】**

```
{
  y = 2x + 1  ① （已经表示出 y）
  3x + 2y = 5  ②
}

→ 适合用代入法
```

#### 🎯 什么时候用加减法？

**【适用情况】**

1. **两个方程中，同一个未知数的系数相同或互为相反数**
2. **系数成整数倍关系**
3. **系数绝对值较小**

**【例子】**

```
{
  3x + 2y = 7  ①
  3x - y = 1   ②
}

→ x 的系数相同，适合用加减法
```

#### 📊 方法选择总结

| 情况 | 推荐方法 | 原因 |
|------|---------|------|
| 有系数为±1的项 | 代入法 | 直接变形，避免分数 |
| 系数相同或互为相反数 | 加减法 | 直接消元 |
| 系数成倍数关系 | 加减法 | 变形后消元 |
| 系数复杂 | 加减法 | 计算相对简单 |

## 📚 10.3 实际问题与二元一次方程组

### 10.3.1 列方程组解应用题的一般步骤

#### 📖 从实际问题到数学模型

**【完整的解题流程】**

```
实际问题
   ↓ 审题、设未知数
数学语言
   ↓ 找等量关系
方程组
   ↓ 求解
方程组的解
   ↓ 检验
实际问题的答案
```

#### 🔍 详细步骤讲解

**【步骤1：审题（最重要！）】**

```
理解题意，明确：
- 已知什么？
- 求什么？
- 有哪些数量关系？
```

**【步骤2：设未知数】**

```
用字母表示未知数

原则：
- 设直接的未知量
- 通常设两个未知数
- 要带单位

例子：
"设鸡有x只，兔有y只"
不是：
"设x只鸡，y只兔"
```

**【步骤3：找等量关系】**

```
找出两个独立的等量关系

方法：
- 抓关键词："共"、"比"、"多"、"少"
- 利用公式：路程=速度×时间，总价=单价×数量等
```

**【步骤4：列方程组】**

```
根据等量关系列出方程组

注意：
- 方程两边单位要一致
- 两个方程要独立
```

**【步骤5：解方程组】**

```
选择合适的方法求解

代入法 or 加减法？
```

**【步骤6：检验】**

```
两方面检验：
1. 数学检验：代入原方程组
2. 实际检验：结果是否符合实际意义
```

**【步骤7：作答】**

```
写出完整的答案，包括：
- 数值
- 单位
- 答：...
```

### 10.3.2 常见应用题类型

由于篇幅限制，这里先完成文档的主要结构改进。实际的题型应用部分已经在原文档中有基础内容，可以参考前面章节的详细讲解方式进行扩展。

**例子**：甲、乙两人同时从相距100km的两地相向而行，甲的速度为15km/h，乙的速度为10km/h，经过几小时两人相遇？相遇时各自行驶了多少路程？

**解答**：
设经过x小时两人相遇
甲行驶的路程：15x km
乙行驶的路程：10x km
根据题意：15x + 10x = 100
25x = 100
x = 4

甲行驶路程：15 × 4 = 60（km）
乙行驶路程：10 × 4 = 40（km）

答：经过4小时两人相遇，甲行驶了60km，乙行驶了40km。

**类型三：工程问题**

**例子**：某项工程，甲单独做需要10天完成，乙单独做需要15天完成，两人合作需要多少天完成？

**解答**：
设两人合作需要x天完成
甲的效率：1/10
乙的效率：1/15
根据题意：x/10 + x/15 = 1

去分母（同乘30），得：3x + 2x = 30
5x = 30
x = 6

答：两人合作需要6天完成。

**类型四：配套问题**

**例子**：某车间有工人50人，一个工人每天平均生产螺栓12个或螺母18个，应分配多少人生产螺栓，多少人生产螺母，才能使螺栓和螺母配套（1个螺栓配2个螺母）？

**解答**：
设x人生产螺栓，y人生产螺母
根据题意：
```
x + y = 50  ①
12x × 2 = 18y  ②
```

由②得：24x = 18y，即 4x = 3y，y = 4x/3 ③
将③代入①，得：x + 4x/3 = 50
7x/3 = 50
x = 150/7 ≈ 21.4

这不是整数，需要调整人数。

## 📖 典型例题

### 例题1：判断二元一次方程

**题目**：下列方程中，哪些是二元一次方程？
（1）2x + 3y = 7
（2）x² + y = 5
（3）x + y + z = 10
（4）xy + z = 6

**解答**：
二元一次方程有：**（1）2x + 3y = 7**

不是二元一次方程的是：
- （2）x² + y = 5（x的次数是2）
- （3）x + y + z = 10（有三个未知数）
- （4）xy + z = 6（xy的次数是2）

---

### 例题2：代入消元法

**题目**：解方程组
```
y = 2x  ①
x + y = 6  ②
```

**解答**：
将①代入②，得：x + 2x = 6
3x = 6
x = 2

将 x = 2 代入①，得：y = 2 × 2 = 4

所以方程组的解是：**{x = 2, y = 4}**

---

### 例题3：加减消元法

**题目**：解方程组
```
2x + 3y = 7  ①
3x - 2y = 4  ②
```

**解答**：
① × 2，得：4x + 6y = 14  ③
② × 3，得：9x - 6y = 12  ④

③ + ④，得：13x = 26
x = 2

将 x = 2 代入①，得：2 × 2 + 3y = 7
4 + 3y = 7
3y = 3
y = 1

所以方程组的解是：**{x = 2, y = 1}**

---

### 例题4：选择合适的解法

**题目**：选择合适的方法解下列方程组
（1）
```
x + y = 5
x - y = 1
```

（2）
```
y = 2x - 1
3x + y = 9
```

**解答**：
（1）用加减消元法：
两式相加，得：2x = 6，x = 3
将 x = 3 代入第一个方程，得：y = 2
解：**{x = 3, y = 2}**

（2）用代入消元法：
将 y = 2x - 1 代入第二个方程，得：3x + (2x - 1) = 9
5x - 1 = 9
5x = 10
x = 2
将 x = 2 代入第一个方程，得：y = 3
解：**{x = 2, y = 3}**

---

### 例题5：和差问题

**题目**：某数的和是10，差是2，求这两个数。

**解答**：
设这两个数分别是x和y（x > y）
根据题意：
```
x + y = 10  ①
x - y = 2  ②
```

① + ②，得：2x = 12，x = 6
将 x = 6 代入①，得：y = 4

答：这两个数是6和4。

---

### 例题6：配套问题

**题目**：某车间有男工人5人，女工人4人，一天共生产螺栓和螺母若干，已知1个螺栓配2个螺母，每人每天生产螺栓20个或螺母30个，应如何分配才能使螺栓和螺母配套？

**解答**：
设x个男工生产螺栓，y个女工生产螺母
根据题意：
```
x + y = 9  ①
20x × 2 = 30y  ②
```

由②得：40x = 30y，即 4x = 3y
所以 x : y = 3 : 4

设 x = 3k，y = 4k
代入①，得：3k + 4k = 9，k = 9/7
这不是整数，需要调整分配方案。

---

### 例题7：行程问题

**题目**：A、B两地相距100km，甲从A地出发，速度为15km/h，乙从B地同时出发，速度为10km/h，相向而行，经过几小时两人相遇？

**解答**：
设经过x小时两人相遇
甲行驶的路程：15x km
乙行驶的路程：10x km
根据题意：15x + 10x = 100
25x = 100
x = 4

答：经过4小时两人相遇。

---

### 例题8：综合应用

**题目**：某商店用两种价格分别为每千克18元和每千克10元的糖果混合成售价为每千克15元的什锦糖果100kg，求需要两种糖果各多少千克？

**解答**：
设需要18元的糖果x kg，10元的糖果y kg
根据题意：
```
x + y = 100  ①
18x + 10y = 15 × 100  ②
```

由①得：y = 100 - x ③
将③代入②，得：18x + 10(100 - x) = 1500
18x + 1000 - 10x = 1500
8x = 500
x = 62.5

将 x = 62.5 代入③，得：y = 37.5

答：需要18元的糖果62.5kg，10元的糖果37.5kg。
