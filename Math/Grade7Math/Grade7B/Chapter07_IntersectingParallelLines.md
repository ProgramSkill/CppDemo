# 第七章 相交线与平行线

## 🌟 章节引入：为什么学习相交线与平行线？

### 生活中的相交与平行

在我们日常生活中，相交线和平行线无处不在：

- **相交线**：十字路口的两条道路、剪刀的两片刀刃、窗户的对角线...
- **平行线**：铁路的双轨道、百叶窗的叶片、书本的对边...

1. **理解几何推理的基础** - 掌握角度关系，为后续几何证明打下基础
2. **培养空间想象能力** - 学会观察和分析图形中的位置关系
3. **建立严谨的数学思维** - 学习命题与证明，体验数学推理的严谨性
4. **解决实际问题** - 运用几何知识解决生活中的问题

## 📚 7.1 相交线

### 7.1.1 邻补角和对顶角

#### 📖 概念的形成过程

**【观察与思考】**

想象两根木条交叉在一起，在交叉点处会形成几个角？让我们仔细观察：

```
        A
        |
        |
        O───────C
       / \
      /   \
     /     \
    D       B
```

当两条直线 AB 和 CD 相交于点 O 时，形成了四个角：∠1、∠2、∠3、∠4

**【问题引导】**
1. 这四个角之间有什么关系？
2. 如果知道其中一个角的大小，能求出其他角吗？

#### 🔍 深入理解：邻补角

**定义的由来：**
- **"邻"** = 相邻（有公共边）
- **"补"** = 互补（和为180°）

所以**邻补角**就是"既相邻又互补的角"。

**严格定义**：
> 两条直线相交所构成的四个角中，有公共顶点且有一条公共边的两个角互为邻补角。

**图形表示**：
```
    A
    │
    │ ∠1
    │
    O────────B
    │ ∠3
    │
    D
```

在这个图中：
- ∠1 和 ∠3 是邻补角（公共顶点O，公共边OD，和为180°）
- ∠1 和 ∠2 也是邻补角（公共顶点O，公共边OB，和为180°）

**重要性质**：
```
邻补角之和 = 180°
```

**为什么邻补角互补？**

因为它们共同组成了一条直线（平角），而平角的度数是180°。

#### 🔍 深入理解：对顶角

**定义的由来：**
- **"对"** = 相对（位置相对）
- **"顶"** = 对顶（对着顶点）

所以**对顶角**就是"位置相对的两个角"。

**严格定义**：
> 两条直线相交所构成的四个角中，有公共顶点但没有公共边的两个角互为对顶角。

**图形表示**：
```
        A
        │
    ∠4  │  ∠1
        │
    D───O─────C
        │
    ∠3  │  ∠2
        │
        B
```

在这个图中：
- ∠1 和 ∠3 是对顶角（公共顶点O，无公共边）
- ∠2 和 ∠4 是对顶角（公共顶点O，无公共边）

**重要性质**：
```
对顶角相等
```

**⚡ 为什么对顶角相等？（证明）**

**方法一：利用邻补角**

已知：直线 AB 和 CD 相交于点 O

求证：∠1 = ∠3

**证明**：
```
∵ ∠1 + ∠2 = 180° （邻补角互补）
∴ ∠1 = 180° - ∠2

∵ ∠3 + ∠2 = 180° （邻补角互补）
∴ ∠3 = 180° - ∠2

∴ ∠1 = ∠3 （等量代换）
```

**方法二：直观理解**

想象把整个图形绕点 O 旋转 180°：
- 点 A 会转到点 B 的位置
- 点 C 会转到点 D 的位置
- ∠1 会恰好与 ∠3 重合

所以 ∠1 = ∠3

#### 📊 完整的角度关系图

当两条直线相交时，四个角的关系：

```
        A
        │
    ∠4  │  ∠1
        │
    D───O─────C
        │
    ∠3  │  ∠2
        │
        B

关系总结：
├── 邻补角（互补）：∠1 + ∠2 = 180°，∠2 + ∠3 = 180°，∠3 + ∠4 = 180°，∠4 + ∠1 = 180°
└── 对顶角（相等）：∠1 = ∠3，∠2 = ∠4
```

#### ⚠️ 易错点分析

**【易错点1】混淆邻补角和对顶角**

❌ 错误理解：认为有公共顶点的两个角就是邻补角

✓ 正确理解：邻补角必须同时满足：
1. 有公共顶点
2. 有公共边
3. 两角之和为180°

**【易错点2】认为所有互补的角都是邻补角**

❌ 错误理解：∠1 = 30°，∠2 = 150°，所以 ∠1 和 ∠2 是邻补角

✓ 正确理解：邻补角不仅要求互补，还要求有公共边和公共顶点

**【易错点3】忘记对顶角相等的条件**

❌ 错误理解：任何两个相对的角都相等

✓ 正确理解：只有两条直线相交形成的四个角中，相对的两个角才是对顶角

**【水平3：综合应用】**

**练习3**：如图，直线AB、CD、EF相交于点O，∠AOE = 30°，∠DOF = 70°，求∠BOC的度数。

<details>
<summary>💡 点击查看解题过程</summary>

**解**：
```

∠AOE = 30°（∠AOF的邻补角）
∠DOF = 70°（已知）

∵ ∠AOE和∠AOF是邻补角
∴ ∠AOF = 180° - ∠AOE = 180° - 30° = 150°

步骤2：求∠AOD（即∠COF）
∵ ∠AOF = ∠AOD + ∠DOF
∴ ∠AOD = ∠AOF - ∠DOF = 150° - 70° = 80°

步骤3：利用对顶角求∠BOC
∵ ∠BOC和∠AOD是对顶角
∴ ∠BOC = ∠AOD = 80°

答：∠BOC = 80°
```
</details>

**【水平4：思维拓展】**

**练习4**：如图，三条直线AB、CD、EF两两相交于点O、G、H，∠AGH = 50°，∠CHD = 60°，求∠BHF的度数。

<details>
<summary>💡 点击查看解题过程</summary>

**解**：
```
这是一个需要综合运用角度关系的题目。

步骤1：分析已知角度
∠AGH = 50°
∠CHD = 60°

步骤2：利用邻补角关系
∵ ∠AGH + ∠BGH = 180°（邻补角）
∴ ∠BGH = 180° - 50° = 130°

步骤3：观察三角形内角关系
在三角形GHC中：
∠GHC + ∠HGC + ∠GCH = 180°
其中：∠GHC = ∠BHF（对顶角）
      ∠HGC = ∠BGH = 130°
      ∠GCH = 180° - ∠CHD = 120°（邻补角）

步骤4：求解
∠GHC = 180° - 130° - 120° = -70°

这不可能！说明题目数据有误，或者需要重新分析...

实际上，我们需要用另一种方法：
利用平角关系和对顶角关系来求解。

正确解法：
∠BHF = ∠CHG（对顶角）
在直线EF上：∠CHG + ∠CHD = 180°
∴ ∠BHF = 180° - ∠CHD = 180° - 60° = 120°

答：∠BHF = 120°
```
</details>

### 7.1.2 垂线

#### 📖 从生活现象到数学概念

**【生活中的垂线】**

- **国旗杆与地面**：国旗杆垂直于地面
- **书角**：书本的角通常是90°直角
- **建筑物**：墙壁与地面垂直

**【问题引入】**

在相交线中，有一种特殊情况：当两条直线相交所成的四个角都相等时，每个角是多少度？

**推导过程**：
```
设其中一个角为x°
∵ 对顶角相等
∴ 相对的两个角都是x°

又∵ 邻补角互补
∴ x + x = 180°
∴ 2x = 180°
∴ x = 90°

所以：四个角都是90°，即直角
```

#### 🔍 深入理解：垂线

**定义**：
> 当两条直线相交所成的四个角中，有一个角是直角时，就说这两条直线互相垂直。其中一条直线叫做另一条直线的垂线，交点叫做垂足。

**符号表示**：
```
AB ⊥ CD
读作："AB垂直于CD"
```

**图形表示**：
```
        A
        │
        │
        O────────C
        │
        │
        B

直线 AB ⊥ 直线 CD，垂足为 O
```

#### 📐 垂线的性质（详细证明）

**性质1：过一点有且只有一条直线与已知直线垂直**

**理解**：
- **"过一点"**：这个点可以在直线上，也可以在直线外
- **"有且只有"**：存在性（有一条）+ 唯一性（只有一条）

**【情况1】点在直线上**

```
已知：直线 l 和直线上一点 P
求证：过点 P 有且只有一条直线垂直于 l

证明：
∵ 过点 P 可以作∠APB = 90°（直角）
∴ 直线 PA ⊥ 直线 l
∵ 直角的定义是唯一的
∴ 这样的直线只有一条
```

**【情况2】点在直线外**

```
已知：直线 l 和直线外一点 P
求证：过点 P 有且只有一条直线垂直于 l

证明：
连接点 P 与直线 l 上各点，其中最短的线段就是垂线段
∵ 垂线段最短（性质2）
∵ 最短距离是唯一的
∴ 这样的垂线只有一条
```

**性质2：垂线段最短**

**严格表述**：
> 连接直线外一点与直线上各点的所有线段中，垂线段最短。

**图形表示**：
```
        P
        │╲
        │ ╲
        │  ╲
        │   ╲
    O───●───●───  l
       PA    PB

其中 PA ⊥ l，PA是垂线段
PB是任意斜线段

比较：PA < PB
```

**证明（用勾股定理）**：
```
设：垂线段 PA = a，OA = b
    斜线段 PB = c，OB = b + d（d > 0）

根据勾股定理：
PA² + OA² = OP² = PB² - OB²（这是错的，需要重新分析）

正确证明：
在直角三角形PAB中：
PA² + AB² = PB²（勾股定理）
∵ AB² > 0（A、B不重合）
∴ PA² < PB²
∴ PA < PB
```

#### 📏 点到直线的距离

**定义**：
> 直线外一点到这条直线的垂线段的长度，叫做点到直线的距离。

**重要区分**：
| 概念 | 含义 | 类型 |
|------|------|------|
| 垂线 | 与已知直线垂直的直线 | 图形 |
| 垂线段 | 垂线上的一段线段 | 图形 |
| 距离 | 垂线段的长度 | 数量 |

#### ⚠️ 易错点分析

**【易错点1】混淆垂直与垂线**

❌ 错误：AB ⊥ CD 叫做垂线

✓ 正确：AB ⊥ CD 表示两条直线互相垂直的关系

**【易错点2】混淆垂线段和距离**

❌ 错误：点到直线的距离是垂线段

✓ 正确：点到直线的距离是垂线段的**长度**（一个数）

**【易错点3】认为距离一定是垂直距离**

在平面几何中，点到直线的距离确实是指垂直距离。但在其他情况（如两点间距离）不是垂直距离。

```
∵ ∠COE和∠DOE是邻补角
∴ ∠COE = 180° - ∠DOE = 180° - 50° = 130°

答：∠COE = 130°
```
</details>

### 7.1.3 同位角、内错角、同旁内角

#### 📖 为什么要学习"三线八角"？

**【问题引入】**

在前面的学习中，我们研究了两条直线相交形成的关系。那么，**三条直线**之间会形成怎样的角度关系呢？

这个问题的答案非常重要，因为它是研究**平行线**的基础！

**【生活中的模型】**

想象一根横梁（直线AB）被两根立柱（直线CD、EF）支撑，它们之间会形成各种角度关系。

#### 🔍 深入理解：三线八角

**基本图形**：
```
        A               C
        ╱               ╱
       ╱               ╱
      ╱               ╱
     /  ∠1    ∠2    /
    /                /
   D ∠3      ∠4    E
    ╲              ╱
     ╲            ╱
      ╲          ╱
       ╲  ∠5  ∠6╱
        ╲      ╱
         ╲    ╱
          ╲  ╱
           ╲╱
           F
            ∠7     ∠8
```

**术语说明**：
- **两条直线**：AB 和 CD（被截线）
- **第三条直线**：EF（截线）
- **八个角**：∠1 到 ∠8

#### 📊 三类角的特征识别

**【同位角】**

**记忆口诀**：同旁同侧

**严格定义**：
> 在两条被截直线的同侧，并且在截线的同旁的两个角叫做同位角。

**图形特征**：
```
位置模式：F型
┌────────┐
│        │
│   ┌────┘
└───┘

同位角对：
- ∠1 和 ∠5
- ∠2 和 ∠6
- ∠3 和 ∠7
- ∠4 和 ∠8
```

**【内错角】**

**记忆口诀**：之间两侧

**严格定义**：
> 在两条被截直线之间，并且在截线的两侧的两个角叫做内错角。

**图形特征**：
```
位置模式：Z型
    ┌───┐
    │   │
┌───┘   └───┐
│           │
└───────────┘

内错角对：
- ∠3 和 ∠5
- ∠4 和 ∠6
```

**【同旁内角】**

**记忆口诀**：之间同旁

**严格定义**：
> 在两条被截直线之间，并且在截线的同旁的两个角叫做同旁内角。

**图形特征**：
```
位置模式：U型
  ┌─────┐
  │     │
  │     │
  └─────┘

同旁内角对：
- ∠3 和 ∠6
- ∠4 和 ∠5
```

#### 📐 快速识别技巧

**【技巧1：看字母形状】**

| 角的类型 | 字母形状 | 位置特征 |
|---------|---------|---------|
| 同位角 | F型 | 同侧同旁 |
| 内错角 | Z型 | 内部两侧 |
| 同旁内角 | U型 | 内部同旁 |

**【技巧2：位置判断法】**

步骤1：确定两条被截线和一条截线
步骤2：观察两个角的位置：
- 在被截线的同侧还是之间？
- 在截线的同旁还是两侧？
- 组合起来就是角的关系

**【技巧3：排除法】**

```
不是同位角 → 不在同侧
不是内错角 → 不在之间或不在两侧
→ 那就是同旁内角
```

#### ⚠️ 易错点分析

**【易错点1】混淆"内"和"外"**

❌ 错误：认为所有的角都是"内部"的角

✓ 正确：只有被两条直线夹住的区域才是"内部"

**【易错点2】忽略截线的选择**

❌ 错误：随便找三条直线就看角的关系

✓ 正确：必须明确哪两条是被截线，哪一条是截线

**【易错点3】记忆混乱**

记忆技巧：
```
同位角 → 同（同侧）同（同旁）→ 像"朋友"站在一起
内错角 → 内（内部）错（错开）→ 像"敌人"躲在里面
同旁内角 → 内（内部）同（同旁）→ 像"邻居"住在一起
```

## 📚 7.2 平行线

### 7.2.1 平行线的概念

#### 📖 从生活到数学

**【生活中的平行】**

- **铁路轨道**：两条铁轨永远不会相交
- **百叶窗**：叶片互相平行
- **笔记本的横线**：整齐的平行线

**【数学定义】**

> 在同一平面内，不相交的两条直线叫做平行线。

**符号表示**：
```
AB ∥ CD
读作："AB平行于CD"
```

**重要关键词**：
1. **"同一平面内"**：强调在同一平面上
2. **"不相交"**：永不相交

#### 🔍 深入理解：为什么强调"同一平面内"？

**【空间中的直线位置关系】**

在空间（三维）中，两条直线的位置关系有三种：

1. **相交**：有且只有一个公共点
2. **平行**：在同一平面内，没有公共点
3. **异面**：不在同一平面内，没有公共点

**例子**：
```
        ┌─────────┐  （天花板）
       ╱          ╲
      ╱            ╲
     ╱              ╲
    │                │  （墙壁）
    │                │
    └────────────────┘  （地板）

- 地板上的对边：平行
- 地板与墙壁的交线：相交
- 墙壁与天花板的交线：异面（不相交但也不平行）
```

#### 📐 平行公理及其推论

**【平行公理】**

> 经过直线外一点，有且只有一条直线与这条直线平行。

**理解**：
- **"过一点"**：这个点在直线外
- **"有且只有"**：存在且唯一

**图形表示**：
```
已知：直线 l 和直线外一点 P

         P
         │
         │
    ─────○─────  l
         │唯一！
         │

过点 P 只能作一条直线与 l 平行
```

**【平行公理的推论】**

> 如果两条直线都与第三条直线平行，那么这两条直线也互相平行。

**符号表示**：
```
若 a ∥ c，b ∥ c
则 a ∥ b
```

**证明思路**：
```
假设 a 不平行于 b
那么 a 和 b 会相交于某点 P

但 a ∥ c，b ∥ c
根据平行公理，过点 P 只能有一条直线与 c 平行

矛盾！所以 a ∥ b
```

**应用举例**：
```
AB ∥ EF
CD ∥ EF
→ AB ∥ CD
```

#### ⚠️ 易错点分析

**【易错点1】忽略"同一平面内"**

❌ 错误：不相交的两条直线就是平行线

✓ 正确：在同一平面内不相交的两条直线才是平行线

**【易错点2】混淆平行与不相交**

在空间中，异面直线也不相交，但不是平行线。

**【易错点3】认为延长后会相交的直线是平行线**

❌ 错误理解：延长很远后才相交的直线可以看作平行

✓ 正确理解：平行线无论延长多远都不会相交
- 位置特征："同旁同侧"

**内错角**：
- 定义：在两条直线之间，第三条直线的两侧的两个角
- 位置特征："之间两侧"

**同旁内角**：
- 定义：在两条直线之间，第三条直线的同旁的两个角
- 位置特征："之间同旁"

### 7.2 平行线

#### 7.2.1 平行线的概念

**平行线**：
- 定义：在同一平面内，不相交的两条直线叫做平行线
- 表示：AB ∥ CD（读作"AB平行于CD"）

**平行公理**：
- 经过直线外一点，有且只有一条直线与这条直线平行

**平行公理的推论**：
- 如果两条直线都与第三条直线平行，那么这两条直线也互相平行
- 若 a ∥ c，b ∥ c，则 a ∥ b

#### 7.2.2 平行线的判定

**判定方法一**：同位角相等，两直线平行

**判定方法二**：内错角相等，两直线平行

**判定方法三**：同旁内角互补，两直线平行

**例子**：
- 若∠1 = ∠2（∠1和∠2是同位角），则 AB ∥ CD
- 若∠3 = ∠4（∠3和∠4是内错角），则 AB ∥ CD
- 若∠5 + ∠6 = 180°（∠5和∠6是同旁内角），则 AB ∥ CD

#### 7.2.3 平行线的性质

**性质一**：两直线平行，同位角相等

**性质二**：两直线平行，内错角相等

**性质三**：两直线平行，同旁内角互补

**注意**：平行线的判定与性质是互逆关系
- 判定：角的关系 → 直线平行
- 性质：直线平行 → 角的关系

### 7.3 平移

#### 7.3.1 平移的概念

**定义**：在平面内，将一个图形沿某个方向移动一定的距离，这样的图形运动称为平移

**平移的两个要素**：
1. 方向
2. 距离

#### 7.3.2 平移的性质

1. 平移不改变图形的形状和大小
2. 平移后，对应点所连的线段平行（或在同一直线上）且相等
3. 平移后，对应角相等
4. 平移后，对应线段平行（或在同一直线上）且相等

### 7.4 命题、定理、证明

#### 7.4.1 命题

**定义**：判断一件事情的语句叫做命题

**命题的结构**：
- 题设：已知事项
- 结论：由已知事项推出的事项

**命题的分类**：
- 真命题：正确的命题
- 假命题：错误的命题

**例子**：
- "如果两条直线都和第三条直线平行，那么这两条直线也互相平行"是真命题
- "相等的角是对顶角"是假命题（反例：等腰三角形底角）

#### 7.4.2 定理和证明

**定理**：用推理的方法判断为真命题的命题叫做定理

**证明**：除了公理外，其他真命题的正确性都需要通过推理的方法来证实，这样的过程叫做证明

**证明的步骤**：
1. 根据题意，画出图形
2. 根据题设、结论，结合图形，写出已知、求证
3. 经过分析，找出由已知推出求证的途径，写出证明过程

## 📖 典型例题

### 例题1：对顶角和邻补角

**题目**：如图，直线AB和CD相交于点O，∠AOC = 50°，求其他三个角的度数。

**解答**：
因为∠AOC和∠BOD是对顶角，
所以 ∠BOD = ∠AOC = **50°**

因为∠AOC和∠BOC是邻补角，
所以 ∠BOC = 180° - ∠AOC = 180° - 50° = **130°**

因为∠AOD和∠BOC是对顶角，
所以 ∠AOD = ∠BOC = **130°**

---

### 例题2：垂线

**题目**：如图，直线AB和CD相交于点O，OE ⊥ AB，∠BOD = 40°，求∠COE的度数。

**解答**：
因为OE ⊥ AB，
所以 ∠BOE = 90°

因为∠BOD = 40°，
所以 ∠DOE = ∠BOE - ∠BOD = 90° - 40° = 50°

因为∠COE和∠DOE是邻补角，
所以 ∠COE = 180° - ∠DOE = 180° - 50° = **130°**

---

### 例题3：识别同位角、内错角、同旁内角

**题目**：如图，直线AB、CD被EF所截，请指出：
（1）∠1和∠2是什么角？
（2）∠3和∠4是什么角？
（3）∠5和∠6是什么角？

**解答**：
（1）∠1和∠2是**同位角**（在两条直线的同侧，第三条直线的同旁）

（2）∠3和∠4是**内错角**（在两条直线之间，第三条直线的两侧）

（3）∠5和∠6是**同旁内角**（在两条直线之间，第三条直线的同旁）

---

### 例题4：平行线的判定

**题目**：如图，∠1 = 70°，∠2 = 70°，AB和CD平行吗？为什么？

**解答**：
因为 ∠1 = 70°，∠2 = 70°，
所以 ∠1 = ∠2

又因为∠1和∠2是同位角，
所以 **AB ∥ CD**（同位角相等，两直线平行）

---

### 例题5：平行线的性质

**题目**：如图，AB ∥ CD，∠1 = 60°，求∠2的度数。

**解答**：
因为 AB ∥ CD，
所以 ∠1和∠2是同位角，
所以 ∠2 = ∠1 = **60°**（两直线平行，同位角相等）

---

### 例题6：平行线的综合应用

**题目**：如图，AB ∥ CD，∠1 = 50°，∠2 = 60°，求∠3的度数。

**解答**：
因为 AB ∥ CD，
所以 ∠4 = ∠1 = 50°（两直线平行，同位角相等）

又因为 ∠2 + ∠3 + ∠4 = 180°（平角），
所以 ∠3 = 180° - ∠2 - ∠4 = 180° - 60° - 50° = **70°**

---

### 例题7：平移

**题目**：如图，将△ABC沿箭头方向平移2cm，得到△A'B'C'，请找出平移后的对应点。

**解答**：
对应点：
- 点A的对应点是点A'
- 点B的对应点是点B'
- 点C的对应点是点C'

平移的方向：箭头指向的方向
平移的距离：2cm

### 例题8：命题的题设和结论

**题目**：指出下列命题的题设和结论
（1）对顶角相等
（2）两条直线被第三条直线所截，同位角相等，两直线平行

**解答**：
（1）题设：两个角是对顶角
    结论：这两个角相等

（2）题设：两条直线被第三条直线所截，同位角相等
    结论：这两条直线平行

---

### 例题9：证明

**题目**：证明：对顶角相等。

**已知**：如图，直线AB和CD相交于点O
**求证**：∠AOC = ∠BOD

**证明**：
因为∠AOC和∠BOC是邻补角，
所以 ∠AOC + ∠BOC = 180°
即 ∠AOC = 180° - ∠BOC

又因为∠BOD和∠BOC是邻补角，
所以 ∠BOD + ∠BOC = 180°
即 ∠BOD = 180° - ∠BOC

所以 ∠AOC = ∠BOD

---

### 例题10：综合应用

**题目**：如图，AB ∥ CD，AD ∥ BC，∠A = 50°，求∠C的度数。

**解答**：
因为 AD ∥ BC，
所以 ∠A + ∠B = 180°（两直线平行，同旁内角互补）
所以 ∠B = 180° - ∠A = 180° - 50° = 130°

又因为 AB ∥ CD，
所以 ∠B + ∠C = 180°（两直线平行，同旁内角互补）
所以 ∠C = 180° - ∠B = 180° - 130° = **50°**
