## Competition Formulas | 竞赛公式

### Cramer's Rule | 克拉默法则
```
For system: ax + by = c, dx + ey = f

x = (ce - bf) / (ae - bd)
y = (af - cd) / (ae - bd)
```

### Diophantine Equation | 丢番图方程
```
ax + by = c has integer solutions iff gcd(a,b) | c

General solution:
x = x₀ + (b/d)t
y = y₀ - (a/d)t
where d = gcd(a,b)
```

### Frobenius Number | 弗罗贝尼乌斯数
```
For coprime a, b:
g(a,b) = ab - a - b
```
