## 📚 12.1 统计调查

### 12.1.1 数据的收集

#### 📖 为什么要收集数据？

**【问题引入】**

```
你想了解全班同学最喜欢的运动是什么

怎么办？

方法1：一个一个问（收集数据）
方法2：凭感觉猜（不准确！）

显然应该选择方法1！
```

#### 🔍 深入理解：两种调查方式

**【方式一：全面调查（普查）】**

**定义**：
> 考察全体对象的调查方式

**【生活中的例子】**

```
例子1：全国人口普查
- 每10年进行一次
- 调查全国每一个人
- 了解国家人口状况

例子2：期末考试
- 每个学生都参加考试
- 了解每个学生的学习情况

例子3：班级视力检查
- 检查每个学生的视力
- 了解班级整体视力状况
```

**【全面调查的优缺点】**

| 优点 | 缺点 |
|------|------|
| ✓ 结果准确、全面 | ✗ 工作量大 |
| ✓ 获得第一手资料 | ✗ 耗费时间、人力、物力 |
| ✓ 没有抽样误差 | ✗ 有时无法实现 |
| | ✗ 破坏性检验不适用 |

**【什么情况用全面调查？】**

```
✓ 适用于：
- 总体数量不大
- 要求结果精确
- 无破坏性

例子：
- 班级人数统计
- 学校运动会成绩
- 小规模产品质量检验
```

**【方式二：抽样调查】**

**定义**：
> 抽取一部分对象进行调查，然后根据调查数据推断全体对象的情况

**【生活中的例子】**

```
例子1：灯泡寿命检验
- 灯泡用完就坏了（破坏性）
- 不能全部测试
- 只能抽取部分测试

例子2：收视率调查
- 不可能调查所有观众
- 抽取部分家庭调查
- 推断整体收视率

例子3：产品质量检验
- 一批产品有10000个
- 抽取100个检验
- 推断整批产品的质量
```

**【抽样调查的优缺点】**

| 优点 | 缺点 |
|------|------|
| ✓ 节省时间、人力、物力 | ✗ 结果可能有误差 |
| ✓ 适用于大规模调查 | ✗ 样本选取不当会有偏差 |
| ✓ 破坏性检验可用 | ✗ 不如全面调查准确 |

**【什么情况用抽样调查？】**

```
✓ 适用于：
- 总体数量很大
- 调查具有破坏性
- 时间和经费有限
- 不需要或无法全面调查

例子：
- 全国粮食产量估计
- 大规模民意调查
- 产品质量检验
```

#### 📊 统计的基本概念

**【1. 总体】**

```
定义：要考察的全体对象

例子：
调查某班学生体重
→ 总体：该班全体学生的体重

调查一批灯泡寿命
→ 总体：这批灯泡中每个灯泡的寿命
```

**【2. 个体】**

```
定义：总体中的每一个考察对象

例子：
调查某班学生体重
→ 个体：每个学生的体重

调查一批灯泡寿命
→ 个体：每个灯泡的寿命
```

**【3. 样本】**

```
定义：从总体中抽取的一部分个体

例子：
从某班50名学生中抽取10名
→ 样本：这10名学生的体重

从1000个灯泡中抽取50个
→ 样本：这50个灯泡的寿命
```

**【4. 样本容量】** ⚠️ 重要！

```
定义：样本中个体的数目

例子：
抽取10名学生
→ 样本容量：10

抽取50个灯泡
→ 样本容量：50

重要提醒：
样本容量是一个数字，没有单位！
```

#### 📐 概念辨析

**【易混淆概念对比】**

| 概念 | 含义 | 例子 | 单位 |
|------|------|------|------|
| **总体** | 全体对象 | 全班50名学生 | 有 |
| **个体** | 单个对象 | 每个学生 | 有 |
| **样本** | 部分对象 | 抽取的10名学生 | 有 |
| **样本容量** | 样本的个数 | 10 | **无** |

**【典型例题】**

```
题目：
为了解某市20000名初中生的视力情况，
从中抽取了500名学生进行视力检查。

指出总体、个体、样本、样本容量。

解答：
总体：该市20000名初中生的视力情况
个体：每名初中生的视力情况
样本：抽取的500名初中生的视力情况
样本容量：500（注意：没有单位！）
```

#### ⚠️ 易错点分析

**【易错点1：混淆对象和对象的属性】**

❌ 错误：
```
调查某班学生体重

总体：某班全体学生（错误！）
```

✓ 正确：
```
总体：某班全体学生的体重（要包括属性！）
```

**【易错点2：样本容量带单位】**

❌ 错误：
```
样本容量：50人（错误！）
```

✓ 正确：
```
样本容量：50（没有单位）
```

**【易错点3：混淆总体和样本】**

记忆：
```
总体 = 全体（全部）
样本 = 部分（一部分）
```

### 12.1.2 数据的整理

#### 📖 为什么要整理数据？

**【问题引入】**

```
调查了全班30名学生最喜欢的科目：

语文、数学、英语、数学、英语、英语、
数学、语文、英语、数学、英语、英语、
语文、数学、英语、数学、英语、英语、
语文、英语、数学、英语、英语、数学、
语文、数学、英语、英语、数学、英语

这样看，很难发现规律！

如果整理一下：
语文：5人
数学：10人
英语：15人

一目了然！
```

#### 🔍 深入理解：数据整理的方法

**【方法一：划记法】**

```
定义：用"正"字的每一划代表一个数据

优点：方便计数，不容易出错

例子：
调查某班学生最喜欢的科目

语文：正 （5人）
数学：正正 （10人）
英语：正正正 （15人）
```

**【方法二：表格法】**

```
定义：将数据整理成表格

优点：清晰明了，便于比较

例子：

| 科目 | 划记 | 人数 | 百分比 |
|------|------|------|--------|
| 语文 | 正 | 5 | 16.7% |
| 数学 | 正正 | 10 | 33.3% |
| 英语 | 正正正 | 15 | 50.0% |
| 总计 | | 30 | 100% |

计算百分比：
语文：5 ÷ 30 × 100% = 16.7%
数学：10 ÷ 30 × 100% = 33.3%
英语：15 ÷ 30 × 100% = 50.0%
```

#### 📊 数据整理的步骤

**【步骤1：收集数据】**

```
通过调查、观察、测量等方式收集原始数据

例子：
问卷调查、访谈、观察记录
```

**【步骤2：分类整理】**

```
按照一定的标准对数据进行分类

例子：
- 按科目分类
- 按成绩等级分类
- 按年龄段分类
```

**【步骤3：划记计数】**

```
用划记法记录每一类的数据个数

注意：
- 一个"正"字代表5个数据
- 划记要清楚、准确
```

**【步骤4：制成表格】**

```
将整理结果填入表格

包括：
- 类别
- 划记
- 数量
- 百分比（可选）
```

## 📚 12.2 统计图

### 12.2.1 条形统计图

#### 📖 什么是条形统计图？

**【直观理解】**

```
用直条的长短来表示数量的大小

就像比赛时用的记分牌：
谁的分高，谁的条就长！
```

**【基本结构】**

```
       ▲
   数量 │
       │     █
       │     █
       │     █  █
       │     █  █
       │     █  █  █
       └─────┴──┴──┴──→ 类别
         A   B   C   D

纵轴：表示数量
横轴：表示类别
直条：高度表示数量大小
```

#### 🔍 深入理解：条形图的特点

**【特点】**

1. **用一个单位长度表示一定的数量**
2. **用直条的长短表示数量的多少**
3. **能够清楚地看出每个项目的具体数目**

**【适用场景】**

```
✓ 适用于：
- 比较不同类别的数量
- 显示具体数值
- 各类别之间相对独立

例子：
- 各班人数比较
- 不同商品销量
- 各种运动喜好人数
```

#### 📊 优缺点对比

| 优点 | 缺点 |
|------|------|
| ✓ 清楚表示具体数目 | ✗ 不能表示部分占总体的比例 |
| ✓ 便于比较大小 | ✗ 不易显示变化趋势 |
| ✓ 制作简单 | ✗ 数据多时显得复杂 |

### 12.2.2 扇形统计图

#### 📖 什么是扇形统计图？

**【直观理解】**

```
用整个圆表示总体
用扇形表示各部分

就像切蛋糕：
每个人分多大一块，用扇形表示！
```

**【基本结构】**

```
      ╱────╲
    ╱   A   ╲
   │    │    │
  ╱     │     ╲
 │   B  │  C   │
  ╲     │     ╱
   │    │    │
    ╲   │   ╱
      ╰──╯

圆心角大小表示部分占总体比例
```

#### 🔍 深入理解：扇形图的特点

**【特点】**

1. **用圆代表总体（100%）**
2. **用扇形代表各部分**
3. **扇形的大小反映部分占总体百分比的大小**

**【核心公式】**

```
百分比 = (部分数量 ÷ 总体数量) × 100%

圆心角 = 360° × 百分比
```

**【适用场景】**

```
✓ 适用于：
- 显示部分与整体的关系
- 表示各部分占总体的比例
- 比较各部分的相对大小

例子：
- 班级男女生比例
- 支出构成
- 人口年龄结构
```

#### 📊 完整例题

**题目**：
某班学生最喜欢的科目统计：语文5人，数学10人，英语15人。

请绘制扇形统计图。

**解**：

**步骤1：计算总数**
```
总数 = 5 + 10 + 15 = 30（人）
```

**步骤2：计算各部分百分比**
```
语文：5 ÷ 30 × 100% = 16.7%
数学：10 ÷ 30 × 100% = 33.3%
英语：15 ÷ 30 × 100% = 50.0%
```

**步骤3：计算各扇形圆心角**
```
语文：360° × 16.7% = 60°
数学：360° × 33.3% = 120°
英语：360° × 50.0% = 180°
```

**步骤4：绘制扇形图**
```
      ╱───────╲  英语 50%
    ╱   180°   ╲
   │      │      │
  ╱  60° │ 120° ╲
 │ 语文   │ 数学  │
  ╲       │       ╱
   │      │      │
    ╲     │     ╱
      ╰───╯───╯
```

#### 📊 优缺点对比

| 优点 | 缺点 |
|------|------|
| ✓ 清楚表示部分占总体比例 | ✗ 不能表示具体数目 |
| ✓ 直观显示各部分关系 | ✗ 不易比较大小差异 |
| ✓ 美观、易懂 | ✗ 绘制较复杂 |

### 12.2.3 折线统计图

#### 📖 什么是折线统计图？

**【直观理解】**

```
用点表示数据
用线段连接各点

就像股票K线图：
清楚地看到变化趋势！
```

**【基本结构】**

```
       ▲
   数量 │    *───*
       │   ╱     ╲
       │  *       *
       │ *         *
       └──────────────→ 时间
        T1  T2  T3  T4

点：表示某一时刻的数量
线：表示变化趋势
```

#### 🔍 深入理解：折线图的特点

**【特点】**

1. **用一个单位长度表示一定的数量**
2. **根据数量的多少描出各点**
3. **用线段依次连接各点**

**【适用场景】**

```
✓ 适用于：
- 显示数据随时间的变化
- 表示变化趋势
- 预测未来走向

例子：
- 气温变化
- 股票价格
- 学生成绩变化
```

#### 📊 优缺点对比

| 优点 | 缺点 |
|------|------|
| ✓ 清楚显示变化趋势 | ✗ 不能表示具体数目（需估算） |
| ✓ 便于比较不同时期数据 | ✗ 不易表示部分与整体关系 |
| ✓ 可以预测未来走向 | ✗ 数据点少时不准确 |

### 12.2.4 三种统计图的对比

#### 📊 如何选择合适的统计图？

**【对比表】**

| 统计图 | 主要用途 | 优势 | 劣势 | 适用场景 |
|--------|---------|------|------|---------|
| **条形图** | 比较数量 | 清楚显示具体数值 | 不能表示比例 | 各类数据对比 |
| **扇形图** | 显示比例 | 清楚显示占比关系 | 不能显示具体数值 | 部分与整体关系 |
| **折线图** | 显示趋势 | 清楚显示变化 | 不能表示具体数值 | 时间序列数据 |

**【选择指南】**

```
问题1：想比较不同类别的数量？
→ 用条形统计图

问题2：想显示部分占总体的比例？
→ 用扇形统计图

问题3：想显示数据随时间的变化？
→ 用折线统计图

问题4：想全面展示数据？
→ 用多种统计图配合使用
```

#### 📐 综合应用实例

**【题目】**

某商店一周的销量如下（单位：件）：

| 星期 | 一 | 二 | 三 | 四 | 五 | 六 | 日 |
|------|---|---|---|---|---|---|---|
| 销量 | 20 | 25 | 18 | 30 | 35 | 50 | 45 |

请选择合适的统计图展示数据，并分析。

**解答**：

**方案1：用条形图**
```
目的：比较每天的具体销量

优点：清楚看到每天的销量
      容易找出销量最好和最差的日子
```

**方案2：用折线图**
```
目的：显示一周销量变化趋势

优点：清楚看到销量变化规律
      可以预测下周趋势
```

**分析**：
```
从数据可以看出：
1. 周末销量最高（周六50件，周日45件）
2. 周三销量最低（18件）
3. 整体呈上升趋势（工作日→周末）
4. 周五到周六是销量高峰期

建议：
- 周末可以增加进货
- 周三可以搞促销活动
```
