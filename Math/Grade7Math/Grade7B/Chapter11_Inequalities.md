## 📚 11.1 不等式

### 11.1.1 不等式的概念

#### 📖 什么是不等式？

**【回顾：等式】**

```
用等号"="连接的式子

例子：
3 + 2 = 5
x + 1 = 3
```

**【不等式的定义】**

> 用"<"、">"、"≤"、"≥"等不等号连接的式子叫做不等式。

#### 🔍 深入理解：不等号的含义

**【常见不等号】**

| 符号 | 读法 | 含义 | 例子 |
|------|------|------|------|
| **<** | 小于 | 左边的数小于右边的数 | 3 < 5 |
| **>** | 大于 | 左边的数大于右边的数 | 7 > 2 |
| **≤** | 小于或等于 | 左边的数不超过右边的数 | x ≤ 10 |
| **≥** | 大于或等于 | 左边的数不低于右边的数 | y ≥ 0 |
| **≠** | 不等于 | 左边的数不等于右边的数 | a ≠ b |

**【"≤"和"≥"的深入理解】**

```
x ≤ 10 表示：
x 可以等于 10
x 也可以小于 10

即：x < 10 或 x = 10
```

**【不等式的例子】**

```
数字不等式：
3 > 2
-1 < 0
5 ≥ 5

含未知数的不等式：
x + 1 > 3
2x - 1 ≤ 5
x² > 4
```

#### 📊 不等式与等式的对比

| 比较项目 | 等式 | 不等式 |
|---------|------|--------|
| **符号** | = | <, >, ≤, ≥, ≠ |
| **含义** | 相等 | 不相等 |
| **例子** | x + 1 = 3 | x + 1 > 3 |
| **解的特点** | 通常一个或几个解 | 通常是无数个解（范围） |

### 11.1.2 不等式的解

#### 📖 什么是不等式的"解"？

**【回顾：方程的解】**

```
方程：x + 1 = 3

解：x = 2

因为：当 x = 2 时，方程成立
```

**【不等式的解】**

```
不等式：x + 1 > 3

尝试一些值：
x = 1 → 1 + 1 = 2 > 3？✗ 不成立
x = 2 → 2 + 1 = 3 > 3？✗ 不成立
x = 3 → 3 + 1 = 4 > 3？✓ 成立！
x = 4 → 4 + 1 = 5 > 3？✓ 成立！
...

所以 x = 3, 4, 5, 10, 100... 都是解！
```

#### 🔍 深入理解：不等式的解集

**【定义】**

> 使不等式成立的未知数的值叫做不等式的解。一个含有未知数的不等式的所有解，组成这个不等式的解集。

**【重要特征】**

1. **无数个解**：通常是一个范围
2. **连续性**：解集中的数通常是连续的
3. **表示方法**：
   - 用式子表示：x > 3
   - 用数轴表示
   - 用集合表示：{x | x > 3}

**【直观理解】**

```
不等式：x > 3

解集：所有大于3的数

数轴表示：
      2     3     4
  ────○━━━━○━━━→
           3

说明：
- 3的位置画空心圆圈（不包含3）
- 向右画粗线（大于3的方向）
```

**【包含与不包含】**

| 符号 | 是否包含边界点 | 数轴表示 |
|------|---------------|---------|
| **>** | 不包含 | 空心圆点 ○ |
| **<** | 不包含 | 空心圆点 ○ |
| **≥** | 包含 | 实心圆点 ● |
| **≤** | 包含 | 实心圆点 ● |

### 11.1.3 不等式的基本性质

#### 📖 从等式性质到不等式性质

**【回顾：等式的性质】**

```
性质1：等式两边都加（或减）同一个数或式子，
      结果仍相等

      如果 a = b，那么 a ± c = b ± c

性质2：等式两边都乘同一个数，或除以同一个
      不为0的数，结果仍相等

      如果 a = b，那么 ac = bc, a/c = b/c
```

**【不等式也有类似的性质！】**

但要注意：乘除负数时不等号方向要改变！

#### 🔍 深入理解：不等式的三大性质

**【性质一：加减同一个数】**

> 不等式两边都加（或减）同一个数或式子，不等号的方向不变。

**数学表达**：
```
如果 a > b，那么 a ± c > b ± c
```

**【为什么不变号？】

```
例子：5 > 3

两边同时加2：
5 + 2 = 7
3 + 2 = 5
7 > 5 ✓ 不等号方向不变

直观理解：
就像在两个数的"高度"上同时加上一层楼
它们的相对高低关系不变！
```

**【性质二：乘除正数】**

> 不等式两边都乘（或除以）同一个正数，不等号的方向不变。

**数学表达**：
```
如果 a > b，c > 0，那么 ac > bc，a/c > b/c
```

**【为什么不变号？】

```
例子：5 > 3

两边同时乘2（正数）：
5 × 2 = 10
3 × 2 = 6
10 > 6 ✓ 不等号方向不变

直观理解：
就像把两个数的"距离"同时放大
它们的相对大小关系不变！
```

**【性质三：乘除负数】** ⚠️ 重点！

> 不等式两边都乘（或除以）同一个负数，不等号的方向改变。

**数学表达**：
```
如果 a > b，c < 0，那么 ac < bc，a/c < b/c
```

**【为什么要变号？】

```
例子：5 > 3

两边同时乘-1（负数）：
5 × (-1) = -5
3 × (-1) = -3
-5 < -3 ✓ 不等号方向改变了！

直观理解：
乘以负数就像"翻转到负数世界"
原本大的数变小了，原本小的数变大了！
```

**【用数轴理解】**

```
正数世界：
    0     3     5
────┬─────○─────○────→
    │
    原点

翻转后（乘-1）：
   -5    -3     0
────○─────○────┬────→
              │
              原点

5 > 3 变成了 -5 < -3
顺序完全颠倒！
```

#### 📊 三大性质总结表

| 性质 | 操作 | 条件 | 不等号方向 | 记忆口诀 |
|------|------|------|-----------|---------|
| **性质一** | 加减 | 任意数 | 不变 | 加减不变号 |
| **性质二** | 乘除 | 正数 | 不变 | 乘正不变号 |
| **性质三** | 乘除 | 负数 | **改变** | **乘负要变号** |

#### ⚠️ 易错点分析

**【易错点1：忘记负数要变号】**

❌ 错误：
```
-2x < 6
x < -3  （两边除以-2，没变号！）

✓ 正确：
-2x < 6
x > -3  （两边除以-2，变号！）
```

**【易错点2：混淆"≤"和"<"的变号】**

```
-2x ≤ 6

变号后：x ≥ -3

注意：只是不等号方向改变
"≤"变成"≥"，而不是变成"＞"
```

**【易错点3：应用性质时符号判断错误】**

```
如果 -a > 3

那么 a < -3（乘-1要变号）

✓ 正确
✗ 错误：a > -3（忘记变号）
```

## 📚 11.2 一元一次不等式

### 11.2.1 一元一次不等式的概念

#### 📖 从"方程"到"不等式"

**【回顾：一元一次方程】**

```
定义：含有一个未知数，未知数的次数是1的方程

标准形式：ax + b = 0（a ≠ 0）

例子：
2x + 3 = 7
x - 1 = 5
```

**【一元一次不等式】**

```
定义：含有一个未知数，未知数的次数是1的不等式

标准形式：ax + b > 0 或 ax + b < 0（a ≠ 0）

例子：
2x + 3 > 7
x - 1 ≤ 5
```

#### 🔍 深入理解：定义的三个关键词

**【1. "一元"】**

```
只含有一个未知数

例子：
✓ 2x + 3 > 7（一个未知数x）
✗ x + y > 5（两个未知数）
✗ 2x + 3y < 7（两个未知数）
```

**【2. "一次"】**

```
未知数的最高次数是1

例子：
✓ 2x + 3 > 7（x的次数是1）
✗ x² > 4（x的次数是2）
✗ x³ < 8（x的次数是3）
✗ 2x² + x > 3（最高次数是2）
```

**【3. "不等式"】**

```
用不等号连接

例子：
✓ 2x + 3 > 7
✓ x - 1 ≤ 5
✗ 2x + 3 = 7（这是方程，不是不等式）
```

#### 📊 判断是否为一元一次不等式

| 式子 | 是否是一元一次不等式 | 原因 |
|------|---------------------|------|
| 2x + 3 > 7 | ✓ 是 | 一元一次不等式 |
| x - 1 ≤ 5 | ✓ 是 | 一元一次不等式 |
| x² > 4 | ✗ 不是 | x的次数是2 |
| 2x + 3y < 7 | ✗ 不是 | 有两个未知数 |
| x + 1/x > 2 | ✗ 不是 | 不是整式（分母有未知数） |
| √x > 2 | ✗ 不是 | 不是整式（根号下有未知数） |

### 11.2.2 一元一次不等式的解法

#### 📖 解不等式的思想

**【类比：解一元一次方程】**

```
方程：2x + 3 = 7

解：
2x = 7 - 3
2x = 4
x = 2
```

**【解不等式：类似但不完全相同】**

```
不等式：2x + 3 > 7

解：
2x > 7 - 3
2x > 4
x > 2  （注意：除以正数，不变号）

结果：x > 2（无数个解）
```

#### 🔍 详细步骤：五步法

**【步骤1：去分母】**

```
不等式：x/2 + 1 > 3

两边同乘2（最小公倍数）：
2 × (x/2 + 1) > 2 × 3
x + 2 > 6

注意：两边每一项都要乘！
```

**【步骤2：去括号】**

```
不等式：2(x + 1) > 6

去括号：
2x + 2 > 6

注意：括号前的系数要乘到括号内每一项！
```

**【步骤3：移项】**

```
不等式：2x + 2 > 6

移项（把常数项移到右边）：
2x > 6 - 2
2x > 4

注意：移项要变号！
```

**【步骤4：合并同类项】**

```
不等式：2x > 4

已经是最简形式

如果还有同类项：
3x - x > 4
2x > 4
```

**【步骤5：系数化为1】** ⚠️ 最关键！

```
不等式：2x > 4

两边除以2（正数，不变号）：
x > 2

如果系数是负数：
-2x > 4
两边除以-2（负数，要变号！）：
x < -2
```

#### 📊 完整例题详解

**【例题1：基础型】**

解不等式：3x - 5 > x + 3

**解**：
```
步骤1：移项（把含x的项移到左边）
3x - x > 3 + 5

步骤2：合并同类项
2x > 8

步骤3：系数化为1（除以正数2，不变号）
x > 4

解集：x > 4

数轴表示：
      3     4     5
  ────○━━━━○━━━→
           4
```

**【例题2：系数为负】** ⚠️ 重点

解不等式：-2x + 3 < 7

**解**：
```
步骤1：移项
-2x < 7 - 3
-2x < 4

步骤2：系数化为1（除以负数-2，要变号！）
x > -2  （注意：< 变成了 >）

解集：x > -2

数轴表示：
   -3    -2    -1
  ───○━━━━●━━━→
        -2
```

**【例题3：去分母型】**

解不等式：(x + 1)/2 - (x - 1)/3 < 1

**解**：
```
步骤1：去分母（同乘6，2和3的最小公倍数）
6 × [(x + 1)/2 - (x - 1)/3] < 6 × 1
3(x + 1) - 2(x - 1) < 6

步骤2：去括号
3x + 3 - 2x + 2 < 6

步骤3：合并同类项
x + 5 < 6

步骤4：移项
x < 6 - 5
x < 1

解集：x < 1

数轴表示：
     0     1     2
  ───●━━━○━━━━→
        1
```

#### ⚠️ 易错点分析

**【易错点1：去分母漏乘】**

❌ 错误：
```
x/2 + 1 > 3

去分母（同乘2）：
x + 1 > 6  （漏乘了1！）

✓ 正确：
x + 2 > 6  （每一项都要乘）
```

**【易错点2：去括号分配不均】**

❌ 错误：
```
2(x + 1) > 6

去括号：
2x + 1 > 6  （2只乘了x，没乘1！）

✓ 正确：
2x + 2 > 6
```

**【易错点3：移项不变号】**

❌ 错误：
```
x + 3 > 5

移项：
x > 5 + 3  （+3移过去还是+3！）

✓ 正确：
x > 5 - 3  （移项要变号）
```

**【易错点4：系数化1忘记变号】** ⚠️ 最常见！

❌ 错误：
```
-2x < 6

系数化1：
x < -3  （除以负数没变号！）

✓ 正确：
x > -3  （除以负数要变号）
```

#### 📐 循序渐进练习

**练习4**：解下列不等式
（1）3x - 5 < 7
（2）2x + 3 ≥ 5x - 6
（3）(x - 1)/2 + (x + 1)/3 < 2

<details>
<summary>💡 点击查看详细解答</summary>

**解答**：

**（1）3x - 5 < 7**
```
移项：3x < 7 + 5
合并：3x < 12
系数化1：x < 4

答案：x < 4
```

**（2）2x + 3 ≥ 5x - 6**
```
移项：2x - 5x ≥ -6 - 3
合并：-3x ≥ -9
系数化1（除以-3，变号）：x ≤ 3

答案：x ≤ 3
```

**（3）(x - 1)/2 + (x + 1)/3 < 2**
```
去分母（同乘6）：
3(x - 1) + 2(x + 1) < 12

去括号：
3x - 3 + 2x + 2 < 12

合并同类项：
5x - 1 < 12

移项：
5x < 13

系数化1：
x < 13/5 = 2.6

答案：x < 2.6
```
</details>

**解**：
去分母（同乘6），得：3(x + 1) - 2(x - 1) < 6
去括号，得：3x + 3 - 2x + 2 < 6
移项，得：3x - 2x < 6 - 3 - 2
合并同类项，得：x < 1

所以不等式的解集是：**x < 1**

#### 11.2.3 不等式解集的表示

**方法一**：用式子表示
- x > 3（大于3的数）
- x < 2（小于2的数）
- x ≥ 1（大于或等于1的数）
- x ≤ 0（小于或等于0的数）

**方法二**：用数轴表示
- **空心圆点**：表示不包含这个数（> 或 <）
- **实心圆点**：表示包含这个数（≥ 或 ≤）

**例子**：
- x > 3 在数轴上表示为：
  ```
       3
  ─────○━━━━━━>
  ```
  （3的位置是空心圆点，向右画粗线）

- x ≤ 2 在数轴上表示为：
  ```
        2
  <━━━━●─────
  ```
  （2的位置是实心圆点，向左画粗线）

## 📚 11.3 一元一次不等式组

### 11.3.1 一元一次不等式组的概念

#### 📖 从"一个不等式"到"多个不等式"

**【生活中的限制】**

```
游泳池的规定：
"身高必须超过1.2米，但不能超过1.8米"

如果身高是 h 米：
h > 1.2 且 h ≤ 1.8

这就是不等式组！
```

**【问题引入】**

```
我们学过不等式：
x > 3（x的取值范围）

但实际问题往往有多个限制：
x > 3
x < 5

同时满足这两个条件的x是什么？
```

#### 🔍 深入理解：不等式组的定义

**【定义】**

> 含有相同未知数的几个一元一次不等式合在一起，就组成了一个一元一次不等式组。

**【标准形式】**

```
{
  ax + b > c
  dx + e < f
}

或简写为：
ax + b > c，dx + e < f
```

**【例子】**

```
不等式组：
{
  x > 2
  x < 5
}

含义：
x 既要大于2，又要小于5
```

### 11.3.2 一元一次不等式组的解集

#### 📖 什么是"公共解集"？

**【理解】**

```
不等式组：
{
  x > 2  ①
  x < 5  ②
}

①的解集：所有大于2的数
②的解集：所有小于5的数

公共解集：既大于2又小于5的数
即：2 < x < 5
```

#### 🔍 深入理解：四种基本情况

**【情况1：同大取大】**

```
不等式组：
{
  x > a  （a < b）
  x > b
}

数轴表示：
     a     b
  ───○━━━○━━━→

公共部分：x > b

口诀：同大取大
```

**【情况2：同小取小】**

```
不等式组：
{
  x < a  （a < b）
  x < b
}

数轴表示：
     a     b
  ○━━━○━━━━━━→

公共部分：x < a

口诀：同小取小
```

**【情况3：大小小大取中间】**

```
不等式组：
{
  x > a  （a < b）
  x < b
}

数轴表示：
     a     b
  ───○━━━○━━━→
     ╰━━━╯

公共部分：a < x < b

口诀：大小小大取中间
```

**【情况4：大大小小无解】**

```
不等式组：
{
  x > a  （a > b）
  x < b
}

数轴表示：
     b     a
  ───○━━━○━━━→
     ╳  ╳
      （没有公共部分）

公共部分：无解

口诀：大大小小无解
```

#### 📊 四种情况总结表

| 类型 | 条件 | 图示 | 解集 | 口诀 |
|------|------|------|------|------|
| **同大** | x > a, x > b (a < b) | ⚫━━━⚫➡ | x > b | 同大取大 |
| **同小** | x < a, x < b (a < b) | ⚫➡━⚫━ | x < a | 同小取小 |
| **中间** | x > a, x < b (a < b) | ⚫━╾━⚫➡ | a < x < b | 大小小大取中间 |
| **无解** | x > a, x < b (a > b) | ⚫━╬━⚫➡ | 无解 | 大大大小无解 |

#### 💡 快速判断技巧

**【技巧1：看不等号方向】**

```
两个都是 > → 同大取大
两个都是 < → 同小取小
一个 > 一个 < → 看大小关系
```

**【技巧2：画简易数轴】**

```
快速画出数轴，找出公共部分

例如：
x > 2，x < 5

数轴：
  2     5
──○━━━○━→
  ╰━━━╯

一目了然：2 < x < 5
```

### 11.3.3 一元一次不等式组的解法

#### 📖 解不等式组的步骤

**【三步法】**

**步骤1：分别求解**
```
分别解出不等式组中各个不等式的解集

注意：
- 熟练运用不等式的性质
- 特别注意系数化1时的变号问题
```

**步骤2：画数轴**
```
利用数轴表示各个不等式的解集

技巧：
- 用不同颜色或符号区分
- 找出公共部分（重叠区域）
```

**步骤3：写解集**
```
根据数轴上的公共部分，写出不等式组的解集

格式：
- 用不等式表示
- 或用区间表示
```

#### 📊 完整例题详解

**【例题1：同大取大型】**

解不等式组：
```
{
  2x - 1 > x + 1  ①
  x + 8 < 4x - 1  ②
}
```

**解**：
```
步骤1：分别求解

解①：
2x - 1 > x + 1
2x - x > 1 + 1
x > 2

解②：
x + 8 < 4x - 1
x - 4x < -1 - 8
-3x < -9
x > 3  （除以-3，变号！）

步骤2：画数轴
     2     3
  ───○━━━○━━━→
     ╰━━━╯

公共部分：x > 3

步骤3：写解集
不等式组的解集是：x > 3
```

**【例题2：取中间型】**

解不等式组：
```
{
  x - 3 < 0  ①
  2x + 1 > 3  ②
}
```

**解**：
```
步骤1：分别求解

解①：
x - 3 < 0
x < 3

解②：
2x + 1 > 3
2x > 2
x > 1

步骤2：画数轴
     1     3
  ───○━━━○━━━→
     ╰━━━╯

公共部分：1 < x < 3

步骤3：写解集
不等式组的解集是：1 < x < 3
```

**【例题3：无解型】**

解不等式组：
```
{
  x + 2 > 4  ①
  x - 1 < 1  ②
}
```

**解**：
```
步骤1：分别求解

解①：
x + 2 > 4
x > 2

解②：
x - 1 < 1
x < 2

步骤2：画数轴
     2
  ───○━━━○━━━→
     ╳╳
  （没有公共部分）

步骤3：写解集
不等式组无解
```

#### ⚠️ 易错点分析

**【易错点1：忘记画数轴】**

❌ 不好：直接口算判断

✓ 好习惯：画数轴辅助判断

**【易错点2：解集写错】**

     1     3
  ───○━━━○━━━→
     ╰━━━╯

}

注意：
2 是包含的（≥）
5 是不包含的（<）
```

## 📖 典型例题

### 例题1：不等式的基本性质

**题目**：已知 a > b，用">"或"<"填空
（1）a + 3 ____ b + 3
（2）a - 2 ____ b - 2
（3）3a ____ 3b
（4）-2a ____ -2b

**解答**：
（1）a + 3 **>** b + 3（性质一）

（2）a - 2 **>** b - 2（性质一）

（3）3a **>** 3b（性质二）

（4）-2a **<** -2b（性质三，乘负数变号）

---

### 例题2：解一元一次不等式

**题目**：解下列不等式
（1）2x - 3 < 5
（2）3x + 2 ≥ 5x - 4
（3）(x - 1)/2 + (x + 1)/3 < 2

**解答**：
（1）2x - 3 < 5
    移项，得：2x < 5 + 3
    合并同类项，得：2x < 8
    系数化为1，得：**x < 4**

（2）3x + 2 ≥ 5x - 4
    移项，得：3x - 5x ≥ -4 - 2
    合并同类项，得：-2x ≥ -6
    系数化为1（除以-2，变号），得：**x ≤ 3**

（3）(x - 1)/2 + (x + 1)/3 < 2
    去分母（同乘6），得：3(x - 1) + 2(x + 1) < 12
    去括号，得：3x - 3 + 2x + 2 < 12
    移项，得：3x + 2x < 12 + 3 - 2
    合并同类项，得：5x < 13
    系数化为1，得：**x < 13/5**

### 例题4：解不等式组

**题目**：解下列不等式组
（1）
```
x > 2
x > 3
```

（2）
```
x < 5
x < 3
```

（3）
```
x > 1
x < 4
```

（4）
```
x > 5
x < 3
```

**解答**：
（1）x > 2 和 x > 3 的公共部分是：**x > 3**（同大取大）

（2）x < 5 和 x < 3 的公共部分是：**x < 3**（同小取小）

（3）x > 1 和 x < 4 的公共部分是：**1 < x < 4**（大小小大取中间）

（4）x > 5 和 x < 3 没有公共部分：**无解**（大大小小无解）

### 例题7：列不等式解应用题

**题目**：某班有学生50人，其中男生人数多于女生人数，求男生至少有多少人？

**解答**：
设男生有x人，则女生有(50 - x)人
根据题意：x > 50 - x
2x > 50
x > 25

因为x是整数，
所以 x ≥ 26

答：男生至少有26人。
