# 第十一章 不等式与不等式组

## 📚 知识要点

### 11.1 不等式

#### 11.1.1 不等式的概念

**定义**：用"<"、">"、"≤"、"≥"等不等号连接的式子叫做不等式

**常见不等号**：
- <：小于
- >：大于
- ≤：小于或等于
- ≥：大于或等于
- ≠：不等于

**例子**：
- 3 > 2（不等式）
- x + 1 > 3（不等式）
- 2x - 1 ≤ 5（不等式）

#### 11.1.2 不等式的解

**定义**：使不等式成立的未知数的值叫做不等式的解

**不等式的解集**：一个含有未知数的不等式的所有解，组成这个不等式的解集

**注意**：
- 不等式的解集通常是一个范围，有无数个解
- 不等式的解集可以在数轴上表示

#### 11.1.3 不等式的基本性质

**性质一**：不等式两边都加（或减）同一个数或式子，不等号的方向不变

```
如果 a > b，那么 a ± c > b ± c
```

**性质二**：不等式两边都乘（或除以）同一个正数，不等号的方向不变

```
如果 a > b，c > 0，那么 ac > bc，a/c > b/c
```

**性质三**：不等式两边都乘（或除以）同一个负数，不等号的方向改变

```
如果 a > b，c < 0，那么 ac < bc，a/c < b/c
```

**注意**：性质三是容易出错的地方，乘除负数要变号！

### 11.2 一元一次不等式

#### 11.2.1 一元一次不等式的概念

**定义**：含有一个未知数，未知数的次数是1的不等式，叫做一元一次不等式

**标准形式**：ax + b > 0 或 ax + b < 0（a ≠ 0）

**例子**：
- 2x + 3 > 7（一元一次不等式）
- x - 1 ≤ 5（一元一次不等式）
- x² > 4（不是，x的次数是2）

#### 11.2.2 一元一次不等式的解法

**步骤**：
1. **去分母**：不等式两边同乘各分母的最小公倍数
2. **去括号**：根据去括号法则
3. **移项**：把含有未知数的项移到不等式的一边，常数项移到另一边
4. **合并同类项**：将不等式化为 ax > b 或 ax < b（a ≠ 0）的形式
5. **系数化为1**：
   - 若 a > 0，不等号方向不变
   - 若 a < 0，不等号方向改变

**注意**：解不等式时，如果乘除负数，一定要变号！

**例子1**：解不等式 3x - 5 > x + 3

**解**：
移项，得：3x - x > 3 + 5
合并同类项，得：2x > 8
系数化为1，得：**x > 4**

---

**例子2**：解不等式 -2x + 3 < 7

**解**：
移项，得：-2x < 7 - 3
合并同类项，得：-2x < 4
系数化为1（两边同除以-2，不等号方向改变），得：**x > -2**

---

**例子3**：解不等式 (x + 1)/2 - (x - 1)/3 < 1

**解**：
去分母（同乘6），得：3(x + 1) - 2(x - 1) < 6
去括号，得：3x + 3 - 2x + 2 < 6
移项，得：3x - 2x < 6 - 3 - 2
合并同类项，得：x < 1

所以不等式的解集是：**x < 1**

#### 11.2.3 不等式解集的表示

**方法一**：用式子表示
- x > 3（大于3的数）
- x < 2（小于2的数）
- x ≥ 1（大于或等于1的数）
- x ≤ 0（小于或等于0的数）

**方法二**：用数轴表示
- **空心圆点**：表示不包含这个数（> 或 <）
- **实心圆点**：表示包含这个数（≥ 或 ≤）

**例子**：
- x > 3 在数轴上表示为：
  ```
       3
  ─────○━━━━━━>
  ```
  （3的位置是空心圆点，向右画粗线）

- x ≤ 2 在数轴上表示为：
  ```
        2
  <━━━━●─────
  ```
  （2的位置是实心圆点，向左画粗线）

### 11.3 一元一次不等式组

#### 11.3.1 一元一次不等式组的概念

**定义**：含有相同未知数的几个一元一次不等式合在一起，就组成了一个一元一次不等式组

**例子**：
```
x > 2
x < 5
```

#### 11.3.2 一元一次不等式组的解集

**定义**：不等式组中各个不等式的解集的公共部分，叫做这个不等式组的解集

**四种基本情况**：
1. **同大取大**：
   ```
   x > a
   x > b
   （a < b）
   ```
   解集：x > b

2. **同小取小**：
   ```
   x < a
   x < b
   （a < b）
   ```
   解集：x < a

3. **大小小大取中间**：
   ```
   x > a
   x < b
   （a < b）
   ```
   解集：a < x < b

4. **大大小小无解**：
   ```
   x > a
   x < b
   （a > b）
   ```
   解集：无解

#### 11.3.3 一元一次不等式组的解法

**步骤**：
1. 分别解出不等式组中各个不等式的解集
2. 利用数轴求出各个不等式解集的公共部分
3. 写出不等式组的解集

**例子**：解不等式组
```
2x - 1 > x + 1  ①
x + 8 < 4x - 1  ②
```

**解**：
解①，得：x > 2
解②，得：x > 3

在数轴上表示：
```
     2     3
━━━━○━━━○━━━>
```

公共部分是：**x > 3**

## 📖 典型例题

### 例题1：不等式的基本性质

**题目**：已知 a > b，用">"或"<"填空
（1）a + 3 ____ b + 3
（2）a - 2 ____ b - 2
（3）3a ____ 3b
（4）-2a ____ -2b

**解答**：
（1）a + 3 **>** b + 3（性质一）

（2）a - 2 **>** b - 2（性质一）

（3）3a **>** 3b（性质二）

（4）-2a **<** -2b（性质三，乘负数变号）

---

### 例题2：解一元一次不等式

**题目**：解下列不等式
（1）2x - 3 < 5
（2）3x + 2 ≥ 5x - 4
（3）(x - 1)/2 + (x + 1)/3 < 2

**解答**：
（1）2x - 3 < 5
    移项，得：2x < 5 + 3
    合并同类项，得：2x < 8
    系数化为1，得：**x < 4**

（2）3x + 2 ≥ 5x - 4
    移项，得：3x - 5x ≥ -4 - 2
    合并同类项，得：-2x ≥ -6
    系数化为1（除以-2，变号），得：**x ≤ 3**

（3）(x - 1)/2 + (x + 1)/3 < 2
    去分母（同乘6），得：3(x - 1) + 2(x + 1) < 12
    去括号，得：3x - 3 + 2x + 2 < 12
    移项，得：3x + 2x < 12 + 3 - 2
    合并同类项，得：5x < 13
    系数化为1，得：**x < 13/5**

---

### 例题4：解不等式组

**题目**：解下列不等式组
（1）
```
x > 2
x > 3
```

（2）
```
x < 5
x < 3
```

（3）
```
x > 1
x < 4
```

（4）
```
x > 5
x < 3
```

**解答**：
（1）x > 2 和 x > 3 的公共部分是：**x > 3**（同大取大）

（2）x < 5 和 x < 3 的公共部分是：**x < 3**（同小取小）

（3）x > 1 和 x < 4 的公共部分是：**1 < x < 4**（大小小大取中间）

（4）x > 5 和 x < 3 没有公共部分：**无解**（大大小小无解）

### 例题6：不等式的整数解

**题目**：求不等式 3(x + 1) < 5x + 9 的正整数解。

**解答**：
3(x + 1) < 5x + 9
3x + 3 < 5x + 9
3x - 5x < 9 - 3
-2x < 6
x > -3（除以-2，变号）

正整数解有：**1，2，3，4，5...**

### 例题7：列不等式解应用题

**题目**：某班有学生50人，其中男生人数多于女生人数，求男生至少有多少人？

**解答**：
设男生有x人，则女生有(50 - x)人
根据题意：x > 50 - x
2x > 50
x > 25

因为x是整数，
所以 x ≥ 26

答：男生至少有26人。

### 例题8：不等式组的应用

**题目**：某商店出售一种商品，原价每件100元，现在打8折出售，但成本价不低于60元，求售价的取值范围。

**解答**：
设成本价为x元
根据题意：60 ≤ x ≤ 100 × 0.8
60 ≤ x ≤ 80

答：成本价的取值范围是60元到80元之间。
