# 第十三章 轴对称

## 本章导引

**轴对称**不仅是几何学的重要内容，更是一种重要的数学思想方法。通过研究轴对称，我们可以：
- 理解图形的对称美

## 13.1 轴对称

### 13.1.1 从生活到数学

#### 🌨 生活中的对称现象

**自然界中的对称**：
- 🦋 蝴蝶的翅膀
- 🍀 叶子的形状
- 🌸 花瓣的排列
- 🏔️ 雪花的晶体
- 👤 人的身体

**人工创造的对称**：
- 🏛️ 古典建筑的对称设计
- 🖼️ 对称的装饰图案
- ✂️ 剪纸艺术
- 🔤 字母和数字（A、H、8等）
- 🚗 汽车的外观设计

**思考**：为什么这么多事物都呈对称形状？
- 对称给人**美感**和**平衡感**
- 对称使结构更加**稳定**
- 对称在工程上有**实用价值**

#### 📖 轴对称图形

**定义**：如果一个图形沿一条直线折叠，直线两旁的部分能够互相重合，这个图形就叫做轴对称图形。

**理解定义的关键**：
1. **有一条对称轴**：这条直线是对称轴
2. **能够折叠重合**：沿对称轴对折后，两部分完全重合
3. **不是所有图形都轴对称**：需要具体判断

**动手体验**：
1. 拿一张纸，对折后剪出一个图形
2. 展开后观察：这个图形就是轴对称图形
3. 折痕就是对称轴

#### 📐 对称轴

**定义**：轴对称图形的折痕所在的直线叫做对称轴。

**注意**：
- 对称轴是一条**直线**，不是线段
- 一个轴对称图形可能有**一条或多条**对称轴
- 对称轴通常用**虚线**表示

#### 🔍 常见轴对称图形及其对称轴

| 图形 | 对称轴 | 对称轴条数 | 图示说明 |
|------|--------|-----------|----------|
| **线段** | 线段的垂直平分线 | 1条 | ⟷ |
| **角** | 角平分线所在的直线 | 1条 | ∠ |
| **等腰三角形** | 顶角平分线所在的直线 | 1条 | △ |
| **等边三角形** | 每条边的高、中线、角平分线所在的直线 | 3条 | △ |
| **矩形** | 对边中点连线所在的直线 | 2条 | ▭ |
| **菱形** | 对角线所在的直线 | 2条 | ◇ |
| **正方形** | 对边中点连线和对角线所在的直线 | 4条 | □ |
| **圆** | 任意一条直径所在的直线 | 无数条 | ○ |
| **等腰梯形** | 两底中点连线所在的直线 | 1条 | ˢᴎ |
| **正n边形** | 连接顶点和对面中点或对角顶点 | n条 | ⬡ |

#### 💡 判断轴对称图形的方法

**方法一：折叠法**
想象或实际操作，看能否沿某条直线对折后重合

**方法二：观察法**
观察图形是否有"左右对称"或"上下对称"的特征

**方法三：找对称轴**
尝试找到所有可能的对称轴

#### 📝 典型例题

**例1**：下列图形中，是轴对称图形的是（  ）
A. 平行四边形
B. 三角形
C. 线段
D. 梯形

**解答**：
**答案：C**

**分析**：
- **A. 平行四边形**：一般的平行四边形不是轴对称图形（特殊的如矩形、菱形才是）
- **B. 三角形**：一般的三角形不是轴对称图形（等腰三角形才是）
- **C. 线段**：是轴对称图形，对称轴是它的垂直平分线 ✅
- **D. 梯形**：一般的梯形不是轴对称图形（等腰梯形才是）

**注意**：不要认为"平行四边形"、"三角形"、"梯形"这些大类都是轴对称图形！

### 13.2 两个图形成轴对称

#### 📖 概念辨析

**两个图形关于某条直线对称**：把一个图形沿某一条直线折叠，如果它能够与另一个图形重合，那么就说这两个图形关于这条直线对称。

**理解要点**：
1. 这里说的是**两个图形**之间的关系
2. 不是"一个图形内部的对称"
3. 强调的是**相对位置**的对称性

#### 🔄 轴对称图形 vs 两个图形成轴对称

| 对比项 | 轴对称图形 | 两个图形成轴对称 |
|--------|-----------|----------------|
| **对象** | 一个图形 | 两个图形 |
| **关系** | 图形自身内部的对称 | 两个图形之间的对称关系 |
| **对称轴** | 在图形内部 | 在两个图形之间 |
| **例子** | 等腰三角形 | 两只完全相同的手 |

**联系**：
- 轴对称图形沿对称轴分开，得到的两部分关于对称轴成轴对称
- 把两个成轴对称的图形看作一个整体，它就是轴对称图形

#### 📐 轴对称的性质

**性质1**：关于某条直线对称的两个图形是全等图形

**性质2**：对称轴是对称点连线的垂直平分线

**性质3**：对称轴垂直平分连接对称点的线段

**性质4**：两个图形关于某条直线对称，如果它们对应线段或延长线相交，那么交点在对称轴上

#### 📝 典型例题

**例2**：如图，△ABC和△ABC'关于直线l对称，点A和B的对称点分别是A'和B'。
（1）找出图中所有的对称点
（2）连接AA'，BB'，CC'，它们与直线l有什么关系？
（3）延长AB和A'B'，它们的交点在哪里？

**解答**：
（1）对称点：
A ↔ A'，B ↔ B'，C ↔ C'

（2）AA'，BB'，CC'都与直线l**垂直**，并且都被直线l**平分**

（3）AB和A'B'的交点在**直线l上**

**说明**：这道题综合考查了轴对称的多个性质。

## 13.3 画轴对称图形

### 13.3.1 画对称点的方法

#### 📐 基本方法

**步骤**：
1. 过已知点A作对称轴l的垂线，垂足为O
2. 在垂线上截取OA' = OA
3. 点A'就是点A关于直线l的对称点

**几何语言**：
```
已知：点A和直线l
求作：点A关于直线l的对称点A'

作法：
1. 过点A作直线l的垂线，垂足为O
2. 在AO的延长线上截取OA' = OA
3. 点A'即为所求
```

#### 🎯 作图要点

1. **垂线要垂直**：使用三角板确保垂线垂直于对称轴
2. **距离要相等**：OA' = OA，可以用圆规或刻度尺测量
3. **点要在另一侧**：A'和A在对称轴的两侧

#### 📝 典型例题

**例3**：如图，已知点A和直线l，画出点A关于直线l的对称点A'。

**作法**：
1. 过点A作直线l的垂线，垂足为O
2. 在AO的延长线上截取OA' = OA
3. 点A'即为所求

### 13.3.2 画轴对称图形

#### 📐 基本步骤

**步骤**：
1. **找关键点**：找出图形的关键点（顶点、端点、拐点等）
2. **作对称点**：分别作出这些关键点关于对称轴的对称点
3. **连接对称点**：按原图形的连接方式顺次连接各对称点
4. **得到对称图形**：得到的图形就是原图形的轴对称图形

#### 💡 作图技巧

**技巧1**：关键点要找全
- 对于三角形：找3个顶点
- 对于四边形：找4个顶点
- 对于复杂图形：找所有转折点

**技巧2**：连接顺序要一致
- 按原图形的连接方式连接
- 不要改变连接顺序

**技巧3**：特殊点在对称轴上
- 如果关键点在对称轴上，它的对称点就是它本身

#### 📝 典型例题

**例4**：画出△ABC关于直线l的对称图形。

**作法**：
1. 作点A关于直线l的对称点A'
2. 作点B关于直线l的对称点B'
3. 作点C关于直线l的对称点C'
4. 连接A'B'、B'C'、C'A'
5. △A'B'C'就是△ABC关于直线l的对称图形

**例5**：如图，画出四边形ABCD关于直线l的对称图形。

**作法**：
1. 分别作点A、B、C、D关于直线l的对称点A'、B'、C'、D'
2. 连接A'B'、B'C'、C'D'、D'A'
3. 四边形A'B'C'D'就是四边形ABCD关于直线l的对称图形

## 13.4 等腰三角形

### 13.4.1 等腰三角形的性质

#### 📖 等腰三角形的定义

**定义**：有两条边相等的三角形叫做等腰三角形。

**相关术语**：
- **腰**：相等的两条边
- **底边**：另一条边
- **顶角**：两腰的夹角
- **底角**：底边上的两个角

#### 🔍 性质1：等边对等角

**定理**：等腰三角形的两个底角相等（简称"等边对等角"）

**数学表达**：
```
∵ AB = AC
∴ ∠B = ∠C
```

#### 📝 定理的证明

**证明**：
```
已知：在△ABC中，AB = AC
求证：∠B = ∠C

证明：作△ABC的角平分线AD

在△BAD和△CAD中
AB = AC（已知）
∠BAD = ∠CAD（角平分线的定义）
AD = AD（公共边）
∴ △BAD ≅ △CAD（SAS）
∴ ∠B = ∠C（全等三角形的对应角相等）
```

#### 🔍 性质2：三线合一

**定理**：等腰三角形的顶角平分线、底边上的中线、底边上的高相互重合（简称"三线合一"）

**说明**：在等腰三角形ABC中，AB = AC
- AD是顶角平分线 ⇒ AD也是底边BC的中线和高
- AD是底边BC的中线 ⇒ AD也是顶角平分线和高
- AD是底边BC的高 ⇒ AD也是顶角平分线和中线

这三条线重合为一条线！

#### 📝 定理的证明

**证明**：
```
已知：在△ABC中，AB = AC，AD是顶角平分线
求证：BD = CD，AD⊥BC

证明：在△ABD和△ACD中
AB = AC（已知）
∠BAD = ∠CAD（AD是角平分线）
AD = AD（公共边）
∴ △ABD ≅ △ACD（SAS）
∴ BD = CD，∠ADB = ∠ADC（全等三角形的对应边、对应角相等）

又∵ ∠ADB + ∠ADC = 180°（平角的定义）
∴ ∠ADB = ∠ADC = 90°
∴ AD⊥BC

因此，AD既是顶角平分线，又是底边的中线和高，三线合一。
```

#### 🔍 性质3：轴对称图形

**定理**：等腰三角形是轴对称图形，对称轴是顶角平分线所在的直线。

#### 🎯 三线合一的应用

**应用场景**：
1. 已知高求中线，或已知中线求高
2. 证明线段相等或角相等
3. 计算角度或线段长度
4. 添加辅助线

#### 📝 典型例题

**例6**：如图，在△ABC中，AB = AC，∠A = 40°，求∠B和∠C的度数。

**解答**：
∵ AB = AC
∴ ∠B = ∠C（等边对等角）

又∵ ∠A + ∠B + ∠C = 180°
40° + 2∠B = 180°
2∠B = 140°
∠B = 70°

∴ ∠B = ∠C = **70°**

**例7**：如图，在△ABC中，AB = AC，AD是角平分线，∠B = 70°，求∠BAD的度数。

**解答**：
∵ AB = AC
∴ ∠B = ∠C（等边对等角）
又∵ ∠B = 70°
∴ ∠C = 70°

∴ ∠BAC = 180° - ∠B - ∠C = 180° - 70° - 70° = 40°

∵ AD是角平分线
∴ ∠BAD = ½∠BAC = ½ × 40° = **20°**

**例8**：如图，在△ABC中，AB = AC，AD⊥BC于D，BD = 5cm，求BC的长。

**解答**：
∵ AB = AC，AD⊥BC
∴ BD = CD（等腰三角形"三线合一"）
又∵ BD = 5cm
∴ CD = 5cm
∴ BC = BD + CD = 5 + 5 = **10cm**

### 13.4.2 等腰三角形的判定

#### 📖 判定定理

**定理**：如果一个三角形有两个角相等，那么这两个角所对的边也相等（简称"等角对等边"）

**数学表达**：
```
∵ ∠B = ∠C
∴ AB = AC
```

#### 📝 定理的证明

**证明**：
```
已知：在△ABC中，∠B = ∠C
求证：AB = AC

证明：作BC边的垂直平分线DE，交BC于D，交AC于E

在△BDE和△CDE中
BD = CD（DE是BC的垂直平分线）
∠BDE = ∠CDE = 90°（垂直的定义）
DE = DE（公共边）
∴ △BDE ≅ △CDE（SAS）
∴ ∠DBE = ∠DCE（全等三角形的对应角相等）

又∵ ∠ABC = ∠DBE + ∠DBE
    ∠ACB = ∠DCE + ∠DCE
    ∠ABC = ∠ACB（已知）
∴ ∠DBE = ∠DCE
∴ ∠DBE = ∠DCE = ½∠ABC

同理可证：∠EBD = ∠FCD

在△ABD和△ACE中
∠ABD = ∠ACE（已证）
∠BAD = ∠CAE（等角的余角相等）
AD = AE（公共边）
∴ △ABD ≅ △ACE（AAS）
∴ AB = AC（全等三角形的对应边相等）
```

#### 🔄 性质与判定的关系

**性质定理**（等边对等角）：
- 条件：AB = AC
- 结论：∠B = ∠C

**判定定理**（等角对等边）：
- 条件：∠B = ∠C
- 结论：AB = AC

它们互为**逆定理**！

#### 📝 典型例题

**例9**：如图，在△ABC中，∠B = 70°，∠C = 70°，求证：AB = AC。

**证明**：
∵ ∠B = ∠C = 70°
∴ AB = AC（等角对等边）

### 13.4.3 等边三角形

#### 📖 等边三角形的定义

**定义**：三条边都相等的三角形叫做等边三角形（也叫正三角形）

**特殊性质**：
等边三角形是特殊的等腰三角形，它有三条相等的边。

#### 🔍 等边三角形的性质

**性质1**：等边三角形的三个内角都相等，都等于60°

**证明**：
```
∵ AB = BC = CA
∴ ∠A = ∠B（等边对等角）
   ∠B = ∠C（等边对等角）
∴ ∠A = ∠B = ∠C

又∵ ∠A + ∠B + ∠C = 180°
∴ 3∠A = 180°
∴ ∠A = 60°
∴ ∠A = ∠B = ∠C = 60°
```

**性质2**：等边三角形有三条对称轴

**性质3**：等边三角形的重心、垂心、内心、外心重合（四心合一）

#### 📐 等边三角形的判定

**判定1**：三条边都相等的三角形是等边三角形
（定义判定）

**判定2**：三个角都相等的三角形是等边三角形

**证明**：
```
∵ ∠A = ∠B = ∠C = 60°
∴ AB = AC（等角对等边）
   BC = AC（等角对等边）
∴ AB = BC = CA
∴ △ABC是等边三角形
```

**判定3**：有一个角是60°的等腰三角形是等边三角形

**证明**：
```
已知：在△ABC中，AB = AC，∠A = 60°
求证：△ABC是等边三角形

证明：∵ AB = AC
∴ ∠B = ∠C（等边对等角）

又∵ ∠A + ∠B + ∠C = 180°
   60° + 2∠B = 180°
   ∠B = 60°
∴ ∠B = ∠C = 60°
∴ ∠A = ∠B = ∠C = 60°
∴ △ABC是等边三角形（三个角都相等）
```

#### 📝 典型例题

**例10**：如图，△ABC是等边三角形，D是BC的中点，连接AD，求∠BAD的度数。

**解答**：
∵ △ABC是等边三角形
∴ AB = BC = CA，∠B = ∠C = ∠BAC = 60°

又∵ D是BC的中点，AB = AC
∴ AD⊥BC（等腰三角形"三线合一"）
∴ ∠ADB = 90°

在Rt△ABD中：
∠BAD = 90° - ∠B = 90° - 60° = **30°**

**例11**：如图，在△ABC中，AB = AC，∠A = 60°，求证：△ABC是等边三角形。

**证明**：
∵ AB = AC
∴ ∠B = ∠C（等边对等角）

又∵ ∠A + ∠B + ∠C = 180°
   60° + 2∠B = 180°
   ∠B = 60°
∴ ∠B = ∠C = 60°
∴ ∠A = ∠B = ∠C = 60°
∴ △ABC是等边三角形

## 13.5 线段的垂直平分线

### 13.5.1 垂直平分线的性质

#### 📖 性质定理

**定理**：线段垂直平分线上的点到这条线段两个端点的距离相等。

**数学表达**：
```
∵ 直线l是线段AB的垂直平分线，点P在l上
∴ PA = PB
```

#### 📝 定理的证明

**证明**：
```
已知：直线l是线段AB的垂直平分线，点P在l上
求证：PA = PB

证明：设直线l与AB相交于点O
∵ l是AB的垂直平分线
∴ AO = BO，l⊥AB

在△PAO和△PBO中
AO = BO（已证）
∠POA = ∠POB = 90°（垂直的定义）
PO = PO（公共边）
∴ △PAO ≅ △PBO（SAS）
∴ PA = PB（全等三角形的对应边相等）
```

#### 🎯 性质的应用

**应用1**：证明线段相等
**应用2**：求线段的长度
**应用3**：确定点的轨迹

#### 📝 典型例题

**例12**：如图，直线l是线段AB的垂直平分线，点P在l上，PA = 6cm，求PB的长。

**解答**：
∵ 直线l是线段AB的垂直平分线，点P在l上
∴ PB = PA（线段垂直平分线上的点到线段两端点的距离相等）
又∵ PA = 6cm
∴ PB = **6cm**

### 13.5.2 垂直平分线的判定

#### 📖 判定定理

**定理**：到一条线段两个端点距离相等的点，在这条线段的垂直平分线上。

**数学表达**：
```
∵ PA = PB
∴ 点P在AB的垂直平分线上
```

#### 🔄 性质与判定的关系

**性质定理**：垂直平分线上的点到线段两端点距离相等
- 条件：点在垂直平分线上
- 结论：到两端点距离相等

**判定定理**：到线段两端点距离相等的点在垂直平分线上
- 条件：到两端点距离相等
- 结论：点在垂直平分线上

它们互为**逆定理**！

## 13.6 最短路径问题

### 13.6.1 问题引入

#### 🤔 将军饮马问题

**故事**：一位将军从A地出发，要到河边l饮马，然后去B地，问：怎样走路径最短？

**分析**：
1. 这是一个最优化问题
2. 需要在河边l上找一点P
3. 使得 AP + PB 最小

### 13.6.2 基本模型

#### 📐 模型1：两点在直线同侧

**问题**：A、B两地在直线l的同侧，在直线l上求一点P，使PA + PB最小。

**解法**：
1. 作点A关于直线l的对称点A'
2. 连接A'B，与直线l交于点P
3. 点P即为所求

**原理**：
```
∵ A'是A关于直线l的对称点
∴ PA = PA'
∴ PA + PB = PA' + PB = A'B

在直线l上任取一点P'（不同于点P）
连接P'A、P'B

∵ PA' = PA，P'A' = P'A
∴ P'A + P'B = P'A' + P'B

根据两点之间线段最短：
A'B < P'A' + P'B
∴ PA + PB < P'A + P'B
∴ PA + PB最小
```

#### 📐 模型2：两点在直线异侧

**问题**：A、B两地在直线l的异侧，在直线l上求一点P，使PA + PB最小。

**解法**：
连接AB，与直线l交于点P，点P即为所求。

**原理**：根据两点之间线段最短，AB就是最短路径。

#### 📐 模型3：最大差问题

**问题**：A、B两地在直线l的异侧，在直线l上求一点P，使|PA - PB|最大。

**解法**：
1. 连接AB，与直线l交于点P
2. 点P即为所求

**原理**：根据三角形两边之差小于第三边，当点P在AB上时，|PA - PB| = AB最大。

### 13.6.3 变式问题

#### 🔄 变式1：造桥选址问题

**问题**：A、B两地在河流l的两侧，河宽为d，要在河上架一座桥（桥垂直于河岸），使从A到B的路径最短。

**解法**：
1. 过点A作l的平行线l'，使l'与l的距离为d
2. 过点B作l的垂线，交l'于点B'
3. 连接AB'，与l交于点P
4. 在点P处架桥

#### 🔄 变式2：多个折点问题

**问题**：从A地到B地，必须经过两条直线l₁和l₂，求最短路径。

**解法**：
1. 作点A关于l₁的对称点A'
2. 作A'关于l₂的对称点A''
3. 连接A''B，与l₂交于点P₂
4. 连接A'P₂，与l₁交于点P₁
5. 路径：A → P₁ → P₂ → B

#### 📝 典型例题

**例13**：如图，A、B两地在直线l的同侧，在直线l上求一点P，使PA + PB最小。

**作法**：
1. 作点A关于直线l的对称点A'
2. 连接A'B，与直线l交于点P
3. 点P即为所求

**证明**：
在直线l上任取一点P'（不同于点P）
连接P'A、P'B、PA、PB

∵ A'是A关于直线l的对称点
∴ PA = PA'，P'A = P'A'
∴ PA + PB = PA' + PB = A'B
   P'A + P'B = P'A' + P'B

根据两点之间线段最短：
A'B < P'A' + P'B
∴ PA + PB < P'A + P'B
∴ PA + PB最小

## 13.7 常见错误与注意事项

### ⚠️ 错误归纳

**错误类型1：混淆轴对称图形和两个图形成轴对称**
- 错误：认为这两个概念是一样的
- 正确：轴对称图形是一个图形自身的对称，两个图形成轴对称是两个图形之间的对称关系

**错误类型2：错误判断轴对称图形**
- 错误：认为所有三角形都是轴对称图形
- 正确：只有等腰三角形（包括等边三角形）才是轴对称图形

**错误类型3：三线合一的应用错误**
- 错误：认为任何三角形都有三线合一性质
- 正确：只有等腰三角形才有三线合一性质

**错误类型4：等边三角形的判定错误**
- 错误：认为有一个角是60°的三角形就是等边三角形
- 正确：必须同时是等腰三角形，有一个角是60°，才是等边三角形

**错误类型5：最短路径问题建模错误**
- 错误：不分类讨论点在同侧还是异侧
- 正确：先判断点的位置关系，再选择合适的方法
