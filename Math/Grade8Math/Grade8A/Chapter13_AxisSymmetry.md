# 第十三章 轴对称

## 📚 知识要点

### 13.1 轴对称

#### 13.1.1 轴对称图形

**定义**：如果一个图形沿一条直线折叠，直线两旁的部分能够互相重合，这个图形就叫做轴对称图形

**对称轴**：这条直线就是它的对称轴

**常见轴对称图形**：
- **线段**：对称轴是线段的垂直平分线（有1条对称轴）
- **角**：对称轴是角平分线所在的直线（有1条对称轴）
- **等腰三角形**：对称轴是顶角平分线所在的直线（有1条对称轴）
- **等边三角形**：对称轴是每条边的高、中线和角平分线所在的直线（有3条对称轴）
- **矩形**：对称轴是对边中点连线所在的直线（有2条对称轴）
- **菱形**：对称轴是对角线所在的直线（有2条对称轴）
- **正方形**：对称轴是对边中点连线和对角线所在的直线（有4条对称轴）
- **圆**：对称轴是任意一条直径所在的直线（有无数条对称轴）

#### 13.1.2 两个图形关于某条直线对称

**定义**：把一个图形沿某一条直线折叠，如果它能够与另一个图形重合，那么就说这两个图形关于这条直线对称

**性质**：
1. 关于某条直线对称的两个图形是全等图形
2. 对称轴是对称点连线的垂直平分线
3. 对称轴垂直平分连接对称点的线段
4. 两个图形关于某条直线对称，如果它们对应线段或延长线相交，那么交点在对称轴上

#### 13.1.3 轴对称的性质

**性质**：
1. 如果两个图形关于某条直线对称，那么对称轴是任何一对对应点所连线段的垂直平分线
2. 轴对称图形的对称轴，是任何一对对应点所连线段的垂直平分线

### 13.2 画轴对称图形

#### 13.2.1 画对称点的方法

**步骤**：
1. 过已知点A作对称轴l的垂线，垂足为O
2. 在垂线上截取OA' = OA
3. 点A'就是点A关于直线l的对称点

**几何语言**：
```
已知：点A和直线l
求作：点A关于直线l的对称点A'

作法：
1. 过点A作直线l的垂线，垂足为O
2. 在AO的延长线上截取OA' = OA
3. 点A'即为所求
```

#### 13.2.2 画轴对称图形

**步骤**：
1. 找出图形的关键点（顶点、端点等）
2. 分别作出这些关键点关于对称轴的对称点
3. 按原图形的连接方式顺次连接各对称点
4. 得到的图形就是原图形的轴对称图形

**例子**：画△ABC关于直线l的对称图形
1. 作点A关于直线l的对称点A'
2. 作点B关于直线l的对称点B'
3. 作点C关于直线l的对称点C'
4. 连接A'B'、B'C'、C'A'
5. △A'B'C'就是△ABC关于直线l的对称图形

### 13.3 等腰三角形

#### 13.3.1 等腰三角形的性质

**性质1**：等腰三角形的两个底角相等（简称"等边对等角"）

**数学表达式**：
```
∵ AB = AC
∴ ∠B = ∠C
```

**性质2**：等腰三角形的顶角平分线、底边上的中线、底边上的高相互重合（简称"三线合一"）

**说明**：在等腰三角形ABC中，AB = AC
- AD是顶角平分线
- AD是底边BC的中线
- AD是底边BC的高
这三条线重合为一条线

**性质3**：等腰三角形是轴对称图形，对称轴是顶角平分线所在的直线

#### 13.3.2 等腰三角形的判定

**判定定理**：如果一个三角形有两个角相等，那么这两个角所对的边也相等（简称"等角对等边"）

**数学表达式**：
```
∵ ∠B = ∠C
∴ AB = AC
```

**应用**：判定一个三角形是等腰三角形

#### 13.3.3 等边三角形

**定义**：三条边都相等的三角形叫做等边三角形（也叫正三角形）

**性质**：
1. 等边三角形的三个内角都相等，都等于60°
2. 等边三角形是特殊的等腰三角形
3. 等边三角形有三条对称轴

**判定**：
1. 三条边都相等的三角形是等边三角形
2. 三个角都相等的三角形是等边三角形
3. 有一个角是60°的等腰三角形是等边三角形

### 13.4 最短路径问题

#### 13.4.1 线段的垂直平分线的性质

**性质**：线段垂直平分线上的点到这条线段两个端点的距离相等

**数学表达式**：
```
∵ 直线l是线段AB的垂直平分线，点P在l上
∴ PA = PB
```

**判定**：到一条线段两个端点距离相等的点，在这条线段的垂直平分线上

#### 13.4.2 最短路径问题

**问题1**：在直线l上求一点P，使PA + PB最小（A、B在直线l的同侧）

**解法**：
1. 作点A关于直线l的对称点A'
2. 连接A'B，与直线l交于点P
3. 点P即为所求

**原理**：
∵ A'是A关于直线l的对称点
∴ PA = PA'
∴ PA + PB = PA' + PB = A'B
根据两点之间线段最短，A'B最短，此时PA + PB最小

**问题2**：在直线l上求一点P，使|PA - PB|最大（A、B在直线l的异侧）

**解法**：
1. 连接AB，与直线l交于点P
2. 点P即为所求

**问题3**：将军饮马问题
将军从A地出发，到河边l饮马，再到B地，怎样走路径最短？

**解法**：
1. 作点A关于直线l的对称点A'
2. 连接A'B，与直线l交于点P
3. 路径：A → P → B

## 📖 典型例题

### 例题1：轴对称图形的判断

**题目**：下列图形中，是轴对称图形的是（  ）
A. 平行四边形
B. 三角形
C. 线段
D. 梯形

**解答**：
**答案：C**

解析：
- 平行四边形不一定是轴对称图形
- 三角形不一定是轴对称图形（等腰三角形才是）
- 线段是轴对称图形（对称轴是垂直平分线）
- 梯形不一定是轴对称图形（等腰梯形才是）

### 例题3：等腰三角形的性质

**题目**：如图，在△ABC中，AB = AC，∠A = 40°，求∠B和∠C的度数。

**解答**：
∵ AB = AC
∴ ∠B = ∠C（等边对等角）

又∵ ∠A + ∠B + ∠C = 180°
40° + 2∠B = 180°
2∠B = 140°
∠B = 70°

∴ ∠B = ∠C = **70°**

### 例题4：等腰三角形的判定

**题目**：如图，在△ABC中，∠B = 70°，∠C = 70°，求证：AB = AC。

**证明**：
∵ ∠B = ∠C = 70°
∴ AB = AC（等角对等边）

---

### 例题5：三线合一的应用

**题目**：如图，在△ABC中，AB = AC，AD⊥BC于D，BD = 5cm，求BC的长。

**解答**：
∵ AB = AC，AD⊥BC
∴ BD = CD（等腰三角形"三线合一"）
又∵ BD = 5cm
∴ CD = 5cm
∴ BC = BD + CD = 5 + 5 = **10cm**

---

### 例题6：线段垂直平分线的性质

**题目**：如图，直线l是线段AB的垂直平分线，点P在l上，PA = 6cm，求PB的长。

**解答**：
∵ 直线l是线段AB的垂直平分线，点P在l上
∴ PB = PA（线段垂直平分线上的点到线段两端点的距离相等）
又∵ PA = 6cm
∴ PB = **6cm**

### 例题7：最短路径问题

**题目**：如图，A、B两地在直线l的同侧，在直线l上求一点P，使PA + PB最小。

**作法**：
1. 作点A关于直线l的对称点A'
2. 连接A'B，与直线l交于点P
3. 点P即为所求

**证明**：
在直线l上任取一点P'（不同于点P）
连接P'A、P'B、PA、PB

∵ A'是A关于直线l的对称点
∴ PA = PA'，P'A = P'A'
∴ PA + PB = PA' + PB = A'B
   P'A + P'B = P'A' + P'B

根据两点之间线段最短：
A'B < P'A' + P'B
∴ PA + PB < P'A + P'B
∴ PA + PB最小

### 例题8：等边三角形的性质

**题目**：如图，△ABC是等边三角形，D是BC的中点，连接AD，求∠BAD的度数。

**解答**：
∵ △ABC是等边三角形
∴ AB = BC = CA，∠B = ∠C = ∠BAC = 60°

又∵ D是BC的中点，AB = AC
∴ AD⊥BC（等腰三角形"三线合一"）
∴ ∠ADB = 90°

在Rt△ABD中：
∠BAD = 90° - ∠B = 90° - 60° = **30°**

---

### 例题9：综合应用

**题目**：如图，在△ABC中，AB = AC，D是BC上一点，DE⊥AB于E，DF⊥AC于F
求证：DE = DF

**证明**：
∵ AB = AC
∴ ∠B = ∠C（等边对等角）

∵ DE⊥AB，DF⊥AC
∴ ∠DEB = ∠DFC = 90°

在△DEB和△DFC中
∠DEB = ∠DFC（已证）
∠B = ∠C（已证）
∴ ∠BDE = ∠CDF（三角形内角和定理）

在△DEB和△DFC中
∠BDE = ∠CDF（已证）
∠B = ∠C（已证）
∴ △DEB ≅ △DFC（AAS）
∴ DE = DF

## 📚 学习建议

1. **区分概念**：轴对称图形和两个图形成轴对称是两个不同概念
2. **掌握画法**：熟练掌握画对称点和对称图形的方法
3. **记忆性质**：等腰三角形的性质（等边对等角、三线合一）要牢记
4. **理解对称**：利用对称思想可以简化很多几何问题
5. **最值问题**：最短路径问题本质是利用对称将折线段转化为直线段
6. **注重证明**：八年级几何证明要求更严格，要养成规范书写的习惯
