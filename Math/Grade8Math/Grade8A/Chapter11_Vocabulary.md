## Key Formulas | 关键公式

### Triangle Inequality | 三角形三边关系
```
For sides a, b, c:
a + b > c
a + c > b
b + c > a

Third side range: |a - b| < c < a + b
```

### Interior Angle Sum | 内角和公式
```
Triangle: ∠A + ∠B + ∠C = 180°
n-gon: (n - 2) × 180°
```

### Exterior Angle | 外角公式
```
Exterior angle = sum of two non-adjacent interior angles
Exterior angle sum of any polygon = 360°
```

### Polygon Diagonals | 多边形对角线
```
From one vertex: n - 3
Total diagonals: n(n - 3) / 2
```

### Regular Polygon | 正多边形
```
Each interior angle: (n - 2) × 180° / n
Each exterior angle: 360° / n
```
