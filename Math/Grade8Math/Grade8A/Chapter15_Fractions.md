# 第十五章 分式

## 本章导引

在小学阶段，我们已经学习了分数，知道了如何进行分数的加减乘除运算。但是，在实际问题中，我们经常遇到更加复杂的情况：

- 🏃‍♂️ 甲、乙两人同时从两地出发，相向而行，甲的速度是乙的1.5倍，相遇时甲比乙多走了2km，求两地距离
- 🍰 把一个蛋糕平均分给x个人，每人分到1/5个，后来又来了y个人，重新分配后每人分到1/8个，求原来有多少人
- 🏭 一个工厂生产一批零件，原计划每天生产x个，实际每天多生产20%，结果提前5天完成，求原计划需要多少天

这些问题都需要用到一种新的数学工具——**分式**。

**为什么需要学习分式？**
1. **从分数到分式**：分式是分数的自然推广，让分母也从具体的数变成含字母的式子
2. **解决实际问题**：很多实际问题无法用整数方程解决，必须用分式方程
3. **代数运算的基础**：分式运算是学习后续内容（如函数、不等式）的基础
4. **培养抽象思维**：从具体到抽象，从数字到字母，思维能力的重要提升

**学习目标**：
- 理解分式的概念，掌握分式有意义的条件
- 掌握分式的基本性质，能熟练进行约分和通分
- 掌握分式的四则运算法则，能进行混合运算
- 理解分式方程的解法，掌握增根的概念
- 能列出分式方程解决实际问题
- 培养严谨的思维习惯和验根意识

## 15.1 分式的概念

### 15.1.1 从分数到分式

#### 🌟 生活实例引入

**实例1**：分蛋糕问题
- 把1个蛋糕平均分给5个人，每人分到多少？
  答案：1/5（这是**分数**）

- 把1个蛋糕平均分给x个人，每人分到多少？
  答案：1/x（这就是**分式**！）

**实例2**：行程问题
- 汽车以60km/h的速度行驶，120km需要多长时间？
  答案：120÷60 = 2小时（这是整数运算）

- 汽车以x km/h的速度行驶，120km需要多长时间？
  答案：120/x 小时（这就是**分式**！）

#### 📖 什么是分式？

**定义**：如果A、B表示两个整式，并且B中含有字母，那么式子A/B叫做分式

**理解要点**：
1. **形式**：A/B，分子是整式，分母是含字母的整式
2. **本质**：分式是分数的推广
3. **关键**：分母必须含有字母

#### 🔄 分式与分数的关系

**相同点**：
- 都是"商"的形式
- 都有分子和分母
- 分母都不能为0

**不同点**：

| 对比项 | 分数 | 分式 |
|--------|------|------|
| **分母** | 具体的数（不含字母） | 含有字母的整式 |
| **值** | 可以算出一个具体的数 | 一般情况下是一个代数式 |
| **举例** | 3/5, 7/2 | x/(x+1), (a+b)/c |

#### 📊 分式的识别

**例1（基础题）**：下列各式中，哪些是分式？
（1）3/x
（2）x/3
（3）(x+1)/(x-1)
（4）5/7
（5）(x²+1)/2
（6）1/(x+y)

**解答**：

**分析思路**：看分母是否含有字母

（1）3/x
分母是x，含字母 → **是分式** ✓

（2）x/3
分母是3，不含字母 → 不是分式（这是整式）

（3）(x+1)/(x-1)
分母是x-1，含字母 → **是分式** ✓

（4）5/7
分母是7，不含字母 → 不是分式（这是分数）

（5）(x²+1)/2
分母是2，不含字母 → 不是分式（这是整式）

（6）1/(x+y)
分母是x+y，含字母 → **是分式** ✓

**答案**：是分式的有 **（1）（3）（6）**

---

**例2（辨析题）**：判断下列说法是否正确
（1）分母中含有字母的式子一定是分式
（2）分式和分数没有任何关系
（3）x/(x+1) 是分式，因为分母含有字母

**解答**：

（1）❌ **错误**
反例：如果分母是字母但分子也是字母，且整个式子可以约分化简为整式，就不一定是分式
比如：x/x = 1（虽然原式分母含字母，但约分后是整数1）

（2）❌ **错误**
分式是分数的推广，两者有密切的关系，运算规则也类似

（3）✅ **正确**
分母x+1含有字母，符合分式的定义

### 15.1.2 分式有意义的条件

#### 🌟 为什么要讨论"有意义"？

在小学学分数时，我们从未担心过分母为0的问题，因为分母都是具体的、非零的数。

但是分式就不同了！分母含有字母，字母可以取不同的值，有些值会让分母等于0，这时分式就没有意义了。

**为什么分母不能为0？**
回顾除法的定义：除法是"已知积和其中一个因数，求另一个因数的运算"
如果除数（分母）为0：
- 当被除数（分子）不为0时，商不存在（因为任何数×0≠非零数）
- 当被除数（分子）也为0时，商不唯一（因为任何数×0=0）

所以：**分母不能为0！**

#### 📖 分式有意义的条件

**结论**：当分母不等于0时，分式有意义

**一般步骤**：
1. 找出分母
2. 令分母 ≠ 0
3. 求出字母的取值范围

#### 📝 典型例题

**例3（基础题）**：当 x 取何值时，下列分式有意义？
（1）1/x
（2）(x+1)/(x-1)
（3）2/(x²-1)
（4）x/(x+2)

**解答**：

（1）1/x

**步骤**：
1. 分母是：x
2. 令分母 ≠ 0：x ≠ 0

**答案**：当 x ≠ 0 时，分式有意义

---

（2）(x+1)/(x-1)

**步骤**：
1. 分母是：x-1
2. 令分母 ≠ 0：x-1 ≠ 0
3. 解得：x ≠ 1

**答案**：当 x ≠ 1 时，分式有意义

---

（3）2/(x²-1)

**步骤**：
1. 分母是：x²-1
2. 令分母 ≠ 0：x²-1 ≠ 0
3. 解方程：x²-1 = 0 → x² = 1 → x = ±1
4. 所以：x ≠ 1 且 x ≠ -1

**答案**：当 x ≠ ±1 时，分式有意义

**注意**：x²-1 = 0 有两个解，都要排除！

---

（4）x/(x+2)

**步骤**：
1. 分母是：x+2
2. 令分母 ≠ 0：x+2 ≠ 0
3. 解得：x ≠ -2

**答案**：当 x ≠ -2 时，分式有意义

---

**例4（进阶题）**：当 x 取何值时，下列分式有意义？
（1）1/(x-2) + 1/(x+3)
（2）(x+1)/[(x-1)(x+2)]
（3）5/|x-1|

**解答**：

（1）1/(x-2) + 1/(x+3)

**分析**：这是两个分式相加，两个分式都要有意义

分式1有意义：x-2 ≠ 0 → x ≠ 2
分式2有意义：x+3 ≠ 0 → x ≠ -3

**答案**：当 x ≠ 2 且 x ≠ -3 时，分式有意义

---

（2）(x+1)/[(x-1)(x+2)]

**步骤**：
1. 分母是：(x-1)(x+2)
2. 令分母 ≠ 0：(x-1)(x+2) ≠ 0
3. 解得：x-1 ≠ 0 且 x+2 ≠ 0
4. 即：x ≠ 1 且 x ≠ -2

**答案**：当 x ≠ 1 且 x ≠ -2 时，分式有意义

---

（3）5/|x-1|

**步骤**：
1. 分母是：|x-1|
2. 令分母 ≠ 0：|x-1| ≠ 0
3. 绝对值等于0时为0，所以：x-1 ≠ 0
4. 即：x ≠ 1

**答案**：当 x ≠ 1 时，分式有意义

---

### 15.1.3 分式的值为0

#### 🌟 什么时候分式的值为0？

**思考**：什么时候分数为0？
- 0/5 = 0 ✓（分子为0，分母不为0）
- 5/0 不存在 ✗（分母为0，无意义）
- 0/0 不确定 ✗（分母为0，无意义）

**结论**：**分子为0且分母不为0时，分式的值为0**

#### ⚠️ 重要提醒

**两个条件缺一不可！**
1. 分子 = 0
2. 分母 ≠ 0

#### 📝 典型例题

**例5（基础题）**：当 x 取何值时，下列分式的值为0？
（1）(x-1)/x
（2）(x+2)/(x-3)
（3）(x²-4)/(x+2)
（4）(x-2)/(x²-4)

**解答**：

（1）(x-1)/x

**步骤**：
1. 分子 = 0：x-1 = 0 → x = 1
2. 检验分母：当 x = 1 时，分母 x = 1 ≠ 0 ✓
3. 两个条件都满足

**答案**：当 x = 1 时，分式的值为0

---

（2）(x+2)/(x-3)

**步骤**：
1. 分子 = 0：x+2 = 0 → x = -2
2. 检验分母：当 x = -2 时，分母 x-3 = -5 ≠ 0 ✓
3. 两个条件都满足

**答案**：当 x = -2 时，分式的值为0

---

（3）(x²-4)/(x+2)

**步骤**：
1. 分子 = 0：x²-4 = 0 → x² = 4 → x = ±2
2. 检验分母：
   - 当 x = 2 时，分母 x+2 = 4 ≠ 0 ✓
   - 当 x = -2 时，分母 x+2 = 0 ✗
3. x = -2 不满足条件

**答案**：当 x = 2 时，分式的值为0

---

（4）(x-2)/(x²-4)

**步骤**：
1. 分子 = 0：x-2 = 0 → x = 2
2. 检验分母：当 x = 2 时，分母 x²-4 = 0 ✗
3. 不满足条件

**答案**：**不存在**使分式的值为0的x值

**注意**：如果让分子为0的值恰好也让分母为0，那么分式的值不存在（无意义），更不可能为0

---

**例6（易错题）**：判断下列说法是否正确
（1）分子为0时，分式的值一定为0
（2）分式的值为0时，分子一定为0
（3）当 x = 0 时，分式 x/x 的值为0

**解答**：

（1）❌ **错误**
正确说法：分子为0且分母不为0时，分式的值为0
反例：当 x = 0 时，0/x 无意义，更谈不上值为0

（2）✅ **正确**
分式的值为0 → 分子÷分母 = 0 → 分子必须为0（且分母不为0）

（3）❌ **错误**
当 x = 0 时，分式 x/x = 0/0，这是无意义的，不能说值为0
实际上，x/x = 1（当 x ≠ 0 时）

---

**例7（综合题）**：已知分式 (x²-4)/(x+2)
（1）当 x 取何值时，分式有意义？
（2）当 x 取何值时，分式的值为0？

**解答**：

（1）分式有意义
分母 x+2 ≠ 0 → x ≠ -2

**答案**：当 x ≠ -2 时，分式有意义

---

（2）分式的值为0
分子 x²-4 = 0 → x² = 4 → x = ±2

检验分母：
- 当 x = 2 时，分母 x+2 = 4 ≠ 0 ✓
- 当 x = -2 时，分母 x+2 = 0 ✗

**答案**：当 x = 2 时，分式的值为0

## 15.2 分式的基本性质

### 15.2.1 分式的基本性质

#### 🌟 从分数的基本性质说起

**回顾分数的基本性质**：
```
分数的分子和分母同时乘（或除以）同一个不等于0的数，
分数的值不变
```

**例子**：
```
1/2 = (1×3)/(2×3) = 3/6 ✓
1/2 = (1÷2)/(2÷2) = 1/2 ✓
```

#### 📖 分式的基本性质

**性质**：分式的分子与分母同时乘（或除以）同一个不等于0的整式，分式的值不变

**公式表示**：
```
A/B = (A×M)/(B×M)  （B≠0，M≠0）
A/B = (A÷M)/(B÷M)  （B≠0，M≠0）
```

**关键理解**：
1. **同时**：分子和分母都要乘（或除以）
2. **同一个**：必须是相同的整式M
3. **不等于0**：M不能为0

#### 🔍 为什么M不能为0？

**反证法**：
如果 M = 0，那么：
- A/B = (A×0)/(B×0) = 0/0（无意义）
- A/B = (A÷0)/(B÷0)（除数不能为0）

所以：**M ≠ 0**

#### 📝 典型例题

**例8（基础题）**：填空
（1）(x+1)/(x-1) = [(x+1)(x+2)]/[______]
（2）(x²-1)/x = (x-1)/[______]
（3）a/(b²) = (ab)/[______]

**解答**：

（1）根据分式的基本性质，分子乘了(x+2)，分母也要乘(x+2)

(x+1)/(x-1) = [(x+1)(x+2)]/[(x-1)(x+2)]

答案：**(x-1)(x+2)**

---

（2）分子从(x²-1)变成了(x-1)，这是因式分解

(x²-1) = (x+1)(x-1)

所以分子除以了(x+1)，分母也要除以(x+1)

(x²-1)/x = [(x+1)(x-1)]/x = (x-1)/[x÷(x+1)] = (x-1)/[x/(x+1)]

答案：**x/(x+1)**

---

（3）分子从a变成了ab，这是乘了b

a/(b²) = (a×b)/(b²×b) = (ab)/(b³)

答案：**b³**

---

**例9（应用题）**：不改变分式的值，使下列分式的分子与分母都不含"-"号
（1）-2x/3y
（2）(a-b)/[-(c+d)]
（3）(-x)²/(-y)³

**解答**：

（1）-2x/3y

方法：分子分母同时乘-1

-2x/3y = (-2x×(-1))/(3y×(-1)) = 2x/(-3y)

或者把负号提到分式前面：
-2x/3y = -(2x/3y)

答案：**2x/(-3y)** 或 **-(2x/3y)**

---

（2）(a-b)/[-(c+d)]

方法：分子分母同时乘-1

(a-b)/[-(c+d)] = [(a-b)×(-1)]/[-(c+d)×(-1)] = (b-a)/(c+d)

或者：
(a-b)/[-(c+d)] = -(a-b)/(c+d) = (b-a)/(c+d)

答案：**(b-a)/(c+d)**

---

（3）(-x)²/(-y)³

先化简：
(-x)² = x²
(-y)³ = -y³

原式 = x²/(-y³)

分子分母同时乘-1：
= [x²×(-1)]/[(-y³)×(-1)] = (-x²)/y³

答案：**(-x²)/y³**

---

### 15.2.2 分式的约分

#### 🌟 从分数的约分说起

**回顾分数的约分**：
```
6/8 = (6÷2)/(8÷2) = 3/4
12/18 = (12÷6)/(18÷6) = 2/3
```

**约分的依据**：分数的基本性质

**约分的目的**：把分数化成最简分数

#### 📖 分式的约分

**定义**：根据分式的基本性质，把一个分式的分子和分母的公因式约去，这种变形叫做分式的约分

**最简分式**：分子和分母没有公因式的分式叫做最简分式

#### 🔍 如何约分？

**情况一：分子和分母都是单项式**

**步骤**：
1. 找出系数的最大公约数
2. 找出相同字母
3. 约去相同字母的最低次幂

**例子**：约分 6x²y / 9xy²

**分析**：
- 系数：6 和 9，最大公约数是 3
- 字母x：x² 和 x，约去 x（最低次幂）
- 字母y：y 和 y²，约去 y（最低次幂）
- 公因式：3xy

6x²y / 9xy²
= (6x²y ÷ 3xy) / (9xy² ÷ 3xy)
= 2x / 3y

---

**情况二：分子和分母都是多项式**

**步骤**：
1. 分子和分母分别因式分解
2. 找出公因式
3. 约去公因式

**例子**：约分 (x²-1) / (x+1)

**分析**：
- 分子因式分解：x²-1 = (x+1)(x-1)
- 分母：x+1
- 公因式：x+1

(x²-1) / (x+1)
= [(x+1)(x-1)] / (x+1)
= x-1

#### ⚠️ 常见错误

**错误1**：约分不彻底
```
❌ 错误：(x²-1)/(x²+x) = (x-1)/x
✓ 正确：先因式分解，(x²-1)/(x²+x) = (x+1)(x-1)/[x(x+1)] = (x-1)/x
```

**错误2**：符号错误
```
❌ 错误：(-x)²/(-y)³ = x²/y³
✓ 正确：(-x)²/(-y)³ = x²/(-y³) = -x²/y³
```

**错误3**：忘记条件
```
❌ 错误：约分时忽略M≠0的条件
✓ 正确：约分时要注意M≠0
```

#### 📝 典型例题

**例10（基础题）**：约分下列各式
（1）6x²y / 9xy²
（2）(x²-1) / (x+1)
（3）(x²-4x+4) / (x-2)
（4）(a²-b²) / (a+b)²

**解答**：

（1）6x²y / 9xy²

**找公因式**：
- 系数：6 和 9 → 最大公约数是 3
- 字母x：x² 和 x → 最低次幂是 x
- 字母y：y 和 y² → 最低次幂是 y
- 公因式：3xy

6x²y / 9xy²
= (6x²y ÷ 3xy) / (9xy² ÷ 3xy)
= **2x / 3y**

---

（2）(x²-1) / (x+1)

**因式分解**：
分子：x²-1 = (x+1)(x-1)
分母：x+1

**约分**：
= [(x+1)(x-1)] / (x+1)
= **x-1**

---

（3）(x²-4x+4) / (x-2)

**因式分解**：
分子：x²-4x+4 = (x-2)²
分母：x-2

**约分**：
= (x-2)² / (x-2)
= **x-2**

---

（4）(a²-b²) / (a+b)²

**因式分解**：
分子：a²-b² = (a+b)(a-b)
分母：(a+b)² = (a+b)(a+b)

**约分**：
= [(a+b)(a-b)] / [(a+b)(a+b)]
= **(a-b)/(a+b)**

---

**例11（进阶题）**：约分下列各式
（1）(x²-y²) / (x²+2xy+y²)
（2）(x²-3x+2) / (x²-4)
（3）(a³-1) / (a²-1)

**解答**：

（1）(x²-y²) / (x²+2xy+y²)

**因式分解**：
分子：x²-y² = (x+y)(x-y)
分母：x²+2xy+y² = (x+y)²

**约分**：
= [(x+y)(x-y)] / (x+y)²
= **(x-y)/(x+y)**

---

（2）(x²-3x+2) / (x²-4)

**因式分解**：
分子：x²-3x+2 = (x-1)(x-2)
分母：x²-4 = (x+2)(x-2)

**约分**：
= [(x-1)(x-2)] / [(x+2)(x-2)]
= **(x-1)/(x+2)**

---

（3）(a³-1) / (a²-1)

**因式分解**：
分子：a³-1 = (a-1)(a²+a+1)
分母：a²-1 = (a-1)(a+1)

**约分**：
= [(a-1)(a²+a+1)] / [(a-1)(a+1)]
= **(a²+a+1)/(a+1)**

**注意**：a²+a+1 不能再因式分解了

---

### 15.2.3 分式的通分

#### 🌟 从分数的通分说起

**回顾分数的通分**：
```
1/2 + 1/3 = 3/6 + 2/6 = 5/6
1/4 + 1/6 = 3/12 + 2/12 = 5/12
```

**通分的依据**：分数的基本性质

**通分的目的**：把异分母的分数化成同分母的分数

#### 📖 分式的通分

**定义**：根据分式的基本性质，把几个异分母的分式化成与原来的分式相等的同分母的分式，叫做分式的通分

**最简公分母**：各分母所有因式的最高次幂的积

#### 🔍 如何找最简公分母？

**步骤**：
1. **取各分母系数的最小公倍数**
2. **取各分母出现的字母（或多项式）**
3. **取各字母（或多项式）的最高次幂**
4. **相乘得到最简公分母**

#### 📝 典型例题

**例12（基础题）**：通分下列各式
（1）1/x 和 1/y
（2）2/(x+1) 和 1/x
（3）a/(x-2) 和 b/(x+2)

**解答**：

（1）1/x 和 1/y

**找最简公分母**：
- 分母分别是：x 和 y
- 最简公分母：xy

**通分**：
1/x = (1×y)/(x×y) = **y/(xy)**
1/y = (1×x)/(y×x) = **x/(xy)**

---

（2）2/(x+1) 和 1/x

**找最简公分母**：
- 分母分别是：x+1 和 x
- 最简公分母：x(x+1)

**通分**：
2/(x+1) = [2×x]/[(x+1)×x] = **2x/[x(x+1)]**
1/x = [1×(x+1)]/[x×(x+1)] = **(x+1)/[x(x+1)]**

---

（3）a/(x-2) 和 b/(x+2)

**找最简公分母**：
- 分母分别是：x-2 和 x+2
- 最简公分母：(x-2)(x+2)

**通分**：
a/(x-2) = [a(x+2)]/[(x-2)(x+2)] = **a(x+2)/[(x-2)(x+2)]**
b/(x+2) = [b(x-2)]/[(x+2)(x-2)] = **b(x-2)/[(x-2)(x+2)]**

---

**例13（进阶题）**：通分下列各式
（1）1/(x²-1) 和 1/(x+1)
（2）x/(x²-4) 和 2/(x-2)
（3）1/(a-b) 和 1/(b-a)

**解答**：

（1）1/(x²-1) 和 1/(x+1)

**找最简公分母**：
- 分母1：x²-1 = (x+1)(x-1)
- 分母2：x+1
- 最简公分母：(x+1)(x-1) = x²-1

**通分**：
1/(x²-1) = **1/(x²-1)**（已经是这个分母）
1/(x+1) = [(x-1)]/[(x+1)(x-1)] = **(x-1)/(x²-1)**

---

（2）x/(x²-4) 和 2/(x-2)

**找最简公分母**：
- 分母1：x²-4 = (x+2)(x-2)
- 分母2：x-2
- 最简公分母：(x+2)(x-2) = x²-4

**通分**：
x/(x²-4) = **x/(x²-4)**（已经是这个分母）
2/(x-2) = [2(x+2)]/[(x-2)(x+2)] = **2(x+2)/(x²-4)**

---

（3）1/(a-b) 和 1/(b-a)

**注意**：a-b = -(b-a)

**找最简公分母**：
- 分母1：a-b
- 分母2：b-a = -(a-b)
- 最简公分母：a-b（或b-a）

**通分**：
1/(a-b) = **1/(a-b)**
1/(b-a) = -1/(a-b) = **-1/(a-b)**

或者：
1/(a-b) = **-1/(b-a)**
1/(b-a) = **1/(b-a)**

---

## 15.3 分式的运算

### 15.3.1 分式的乘法

#### 📖 分式的乘法法则

**法则**：分式乘分式，用分子的积作为积的分子，分母的积作为积的分母

**公式表示**：
```
A/B × C/D = (A×C) / (B×D)
```

**直观理解**：
```
┌─────┐   ┌─────┐
│  A  │ × │  C  │
│ ─── │   │ ─── │
│  B  │   │  D  │
└─────┘   └─────┘
    ┌───────┐
    │  A×C  │
    │ ───── │
    │  B×D  │
    └───────┘
```

#### 📝 典型例题

**例14（基础题）**：计算下列各式
（1）(2x/y) × (3y/x)
（2）(a/b) × (b/c)
（3）[(x+1)/x] × [x/(x+1)]

**解答**：

（1）(2x/y) × (3y/x)

**方法**：分子乘分子，分母乘分母
= (2x×3y) / (y×x)
= 6xy / xy

**约分**：= **6**

---

（2）(a/b) × (b/c)

**方法**：分子乘分子，分母乘分母
= (a×b) / (b×c)

**约分**：= **a/c**

---

（3）[(x+1)/x] × [x/(x+1)]

**方法**：分子乘分子，分母乘分母
= [(x+1)×x] / [x×(x+1)]
= [x(x+1)] / [x(x+1)]

**约分**：= **1**

---

**例15（进阶题）**：计算下列各式
（1）[(x²-1)/x] × [x/(x-1)]
（2）[(a+b)/a] × [a/(a²-b²)]
（3）(x²-4x+4)/(x-2) × (x+2)/(x+2)

**解答**：

（1）[(x²-1)/x] × [x/(x-1)]

**步骤**：
1. 先因式分解：x²-1 = (x+1)(x-1)
2. 分子乘分子，分母乘分母
3. 约分

= [(x+1)(x-1)×x] / [x×(x-1)]
= [x(x+1)(x-1)] / [x(x-1)]
= **x+1**

---

（2）[(a+b)/a] × [a/(a²-b²)]

**步骤**：
1. 先因式分解：a²-b² = (a+b)(a-b)
2. 分子乘分子，分母乘分母
3. 约分

= [(a+b)×a] / [a×(a+b)(a-b)]
= [a(a+b)] / [a(a+b)(a-b)]
= **1/(a-b)**

---

（3）(x²-4x+4)/(x-2) × (x+2)/(x+2)

**步骤**：
1. 先因式分解：x²-4x+4 = (x-2)²
2. 约分：(x²-4x+4)/(x-2) = (x-2)²/(x-2) = x-2
3. (x+2)/(x+2) = 1

= (x-2) × 1
= **x-2**

---

### 15.3.2 分式的除法

#### 📖 分式的除法法则

**法则**：分式除以分式，把除式的分子、分母颠倒位置后，与被除式相乘

**公式表示**：
```
A/B ÷ C/D = A/B × D/C = (A×D) / (B×C)
```

**记忆口诀**：**除法变乘法，除号后面倒个个**

#### ⚠️ 重要提醒

**除法转化为乘法后，再按照乘法的规则进行运算**

#### 📝 典型例题

**例16（基础题）**：计算下列各式
（1）(2x/y) ÷ (3y/x)
（2）(a/b) ÷ (b/a)
（3）[x/(x+1)] ÷ [(x-1)/x]

**解答**：

（1）(2x/y) ÷ (3y/x)

**步骤**：除法变乘法，除号后面倒个个
= (2x/y) × (x/3y)
= (2x×x) / (y×3y)
= 2x² / 3y²

**答案**：**2x²/3y²**

---

（2）(a/b) ÷ (b/a)

**步骤**：除法变乘法，除号后面倒个个
= (a/b) × (a/b)
= (a×a) / (b×b)
= a²/b²

**答案**：**a²/b²**

---

（3）[x/(x+1)] ÷ [(x-1)/x]

**步骤**：除法变乘法，除号后面倒个个
= [x/(x+1)] × [x/(x-1)]
= (x×x) / [(x+1)(x-1)]
= x² / (x²-1)

**答案**：**x²/(x²-1)**

---

**例17（进阶题）**：计算下列各式
（1）[(x+1)/(x-1)] ÷ [(x+1)/(x+2)]
（2）(x²-1)/x ÷ (x+1)
（3）1 ÷ (x-1)/x

**解答**：

（1）[(x+1)/(x-1)] ÷ [(x+1)/(x+2)]

**步骤**：除法变乘法
= [(x+1)/(x-1)] × [(x+2)/(x+1)]
= [(x+1)(x+2)] / [(x-1)(x+1)]
= **(x+2)/(x-1)**

---

（2）(x²-1)/x ÷ (x+1)

**步骤**：
1. 把(x+1)看作 (x+1)/1
2. 除法变乘法

= [(x²-1)/x] × [1/(x+1)]
= (x²-1) / [x(x+1)]

**因式分解**：
x²-1 = (x+1)(x-1)

= [(x+1)(x-1)] / [x(x+1)]
= **(x-1)/x**

---

（3）1 ÷ (x-1)/x

**步骤**：
1. 把1看作 1/1
2. 除法变乘法

= (1/1) × [x/(x-1)]
= **x/(x-1)**

---

### 15.3.3 分式的加减法

#### 🌟 从分数的加减法说起

**回顾分数的加减法**：

**同分母**：
```
1/5 + 2/5 = (1+2)/5 = 3/5
3/7 - 2/7 = (3-2)/7 = 1/7
```

**异分母**：
```
1/2 + 1/3 = 3/6 + 2/6 = 5/6
```

#### 📖 同分母分式的加减法

**法则**：同分母分式相加减，分母不变，把分子相加减

**公式表示**：
```
A/C ± B/C = (A±B) / C
```

#### 📖 异分母分式的加减法

**法则**：异分母分式相加减，先通分，变为同分母的分式，再加减

**公式表示**：
```
A/B ± C/D = (AD±BC) / (BD)
```

**注意**：分子相加减时，要加括号！

#### 📝 典型例题

**例18（基础题）**：计算下列各式
（1）1/x + 2/x
（2）1/x + 1/y
（3）(x+2)/(x-1) - x/(x-1)
（4）1/x + 1/(x+1)

**解答**：

（1）1/x + 2/x

**同分母**：分母不变，分子相加
= (1+2)/x
= **3/x**

---

（2）1/x + 1/y

**异分母**：先通分
最简公分母：xy

= (1×y)/(x×y) + (1×x)/(y×x)
= y/(xy) + x/(xy)

**同分母相加**
= (y+x)/(xy)
= **(x+y)/(xy)**

---

（3）(x+2)/(x-1) - x/(x-1)

**同分母**：分母不变，分子相减
= [(x+2)-x] / (x-1)
= **2/(x-1)**

**注意**：分子相减时要加括号！

---

（4）1/x + 1/(x+1)

**异分母**：先通分
最简公分母：x(x+1)

= [1×(x+1)]/[x×(x+1)] + [1×x]/[(x+1)×x]
= (x+1)/[x(x+1)] + x/[x(x+1)]

**同分母相加**
= [(x+1)+x] / [x(x+1)]
= (2x+1) / [x(x+1)]

**答案**：**(2x+1)/[x(x+1)]**

---

**例19（进阶题）**：计算下列各式
（1）a/(a-b) - b/(a-b)
（2）1/(x-1) + 1/(x+1)
（3）x/(x²-1) - 1/(x+1)

**解答**：

（1）a/(a-b) - b/(a-b)

**同分母**：分母不变，分子相减
= (a-b) / (a-b)
= **1**

---

（2）1/(x-1) + 1/(x+1)

**异分母**：先通分
最简公分母：(x-1)(x+1) = x²-1

= [1×(x+1)]/[(x-1)(x+1)] + [1×(x-1)]/[(x+1)(x-1)]
= (x+1)/(x²-1) + (x-1)/(x²-1)

**同分母相加**
= [(x+1)+(x-1)] / (x²-1)
= 2x / (x²-1)

**答案**：**2x/(x²-1)**

---

（3）x/(x²-1) - 1/(x+1)

**异分母**：先通分
最简公分母：x²-1 = (x+1)(x-1)

= x/(x²-1) - [(x-1)]/[(x+1)(x-1)]
= x/(x²-1) - (x-1)/(x²-1)

**同分母相减**
= [x-(x-1)] / (x²-1)
= [x-x+1] / (x²-1)
= **1/(x²-1)**

### 15.3.4 分式的混合运算

#### 📖 运算顺序

**顺序**：
1. **先算乘方**：如果有乘方，先算乘方
2. **再算乘除**：从左到右依次计算
3. **最后算加减**：从左到右依次计算
4. **有括号先算括号**：从小到大，从内到外

#### ⚠️ 注意事项

1. **严格按照运算顺序**
2. **能简化的先简化**：如能约分的先约分
3. **结果要化简**：化成最简分式
4. **注意符号**：特别是负号

#### 📝 典型例题

**例20（基础题）**：计算下列各式
（1）1/x + 2/x × 3/x
（2）(1/x - 1/y) × xy/(y-x)
（3）[(x+1)/(x-1) - 1] ÷ x/(x-1)

**解答**：

（1）1/x + 2/x × 3/x

**步骤**：先算乘法，再算加法
= 1/x + (2×3)/(x×x)
= 1/x + 6/x²
= x/x² + 6/x²
= (x+6)/x²

**答案**：**(x+6)/x²**

---

（2）(1/x - 1/y) × xy/(y-x)

**步骤**：先算括号内，再算乘法
括号内：先通分
= (y-x)/(xy) × xy/(y-x)

乘法：
= [(y-x)×xy] / [(xy)×(y-x)]
= **1**

---

（3）[(x+1)/(x-1) - 1] ÷ x/(x-1)

**步骤**：先算括号内，再算除法

括号内：(x+1)/(x-1) - 1
= (x+1)/(x-1) - (x-1)/(x-1)
= [(x+1)-(x-1)] / (x-1)
= 2/(x-1)

除法：[2/(x-1)] ÷ [x/(x-1)]
= [2/(x-1)] × [(x-1)/x]
= 2/x

**答案**：**2/x**

---

**例21（综合题）**：计算下列各式
（1）[x/(x-1) - 1/(x+1)] × (x²-1)/x
（2）(a/b - b/a) ÷ (a/b + b/a)
（3）[(x+1)/x + x/(x+1)] × x/(x+1)

**解答**：

（1）[x/(x-1) - 1/(x+1)] × (x²-1)/x

**步骤**：先算括号内，再算乘法

**括号内**：
最简公分母：(x-1)(x+1) = x²-1

= [x(x+1)]/(x²-1) - [(x-1)]/(x²-1)
= [x(x+1)-(x-1)] / (x²-1)
= [x²+x-x+1] / (x²-1)
= (x²+1) / (x²-1)

**乘法**：
[(x²+1)/(x²-1)] × [(x²-1)/x]
= (x²+1)/x

**答案**：**(x²+1)/x**

---

（2）(a/b - b/a) ÷ (a/b + b/a)

**步骤**：先算括号内，再算除法

**括号内**：
最简公分母：ab

(a/b - b/a) = (a²-b²)/(ab)
(a/b + b/a) = (a²+b²)/(ab)

**除法**：
[(a²-b²)/(ab)] ÷ [(a²+b²)/(ab)]
= [(a²-b²)/(ab)] × [ab/(a²+b²)]
= **(a²-b²)/(a²+b²)**

---

（3）[(x+1)/x + x/(x+1)] × x/(x+1)

**步骤**：先算括号内，再算乘法

**括号内**：
最简公分母：x(x+1)

= [(x+1)²]/[x(x+1)] + x²/[x(x+1)]
= [(x+1)²+x²] / [x(x+1)]
= [x²+2x+1+x²] / [x(x+1)]
= (2x²+2x+1) / [x(x+1)]

**乘法**：
[(2x²+2x+1)/[x(x+1)]] × [x/(x+1)]
= [(2x²+2x+1)×x] / [x(x+1)(x+1)]
= x(2x²+2x+1) / [x(x+1)²]
= (2x²+2x+1) / [(x+1)²]

**答案**：**(2x²+2x+1)/(x+1)²**

## 15.4 分式方程

### 15.4.1 分式方程的概念

#### 📖 什么是分式方程？

**定义**：分母中含有未知数的方程叫做分式方程

**辨析**：
- **分式方程**：1/x = 2，(x+1)/(x-1) = 3
- **整式方程**：x/2 + 1 = 3，2x + 3 = 5

**关键区别**：分母中是否含有未知数

#### 📝 典型例题

**例22**：下列方程中，哪些是分式方程？
（1）x + 2 = 5
（2）1/x = 2
（3）(x+1)/(x-1) = 3
（4）x/2 + 1 = 3
（5）(x+1)/2 = x

**解答**：

**分析思路**：看分母中是否含有未知数x

（1）x + 2 = 5 → 无分母 → 不是分式方程
（2）1/x = 2 → 分母是x，含未知数 → **是分式方程** ✓
（3）(x+1)/(x-1) = 3 → 分母是x-1，含未知数 → **是分式方程** ✓
（4）x/2 + 1 = 3 → 分母是2，不含未知数 → 不是分式方程
（5）(x+1)/2 = x → 分母是2，不含未知数 → 不是分式方程

**答案**：是分式方程的有 **（2）（3）**

---

### 15.4.2 分式方程的解法

#### 📖 解分式方程的基本思路

**核心思想**：**去分母**，把分式方程转化为整式方程

**步骤**：
1. **去分母**：方程两边同乘各分母的最简公分母
2. **解整式方程**：解所得的整式方程
3. **验根**：把整式方程的根代入最简公分母，看是否为0
4. **写出答案**：排除增根，写出原方程的解

#### 🔍 什么是增根？

**增根**：在去分母的过程中产生的、不适合原方程的根

**为什么会产生增根？**
- 去分母时，两边同乘最简公分母
- 如果这个最简公分母为0，就相当于两边同乘0
- 这样会产生不符合原方程的解

**如何检验增根？**
把整式方程的根代入最简公分母：
- 如果最简公分母 ≠ 0 → 是原方程的根
- 如果最简公分母 = 0 → 是增根，舍去

#### 📝 典型例题

**例23（基础题）**：解下列方程
（1）1/x = 2
（2）3/(x-1) = 2
（3）(x+1)/(x-1) = 2

**解答**：

（1）1/x = 2

**步骤1：去分母**
两边同乘 x：
1 = 2x

**步骤2：解整式方程**
x = 1/2

**步骤3：验根**
当 x = 1/2 时，分母 x = 1/2 ≠ 0 ✓

**步骤4：写出答案**
x = **1/2**

---

（2）3/(x-1) = 2

**步骤1：去分母**
两边同乘 (x-1)：
3 = 2(x-1)

**步骤2：解整式方程**
3 = 2x - 2
2x = 5
x = 5/2

**步骤3：验根**
当 x = 5/2 时，分母 x-1 = 3/2 ≠ 0 ✓

**步骤4：写出答案**
x = **5/2**

---

（3）(x+1)/(x-1) = 2

**步骤1：去分母**
两边同乘 (x-1)：
x+1 = 2(x-1)

**步骤2：解整式方程**
x+1 = 2x - 2
-x = -3
x = 3

**步骤3：验根**
当 x = 3 时，分母 x-1 = 2 ≠ 0 ✓

**步骤4：写出答案**
x = **3**

---

**例24（增根问题）**：解方程 x/(x-1) - 1 = 1/(x-1)

**解答**：

**步骤1：去分母**
两边同乘 (x-1)：
x - (x-1) = 1
x - x + 1 = 1
1 = 1

**分析**：这是一个恒等式，说明什么？
说明只要 x-1 ≠ 0，即 x ≠ 1，方程都成立

**步骤2：验根**
需要排除使分母为0的值
x-1 ≠ 0 → x ≠ 1

**步骤3：写出答案**
x ≠ 1 的任何数都是方程的解

---

**例25（增根问题）**：解方程 x/(x²-1) + 1/(x+1) = 0

**解答**：

**步骤1：去分母**
最简公分母：(x+1)(x-1)

两边同乘 (x+1)(x-1)：
x(x+1) + (x-1) = 0

**步骤2：解整式方程**
x² + x + x - 1 = 0
x² + 2x - 1 = 0

用求根公式：
x = [-2 ± √(4+4)]/2 = [-2 ± √8]/2 = [-2 ± 2√2]/2 = -1 ± √2

所以：x₁ = -1 + √2，x₂ = -1 - √2

**步骤3：验根**
- 当 x = -1 + √2 时，分母 (x+1)(x-1) = √2(-2+√2) ≠ 0 ✓
- 当 x = -1 - √2 时，分母 (x+1)(x-1) = -√2(-2-√2) ≠ 0 ✓

**步骤4：写出答案**
x = **-1 ± √2**

---

**例26（增根问题）**：解方程 (x-2)/(x-3) + 1 = 3/(x-3)

**解答**：

**步骤1：去分母**
两边同乘 (x-3)：
(x-2) + (x-3) = 3

**步骤2：解整式方程**
x - 2 + x - 3 = 3
2x - 5 = 3
2x = 8
x = 4

**步骤3：验根**
当 x = 4 时，分母 x-3 = 1 ≠ 0 ✓

**步骤4：写出答案**
x = **4**

### 15.4.3 分式方程的应用

#### 🌟 实际问题中的分式方程

**类型1：工程问题**
**类型2：行程问题**
**类型3：利润问题**
**类型4：比例问题**

#### 📝 典型例题

**例27（工程问题）**：甲、乙两地相距120km，某人从甲地到乙地，步行速度为x km/h，骑自行车速度为3x km/h，步行和骑自行车各用了2h，求x的值。

**解答**：

**分析**：
- 步行距离：2x km
- 骑车距离：2 × 3x = 6x km
- 总距离：2x + 6x = 120 km

**列方程**：
(2x + 6x)/2 = 120
或：2x + 6x = 120

**解方程**：
8x = 120
x = 15

**验根**：
x = 15 ≠ 0，且符合题意 ✓

**答**：x = **15**

**注意**：这个方程其实不是分式方程，直接是整式方程

---

**例28（行程问题）**：甲、乙两人分别从A、B两地同时出发，相向而行。已知A、B两地相距60km，甲的速度是乙的1.5倍，两人相遇后，甲继续前行，用2小时到达B地。求乙的速度。

**解答**：

**设**：乙的速度为 x km/h，则甲的速度为 1.5x km/h

**分析**：
- 相遇时：甲用的时间 = 乙用的时间
- 相遇后：甲用2小时到B地

**相遇时的距离**：
甲走的距离 = 1.5x × t
乙走的距离 = x × t
总距离：1.5x × t + x × t = 60
2.5x × t = 60
t = 60/(2.5x)

相遇后甲到B地的距离：乙在相遇时走的距离 = x × t
所以：1.5x × 2 = x × t
3x = x × t
t = 3

代入t = 60/(2.5x)：
3 = 60/(2.5x)
7.5x = 60
x = 8

**验根**：
x = 8 ≠ 0，且符合题意 ✓

**答**：乙的速度是 **8 km/h**

---

**例29（工作问题）**：某项工程，甲队单独做需要10天完成，乙队单独做需要15天完成。现在甲队先做2天，剩下的由甲、乙两队合作，问还需要多少天才能完成？

**解答**：

**设**：还需要 x 天才能完成

**分析**：
- 甲队的工作效率：1/10（每天完成工程的1/10）
- 乙队的工作效率：1/15
- 甲队先做2天完成的工作量：2/10 = 1/5
- 剩余工作量：1 - 1/5 = 4/5
- 甲、乙合作的工作效率：1/10 + 1/15 = 3/30 + 2/30 = 5/30 = 1/6

**列方程**：
x × (1/10 + 1/15) = 4/5
x × (1/6) = 4/5
x = (4/5) × 6
x = 24/5

**验根**：
x = 24/5 ≠ 0，且符合题意 ✓

**答**：还需要 **24/5 天**（即4.8天）才能完成
