# 第十五章 分式

## 📚 知识要点

### 15.1 分式

#### 15.1.1 分式的概念

**定义**：如果A、B表示两个整式，并且B中含有字母，那么式子A/B叫做分式

**分式与分数的区别**：
- 分数：A、B都是数
- 分式：A是整式，B是含有字母的整式

**分式的有意义**：
- 分母不为0时，分式有意义
- 分母为0时，分式无意义

**分式的值为0**：
- 分子为0且分母不为0时，分式的值为0

**例子**：
- x/(x+1) 是分式（分母含字母）
- 3/5 是分数，不是分式（分母不含字母）
- (x+1)/(x-1) 在 x ≠ 1 时有意义
- 当 x = -1 时，(x+1)/(x-2) = 0

#### 15.1.2 分式的基本性质

**性质**：分式的分子与分母同乘（或除以）一个不等于0的整式，分式的值不变

**公式**：
- A/B = (A×M)/(B×M) （M ≠ 0）
- A/B = (A÷M)/(B÷M) （M ≠ 0）

**应用**：分式的通分和约分

#### 15.1.3 分式的约分

**约分**：根据分式的基本性质，把一个分式的分子和分母的公因式约去

**最简分式**：分子和分母没有公因式的分式叫做最简分式

**约分的方法**：
1. 当分子和分母是单项式时：约去系数的最大公约数，约去相同字母的最低次幂
2. 当分子和分母是多项式时：先因式分解，再约分

**例子**：
- 6x²y / 9xy² = (3·2·x·x·y) / (3·3·x·y·y) = 2x / 3y
- (x²-1) / (x+1) = (x+1)(x-1) / (x+1) = x-1

#### 15.1.4 分式的通分

**通分**：根据分式的基本性质，把几个异分母的分式化成与原来的分式相等的同分母的分式

**最简公分母**：各分母所有因式的最高次幂的积

**通分的步骤**：
1. 找出各分母的最简公分母
2. 把各分式化成以最简公分母为分母的分式

**例子**：通分 1/x 和 1/(x+1)
- 最简公分母：x(x+1)
- 1/x = (x+1) / [x(x+1)]
- 1/(x+1) = x / [x(x+1)]

### 15.2 分式的运算

#### 15.2.1 分式的乘除

**乘法法则**：分式乘分式，用分子的积作为积的分子，分母的积作为积的分母

**公式**：A/B × C/D = (A×C) / (B×D)

**除法法则**：分式除以分式，把除式的分子、分母颠倒位置后，与被除式相乘

**公式**：A/B ÷ C/D = A/B × D/C = (A×D) / (B×C)

**注意**：
- 运算结果要化为最简分式
- 分式除法要转化为乘法

**例子**：
- (2x / y) × (3y / x) = (2x·3y) / (y·x) = 6
- (a/b) ÷ (b/a) = (a/b) × (a/b) = a²/b²

#### 15.2.2 分式的加减

**同分母分式相加减**：分母不变，把分子相加减

**公式**：A/C ± B/C = (A±B) / C

**异分母分式相加减**：先通分，变为同分母的分式，再加减

**公式**：A/B ± C/D = (AD±BC) / (BD)

**注意**：
- 运算结果要化为最简分式
- 分子相加减时，要加括号

**例子**：
- 1/x + 2/x = (1+2)/x = 3/x
- 1/x + 1/(x+1) = (x+1) / [x(x+1)] + x / [x(x+1)] = (2x+1) / [x(x+1)]

#### 15.2.3 分式的混合运算

**运算顺序**：先乘方，再乘除，最后加减；有括号先算括号

**注意**：
- 严格按照运算顺序
- 能简化的先简化
- 结果化为最简分式

**例子**：计算 (1/x - 1/y) ÷ (x-y)/(xy)
解：原式 = (y-x)/(xy) × (xy)/(x-y)
      = -(x-y)/(xy) × (xy)/(x-y)
      = -1

### 15.3 分式方程

#### 15.3.1 分式方程的概念

**定义**：分母中含有未知数的方程叫做分式方程

**例子**：
- 1/x = 2（是分式方程）
- (x+1)/(x-1) = 3（是分式方程）
- x/2 + 1 = 3（不是分式方程）

#### 15.3.2 分式方程的解法

**步骤**：
1. 去分母：方程两边同乘各分母的最简公分母，化成整式方程
2. 解整式方程
3. 验根：把整式方程的根代入最简公分母，看是否为0
4. 写出答案

**注意**：解分式方程必须验根！

**例子**：解方程 (x)/(x-1) - 1 = 1/(x-1)
解：方程两边同乘 (x-1)：
    x - (x-1) = 1
    x - x + 1 = 1
    1 = 1

检验：当 x ≠ 1 时，(x-1) ≠ 0
    ∴ x ≠ 1
    ∴ 方程的解是 x ≠ 1 的任何数

#### 15.3.3 分式方程的应用

**列分式方程解应用题的步骤**：
1. 审题，设未知数
2. 找等量关系，列方程
3. 解方程
4. 验根
5. 写出答案

**注意**：检验不仅要检验是否增根，还要检验是否符合题意

## 📖 典型例题

### 例题1：分式的概念

**题目**：下列各式中，哪些是分式？
（1）3/x
（2）x/3
（3）(x+1)/(x-1)
（4）5/7

**解答**：
是分式的有：**（1）3/x，（3）(x+1)/(x-1)**

不是分式的有：
- （2）x/3（分母不含字母）
- （4）5/7（是分数）

---

### 例题2：分式有意义的条件

**题目**：当 x 取何值时，下列分式有意义？
（1）1/x
（2）(x+1)/(x-1)
（3）2/(x²-1)

**解答**：
（1）x ≠ 0

（2）x - 1 ≠ 0，即 x ≠ **1**

（3）x² - 1 ≠ 0，即 x ≠ **±1**

---

### 例题3：分式的值为0

**题目**：当 x 取何值时，下列分式的值为0？
（1）(x-1)/x
（2）(x+2)/(x-3)

**解答**：
（1）分子为0且分母不为0：
    x - 1 = 0 且 x ≠ 0
    ∴ x = **1**

（2）分子为0且分母不为0：
    x + 2 = 0 且 x - 3 ≠ 0
    ∴ x = **-2**

---

### 例题4：分式的约分

**题目**：约分下列各式
（1）6x²y / 9xy²
（2）(x²-1) / (x+1)

**解答**：
（1）6x²y / 9xy² = (3·2·x·x·y) / (3·3·x·y·y) = **2x / 3y**

（2）(x²-1) / (x+1) = (x+1)(x-1) / (x+1) = **x-1**

---

### 例题5：分式的通分

**题目**：通分下列各式
（1）1/x 和 1/y
（2）2/(x+1) 和 1/x

**解答**：
（1）最简公分母：xy
    1/x = **y/(xy)**
    1/y = **x/(xy)**

（2）最简公分母：x(x+1)
    2/(x+1) = **2x/[x(x+1)]**
    1/x = **(x+1)/[x(x+1)]**

---

### 例题6：分式的乘除

**题目**：计算下列各式
（1）(2x / y) × (3y / x)
（2）(a/b) ÷ (b/a)

**解答**：
（1）(2x / y) × (3y / x) = (2x·3y) / (y·x) = **6**

（2）(a/b) ÷ (b/a) = (a/b) × (a/b) = **a² / b²**

---

### 例题7：分式的加减

**题目**：计算下列各式
（1）1/x + 2/x
（2）1/x + 1/(x+1)
（3）(x+2)/(x-1) - x/(x-1)

**解答**：
（1）1/x + 2/x = (1+2)/x = **3/x**

（2）1/x + 1/(x+1)
    = (x+1) / [x(x+1)] + x / [x(x+1)]
    = **(2x+1) / [x(x+1)]**

（3）(x+2)/(x-1) - x/(x-1)
    = (x+2-x) / (x-1)
    = **2/(x-1)**

---

### 例题8：分式的混合运算

**题目**：计算 (1/x - 1/y) ÷ (x-y)/(xy)

**解答**：
(1/x - 1/y) ÷ (x-y)/(xy)
= (y-x)/(xy) × (xy)/(x-y)
= -(x-y)/(xy) × (xy)/(x-y)
= **-1**

---

### 例题9：分式方程的解法

**题目**：解方程 (x)/(x-2) - 2 = 1/(x-2)

**解答**：
方程两边同乘 (x-2)：
x - 2(x-2) = 1
x - 2x + 4 = 1
-x = -3
x = 3

检验：当 x = 3 时，x-2 = 1 ≠ 0
∴ x = **3** 是原方程的解

---

### 例题10：分式方程的应用

**题目**：甲、乙两地相距100km，某人从甲地到乙地，步行速度为x km/h，骑自行车速度为2x km/h，步行和骑自行车各用了2.5h，求x的值。

**解答**：
根据题意：(2.5x + 2.5×2x) / 2.5 = 100
3x = 100
x = 100/3

检验：x = 100/3 ≠ 0，且符合题意
∴ x = **100/3**

答：步行的速度是 100/3 km/h。
