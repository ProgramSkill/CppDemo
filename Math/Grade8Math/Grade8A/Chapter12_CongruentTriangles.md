# 第十二章 全等三角形

## 📚 知识要点

### 12.1 全等三角形

#### 12.1.1 全等图形

**定义**：能够完全重合的两个图形叫做全等图形

**性质**：全等图形的形状和大小都相同

#### 12.1.2 全等三角形

**定义**：能够完全重合的两个三角形叫做全等三角形

**表示方法**：
- 符号：≅（读作"全等于"）
- 例如：△ABC ≅ △A'B'C'（读作"△ABC全等于△A'B'C'"）

**对应元素**：
- **对应顶点**：互相重合的顶点
- **对应边**：互相重合的边
- **对应角**：互相重合的角

**性质**：全等三角形的对应边相等，对应角相等

**数学表达式**：
```
若 △ABC ≅ △A'B'C'
则 AB = A'B'，BC = B'C'，CA = C'A'
∠A = ∠A'，∠B = ∠B'，∠C = ∠C'
```

**注意**：记两个全等三角形时，通常把表示对应顶点的字母写在对应的位置上

### 12.2 三角形全等的判定

#### 12.2.1 三边分别相等的两个三角形全等（SSS）

**边边边定理**：三边对应相等的两个三角形全等（简称"边边边"或"SSS"）

**数学表达式**：
```
在△ABC和△A'B'C'中
AB = A'B'
BC = B'C'
CA = C'A'
∴ △ABC ≅ △A'B'C'（SSS）
```

**应用**：已知三边，判定三角形全等

#### 12.2.2 两边和它们的夹角对应相等的两个三角形全等（SAS）

**边角边定理**：两边和它们的夹角对应相等的两个三角形全等（简称"边角边"或"SAS"）

**数学表达式**：
```
在△ABC和△A'B'C'中
AB = A'B'
∠A = ∠A'
AC = A'C'
∴ △ABC ≅ △A'B'C'（SAS）
```

**注意**：角必须是两边的夹角

#### 12.2.3 两角和它们的夹边对应相等的两个三角形全等（ASA）

**角边角定理**：两角和它们的夹边对应相等的两个三角形全等（简称"角边角"或"ASA"）

**数学表达式**：
```
在△ABC和△A'B'C'中
∠B = ∠B'
BC = B'C'
∠C = ∠C'
∴ △ABC ≅ △A'B'C'（ASA）
```

#### 12.2.4 两个角和其中一个角的对边对应相等的两个三角形全等（AAS）

**角角边定理**：两个角和其中一个角的对边对应相等的两个三角形全等（简称"角角边"或"AAS"）

**数学表达式**：
```
在△ABC和△A'B'C'中
∠B = ∠B'
∠C = ∠C'
AC = A'C'
∴ △ABC ≅ △A'B'C'（AAS）
```

#### 12.2.5 斜边和一条直角边对应相等的两个直角三角形全等（HL）

**斜边直角边定理**：斜边和一条直角边对应相等的两个直角三角形全等（简称"斜边直角边"或"HL"）

**数学表达式**：
```
在Rt△ABC和Rt△A'B'C'中
AB = A'B'（斜边）
AC = A'C'（直角边）
∴ Rt△ABC ≅ Rt△A'B'C'（HL）
```

**注意**：HL定理仅适用于直角三角形

#### 12.2.6 判定方法总结

| 判定方法 | 简称 | 适用条件 | 注意事项 |
|---------|------|----------|----------|
| 三边对应相等 | SSS | 任意三角形 | 无需角度 |
| 两边夹角对应相等 | SAS | 任意三角形 | 角必须是夹角 |
| 两角夹边对应相等 | ASA | 任意三角形 | 边必须是夹边 |
| 两角及一角对边对应相等 | AAS | 任意三角形 | 常用推论 |
| 斜边直角边对应相等 | HL | 直角三角形 | 仅适用于直角三角形 |

**注意**：**SSA**（边边角）和**AAA**（角角角）不能判定三角形全等

### 12.3 角的平分线的性质

#### 12.3.1 角平分线的作法

**用量角器作角平分线**：
1. 用量角器量出已知角的度数
2. 计算出这个角的半角
3. 用量角器作出半角，画出角平分线

**用尺规作角平分线**：
1. 以点O为圆心，适当长为半径作弧，交OA于点M，交OB于点N
2. 分别以点M、点N为圆心，大于½MN的长为半径作弧，两弧在∠AOB内部交于点C
3. 画射线OC
4. 射线OC就是∠AOB的平分线

#### 12.3.2 角平分线的性质

**性质定理**：角平分线上的点到角的两边的距离相等

**数学表达式**：
```
∵ OC是∠AOB的平分线，点P在OC上，PD⊥OA，PE⊥OB
∴ PD = PE
```

**判定定理**：角的内部到角的两边距离相等的点在角的平分线上

**应用**：
- 证明线段相等
- 求线段的长度
- 证明点在角平分线上

## 📖 典型例题

### 例题1：全等三角形的性质

**题目**：已知△ABC ≅ △A'B'C'，若AB = 5cm，BC = 6cm，∠A = 40°，∠B = 60°
求A'B'、B'C'、∠A'、∠B'的度数。

**解答**：
∵ △ABC ≅ △A'B'C'
∴ A'B' = AB = **5cm**
   B'C' = BC = **6cm**
   ∠A' = ∠A = **40°**
   ∠B' = ∠B = **60°**

---

### 例题2：SSS判定全等

**题目**：如图，已知AB = DE，AC = DF，BC = EF
求证：△ABC ≅ △DEF

**证明**：
在△ABC和△DEF中
AB = DE（已知）
AC = DF（已知）
BC = EF（已知）
∴ △ABC ≅ △DEF（SSS）

---

### 例题3：SAS判定全等

**题目**：如图，已知AB = AC，AD = AE，∠BAC = ∠DAE
求证：△ABD ≅ △ACE

**证明**：
∵ ∠BAC = ∠DAE
∴ ∠BAD = ∠CAE
在△ABD和△ACE中
AB = AC（已知）
∠BAD = ∠CAE（已证）
AD = AE（已知）
∴ △ABD ≅ △ACE（SAS）

---

### 例题4：ASA判定全等

**题目**：如图，已知∠B = ∠E，BC = EF，∠C = ∠F
求证：△ABC ≅ △DEF

**证明**：
在△ABC和△DEF中
∠B = ∠E（已知）
BC = EF（已知）
∠C = ∠F（已知）
∴ △ABC ≅ △DEF（ASA）

---

### 例题5：AAS判定全等

**题目**：如图，已知∠A = ∠D，∠B = ∠E，BC = EF
求证：△ABC ≅ △DEF

**证明**：
∵ ∠A = ∠D，∠B = ∠E
∴ ∠C = 180° - ∠A - ∠B = 180° - ∠D - ∠E = ∠F
在△ABC和△DEF中
∠B = ∠E（已知）
∠C = ∠F（已证）
BC = EF（已知）
∴ △ABC ≅ △DEF（AAS）

---

### 例题6：HL判定全等

**题目**：如图，在Rt△ABC和Rt△DEF中，∠C = ∠F = 90°，AB = DE，AC = DF
求证：Rt△ABC ≅ Rt△DEF

**证明**：
在Rt△ABC和Rt△DEF中
AB = DE（已知）
AC = DF（已知）
∴ Rt△ABC ≅ Rt△DEF（HL）

---

### 例题7：角平分线的性质

**题目**：如图，OC是∠AOB的平分线，点P在OC上，PD⊥OA于D，PE⊥OB于E
已知PD = 3cm，求PE的长。

**解答**：
∵ OC是∠AOB的平分线，点P在OC上，PD⊥OA，PE⊥OB
∴ PE = PD（角平分线上的点到角的两边的距离相等）
又∵ PD = 3cm
∴ PE = **3cm**

---

### 例题8：角平分线的判定

**题目**：如图，点P在∠AOB的内部，PD⊥OA于D，PE⊥OB于E，且PD = PE
求证：点P在∠AOB的平分线上。

**证明**：
∵ PD⊥OA，PE⊥OB（已知）
∴ PD = PE（已知）
∴ 点P在∠AOB的平分线上（到角的两边距离相等的点在角的平分线上）

---

### 例题9：综合应用

**题目**：如图，在△ABC中，AD是角平分线，DE⊥AB于E，DF⊥AC于F
已知DE = 2cm，AC = 8cm，求DF的长和AE的长。

**解答**：
（1）∵ AD是角平分线，DE⊥AB，DF⊥AC
    ∴ DE = DF（角平分线上的点到角的两边的距离相等）
    又∵ DE = 2cm
    ∴ DF = **2cm**

（2）∵ DE⊥AB，DF⊥AC
    ∴ ∠AED = ∠AFD = 90°
    在Rt△ADE和Rt△ADF中
    DE = DF（已证）
    AD = AD（公共边）
    ∴ Rt△ADE ≅ Rt△ADF（HL）
    ∴ AE = AF
    又∵ AC = AE + EC = AE + DF = AE + 2
    ∴ 8 = AE + 2
    ∴ AE = **6cm**

## 📚 学习建议

1. **掌握判定方法**：牢记五种判定方法，知道各自的适用条件
2. **找准对应元素**：判定全等前先确定对应边和对应角
3. **规范书写证明**：几何证明要按照"已知-求证-证明"的格式
4. **灵活选择方法**：根据题目条件选择最合适的判定方法
5. **注意SSA陷阱**：边边角不能判定全等（HL除外）
6. **应用角平分线**：角平分线的性质在证明题中很有用
