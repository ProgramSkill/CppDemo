# 第十一章 三角形

## 📚 知识要点

### 11.1 与三角形有关的线段

#### 11.1.1 三角形的边

**三角形的定义**：由不在同一条直线上的三条线段首尾顺次相接所组成的图形叫做三角形

**表示方法**：
- 顶点：A、B、C
- 边：AB、BC、CA
- 角：∠A、∠B、∠C
- 记作：△ABC（读作"三角形ABC"）

**三角形三边关系**：
- **定理**：三角形两边之和大于第三边
- **推论**：三角形两边之差小于第三边

**数学表达式**：
```
a + b > c
a + c > b
b + c > a

|a - b| < c
|a - c| < b
|b - c| < a
```

**应用**：判断三条线段能否组成三角形
- 方法：检查较短两边之和是否大于最长边

**例子**：下列长度的三条线段能否组成三角形？
（1）3cm, 4cm, 5cm → 能（3 + 4 > 5）
（2）2cm, 3cm, 6cm → 不能（2 + 3 < 6）
（3）4cm, 4cm, 8cm → 不能（4 + 4 = 8）

#### 11.1.2 三角形的高、中线与角平分线

**高**：
- 定义：从三角形的一个顶点向它的对边所在直线作垂线，顶点和垂足之间的线段
- 性质：三角形的三条高交于一点，该点叫做**垂心**
- 锐角三角形：垂心在三角形内部
- 直角三角形：垂心在直角顶点
- 钝角三角形：垂心在三角形外部

**中线**：
- 定义：连接三角形的一个顶点和它对边中点的线段
- 性质：三角形的三条中线交于一点，该点叫做**重心**
- 重心将每条中线分为2:1的两部分（从顶点算起）

**角平分线**：
- 定义：三角形一个内角的平分线与它的对边相交，这个角的顶点与交点之间的线段
- 性质：三角形的三条角平分线交于一点，该点叫做**内心**（内切圆圆心）

#### 11.1.3 三角形的稳定性

**性质**：三角形具有稳定性

**应用**：
- 建筑脚手架
- 房屋的人字梁
- 窗户的防盗网
- 自行车的三脚架

**对比**：四边形不具有稳定性（容易变形），但可以通过添加对角线使其稳定

### 11.2 与三角形有关的角

#### 11.2.1 三角形的内角

**内角和定理**：三角形三个内角的和等于180°

**证明**：
```
已知：△ABC
求证：∠A + ∠B + ∠C = 180°

证明：过点A作直线MN ∥ BC
      ∵ MN ∥ BC
      ∴ ∠1 = ∠B（两直线平行，同位角相等）
          ∠2 = ∠C（两直线平行，内错角相等）
      ∵ ∠1 + ∠BAC + ∠2 = 180°（平角的定义）
      ∴ ∠B + ∠BAC + ∠C = 180°
```

**推论1**：直角三角形的两个锐角互余

**推论2**：三角形的一个外角等于和它不相邻的两个内角的和

#### 11.2.2 三角形的外角

**外角的定义**：三角形的一边与另一边的延长线组成的角，叫做三角形的外角

**外角性质**：
1. **性质1**：三角形的一个外角等于和它不相邻的两个内角的和
   - 例如：∠ACD = ∠A + ∠B

2. **性质2**：三角形的一个外角大于任何一个和它不相邻的内角
   - 例如：∠ACD > ∠A，∠ACD > ∠B

**应用**：求角的度数、证明角的不等关系

#### 11.2.3 三角形的分类

**按角分类**：
1. **锐角三角形**：三个角都是锐角
2. **直角三角形**：有一个角是直角
3. **钝角三角形**：有一个角是钝角

**按边分类**：
1. **不等边三角形**：三边都不相等
2. **等腰三角形**：有两条边相等
   - 相等的两边叫做腰
   - 另一边叫做底边
   - 两腰的夹角叫做顶角
   - 底边上的两个角叫做底角
3. **等边三角形**：三条边都相等（特殊的等腰三角形）

### 11.3 多边形及其内角和

#### 11.3.1 多边形的概念

**定义**：在平面内，由一些线段首尾顺次相接组成的图形叫做多边形

**组成要素**：
- **边**：组成多边形的线段
- **顶点**：相邻两边的公共端点
- **对角线**：连接多边形不相邻的两个顶点的线段
- **内角**：多边形相邻两边组成的角
- **外角**：多边形的一边与另一边的延长线组成的角

**分类**：
- 按边数：三角形、四边形、五边形...
- 按形状：凸多边形、凹多边形

**对角线条数公式**：
从n边形的一个顶点出发，可以引(n-3)条对角线
n边形共有：n(n-3)/2 条对角线

#### 11.3.2 多边形的内角和

**内角和定理**：n边形的内角和等于(n-2)·180°

**证明思路**：
从n边形的一个顶点出发，可以引(n-3)条对角线
这些对角线将n边形分成(n-2)个三角形
每个三角形的内角和是180°
所以n边形的内角和是(n-2)·180°

**例子**：
- 四边形内角和：(4-2)×180° = 360°
- 五边形内角和：(5-2)×180° = 540°
- 六边形内角和：(6-2)×180° = 720°

#### 11.3.3 多边形的外角和

**外角和定理**：n边形的外角和等于360°

**说明**：n边形的外角和与边数n无关，恒等于360°

**应用**：已知多边形的各个外角，求边数

**例子**：一个多边形的每个外角都等于60°，求这个多边形的边数
解：360° ÷ 60° = 6
答：这是一个六边形

## 📖 典型例题

### 例题1：三角形三边关系的应用

**题目**：一个三角形的两边长分别是3和5，第三边长x的取值范围是（  ）
A. 2 < x < 8
B. x > 2
C. x < 8
D. 2 ≤ x ≤ 8

**解答**：
根据三角形三边关系：
- 两边之和大于第三边：3 + 5 > x，即 x < 8
- 两边之差小于第三边：5 - 3 < x，即 x > 2

所以：2 < x < 8

答案：**A**

---

### 例题2：三角形的高、中线、角平分线

**题目**：如图，在△ABC中，AD是高，AE是中线，AF是角平分线。
（1）若BD = 3，CD = 5，求BC的长
（2）若BE = EC = 4，求BC的长
（3）若∠BAF = ∠CAF = 30°，求∠BAC的度数

**解答**：
（1）AD是高，D在BC上
    BC = BD + CD = 3 + 5 = **8**

（2）AE是中线，E是BC的中点
    BC = BE + EC = 4 + 4 = **8**

（3）AF是角平分线
    ∠BAC = ∠BAF + ∠CAF = 30° + 30° = **60°**

---

### 例题3：三角形内角和定理的应用

**题目**：在△ABC中，∠A = 40°，∠B = 60°，求∠C的度数。

**解答**：
根据三角形内角和定理：
∠A + ∠B + ∠C = 180°
40° + 60° + ∠C = 180°
∠C = 180° - 40° - 60° = **80°**

---

### 例题4：直角三角形的性质

**题目**：在Rt△ABC中，∠C = 90°，∠A = 35°，求∠B的度数。

**解答**：
在Rt△ABC中，∠C = 90°
根据推论1：直角三角形的两个锐角互余
∠A + ∠B = 90°
35° + ∠B = 90°
∠B = 90° - 35° = **55°**

---

### 例题5：三角形外角性质的应用

**题目**：如图，在△ABC中，外角∠ACD = 100°，∠A = 40°，求∠B的度数。

**解答**：
根据三角形外角性质1：
外角等于和它不相邻的两个内角的和
∠ACD = ∠A + ∠B
100° = 40° + ∠B
∠B = 100° - 40° = **60°**

---

### 例题6：多边形内角和的应用

**题目**：
（1）求七边形的内角和
（2）已知一个多边形的内角和是1440°，求这个多边形的边数

**解答**：
（1）七边形内角和 = (7 - 2) × 180°
                  = 5 × 180°
                  = **900°**

（2）设这个多边形的边数为n
    (n - 2) × 180° = 1440°
    n - 2 = 1440° ÷ 180° = 8
    n = 8 + 2 = 10

    答：这是一个十边形。

---

### 例题7：多边形外角和的应用

**题目**：一个多边形的每个外角都等于40°，求这个多边形的边数。

**解答**：
多边形的外角和恒等于360°
边数 = 360° ÷ 40° = **9**

答：这是一个九边形。

---

### 例题8：综合应用

**题目**：如图，在△ABC中，AD是高，AE是角平分线，∠B = 50°，∠C = 70°
（1）求∠BAC的度数
（2）求∠DAE的度数

**解答**：
（1）根据三角形内角和定理：
    ∠BAC = 180° - ∠B - ∠C
         = 180° - 50° - 70°
         = **60°**

（2）AD是高，所以 ∠ADB = 90°
    在Rt△ABD中：
    ∠BAD = 90° - ∠B = 90° - 50° = 40°

    AE是角平分线，所以：
    ∠BAE = ½∠BAC = ½ × 60° = 30°

    ∠DAE = ∠BAD - ∠BAE = 40° - 30° = **10°**

## 📚 学习建议

1. **理解三边关系**：记住"两边之和大于第三边"，会判断能否组成三角形
2. **掌握内角和**：三角形内角和180°，多边形内角和(n-2)·180°
3. **区分三类线**：高、中线、角平分线的定义和性质要区分清楚
4. **应用外角性质**：外角等于不相邻两内角之和，这是解题的常用工具
5. **注重几何证明**：八年级开始学习几何证明，要培养严谨的逻辑思维
6. **画图辅助**：几何问题要养成画图的好习惯
