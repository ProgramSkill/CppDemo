# 第十四章 整式的乘法与因式分解

## 📚 知识要点

### 14.1 整式的乘法

#### 14.1.1 同底数幂的乘法

**法则**：同底数幂相乘，底数不变，指数相加

**公式**：aᵐ · aⁿ = aᵐ⁺ⁿ（m、n为正整数）

**注意**：
- 底数必须相同
- 指数相加，不是相乘
- 法则可以逆用：aᵐ⁺ⁿ = aᵐ · aⁿ

**例子**：
- 2³ × 2⁴ = 2³⁺⁴ = 2⁷ = 128
- x² · x³ = x²⁺³ = x⁵
- (a+b)² · (a+b)³ = (a+b)⁵

#### 14.1.2 幂的乘方

**法则**：幂的乘方，底数不变，指数相乘

**公式**：(aᵐ)ⁿ = aᵐⁿ（m、n为正整数）

**注意**：
- 先算乘方，再算乘法
- 法则可以逆用：aᵐⁿ = (aᵐ)ⁿ = (aⁿ)ᵐ

**例子**：
- (2³)² = 2³×² = 2⁶ = 64
- (x²)³ = x²×³ = x⁶
- [(a+b)²]³ = (a+b)⁶

#### 14.1.3 积的乘方

**法则**：积的乘方，等于把积的每一个因式分别乘方，再把所得的幂相乘

**公式**：(ab)ⁿ = aⁿbⁿ（n为正整数）

**注意**：
- 每个因式都要乘方
- 法则可以逆用：aⁿbⁿ = (ab)ⁿ

**例子**：
- (2x)³ = 2³ · x³ = 8x³
- (-3ab)² = (-3)² · a² · b² = 9a²b²
- (2xy)⁴ = 2⁴ · x⁴ · y⁴ = 16x⁴y⁴

#### 14.1.4 整式的乘法

**单项式乘单项式**：
- 法则：系数相乘，同底数幂相乘
- 例子：2x²y · 3xy² = (2×3) · (x²·x) · (y·y²) = 6x³y³

**单项式乘多项式**：
- 法则：m(a + b + c) = ma + mb + mc（分配律）
- 例子：2x(x² - 3x + 1) = 2x³ - 6x² + 2x

**多项式乘多项式**：
- 法则：(a + b)(m + n) = am + an + bm + bn
- 例子：(x + 2)(x + 3) = x² + 3x + 2x + 6 = x² + 5x + 6

### 14.2 乘法公式

#### 14.2.1 平方差公式

**公式**：(a + b)(a - b) = a² - b²

**语言叙述**：两个数的和与这两个数的差的积，等于这两个数的平方差

**特点**：
- 左边是两个二项式的积
- 这两个二项式中，有一项完全相同，另一项互为相反数
- 右边是完全相同项的平方减去互为相反数项的平方

**应用**：
- 正用：计算乘法
  - (x + 3)(x - 3) = x² - 9

- 逆用：因式分解
  - x² - 9 = (x + 3)(x - 3)

**例子**：
- (x + 5)(x - 5) = x² - 25
- (2a + b)(2a - b) = 4a² - b²
- (3x + 2y)(3x - 2y) = 9x² - 4y²

#### 14.2.2 完全平方公式

**公式**：
- (a + b)² = a² + 2ab + b²
- (a - b)² = a² - 2ab + b²

**语言叙述**：两个数的和（或差）的平方，等于它们的平方和，加上（或减去）它们的积的2倍

**特点**：
- 左边是两个数的和（或差）的平方
- 右边有三项
- 首尾两项是平方（符号为正）
- 中间项是积的2倍（符号与左边相同）

**记忆口诀**：首平方，尾平方，首尾两倍在中央

**应用**：
- 正用：计算平方
  - (x + 3)² = x² + 6x + 9

- 逆用：因式分解
  - x² + 6x + 9 = (x + 3)²

**例子**：
- (x + 4)² = x² + 8x + 16
- (x - 5)² = x² - 10x + 25
- (2a + 3b)² = 4a² + 12ab + 9b²

### 14.3 因式分解

#### 14.3.1 因式分解的概念

**定义**：把一个多项式化成几个整式的积的形式，叫做因式分解

**注意**：
- 因式分解是整式乘法的逆变形
- 因式分解的结果必须是几个整式的积
- 要分解到不能再分解为止

**例子**：
- ma + mb + mc = m(a + b + c)
- x² - 4 = (x + 2)(x - 2)
- x² + 4x + 4 = (x + 2)²

#### 14.3.2 提公因式法

**公因式**：一个多项式中各项都含有的因式

**找公因式的方法**：
1. 系数：各项系数的最大公约数
2. 字母：各项都含有的相同字母
3. 指数：相同字母的最低次幂

**提公因式法**：ma + mb + mc = m(a + b + c)

**步骤**：
1. 找公因式
2. 提公因式
3. 检查是否分解彻底

**例子**：
- 3x + 6 = 3(x + 2)
- x² - 2x = x(x - 2)
- 4x³ - 2x² + 6x = 2x(2x² - x + 3)

#### 14.3.3 公式法

**运用平方差公式分解**：
a² - b² = (a + b)(a - b)

**注意**：
- 左边是两项
- 两项都是平方项
- 两项符号相反

**例子**：
- x² - 9 = x² - 3² = (x + 3)(x - 3)
- 4x² - 25 = (2x)² - 5² = (2x + 5)(2x - 5)

**运用完全平方公式分解**：
a² ± 2ab + b² = (a ± b)²

**注意**：
- 左边是三项
- 首尾两项是平方（符号为正）
- 中间项是积的2倍

**例子**：
- x² + 6x + 9 = x² + 2·x·3 + 3² = (x + 3)²
- x² - 4x + 4 = x² - 2·x·2 + 2² = (x - 2)²

#### 14.3.4 十字相乘法（选学）

**适用于**：分解二次三项式 ax² + bx + c

**步骤**：
1. 把二次项分解为两个因数的积
2. 把常数项分解为两个因数的积
3. 交叉相乘，再相加，等于一次项系数
4. 横向写出因式

**例子**：x² + 5x + 6
```
   x      +2
   x      +3
   ─────────────
   x²  + 3x  + 2x  + 6  →  3x + 2x = 5x ✓
```
∴ x² + 5x + 6 = (x + 2)(x + 3)

#### 14.3.5 因式分解的一般步骤

**步骤**：
1. **提公因式**：如果各项有公因式，先提公因式
2. **看公式**：看能否用公式法（平方差、完全平方）
3. **十字相乘**：对于二次三项式，尝试十字相乘
4. **检查**：检查是否分解彻底

**口诀**：一提二套三查四分组

**例子**：分解 2x³ - 8x
解：2x³ - 8x
   = 2x(x² - 4)  ← 提公因式
   = 2x(x + 2)(x - 2)  ← 平方差公式

## 📖 典型例题

### 例题1：同底数幂的乘法

**题目**：计算下列各式
（1）x³ · x⁴
（2）2⁵ × 2³
（3）(a+b)² · (a+b)³

**解答**：
（1）x³ · x⁴ = x³⁺⁴ = **x⁷**

（2）2⁵ × 2³ = 2⁵⁺³ = **2⁸ = 256**

（3）(a+b)² · (a+b)³ = **(a+b)⁵**

---

### 例题2：幂的乘方与积的乘方

**题目**：计算下列各式
（1）(x²)³
（2）(2a)³
（3）( -3x²y )²

**解答**：
（1）(x²)³ = **x⁶**

（2）(2a)³ = 2³ · a³ = **8a³**

（3）( -3x²y )² = (-3)² · (x²)² · y² = **9x⁴y²**

---

### 例题3：整式的乘法

**题目**：计算下列各式
（1）2x²y · 3xy²
（2）2x(x² - 3x + 1)
（3）(x + 2)(x + 3)

**解答**：
（1）2x²y · 3xy² = (2×3) · (x²·x) · (y·y²) = **6x³y³**

（2）2x(x² - 3x + 1) = 2x³ - 6x² + 2x

（3）(x + 2)(x + 3) = x² + 3x + 2x + 6 = **x² + 5x + 6**

---

### 例题4：平方差公式的应用

**题目**：计算下列各式
（1）(x + 5)(x - 5)
（2）(2a + 3b)(2a - 3b)
（3）99 × 101

**解答**：
（1）(x + 5)(x - 5) = **x² - 25**

（2）(2a + 3b)(2a - 3b) = **4a² - 9b²**

（3）99 × 101 = (100 - 1)(100 + 1) = 100² - 1² = 10000 - 1 = **9999**

---

### 例题5：完全平方公式的应用

**题目**：计算下列各式
（1）(x + 4)²
（2）(x - 5)²
（3）(2a + 3b)²

**解答**：
（1）(x + 4)² = x² + 2·x·4 + 4² = **x² + 8x + 16**

（2）(x - 5)² = x² - 2·x·5 + 5² = **x² - 10x + 25**

（3）(2a + 3b)² = (2a)² + 2·(2a)·(3b) + (3b)² = **4a² + 12ab + 9b²**

---

### 例题6：提公因式法

**题目**：分解下列各式
（1）3x + 6
（2）x² - 2x
（3）4x³ - 2x² + 6x

**解答**：
（1）3x + 6 = **3(x + 2)**

（2）x² - 2x = **x(x - 2)**

（3）4x³ - 2x² + 6x = **2x(2x² - x + 3)**

---

### 例题7：公式法分解

**题目**：分解下列各式
（1）x² - 9
（2）4x² - 25
（3）x² + 6x + 9

**解答**：
（1）x² - 9 = x² - 3² = **(x + 3)(x - 3)**

（2）4x² - 25 = (2x)² - 5² = **(2x + 5)(2x - 5)**

（3）x² + 6x + 9 = x² + 2·x·3 + 3² = **(x + 3)²**

---

### 例题8：综合分解

**题目**：分解因式：2x³ - 8x

**解答**：
2x³ - 8x
= 2x(x² - 4)  ← 提公因式
= 2x(x + 2)(x - 2)  ← 平方差公式

---

### 例题9：十字相乘法

**题目**：分解因式：x² + 5x + 6

**解答**：
```
   x      +2
   x      +3
   ─────────────
   x²  + 3x  + 2x  + 6  →  3x + 2x = 5x ✓
```
∴ x² + 5x + 6 = **(x + 2)(x + 3)**

---

### 例题10：综合应用

**题目**：先化简，再求值：(2x + y)² - y(4x + y)，其中 x = 1，y = 2

**解答**：
(2x + y)² - y(4x + y)
= (4x² + 4xy + y²) - (4xy + y²)
= 4x² + 4xy + y² - 4xy - y²
= 4x²

当 x = 1，y = 2 时：
原式 = 4 × 1² = **4**

## 📚 学习建议

1. **掌握幂的运算**：同底数幂、幂的乘方、积的乘方要牢记
2. **记住公式**：平方差和完全平方公式要会背、会用
3. **区分乘法与因式分解**：乘法是化成和差形式，因式分解是化成积的形式
4. **彻底分解**：因式分解要分解到不能再分解为止
5. **先提后套**：因式分解的一般步骤是先提公因式，再用公式
6. **灵活应用**：乘法公式可以正用，也可以逆用
