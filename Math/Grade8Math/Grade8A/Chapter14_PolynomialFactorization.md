## 14.1 幂的运算

### 14.1.1 同底数幂的乘法

#### 🌟 从生活到数学

**问题引入**：
一个正方形的边长是 2³ cm，另一个更大的正方形边长是 2⁴ cm，如果我们要计算两个正方形面积的比值，该怎么办？

这个问题本质上就是：2³ × 2⁴ 等于多少？

**思考**：
- 2³ = 2 × 2 × 2（3个2相乘）
- 2⁴ = 2 × 2 × 2 × 2（4个2相乘）
- 2³ × 2⁴ = (2×2×2) × (2×2×2×2) = 2⁷

**发现规律**：3个2 × 4个2 = 7个2，即指数相加！

#### 📖 同底数幂的乘法法则

**法则**：同底数幂相乘，底数不变，指数相加

**公式表示**：
```
aᵐ · aⁿ = aᵐ⁺ⁿ  （m、n为正整数）
```

#### 🔍 法则的推导过程

**推导**：
```
aᵐ · aⁿ = (a × a × ... × a) × (a × a × ... × a)
          └───── m个a ─────┘   └───── n个a ─────┘

        = a × a × ... × a × a × a × ... × a
          └────────── m+n个a ──────────┘

        = aᵐ⁺ⁿ
```

**关键理解**：
1. **底数相同**：必须确保是"同底数"才能用这个法则
2. **指数相加**：不是相乘！这是最容易错的地方
3. **法则可逆**：aᵐ⁺ⁿ = aᵐ · aⁿ

#### ⚠️ 常见错误

**错误1**：底数不同时滥用法则
```
❌ 错误：2³ × 3³ = 2⁶（底数不同，不能用）
✅ 正确：2³ × 3³ = 8 × 27 = 216
```

**错误2**：指数相乘而不是相加
```
❌ 错误：x³ · x⁴ = x¹²
✅ 正确：x³ · x⁴ = x⁷
```

**错误3**：忽略底数为1或0的情况
```
1⁵ × 1⁶ = 1¹¹ = 1（✅ 正确）
0³ × 0⁴ = 0⁷ = 0（✅ 正确）
```

#### 📝 典型例题

**例1（基础题）**：计算下列各式
（1）x³ · x⁴
（2）2⁵ × 2³
（3）(a+b)² · (a+b)³

**解答**：
（1）x³ · x⁴ = x³⁺⁴ = **x⁷**

**思路**：底数都是x，指数3+4=7

（2）2⁵ × 2³ = 2⁵⁺³ = 2⁸ = **256**

**思路**：底数都是2，先算指数，再算数值

（3）(a+b)² · (a+b)³ = **(a+b)⁵**

**思路**：把(a+b)看作一个整体，底数相同

---

**例2（进阶题）**：计算
（1）(-3)² × (-3)³
（2）-a⁴ · a³
（3）(x-y)² · (y-x)³

**解答**：
（1）(-3)² × (-3)³ = (-3)²⁺³ = (-3)⁵ = -243

**注意**：负数的奇数次幂为负，偶数次幂为正

（2）-a⁴ · a³ = -(a⁴ · a³) = -a⁷

**注意**：系数-1要提到前面

（3）(x-y)² · (y-x)³
= (x-y)² · [-(x-y)]³
= (x-y)² · [-(x-y)³]
= -(x-y)⁵

**技巧**：遇到(y-x)，转化为(x-y)的相反数

---

**例3（综合题）**：计算 xⁿ⁺¹ · xⁿ⁻¹ · x²

**解答**：
xⁿ⁺¹ · xⁿ⁻¹ · x²
= x^(n+1+n-1+2)
= x^(2n+2)

**方法**：多个同底数幂相乘，指数全部相加

---

**例4（逆用法则）**：已知 2ᵃ = 8，2ᵇ = 16，求 2ᵃ⁺ᵇ 的值

**解答**：
方法一：先求出a和b
∵ 2ᵃ = 8 = 2³，∴ a = 3
∵ 2ᵇ = 16 = 2⁴，∴ b = 4
∴ 2ᵃ⁺ᵇ = 2³⁺⁴ = 2⁷ = 128

方法二：直接逆用法则
2ᵃ⁺ᵇ = 2ᵃ · 2ᵇ = 8 × 16 = 128

### 14.1.2 幂的乘方

#### 🌟 从生活到数学

**问题引入**：
如果一个正方形的边长是 2³ cm，那么这个正方形的面积是多少？
面积 = (2³)² = ?

**思考**：
(2³)² = 2³ × 2³ = 2³⁺³ = 2⁶

**发现规律**：(2³)² = 2⁶，即指数相乘！

#### 📖 幂的乘方法则

**法则**：幂的乘方，底数不变，指数相乘

**公式表示**：
```
(aᵐ)ⁿ = aᵐⁿ  （m、n为正整数）
```

#### 🔍 法则的推导过程

**推导**：
```
(aᵐ)ⁿ = aᵐ · aᵐ · ... · aᵐ
        └────── n个aᵐ ──────┘

      = a^(m+m+...+m)
        └── n个m相加 ──┘

      = aᵐⁿ
```

**关键理解**：
1. **运算顺序**：先算乘方，再算乘法
2. **指数相乘**：注意与同底数幂相乘的区别
3. **法则可逆**：aᵐⁿ = (aᵐ)ⁿ = (aⁿ)ᵐ

#### 📊 对比记忆

| 运算类型 | 公式 | 运算 |
|---------|------|------|
| 同底数幂相乘 | aᵐ · aⁿ | 指数**相加**：m + n |
| 幂的乘方 | (aᵐ)ⁿ | 指数**相乘**：m × n |

#### ⚠️ 常见错误

**错误1**：混淆两种运算
```
❌ 错误：(x³)² = x⁵（把指数相加了）
✅ 正确：(x³)² = x⁶（指数要相乘）
```

**错误2：忘记括号
```
❌ 错误：-a³² = -(a³)² = -a⁶
✅ 正确：-a³² = -(a³)² = -a⁶（这个是对的）
但要注意：-a³² 和 (-a)² 是不同的！
```

#### 📝 典型例题

**例5（基础题）**：计算下列各式
（1）(x²)³
（2）(2³)²
（3）[(a+b)²]³

**解答**：
（1）(x²)³ = x²×³ = **x⁶**

**思路**：底数x不变，指数2×3=6

（2）(2³)² = 2³×² = 2⁶ = **64**

**思路**：先算指数，再算数值

（3）[(a+b)²]³ = (a+b)²×³ = **(a+b)⁶**

**思路**：(a+b)看作整体

---

**例6（进阶题）**：计算
（1）(-x³)⁴
（2）-(a²)³
（3）(x²)³ · (x⁴)²

**解答**：
（1）(-x³)⁴ = (-1)⁴ · (x³)⁴ = 1 · x¹² = **x¹²**

**注意**：系数-1也要乘方

（2）-(a²)³ = -a²×³ = **-a⁶**

**注意**：负号在括号外，不参与乘方

（3）(x²)³ · (x⁴)²
= x⁶ · x⁸
= x¹⁴

**方法**：综合运用幂的乘方和同底数幂相乘

---

**例7（技巧题）**：已知 10ᵃ = 2，10ᵇ = 3，求 10²ᵃ⁺³ᵇ 的值

**解答**：
10²ᵃ⁺³ᵇ = 10²ᵃ · 10³ᵇ
        = (10ᵃ)² · (10ᵇ)³
        = 2² × 3³
        = 4 × 27
        = **108**

**技巧**：逆用幂的运算法则

### 14.1.3 积的乘方

#### 🌟 从生活到数学

**问题引入**：
一个长方形的长是 2x，宽是 3y，求这个长方形面积的平方。

面积 = (2x × 3y)² = ?

**方法一**：先算乘法，再平方
(2x × 3y)² = (6xy)² = 36x²y²

**方法二**：用积的乘方法则
(2x × 3y)² = (2x)² × (3y)² = 4x² × 9y² = 36x²y²

**发现**：积的乘方，可以把每个因式分别乘方！

#### 📖 积的乘方法则

**法则**：积的乘方，等于把积的每一个因式分别乘方，再把所得的幂相乘

**公式表示**：
```
(ab)ⁿ = aⁿbⁿ  （n为正整数）
```

**推广**：
```
(abc)ⁿ = aⁿbⁿcⁿ
```

#### 🔍 法则的推导过程

**推导**：
```
(ab)ⁿ = (ab) · (ab) · ... · (ab)
        └────── n个ab ──────┘

      = (a·a·...·a) · (b·b·...·b)
        └─ n个a ─┘   └─ n个b ─┘

      = aⁿ · bⁿ
```

**关键理解**：
1. **每个因式都要乘方**：不能漏掉任何一个
2. **系数也要乘方**：注意符号
3. **法则可逆**：aⁿbⁿ = (ab)ⁿ

#### ⚠️ 常见错误

**错误1**：漏掉系数
```
❌ 错误：(2x)³ = 2x³
✅ 正确：(2x)³ = 2³ · x³ = 8x³
```

**错误2**：符号错误
```
❌ 错误：(-2x)² = -4x²
✅ 正确：(-2x)² = (-2)² · x² = 4x²
```

**错误3**：只有部分因式乘方
```
❌ 错误：(2xy)³ = 2x³y³
✅ 正确：(2xy)³ = 2³ · x³ · y³ = 8x³y³
```

#### 📝 典型例题

**例8（基础题）**：计算下列各式
（1）(2x)³
（2）(-3ab)²
（3）(xy⁴)³

**解答**：
（1）(2x)³ = 2³ · x³ = 8 · x³ = **8x³**

**步骤**：
1. 系数2乘方：2³ = 8
2. 字母x乘方：x³
3. 相乘

（2）(-3ab)² = (-3)² · a² · b² = **9a²b²**

**注意**：负数的偶数次幂为正

（3）(xy⁴)³ = x³ · (y⁴)³ = x³ · y¹² = **x³y¹²**

**注意**：y⁴也要乘方，指数相乘

---

**例9（进阶题）**：计算
（1）(-2x²y)³
（2）(a²b)⁴ · (ab³)²
（3）(-3x²y)² · (2xy³)

**解答**：
（1）(-2x²y)³
= (-2)³ · (x²)³ · y³
= -8 · x⁶ · y³
= **-8x⁶y³**

**步骤**：
1. 系数-2乘方：(-2)³ = -8
2. x²乘方：(x²)³ = x⁶
3. y乘方：y³
4. 相乘

（2）(a²b)⁴ · (ab³)²
= (a⁸b⁴) · (a²b⁶)
= a¹⁰b¹⁰

**方法**：
1. 分别积的乘方
2. 同底数幂相乘

（3）(-3x²y)² · (2xy³)
= (9x⁴y²) · (2xy³)
= 18x⁵y⁵

**方法**：
1. 先算积的乘方
2. 再单项式乘法

---

**例10（技巧题）**：计算 0.25² × 4³

**解答**：
方法一：直接计算
0.25² × 4³ = 0.0625 × 64 = 4

方法二：逆用积的乘方
0.25² × 4³ = 0.25² × 4² × 4
           = (0.25 × 4)² × 4
           = 1² × 4
           = 4

**技巧**：利用 aⁿbⁿ = (ab)ⁿ

### 14.1.4 整式的乘法

#### 📐 单项式乘单项式

**法则**：系数相乘，相同字母的幂相乘，只在一个单项式里含有的字母，连同它的指数作为积的一个因式

**一般形式**：
```
Axᵐ · Byⁿ = (AB) · xᵐ · yⁿ
```

**例11**：计算 2x²y · 3xy²

**解答**：
2x²y · 3xy²
= (2×3) · (x²·x) · (y·y²)
= 6 · x³ · y³
= **6x³y³**

**步骤**：
1. 系数：2 × 3 = 6
2. x的幂：x² · x = x³
3. y的幂：y · y² = y³
4. 合并

---

#### 📐 单项式乘多项式

**法则**：根据分配律，用单项式去乘多项式的每一项，再把所得的积相加

**公式表示**：
```
m(a + b + c) = ma + mb + mc
```

**直观理解**：
```
     m       m       m
    ┌─┐     ┌─┐     ┌─┐
    │ │     │ │     │ │
┌───┼─┼─────┼─┼─────┼─┼───┐
│   │ │     │ │     │ │   │
│   a │  +  b │  +  c │   │
│       │       │       │   │
└───────┴───────┴───────┴───┘
```

**例12**：计算 2x(x² - 3x + 1)

**解答**：
2x(x² - 3x + 1)
= 2x · x² + 2x · (-3x) + 2x · 1
= 2x³ - 6x² + 2x

**步骤**：
1. 2x 乘 x²：2x³
2. 2x 乘 -3x：-6x²
3. 2x 乘 1：2x
4. 合并同类项

---

#### 📐 多项式乘多项式

**法则**：先用一个多项式的每一项乘另一个多项式的每一项，再把所得的积相加

**公式表示**：
```
(a + b)(m + n) = am + an + bm + bn
```

**直观理解（面积法）**：
```
        ┌─────────┬─────────┐
        │         │         │
        │   am    │   an    │
        │         │         │
├───────┼─────────┼─────────┤
        │         │         │
        │   bm    │   bn    │
        │         │         │
└───────┴─────────┴─────────┘
```

总面积 = am + an + bm + bn

**例13**：计算 (x + 2)(x + 3)

**解答**：
方法一：展开法
(x + 2)(x + 3)
= x·x + x·3 + 2·x + 2·3
= x² + 3x + 2x + 6
= x² + 5x + 6

方法二：竖式计算法
```
      x  +  2
    × x  +  3
    ────────
      3x  +  6  （3乘每一项）
x²  +  2x      （x乘每一项，向左移一位）
───────────────
x²  +  5x  +  6
```

**例14**：计算 (x - 3)(2x + 5)

**解答**：
(x - 3)(2x + 5)
= x·2x + x·5 + (-3)·2x + (-3)·5
= 2x² + 5x - 6x - 15
= 2x² - x - 15

**注意**：符号要小心！

---

**例15（应用题）**：一个长方形的长是 (x + 3) cm，宽是 (x + 2) cm，求面积。

**解答**：
面积 = (x + 3)(x + 2)
     = x² + 2x + 3x + 6
     = x² + 5x + 6 (cm²)

**思考**：如果 x = 4，面积是多少？
面积 = 4² + 5×4 + 6 = 16 + 20 + 6 = 42 (cm²)

## 14.2 乘法公式

### 14.2.1 平方差公式

#### 🌟 从生活到数学

**问题引入**：
有两个正方形，边长分别为 a 和 b（a > b），求阴影部分的面积。

```
┌─────────────────┐
│                 │
│     ┌─────┐     │
│     │     │     │  大正方形边长 = a
│     │     │     │  小正方形边长 = b
│     └─────┘     │  阴影面积 = ?
│                 │
└─────────────────┘
```

**方法一**：直接相减
阴影面积 = a² - b²

**方法二**：分割重组
把阴影部分剪下来，重新拼成一个长方形：
- 长方形的长 = a + b
- 长方形的宽 = a - b
- 阴影面积 = (a + b)(a - b)

**结论**：a² - b² = (a + b)(a - b)

#### 📖 平方差公式

**公式**：
```
(a + b)(a - b) = a² - b²
```

**语言叙述**：两个数的和与这两个数的差的积，等于这两个数的平方差

**公式的结构特征**：
1. **左边**：两个二项式相乘
   - 一项完全相同
   - 另一项互为相反数

2. **右边**：两个数的平方差
   - 完全相同项的平方
   - 减去互为相反数项的平方

#### 🔍 公式的几何证明

**面积法证明**：

```
  ┌──────┬─────┐
  │      │     │
  │   1  │  2  │  大正方形：边长 a
  │      │     │
  ├──────┼─────┤
  │      │     │
  │   3  │  4  │  小正方形：边长 b
  │      │     │  在左上角
  └──────┴─────┘
```

**证明**：
- 大正方形面积 = a²
- 小正方形面积 = b²
- 阴影部分面积 = a² - b²

把阴影部分重新拼合：
```
┌─────────┬─────────┐
│         │         │
│    1    │    2    │  长：a + b
│         │         │  宽：a - b
├─────────┤         │
│         │         │
│    3    │    4    │
│         │         │
└─────────┴─────────┘
```

拼合后的长方形：
- 长 = a + b
- 宽 = a - b
- 面积 = (a + b)(a - b)

所以：a² - b² = (a + b)(a - b)

#### 📝 典型例题

**例16（基础题）**：计算下列各式
（1）(x + 3)(x - 3)
（2）(2a + b)(2a - b)
（3）(3x + 2y)(3x - 2y)

**解答**：
（1）(x + 3)(x - 3)
= x² - 3²
= x² - 9

**识别**：相同项是 x，相反项是 3 和 -3

（2）(2a + b)(2a - b)
= (2a)² - b²
= 4a² - b²

**识别**：相同项是 2a，相反项是 b 和 -b

（3）(3x + 2y)(3x - 2y)
= (3x)² - (2y)²
= 9x² - 4y²

**识别**：相同项是 3x，相反项是 2y 和 -2y

---

**例17（变式题）**：计算
（1）(-x + 3)(-x - 3)
（2）(a - 2)(2 + a)
（3）(3m - 2n)(-3m - 2n)

**解答**：
（1）(-x + 3)(-x - 3)
= (-x)² - 3²
= x² - 9

**技巧**：提取-1，(-x + 3) = -(x - 3)，(-x - 3) = -(x + 3)
= [-(x - 3)][-(x + 3)] = (x - 3)(x + 3) = x² - 9

（2）(a - 2)(2 + a)
= (a - 2)(a + 2)
= a² - 4

**技巧**：先调整顺序

（3）(3m - 2n)(-3m - 2n)
= [-(2n - 3m)][-(2n + 3m)]
= (2n - 3m)(2n + 3m)
= (2n)² - (3m)²
= 4n² - 9m²

**技巧**：-3m - 2n = -(2n + 3m)

---

**例18（巧算题）**：计算
（1）102 × 98
（2）59 × 61
（3）199 × 201

**解答**：
（1）102 × 98

### 14.2.2 完全平方公式

#### 🌟 从生活到数学

**问题引入**：有一个边长为 a 的正方形，如果每边增加 b，面积增加了多少？

**方法**：画图分析

```
原正方形边长 = a，边长增加 b

┌─────────┬─────┐
│         │     │
│   a²    │ ab  │  1. 原正方形：a²
│         │     │  2. 右边长方形：ab
├─────────┼─────┤  3. 下边长方形：ab
│         │     │  4. 右下角正方形：b²
│   ab    │ b²  │
│         │     │
└─────────┴─────┘
```

总面积 = a² + ab + ab + b² = a² + 2ab + b²

**又**：新正方形边长 = a + b
总面积 = (a + b)²

**结论**：(a + b)² = a² + 2ab + b²

#### 📖 完全平方公式

**公式**：
```
(a + b)² = a² + 2ab + b²
(a - b)² = a² - 2ab + b²
```

**语言叙述**：两个数的和（或差）的平方，等于它们的平方和，加上（或减去）它们的积的2倍

**公式的结构特征**：
1. **左边**：二项式的平方
2. **右边**：三项
   - 首项：平方（符号为正）
   - 中间项：积的2倍（符号与左边相同）
   - 尾项：平方（符号为正）

**记忆口诀**：首平方，尾平方，首尾两倍在中央

#### 🔍 公式的几何证明

**证明 (a + b)² = a² + 2ab + b²**：

**面积法**：

```
边长为 (a + b) 的正方形：

┌─────────┬─────┐
│         │     │
│   a²    │ ab  │
│         │     │  面积 = (a + b)²
├─────────┼─────┤
│         │     │
│   ab    │ b²  │
│         │     │
└─────────┴─────┘
```

总面积 = a² + ab + ab + b² = a² + 2ab + b²

所以：(a + b)² = a² + 2ab + b²

---

**证明 (a - b)² = a² - 2ab + b²**：

**面积法**：

```
大正方形边长为 a，小正方形边长为 b

┌─────────────┐
│             │
│     a²      │
│             │  从大正方形 a² 中剪去
│      ┌──┐   │  两个 ab，但多剪了 b²
│      │b² │   │  所以要加回来
│      └──┘   │
│             │
└─────────────┘
```

(a - b)² = a² - ab - ab + b² = a² - 2ab + b²

#### 📊 两个公式对比

| 公式 | 中间项符号 | 图示 |
|------|-----------|------|
| (a + b)² = a² + 2ab + b² | + | a² **+** ab **+** ab **+** b² |
| (a - b)² = a² - 2ab + b² | - | a² **-** ab **-** ab **+** b² |

#### ⚠️ 常见错误

**错误1**：忘记中间项
```
❌ 错误：(x + 3)² = x² + 9
✅ 正确：(x + 3)² = x² + 6x + 9
```

**错误2**：中间项忘记乘2
```
❌ 错误：(x + 3)² = x² + 3x + 9
✅ 正确：(x + 3)² = x² + 6x + 9
```

**错误3**：符号错误
```
❌ 错误：(x - 3)² = x² - 6x - 9
✅ 正确：(x - 3)² = x² - 6x + 9
```

**错误4**：混淆公式
```
❌ 错误：(x + y)² = x² + y²
✅ 正确：(x + y)² = x² + 2xy + y²
```

## 14.3 因式分解

### 14.3.1 因式分解的概念

#### 📖 什么是因式分解？

**定义**：把一个多项式化成几个整式的积的形式，叫做因式分解

**类比理解**：
- 数的分解：15 = 3 × 5
- 式的分解：x² - 4 = (x + 2)(x - 2)

#### 🔄 因式分解 vs 整式乘法

| 对比项 | 整式乘法 | 因式分解 |
|--------|---------|---------|
| **运算** | 积 → 和差 | 和差 → 积 |
| **方向** | 正向 | 逆向 |
| **例子** | (x+2)(x-2) = x²-4 | x²-4 = (x+2)(x-2) |
| **目的** | 展开、化简 | 分解、约分 |

**关系**：互为逆运算

#### ⚠️ 易错辨析

**错误1**：因式分解不彻底
```
❌ 错误：2x³ - 8x = 2x(x² - 4)
✅ 正确：2x³ - 8x = 2x(x + 2)(x - 2)
```

**错误2**：不是积的形式
```
❌ 错误：x² - 4 = x² - 2²（这叫恒等变形，不叫因式分解）
✅ 正确：x² - 4 = (x + 2)(x - 2)
```

**错误3**：混淆整式乘法和因式分解
```
❌ 错误：(x + 2)(x - 2) = x² - 4 叫因式分解
✅ 正确：这叫整式乘法，x² - 4 = (x + 2)(x - 2) 才叫因式分解
```

---

### 14.3.2 提公因式法

#### 🌟 从生活到数学

**问题引入**：分配律 a(b + c) = ab + ac 的逆向应用

ab + ac = a(b + c)

这就是提公因式法！

#### 📖 公因式

**定义**：一个多项式中各项都含有的因式，叫做公因式

**找公因式的方法（三步法）**：

**第一步：找系数的最大公约数**
```
6x²y - 9xy²
系数：6 和 -9
最大公约数：3
```

**第二步：找相同的字母**
```
6x²y - 9xy²
相同字母：x、y
```

**第三步：找相同字母的最低次幂**
```
6x²y - 9xy²
x的最低次幂：x¹
y的最低次幂：y¹
字母部分：xy
```

**公因式 = 3xy**

#### 📐 提公因式法

**公式**：
```
ma + mb + mc = m(a + b + c)
```

**步骤**：
1. **找公因式**：按三步法找
2. **提公因式**：把公因式提到括号外面
3. **检查**：括号内的项是否正确

#### 📝 典型例题

**例25（基础题）**：分解因式
（1）3x + 6
（2）x² - 2x
（3）4x³ - 2x² + 6x

**解答**：
（1）3x + 6
= 3x + 3×2
= 3(x + 2)

**分析**：
- 系数：3、6 → 最大公约数是3
- 字母：只有第一项有x
- 公因式：3

（2）x² - 2x
= x·x - 2·x
= x(x - 2)

**分析**：
- 系数：1、-2 → 最大公约数是1
- 字母：都有x
- 公因式：x

（3）4x³ - 2x² + 6x

**找公因式**：
- 系数：4、-2、6 → 最大公约数是2
- 字母：都有x
- x的最低次幂：x¹
- 公因式：2x

4x³ - 2x² + 6x
= 2x(2x²) - 2x(x) + 2x(3)
= 2x(2x² - x + 3)

**检查**：2x(2x² - x + 3) = 4x³ - 2x² + 6x ✓

---

**例26（进阶题）**：分解因式
（1）-3x² + 6x
（2）a(x - y) + b(x - y)
（3）2a(b - c)³ - 4a²(b - c)²

**解答**：
（1）-3x² + 6x

**方法一**：公因式是 3x
-3x² + 6x = 3x(-x + 2) = 3x(2 - x)

**方法二**：公因式是 -3x（首项为负时，常提负号）
-3x² + 6x = -3x(x - 2)

**技巧**：一般首项为负时，提负的公因式

（2）a(x - y) + b(x - y)
= (x - y)(a + b)

**技巧**：把(x - y)看作整体

（3）2a(b - c)³ - 4a²(b - c)²

**找公因式**：
- 系数：2、-4 → 最大公约数是2
- 字母a：a、a² → 最低次幂是a
- 字母(b - c)：³、² → 最低次幂是²
- 公因式：2a(b - c)²

2a(b - c)³ - 4a²(b - c)²
= 2a(b - c)² · (b - c) - 2a(b - c)² · 2a
= 2a(b - c)²[(b - c) - 2a]
= 2a(b - c)²(b - c - 2a)

---

**例27（技巧题）**：分解因式
（1）2x(x - y) + 3y(x - y)
（2）x(m - n) - y(n - m)
（3）(a - b)² - 2(a - b)

**解答**：
（1）2x(x - y) + 3y(x - y)
= (x - y)(2x + 3y)

**技巧**：(x - y)是公因式

（2）x(m - n) - y(n - m)

**注意**：(m - n) 和 (n - m) 互为相反数！
(m - n) = -(n - m)

x(m - n) - y(n - m)
= x[-(n - m)] - y(n - m)
= -x(n - m) - y(n - m)
= -(n - m)(x + y)
= (m - n)(x + y)

或者：
x(m - n) - y(n - m)
= x(m - n) + y(m - n)
= (m - n)(x + y)

（3）(a - b)² - 2(a - b)
= (a - b)[(a - b) - 2]
= (a - b)(a - b - 2)
= (a - b)(a - b - 2)

### 14.3.3 公式法

#### 📖 运用平方差公式分解

**公式**：
```
a² - b² = (a + b)(a - b)
```

**适用条件**：
1. **二项式**：左边是两项
2. **平方项**：两项都是完全平方
3. **符号相反**：一项为正，一项为负

**识别技巧**：
```
x² - 4    → x² - 2² ✓（可以用）
x² + 4    → 不能用平方差（符号相同）
-x² + 4   → 4 - x² = 2² - x² ✓（可以用）
4x² - 9   → (2x)² - 3² ✓（可以用）
```

**例28**：分解因式
（1）x² - 9
（2）4x² - 25
（3）9a² - 16b²

**解答**：
（1）x² - 9
= x² - 3²
= (x + 3)(x - 3)

**步骤**：
1. 识别平方项：x² = x²，9 = 3²
2. 应用公式

（2）4x² - 25
= (2x)² - 5²
= (2x + 5)(2x - 5)

**注意**：4x² = (2x)²，要写成完全平方

（3）9a² - 16b²
= (3a)² - (4b)²
= (3a + 4b)(3a - 4b)

---

**例29**：分解因式（先提公因式，再用公式）
（1）2x² - 8
（2）3a³ - 27a
（3）x⁴ - 16

**解答**：
（1）2x² - 8
= 2(x² - 4)      ← 先提公因式
= 2(x + 2)(x - 2) ← 再用平方差公式

（2）3a³ - 27a
= 3a(a² - 9)     ← 先提公因式
= 3a(a + 3)(a - 3) ← 再用平方差公式

（3）x⁴ - 16
= (x²)² - 4²     ← 识别平方项
= (x² + 4)(x² - 4) ← 第一次平方差
= (x² + 4)(x + 2)(x - 2) ← 第二次平方差

**注意**：要分解到不能再分解为止！

---

#### 📖 运用完全平方公式分解

**公式**：
```
a² + 2ab + b² = (a + b)²
a² - 2ab + b² = (a - b)²
```

**适用条件**：
1. **三项式**：左边是三项
2. **首尾平方**：首尾两项都是平方项（符号为正）
3. **中间项**：是首尾两数积的2倍

**识别技巧**：
```
x² + 6x + 9
首项：x² ✓
尾项：9 = 3² ✓
中间：6x = 2·x·3 ✓
可以用：= (x + 3)²

---

x² - 4x + 4
首项：x² ✓
尾项：4 = 2² ✓
中间：-4x = -2·x·2 ✓
可以用：= (x - 2)²

---

x² + 4x + 4
首项：x² ✓
尾项：4 = 2² ✓
中间：4x ≠ 2·x·2 = 4x ✓
可以用：= (x + 2)²
```

**例30**：分解因式
（1）x² + 6x + 9
（2）x² - 4x + 4
（3）4x² + 12xy + 9y²

**解答**：
（1）x² + 6x + 9
= x² + 2·x·3 + 3²
= (x + 3)²

**识别步骤**：
1. 首项：x² ✓
2. 尾项：9 = 3² ✓
3. 中间：6x = 2·x·3 ✓

（2）x² - 4x + 4
= x² - 2·x·2 + 2²
= (x - 2)²

（3）4x² + 12xy + 9y²
= (2x)² + 2·(2x)·(3y) + (3y)²
= (2x + 3y)²

---

**例31**：分解因式（先提公因式，再用公式）
（1）2x² + 4x + 2
（2）-3x² + 6x - 3
（3）ax² + 2axy + ay²

**解答**：
（1）2x² + 4x + 2
= 2(x² + 2x + 1)      ← 先提公因式
= 2(x + 1)²            ← 再用完全平方公式

（2）-3x² + 6x - 3
= -3(x² - 2x + 1)     ← 提负的公因式
= -3(x - 1)²

（3）ax² + 2axy + ay²
= a(x² + 2xy + y²)    ← 先提公因式
= a(x + y)²            ← 再用完全平方公式

### 14.3.4 十字相乘法

#### 📖 什么是十字相乘法？

**适用范围**：分解二次三项式 ax² + bx + c

**基本思路**：借助十字交叉线分解系数，帮助我们把二次三项式分解因式

#### 📐 十字相乘法的步骤

**以 x² + 5x + 6 为例**：

**第一步**：分解二次项系数
```
x² = x · x
```
写成：
```
   x
   x
```

**第二步**：分解常数项
```
6 = 1 × 6 = 2 × 3 = (-1) × (-6) = ...
```

**第三步**：尝试组合
```
   x      +2
   x      +3
   ─────────────
   x²  + 3x  + 2x  + 6
```

**第四步**：验证交叉相乘之和
```
3x + 2x = 5x ✓ （等于一次项系数）
```

**第五步**：写出因式
```
x² + 5x + 6 = (x + 2)(x + 3)
```

#### 🎯 如何选择分解方式？

**对于 x² + 5x + 6**：
- 6 = 1 × 6 → 交叉相乘：1x + 6x = 7x ✗
- 6 = 2 × 3 → 交叉相乘：2x + 3x = 5x ✓

所以选 2 和 3

#### 📝 典型例题

**例32**：分解因式
（1）x² + 5x + 6
（2）x² - 5x + 6
（3）x² + x - 6
（4）x² - x - 6

**解答**：
（1）x² + 5x + 6

```
   x      +2
   x      +3
   ─────────────
   x²  + 3x  + 2x  + 6  →  3x + 2x = 5x ✓
```

x² + 5x + 6 = **(x + 2)(x + 3)**

---

（2）x² - 5x + 6

分析：
- 二次项系数：1 = 1 × 1
- 常数项：6 = (-2) × (-3)
- 交叉相乘：(-3x) + (-2x) = -5x ✓

```
   x      -2
   x      -3
   ─────────────
   x²  - 3x  - 2x  + 6  →  -3x - 2x = -5x ✓
```

x² - 5x + 6 = **(x - 2)(x - 3)**

**规律**：常数项为正，中间项为负 → 两个因数都为负

---

（3）x² + x - 6

分析：
- 二次项系数：1 = 1 × 1
- 常数项：-6 = (-2) × 3 或 2 × (-3)
- 尝试 (-2) × 3：交叉相乘 3x + (-2x) = x ✓

```
   x      -2
   x      +3
   ─────────────
   x²  + 3x  - 2x  - 6  →  3x - 2x = x ✓
```

x² + x - 6 = **(x - 2)(x + 3)**

**规律**：常数项为负 → 两个因数一正一负，绝对值大的符号与一次项相同

---

（4）x² - x - 6

分析：
- 二次项系数：1 = 1 × 1
- 常数项：-6 = 2 × (-3) 或 (-2) × 3
- 尝试 2 × (-3)：交叉相乘 (-3x) + 2x = -x ✓

```
   x      +3
   x      -2
   ─────────────
   x²  - 2x  + 3x  - 6  →  -2x + 3x = x ✗

   x      -3
   x      +2
   ─────────────
   x²  + 2x  - 3x  - 6  →  2x - 3x = -x ✓
```

x² - x - 6 = **(x - 3)(x + 2)**

**例34**：分解因式（综合应用）

### 14.3.5 因式分解的一般步骤

#### 📋 标准步骤

**口诀**：一提二套三查四分组

```
┌─────────────────────────────────────┐
│  第一步：提公因式                    │
│  如果多项式各项有公因式，先提公因式  │
├─────────────────────────────────────┤
│  第二步：套公式                      │
│  观察是否能用平方差或完全平方公式    │
├─────────────────────────────────────┤
│  第三步：十字相乘                    │
│  对于二次三项式，尝试十字相乘        │
├─────────────────────────────────────┤
│  第四步：检查                        │
│  检查是否分解到不能再分解为止        │
└─────────────────────────────────────┘
```

#### 📝 综合例题

**例35**：分解因式（综合运用）
（1）2x³ - 8x
（2）3x² - 12
（3）x⁴ - 1
（4）x³ - 6x² + 9x

**解答**：
（1）2x³ - 8x

第一步：提公因式
= 2x(x² - 4)

第二步：用公式（平方差）
= 2x(x + 2)(x - 2)

第三步：检查（已分解彻底）

答案：**2x(x + 2)(x - 2)**

---

（2）3x² - 12

第一步：提公因式
= 3(x² - 4)

第二步：用公式（平方差）
= 3(x + 2)(x - 2)

第三步：检查

答案：**3(x + 2)(x - 2)**

---

（3）x⁴ - 1

第一步：用公式（平方差）
= (x²)² - 1²
= (x² + 1)(x² - 1)

第二步：继续分解
x² - 1 还能用平方差
= (x² + 1)(x + 1)(x - 1)

第三步：检查

答案：**(x² + 1)(x + 1)(x - 1)**

---

（4）x³ - 6x² + 9x

第一步：提公因式
= x(x² - 6x + 9)

第二步：用公式（完全平方）
= x(x - 3)²

第三步：检查

答案：**x(x - 3)²**

---

**例36**：分解因式（分组分解）
（1）ax - ay + bx - by
（2）x² - y² + 2x + 2y
（3）a² - b² + 2a - 2b

**解答**：
（1）ax - ay + bx - by

方法：分组提取公因式
= (ax - ay) + (bx - by)
= a(x - y) + b(x - y)
= (x - y)(a + b)

或者：
= (ax + bx) - (ay + by)
= x(a + b) - y(a + b)
= (a + b)(x - y)

答案：**(x - y)(a + b)** 或 **(a + b)(x - y)**

---

（2）x² - y² + 2x + 2y

方法：先平方差，再提公因式
= (x + y)(x - y) + 2(x + y)
= (x + y)[(x - y) + 2]
= (x + y)(x - y + 2)

答案：**(x + y)(x - y + 2)**

---

（3）a² - b² + 2a - 2b

方法：先平方差，再提公因式
= (a + b)(a - b) + 2(a - b)
= (a - b)[(a + b) + 2]
= (a - b)(a + b + 2)

答案：**(a - b)(a + b + 2)**

### 14.4.1 面积问题

**例37**：一个长方形的长比宽多3cm，面积是40cm²，求长和宽。

**解答**：
设宽为 x cm，则长为 (x + 3) cm

根据题意：
x(x + 3) = 40
x² + 3x - 40 = 0

分解因式：
x² + 3x - 40
= (x + 8)(x - 5) = 0

所以：x + 8 = 0 或 x - 5 = 0
x = -8（舍去）或 x = 5

宽：x = 5 (cm)
长：x + 3 = 8 (cm)

答：长方形的长是8cm，宽是5cm。

**例38**：计算下列各式（巧用因式分解）
（1）57² + 2 × 57 × 43 + 43²
（2）2024² - 2024 × 2023 + 2023²

**解答**：
（1）57² + 2 × 57 × 43 + 43²

观察：这是完全平方公式的形式
= (57 + 43)²
= 100²
= 10000

（2）2024² - 2024 × 2023 + 2023²

方法：把 -2024×2023 看作 -2 × 2024 × 1011.5，不方便

用另一种方法：
设 a = 2024，b = 2023
a² - ab + b² = (a - b)² + ab = 1² + 2024×2023 = 1 + ...

这个不太好算，换个思路：

a² - ab + b² = a² - 2ab + b² + ab = (a - b)² + ab
= 1² + 2024 × 2023

还是不行，再想：

a² - ab + b² = (a + b)² - 3ab
= (2024 + 2023)² - 3 × 2024 × 2023

也不行

直接用：a² - ab + b² = (a³ + b³)/(a + b) 当 a + b ≠ 0时

2024² - 2024 × 2023 + 2023²
= 2024³ + 2023³ / (2024 + 2023)
= ...

太复杂了，让我用更简单的方法：

a² - ab + b²
= a² - 2ab + b² + ab
= (a - b)² + ab
= 1 + 2024 × 2023
= 1 + (2000 + 24)(2000 + 23)
= 1 + 4000000 + 46000 + 552
= 4046553

所以答案是 4046553
