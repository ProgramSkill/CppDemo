# 第十六章 二次根式

## 📚 知识要点

### 16.1 二次根式

#### 16.1.1 二次根式的概念

**定义**：一般地，形如√a（a≥0）的式子叫做二次根式

**注意**：
- 被开方数a必须是非负数（a ≥ 0）
- √a ≥ 0（二次根式的双重非负性）

**例子**：
- √3（是二次根式）
- √(x²+1)（是二次根式）
- √(-2)（不是，因为被开方数是负数）

#### 16.1.2 二次根式的性质

**性质1**：(√a)² = a（a ≥ 0）

**应用**：化简二次根式
- (√5)² = 5
- (√3)² = 3

**性质2**：√(a²) = |a| = {a (a≥0), -a (a<0)}

**注意**：√(a²) = |a|，不一定是a

**例子**：
- √(3²) = 3
- √((-3)²) = 3
- √(x²) = |x|

#### 16.1.3 代数式

**代数式的分类**：
- **有理式**：整式和分式统称为有理式
- **无理式**：根号下含有字母的代数式叫做无理式

**注意**：二次根式是无理式的一种

### 16.2 二次根式的乘除

#### 16.2.1 二次根式的乘法

**法则**：√a · √b = √(ab)（a≥0，b≥0）

**逆用**：√(ab) = √a · √b

**应用**：二次根式的化简
- √2 · √8 = √(2×8) = √16 = 4
- √3 · √12 = √(3×12) = √36 = 6
- 3√2 · 2√6 = 6√12 = 6×2√3 = 12√3

#### 16.2.2 二次根式的除法

**法则**：√a ÷ √b = √(a/b)（a≥0，b>0）

**逆用**：√(a/b) = √a ÷ √b

**商的算术平方根**：√(a²/b²) = a/b（a≥0，b>0）

**例子**：
- √18 ÷ √2 = √(18/2) = √9 = 3
- √27 ÷ √3 = √(27/3) = √9 = 3

### 16.3 二次根式的加减

#### 16.3.1 同类二次根式

**定义**：几个二次根式化成最简二次根式后，如果被开方数相同，这几个二次根式就叫做同类二次根式

**判断方法**：
1. 先化成最简二次根式
2. 比较被开方数
3. 被开方数相同就是同类二次根式

**例子**：
- √2 和 3√2 是同类二次根式
- √8 和 √2 不是同类二次根式（√8 = 2√2）
- √12 和 √3 不是同类二次根式（√12 = 2√3）

#### 16.3.2 二次根式的加减

**法则**：二次根式相加减，先把各个二次根式化成最简二次根式，再把同类二次根式进行合并，合并方法与合并同类项类似

**步骤**：
1. 化简：把每个二次根式化成最简二次根式
2. 判断：找出同类二次根式
3. 合并：合并同类二次根式

**注意**：不是同类二次根式不能合并

**例子**：
- √2 + 3√2 = 4√2
- √3 - √12 = √3 - 2√3 = -√3
- √8 + √18 = 2√2 + 3√2 = 5√2

#### 16.3.3 二次根式的混合运算

**运算顺序**：
1. 乘方（如果有）
2. 乘除
3. 加减
4. 有括号先算括号

**注意**：
- 运算结果要化成最简二次根式
- 能用运算律的要尽量使用运算律

**例子**：
(√6 + √2)(√6 - √2)
= (√6)² - (√2)²
= 6 - 2
= 4

### 16.4 二次根式的化简

#### 16.4.1 最简二次根式

**条件**：
1. 被开方数不含分母
2. 被开方数中不含能开得尽方的因数或因式

**化简方法**：
1. 移进根号：a = √(a²)（a≥0）
2. 移出根号：√(a²b) = a√b（a≥0）
3. 化去分母：√(a/b) = √(ab)/b

**例子**：
- √12 = √(4×3) = 2√3
- √18 = √(9×2) = 3√2
- √20 = √(4×5) = 2√5

#### 16.4.2 分母有理化

**定义**：把分母中的根号化去，叫做分母有理化

**方法**：
1. 单个根号：分子分母同乘分母的根号
   - 1/√2 = √2/2

2. 和差形式：利用平方差公式
   - 1/(√2+1) = (√2-1)/[(√2+1)(√2-1)] = (√2-1)/(2-1) = √2-1

**例子**：
- √2/2（已经是最简二次根式）
- 1/(√3-1) = (√3+1)/[(√3-1)(√3+1)] = (√3+1)/(3-1) = (√3+1)/2

## 📖 典型例题

### 例题1：二次根式的概念

**题目**：下列各式是二次根式的是（  ）
A. √(-3)
B. √(x²+1)
C. √3²
D. ³√2

**解答**：
**答案：B**

解析：
- A：√(-3) 被开方数是负数，不是
- B：√(x²+1) 是二次根式
- C：√3² = 3，不是二次根式
- D：³√2 是立方根，不是二次根式

---

### 例题2：二次根式的性质

**题目**：计算下列各式
（1）(√5)²
（2）√(3²)
（3）√((-3)²)

**解答**：
（1）(√5)² = **5**

（2）√(3²) = **3**

（3）√((-3)²) = **3**

---

### 例题3：二次根式的乘法

**题目**：计算下列各式
（1）√2 · √8
（2）√3 · √12
（3）3√2 · 2√6

**解答**：
（1）√2 · √8 = √(2×8) = √16 = **4**

（2）√3 · √12 = √(3×12) = √36 = **6**

（3）3√2 · 2√6 = 6√12 = 6×2√3 = **12√3**

---

### 例题4：二次根式的除法

**题目**：计算下列各式
（1）√18 ÷ √2
（2）√27 ÷ √3
（3）√50 ÷ √2

**解答**：
（1）√18 ÷ √2 = √(18/2) = √9 = **3**

（2）√27 ÷ √3 = √(27/3) = √9 = **3**

（3）√50 ÷ √2 = √(50/2) = √25 = **5**

---

### 例题5：同类二次根式

**题目**：下列各组中，是同类二次根式的是（  ）
A. √2 和 √3
B. √2 和 √8
C. √12 和 √3
D. √3 和 2√3

**解答**：
**答案：D**

解析：
- A：√2 和 √3 不是（被开方数不同）
- B：√2 和 √8 不是（√8 = 2√2）
- C：√12 和 √3 不是（√12 = 2√3）
- D：√3 和 2√3 是（被开方数相同）

---

### 例题6：二次根式的加减

**题目**：计算下列各式
（1）√2 + 3√2
（2）√3 - √12
（3）√8 + √18

**解答**：
（1）√2 + 3√2 = **4√2**

（2）√3 - √12 = √3 - 2√3 = **-√3**

（3）√8 + √18 = 2√2 + 3√2 = **5√2**

---

### 例题7：化简二次根式

**题目**：化简下列各式
（1）√12
（2）√18
（3）√20

**解答**：
（1）√12 = √(4×3) = **2√3**

（2）√18 = √(9×2) = **3√2**

（3）√20 = √(4×5) = **2√5**

---

### 例题8：分母有理化

**题目**：化简下列各式
（1）1/√2
（2）1/(√3-1)
（3）√2/(√6+√3)

**解答**：
（1）1/√2 = √2/2

（2）1/(√3-1) = (√3+1)/[(√3-1)(√3+1)] = (√3+1)/(3-1) = **(√3+1)/2**

（3）√2/(√6+√3) = √2/[√3(√2+1)] = √2(√2-1)/[√3(√2+1)(√2-1)]
      = (2-√2)/[√3(2-1)] = (2-√2)/√3 = (2√3-√6)/3

---

### 例题9：二次根式的混合运算

**题目**：计算 (√6 + √2)(√6 - √2)

**解答**：
(√6 + √2)(√6 - √2)
= (√6)² - (√2)²
= 6 - 2
= **4**

---

### 例题10：综合应用

**题目**：已知 x = √3 + 1，求 x² - 2x + 1 的值。

**解答**：
x² - 2x + 1 = (x-1)² = (√3+1-1)² = (√3)² = **3**
