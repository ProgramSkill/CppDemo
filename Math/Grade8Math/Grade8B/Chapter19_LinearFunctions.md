# 第十九章 一次函数

## 📚 知识要点

### 19.1 函数

#### 19.1.1 变量与常量

**变量**：在一个变化过程中，数值发生变化的量叫做变量

**常量**：在一个变化过程中，数值始终不变的量叫做常量

**例子**：
- 汽车行驶时，速度不变，时间和路程是变量
- 圆的面积公式 S = πr² 中，r是变量，π是常量

#### 19.1.2 函数的概念

**定义**：在一个变化过程中，如果有两个变量x与y，对于x的每一个确定的值，y都有唯一确定的值与它对应，那么x叫做自变量，y叫做x的函数

**表示方法**：
1. 解析法：用含有自变量的式子表示函数
   - 例如：y = 2x + 1

2. 列表法：用表格表示函数关系
   - 例如：x | 1 | 2 | 3 |
     y | 3 | 5 | 7 |

3. 图象法：用图象表示函数关系

**函数值**：当x取一个确定值时，对应的y的值叫做函数值

**例子**：对于函数 y = 2x + 1
- 当 x = 1 时，y = 2×1 + 1 = 3
- 当 x = 2 时，y = 2×2 + 1 = 5

#### 19.1.3 函数的定义域

**定义域**：函数关系中自变量可以取值的范围

**求定义域的方法**：
1. 解析式有意义：分母不为0，偶次根式的被开方数≥0
2. 实际问题有意义

**例子**：
- y = 2x + 1 的定义域是**全体实数**
- y = 1/(x-1) 的定义域是**x ≠ 1**
- y = √(x-2) 的定义域是**x ≥ 2**

### 19.2 一次函数

#### 19.2.1 一次函数的概念

**定义**：函数y = kx + b（k、b是常数，k ≠ 0）叫做一次函数

**特例**：当 b = 0 时，y = kx 叫做正比例函数

**注意**：
- k ≠ 0（否则不是一次函数）
- x的次数是1
- 一次函数是整式函数

**例子**：
- y = 2x + 3（一次函数）
- y = -3x + 1（一次函数）
- y = 0.5x（正比例函数）

#### 19.2.2 一次函数的图象

**图象**：一次函数 y = kx + b 的图象是一条直线

**画法**：
1. 列表：取两个不同的x值，计算对应的y值
2. 描点：在坐标系中描出这两个点
3. 连线：过这两点作一条直线

**两点法**：
- 与x轴的交点：令 y = 0，求x
- 与y轴的交点：令 x = 0，求y

**例子**：画函数 y = 2x + 1 的图象
- 与x轴的交点：2x + 1 = 0，x = -0.5 → (-0.5, 0)
- 与y轴的交点：x = 0，y = 1 → (0, 1)
- 过这两点作直线

#### 19.2.3 一次函数的性质

**性质1**：y 随 x 的变化情况
- k > 0 时，y 随 x 的增大而增大
- k < 0 时，y 随 x 的增大而减小

**性质2**：图象的特征
- k > 0 时，图象从左到右上升
- k < 0 时，图象从左到右下降

**性质3**：截距
- b 是直线与y轴的交点的纵坐标
- b > 0 时，直线与y轴交于正半轴
- b < 0 时，直线与y轴交于负半轴
- b = 0 时，直线经过原点

**性质4**：k值的影响
- |k| 越大，直线越陡
- |k| 越小，直线越平缓

**性质5**：图象的象限分布
- 经过几个象限取决于k和b的符号

#### 19.2.4 正比例函数

**定义**：函数 y = kx（k是常数，k ≠ 0）叫做正比例函数

**性质**：
1. 图象是经过原点的一条直线
2. k > 0 时，图象在第一、三象限
3. k < 0 时，图象在第二、四象限

**注意**：正比例函数是特殊的一次函数

### 19.3 一次函数与方程、不等式

#### 19.3.1 一次函数与方程

**关系**：一次函数 y = kx + b
- 当 y = 0 时，kx + b = 0（一元一次方程）
- 解为 x = -b/k（这是函数图象与x轴的交点）

**应用**：用函数观点解方程

**例子**：解方程 2x + 1 = 0
解：x = -½（这是函数 y = 2x + 1 与x轴的交点）

#### 19.3.2 一次函数与不等式

**关系**：一次函数 y = kx + b
- 当 y > 0 时，kx + b > 0（一元一次不等式）
- 当 y < 0 时，kx + b < 0（一元一次不等式）

**应用**：用函数观点解不等式

**例子**：解不等式 2x + 1 > 0
解：2x > -1，x > -½
（这是函数 y = 2x + 1 在x轴上方的部分）

#### 19.3.3 一次函数的应用

**应用类型**：
1. 行程问题：路程 = 速度 × 时间
2. 工程问题：工作总量 = 工作效率 × 时间
3. 经济问题：成本、利润、价格
4. 几何问题：图形的周长、面积

**步骤**：
1. 理解题意，找出等量关系
2. 设未知数，列出函数关系式
3. 根据条件求出参数
4. 解决实际问题

## 📖 典型例题

### 例题1：函数的概念

**题目**：下列各关系中，y是x的函数的是（  ）
A. y² = x
B. y = |x|
C. y = ±√x
D. x² + y² = 1

**解答**：
**答案：B**

解析：
- A：对于x=4，y有两个值±2，不是函数
- B：对于每个x，y都有唯一确定的值，是函数
- C：对于x=4，y有两个值±2，不是函数
- D：对于x=0，y有两个值±1，不是函数

---

### 例题2：函数的定义域

**题目**：求下列函数的定义域
（1）y = 2x + 1
（2）y = 1/(x-2)
（3）y = √(x-3)

**解答**：
（1）y = 2x + 1 的定义域是**全体实数**

（2）y = 1/(x-2) 的定义域是**x ≠ 2**

（3）y = √(x-3) 的定义域是**x ≥ 3**

---

### 例题3：一次函数的概念

**题目**：下列函数中，是一次函数的是（  ）
A. y = 2x²
B. y = x/2
C. y = 1/x
D. y = √x

**解答**：
**答案：B**

解析：
- A：y = 2x² 是二次函数
- B：y = x/2 是一次函数（k=½，b=0）
- C：y = 1/x 是反比例函数
- D：y = √x 不是一次函数

**题目**：画出函数 y = 2x - 1 的图象。

**解答**：
1. 列表：
   x = 0，y = -1 → (0, -1)
   x = 1，y = 1 → (1, 1)

2. 描点：在坐标系中描出 (0, -1) 和 (1, 1)

3. 连线：过这两点作一条直线

### 例题5：一次函数的性质

**题目**：对于函数 y = -2x + 3
（1）求图象与x轴的交点
（2）求图象与y轴的交点
（3）判断y随x的增大如何变化

**解答**：
（1）与x轴的交点：令 y = 0
    -2x + 3 = 0
    x = 1.5
    ∴ 交点是**(1.5, 0)**

（2）与y轴的交点：令 x = 0
    y = 3
    ∴ 交点是**(0, 3)**

（3）∵ k = -2 < 0
    ∴ y 随 x 的增大而**减小**

---

### 例题6：正比例函数

**题目**：已知 y - 2 与 x 成正比例，且当 x = 2 时，y = 4
（1）求 y 与 x 的函数关系式
（2）求当 x = 3 时，y 的值

**解答**：
（1）设 y - 2 = kx
    当 x = 2，y = 4 时：4 - 2 = 2k，k = 1
    ∴ y = x + 2

（2）当 x = 3 时：
    y = 3 + 2 = **5**

---

### 例题7：一次函数与方程

**题目**：已知一次函数 y = 2x - 4
（1）求图象与x轴的交点
（2）求方程 2x - 4 = 0 的解
（3）这两个问题有什么关系？

**解答**：
（1）与x轴的交点：令 y = 0
    2x - 4 = 0，x = 2
    ∴ 交点是**(2, 0)**

（2）方程 2x - 4 = 0 的解：**x = 2**

（3）函数图象与x轴的交点的横坐标就是方程的解

---

### 例题8：一次函数与不等式

**题目**：已知一次函数 y = 2x - 4
（1）求当 y > 0 时，x 的取值范围
（2）求不等式 2x - 4 > 0 的解集
（3）这两个问题有什么关系？

**解答**：
（1）y > 0 即 2x - 4 > 0
    2x > 4，x > 2
    ∴ x > **2**

（2）不等式 2x - 4 > 0 的解集：**x > 2**

（3）函数值大于0时，自变量的取值范围就是对应不等式的解集

### 例题9：实际问题

**题目**：某工厂有甲、乙两个仓库，每个仓库最多能存放肥料50吨
甲仓库现有肥料30吨，乙仓库现有肥料10吨
如果每天从甲仓库运出2吨，运入乙仓库1吨
（1）求x天后，甲、乙两仓库的肥料量
（2）求x天后，两仓库的肥料总量

**解答**：
（1）x天后：
    甲仓库：**30 - 2x**（吨）
    乙仓库：**10 + x**（吨）

（2）肥料总量：
    (30 - 2x) + (10 + x) = **40 - x**（吨）

---

### 例题10：综合应用

**题目**：某公司租用汽车和货车每天共运货物360吨
汽车每天运量是货车的1.5倍，货车每天运x吨
（1）求汽车每天运量（用含x的式子表示）
（2）求每天的总运量（用含x的式子表示）

**解答**：
（1）汽车每天运量：**1.5x**（吨）

（2）每天的总运量：
    x + 1.5x = **2.5x**（吨）
