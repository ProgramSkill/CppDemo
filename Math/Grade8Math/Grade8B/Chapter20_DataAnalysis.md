# 第二十章 数据的分析

## 📚 知识要点

### 20.1 数据的代表

#### 20.1.1 平均数

**定义**：一般地，如果有n个数x₁, x₂, ..., xₙ，那么这n个数的平均数是**
$$\bar{x} = \frac{x_1 + x_2 + ... + x_n}{n}$$

**简化公式**：如果n个数中，x₁出现f₁次，x₂出现f₂次，...，xk出现f_k次（f₁ + f₂ + ... + f_k = n），那么这n个数的平均数是**
$$\bar{x} = \frac{x_1f_1 + x_2f_2 + ... + x_kf_k}{f_1 + f_2 + ... + f_k}$$

**性质**：
1. 平均数反映了一组数据的集中趋势
2. 平均数与每一个数据都有关
3. 平均数易受极端值的影响

**例子**：
- 数据1, 2, 3, 4, 5的平均数：**(1+2+3+4+5)/5 = 3**
- 数据1, 1, 2, 2, 3的平均数：**(1+1+2+2+3)/5 = 1.8**

#### 20.1.2 中位数

**定义**：将一组数据按照大小顺序排列，位于中间位置的数叫做中位数

**求法**：
1. 排序：将数据从小到大（或从大到小）排列
2. 确定位置：
   - 数据个数为奇数时，中位数是中间那个数
   - 数据个数为偶数时，中位数是中间两个数的平均数

**性质**：
1. 中位数反映了一组数据的集中趋势
2. 中位数不受极端值的影响
3. 中位数只与数据的排列位置有关

**例子**：
- 数据1, 2, 3, 4, 5的中位数：**3**
- 数据1, 2, 3, 4, 5, 6的中位数：**(3+4)/2 = 3.5**

#### 20.1.3 众数

**定义**：一组数据中出现次数最多的数据叫做这组数据的众数

**性质**：
1. 众数反映了一组数据的集中趋势
2. 众数不一定存在
3. 众数可能不唯一

**例子**：
- 数据1, 2, 2, 3, 3, 3的众数：**3**
- 数据1, 2, 2, 3, 3的众数：**2和3**

#### 20.1.4 平均数、中位数、众数的选用

**平均数**：
- 优点：利用了所有数据，可靠性高
- 缺点：易受极端值的影响
- 适用：数据比较平稳，没有极端值

**中位数**：
- 优点：不受极端值的影响
- 缺点：没有充分利用所有数据
- 适用：数据中有极端值

**众数**：
- 优点：容易计算，不受极端值影响
- 缺点：可能不存在或不唯一
- 适用：数据分布有明显集中趋势

### 20.2 数据的波动

#### 20.2.1 方差

**定义**：设有n个数据x₁, x₂, ..., xₙ，各数据与它们的平均数的差的平方分别是**
$$(x_1 - \bar{x})^2, (x_2 - \bar{x})^2, ..., (x_n - \bar{x})^2$$

**我们用它们的平均数来衡量这组数据的波动大小，把这个平均数叫做这组数据的方差，记作s²**

$$s^2 = \frac{1}{n}[(x_1 - \bar{x})^2 + (x_2 - \bar{x})^2 + ... + (x_n - \bar{x})^2]$$

**性质**：
1. 方差越大，数据的波动越大
2. 方差越小，数据的波动越小
3. 方差为0时，所有数据都相等

**简化公式**：
$$s^2 = \frac{1}{n}[x_1^2 + x_2^2 + ... + x_n^2] - \bar{x}^2$$

#### 20.2.2 标准差

**定义**：方差的算术平方根叫做标准差，记作s**

$$s = \sqrt{s^2} = \sqrt{\frac{1}{n}[(x_1 - \bar{x})^2 + (x_2 - \bar{x})^2 + ... + (x_n - \bar{x})^2]}$$

**注意**：标准差与原数据的单位相同

**例子**：
- 数据1, 2, 3, 4, 5：
  平均数：3
  方差：1/5[(1-3)²+(2-3)²+(3-3)²+(4-3)²+(5-3)²] = **2**
  标准差：**√2 ≈ 1.414**

#### 20.2.3 方差的应用

**应用1**：比较两组数据的波动
- 方差大的波动大，不稳定
- 方差小的波动小，稳定

**应用2**：质量检验
- 方差小说明产品质量稳定
- 方差大说明产品质量不稳定

**应用3**：成绩分析
- 方差小说明成绩差距小
- 方差大说明成绩差距大

## 📖 典型例题

### 例题1：平均数的计算

**题目**：计算数据1, 2, 3, 4, 5的平均数。

**解答**：
平均数 = (1 + 2 + 3 + 4 + 5) ÷ 5 = **3**

---

### 例题2：加权平均数

**题目**：某班一次数学测验，得分为100分的5人，90分的20人，80分的15人，70分的10人，求全班的平均分。

**解答**：
平均分 = (100×5 + 90×20 + 80×15 + 70×10) ÷ (5 + 20 + 15 + 10)
      = (500 + 1800 + 1200 + 700) ÷ 50
      = 4200 ÷ 50
      = **84（分）**

---

### 例题3：中位数的确定

**题目**：求下列数据的中位数
（1）1, 3, 5, 7, 9
（2）1, 2, 3, 4, 5, 6

**解答**：
（1）数据已排序，共5个（奇数个）
    中位数是第3个数：**5**

（2）数据已排序，共6个（偶数个）
    中位数是第3和第4个数的平均数：(3 + 4) ÷ 2 = **3.5**

---

### 例题4：众数的确定

**题目**：求下列数据的众数
（1）1, 2, 2, 3, 3, 3
（2）1, 2, 2, 3, 3

**解答**：
（1）3出现的次数最多（3次）
    ∴ 众数是**3**

（2）2和3各出现2次
    ∴ 众数是**2和3**

---

### 例题5：平均数、中位数、众数的比较

**题目**：某公司10名员工的工资（单位：元）如下：
3000, 4000, 4000, 5000, 5000, 5000, 6000, 6000, 7000, 8000
求（1）平均数；（2）中位数；（3）众数

**解答**：
（1）平均数 = (3000 + 4000 + 4000 + 5000 + 5000 + 5000 + 6000 + 6000 + 7000 + 8000) ÷ 10
          = **5300（元）**

（2）数据已排序，共10个（偶数个）
    中位数是第5和第6个数的平均数：(5000 + 5000) ÷ 2 = **5000（元）**

（3）5000出现的次数最多（3次）
    ∴ 众数是**5000（元）**

---

### 例题6：方差的计算

**题目**：计算数据1, 2, 3, 4, 5的方差和标准差。

**解答**：
平均数：$\bar{x} = 3$

方差：
$s^2 = ⅕[(1-3)^2 + (2-3)^2 + (3-3)^2 + (4-3)^2 + (5-3)^2]$
    = ⅕[4 + 1 + 0 + 1 + 4]$
    = ⅕ × 10
    = **2**

标准差：$s = √2 ≈ 1.414$

---

### 例题7：方差的比较

**题目**：甲、乙两个小组各5名学生的数学成绩如下：
甲组：80, 85, 90, 85, 90
乙组：82, 84, 86, 88, 90
哪个小组的成绩更稳定？

**解答**：
甲组平均数：$(80 + 85 + 90 + 85 + 90) ÷ 5 = 86$
甲组方差：$s_甲^2 = ⅕[(80-86)^2 + (85-86)^2 + (90-86)^2 + (85-86)^2 + (90-86)^2]$
            = ⅕[36 + 1 + 16 + 1 + 16] = 14

乙组平均数：$(82 + 84 + 86 + 88 + 90) ÷ 5 = 86$
乙组方差：$s_乙^2 = ⅕[(82-86)^2 + (84-86)^2 + (86-86)^2 + (88-86)^2 + (90-86)^2]$
            = ⅕[16 + 4 + 0 + 4 + 16] = 8$

∵ $s_甲^2 > s_乙^2$
∴ **乙组成绩更稳定**

### 例题8：实际问题

**题目**：某商场销售A、B、C三种品牌衬衫，A品牌销售8件，B品牌销售12件，C品牌销售10件
（1）求A、B、C三种品牌衬衫的销售数量的平均数
（2）求A、B、C三种品牌衬衫的销售数量的众数

**解答**：
（1）平均数 = (8 + 12 + 10) ÷ 3 = **10（件）**

（2）12是销售最多的数量
    ∴ 众数是**B品牌**

### 例题9：综合应用

**题目**：某班50名学生的体重（单位：kg）如下：
40-45：5人
45-50：15人
50-55：20人
55-60：8人
60-65：2人

（1）求平均体重的估计值
（2）求体重的众数范围

**解答**：
（1）平均体重的估计值：
    各组中值：42.5, 47.5, 52.5, 57.5, 62.5
    平均数 ≈ (42.5×5 + 47.5×15 + 52.5×20 + 57.5×8 + 62.5×2) ÷ 50
          ≈ **52.3（kg）**

（2）50-55kg的人数最多（20人）
    ∴ 众数范围是**50-55kg**

### 例题10：数据分析

**题目**：A、B两台机床生产同一种零件，为了比较它们的性能，各抽取10个零件，测量直径（单位：mm）如下：
A机床：10, 10.1, 10.2, 9.9, 10, 10.1, 10.2, 9.9, 10, 10.1
B机床：10, 10.1, 10, 9.9, 10.2, 10.1, 10, 10.2, 10, 9.9

哪台机床的性能更稳定？

**解答**：
A机床平均数：$\bar{x}_A = 10.05$
A机床方差：$s_A^2 = 0.009$

B机床平均数：$\bar{x}_B = 10.04$
B机床方差：$s_B^2 = 0.0104$

∵ $s_A^2 < s_B^2$
∴ **A机床的性能更稳定**
