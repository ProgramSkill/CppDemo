## 一、立方的基本概念

### 1.1 什么是立方？

**立方**是指一个数自乘三次的运算结果。

**数学表达式**：a³ = a × a × a

**读法**：a³ 读作"a的立方"或"a的三次方"

**例子**：
- 2³ = 2 × 2 × 2 = 8
- 3³ = 3 × 3 × 3 = 27
- 5³ = 5 × 5 × 5 = 125
- (-2)³ = (-2) × (-2) × (-2) = -8

### 1.2 为什么叫"立方"？

**几何意义**：立方来源于几何中的**正方体体积**计算。

- 边长为 a 的正方体，体积 V = a × a × a = a³
- 这就是"立方"名称的由来

**示例**：
- 边长为 2cm 的正方体，体积 = 2³ = 8 cm³
- 边长为 3cm 的正方体，体积 = 3³ = 27 cm³

| 10 | 1000 | 10×10×10 = 1000 |

**建议**：熟记1-10的立方数，对后续学习很有帮助！

### 2.2 立方数的重要性质

#### 性质1：正数的立方是正数，负数的立方是负数

- 正数：5³ = 125 > 0
- 负数：(-5)³ = -125 < 0
- 零：0³ = 0

**规律**：立方运算**保持符号**

#### 性质2：奇数的立方是奇数，偶数的立方是偶数

- 奇数：3³ = 27（奇数）
- 偶数：4³ = 64（偶数）

**证明思路**：
- 奇数 = 2n+1，则 (2n+1)³ 展开后仍是奇数
- 偶数 = 2n，则 (2n)³ = 8n³ 必是偶数

#### 性质3：立方数的个位数规律

| 底数个位 | 立方数个位 | 示例 |
|----------|------------|------|
| 1 | 1 | 11³ = 1331 |
| 2 | 8 | 12³ = 1728 |
| 3 | 7 | 13³ = 2197 |
| 4 | 4 | 14³ = 2744 |
| 5 | 5 | 15³ = 3375 |
| 6 | 6 | 16³ = 4096 |
| 7 | 3 | 17³ = 4913 |
| 8 | 2 | 18³ = 5832 |
| 9 | 9 | 19³ = 6859 |
| 0 | 0 | 20³ = 8000 |

**记忆技巧**：
- 1、4、5、6、9、0 的立方个位数不变
- 2和8互换（2→8，8→2）
- 3和7互换（3→7，7→3）

## 三、立方根

### 3.1 立方根的定义

**定义**：如果 x³ = a，那么 x 叫做 a 的**立方根**（也叫**三次方根**）。

**符号**：∛a 或 a^(1/3)

**读法**：∛a 读作"三次根号a"或"a的立方根"

**例子**：
- ∛8 = 2（因为 2³ = 8）
- ∛27 = 3（因为 3³ = 27）
- ∛(-8) = -2（因为 (-2)³ = -8）

### 3.2 立方根的性质

#### 性质1：任何实数都有唯一的立方根

- 正数的立方根是正数：∛125 = 5
- 负数的立方根是负数：∛(-125) = -5
- 零的立方根是零：∛0 = 0

**对比平方根**：
- 平方根：正数有两个平方根（±），负数没有平方根
- 立方根：任何数都有唯一立方根

#### 性质2：立方根的运算法则

1. **乘法法则**：∛(a·b) = ∛a · ∛b
   - 例：∛(8×27) = ∛8 · ∛27 = 2 × 3 = 6

2. **除法法则**：∛(a/b) = ∛a / ∛b（b≠0）
   - 例：∛(64/8) = ∛64 / ∛8 = 4 / 2 = 2

3. **互为逆运算**：(∛a)³ = a，∛(a³) = a
   - 例：(∛5)³ = 5，∛(5³) = 5

### 3.3 立方根的计算方法

#### 方法1：直接法（完全立方数）

对于完全立方数，直接查表或记忆：
- ∛1 = 1
- ∛8 = 2
- ∛27 = 3
- ∛64 = 4
- ∛125 = 5
- ∛1000 = 10

#### 方法2：估算法（非完全立方数）

**例题**：估算 ∛50的值

**步骤**：
1. 找到相邻的完全立方数：27 < 50 < 64
2. 得出：3 < ∛50 < 4
3. 因为 50 更接近 64，所以 ∛50 ≈ 3.7

**精确值**：∛50 ≈ 3.684

#### 方法3：质因数分解法

**例题**：计算 ∛216

**步骤**：
1. 质因数分解：216 = 2³ × 3³
2. 提取立方：∛216 = ∛(2³ × 3³) = 2 × 3 = 6

## 四、立方公式与恒等式

### 4.1 两数和的立方

**(a + b)³ = a³ + 3a²b + 3ab² + b³**

**记忆方法**：系数是 1, 3, 3, 1（杨辉三角第三行）

**展开过程**：
```
(a + b)³ = (a + b)(a + b)(a + b)
         = (a + b)(a² + 2ab + b²)
         = a³ + 2a²b + ab² + a²b + 2ab² + b³
         = a³ + 3a²b + 3ab² + b³
```

**例题**：计算 (x + 2)³

**解答**：
```
(x + 2)³ = x³ + 3x²·2 + 3x·2² + 2³
         = x³ + 6x² + 12x + 8
```

### 4.2 两数差的立方

**(a - b)³ = a³ - 3a²b + 3ab² - b³**

**记忆方法**：与和的立方相比，奇数项变负号

**例题**：计算 (x - 1)³

**解答**：
```
(x - 1)³ = x³ - 3x²·1 + 3x·1² - 1³
         = x³ - 3x² + 3x - 1
```

### 4.3 立方和公式

**a³ + b³ = (a + b)(a² - ab + b²)**

**证明**：
```
(a + b)(a² - ab + b²) = a³ - a²b + ab² + a²b - ab² + b³
                       = a³ + b³
```

**应用**：因式分解

**例题**：分解 x³ + 8

**解答**：
```
x³ + 8 = x³ + 2³
       = (x + 2)(x² - 2x + 4)
```

### 4.4 立方差公式

**a³ - b³ = (a - b)(a² + ab + b²)**

**证明**：
```
(a - b)(a² + ab + b²) = a³ + a²b + ab² - a²b - ab² - b³
                       = a³ - b³
```

**例题**：分解 27x³ - 1

**解答**：
```
27x³ - 1 = (3x)³ - 1³
         = (3x - 1)(9x² + 3x + 1)
```

### 4.5 完全立方数的判定

一个数是完全立方数，当且仅当它可以表示为某个整数的立方。

**判定方法**：
1. 质因数分解，所有质因数的指数都是3的倍数
2. 例：216 = 2³ × 3³ ✓（是完全立方数）
3. 例：72 = 2³ × 3² ✗（不是完全立方数）

## 五、实际应用

### 5.1 体积计算

**问题1**：一个正方体水箱的体积是 64 立方米，求它的边长。

**解答**：
```
设边长为 a 米
a³ = 64
a = ∛64 = 4
答：边长为 4 米
```

**问题2**：要制作一个体积为 1000 cm³ 的正方体盒子，边长应该是多少？

**解答**：
```
a³ = 1000
a = ∛1000 = 10
答：边长应该是 10 cm
```

### 5.2 增长率问题

**问题**：某城市人口连续3年每年增长10%，如果现在人口是100万，3年后人口是多少？

**解答**：
```
3年后人口 = 100 × (1 + 10%)³
          = 100 × 1.1³
          = 100 × 1.331
          = 133.1 万
```

### 5.3 科学计数

**问题**：1立方米等于多少立方厘米？

**解答**：
```
1米 = 100厘米
1立方米 = (100厘米)³
        = 100³ 立方厘米
        = 1,000,000 立方厘米
```

## 七、答案与解析

### 基础题答案

1. (1) 343  (2) -27  (3) 0.125  (4) -1

2. (1) 5  (2) -4  (3) 0.1  (4) -1

### 进阶题答案

3. (1) x³ + 9x² + 27x + 27
   (2) 8a³ - 12a² + 6a - 1
   (3) 2x

4. (1) (x + 3)(x² - 3x + 9)
   (2) (2a - 1)(4a² + 2a + 1)
   (3) (x - 4)(x² + 4x + 16)

5. **解析**：
   ```
   设边长为 a
   a³ = 343
   a = ∛343 = 7 cm
   表面积 = 6a² = 6 × 49 = 294 cm²
   ```

### 挑战题答案

6. **解析**：
   ```
   a³ + b³ = (a + b)(a² - ab + b²)
           = (a + b)[(a + b)² - 3ab]
           = 5 × (25 - 18)
           = 5 × 7
           = 35
   ```

7. **证明**：
   ```
   (a + 1)³ - a³ = a³ + 3a² + 3a + 1 - a³
                  = 3a² + 3a + 1
   ```

8. **解析**：
   ```
   ∛(x - 1) = 3
   x - 1 = 3³
   x - 1 = 27
   x = 28
   ```
