# Code Examples

## Basic Examples

This directory contains practical code examples for 8051 programming:

1. **LED_Blink.c** - Simple LED blinking
2. **Button_LED.c** - Button-controlled LED
3. **Timer_Delay.c** - Accurate delays using timers
4. **Interrupt_Example.c** - External interrupt handling
5. **Serial_Echo.c** - Serial communication echo
6. **Seven_Segment.c** - Seven-segment display control
7. **LCD_Interface.c** - LCD 16x2 interfacing
8. **ADC_Example.c** - Analog-to-Digital conversion
9. **PWM_Generation.c** - PWM using timers
10. **Keypad_Interface.c** - 4x4 matrix keypad
