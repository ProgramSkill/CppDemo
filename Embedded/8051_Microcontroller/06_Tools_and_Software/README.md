# Development Tools and Software

## Compilers

### Keil µVision (Commercial)
- **Type**: IDE + C Compiler
- **Features**:
  - Integrated debugger
  - Simulator
  - Code optimization
  - Project management
- **License**: Free evaluation (2KB code limit), paid full version
- **Download**: https://www.keil.com/

### SDCC (Open Source)
- **Type**: Small Device C Compiler
- **Features**:
  - Free and open source
  - Cross-platform (Windows, Linux, Mac)
  - Good optimization
- **Download**: http://sdcc.sourceforge.net/

## Programmers

### Flash Magic
- **Type**: ISP programmer for NXP/Philips chips
- **Features**: Easy-to-use GUI, supports AT89S series
- **License**: Free
- **Download**: https://www.flashmagictool.com/

### Progisp
- **Type**: Universal programmer software
- **Features**: Supports multiple chip families
- **License**: Free

## Simulators

### Proteus
- **Type**: Circuit simulation + MCU simulation
- **Features**:
  - Visual circuit design
  - Real-time simulation
  - Virtual instruments
- **License**: Commercial (trial available)

### Keil Simulator
- **Type**: Built-in simulator in Keil µVision
- **Features**: Step-by-step debugging, register viewing

## Hardware Programmers

- **USBasp**: USB programmer for AVR/8051
- **Serial Programmer**: RS232-based programmer
- **Universal Programmer**: TL866 series

## Recommended Setup

1. **Compiler**: SDCC (free) or Keil (evaluation)
2. **Programmer**: Flash Magic + USB-to-Serial adapter
3. **Simulator**: Proteus (for testing before hardware)
4. **Editor**: VS Code with C/C++ extension
