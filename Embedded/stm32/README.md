# STM32 Hardware and Software Development Tutorial

This directory contains a complete tutorial series for STM32 microcontroller development, covering everything from basics to advanced topics.

## Table of Contents

1. [STM32 Introduction](01-STM32-Introduction.md) - STM32 family overview and selection guide
2. [Hardware Basics](02-Hardware-Basics.md) - Circuit design, minimal system, peripheral connections
3. [Development Environment Setup](03-Development-Environment-Setup.md) - IDE installation and toolchain configuration
4. [Basic Programming](04-Basic-Programming.md) - GPIO, clocks, interrupts, and basic peripherals
5. [Communication Interfaces](05-Communication-Interfaces.md) - UART, SPI, I2C protocols
6. [Advanced Peripherals](06-Advanced-Peripherals.md) - ADC, DAC, timers, DMA
7. [RTOS Development](07-RTOS-Development.md) - FreeRTOS on STM32
8. [Project Examples](08-Project-Examples.md) - Complete project case studies

## Target Audience

- Embedded development beginners
- Developers with microcontroller experience wanting to learn STM32
- Engineers needing systematic STM32 knowledge

## Recommended Learning Path

1. Read STM32 Introduction to understand the architecture
2. Study Hardware Basics to grasp circuit principles
3. Set up the development environment
4. Follow chapters sequentially
5. Practice with project examples

## Hardware Requirements

- STM32 development board (STM32F103C8T6 or STM32F407 series recommended)
- ST-Link debugger
- USB-to-Serial converter
- Common electronic components

## Software Requirements

- STM32CubeMX (graphical configuration tool)
- Keil MDK-ARM or STM32CubeIDE
- Serial terminal software
- Logic analyzer software (optional)
