# 01-学而

## 篇名说明

本篇以首章"子曰：学而时习之"的前两字"学而"命名。《论语》各篇皆以篇首前二三字命名。

## 历史背景

《论语》成书于**春秋末期**（约公元前5世纪），这一时期的社会特征深刻影响了孔子的思想与言论：

### 时代特征

| 方面 | 状况 | 影响 |
|------|------|------|
| **政治格局** | 周王室衰微，诸侯割据称雄 | 天子权威丧失，诸侯争霸，战乱频繁 |
| **社会秩序** | 礼崩乐坏，传统制度瓦解 | 等级秩序混乱，臣弑君、子弑父屡见不鲜 |
| **经济状况** | 铁器普及，生产力发展 | 井田制瓦解，私田出现，商业兴起 |
| **思想文化** | 官学下移，私学兴起 | 士阶层崛起，百家争鸣的前奏 |

### 孔子的时代使命

- **时间跨度**：孔子生于公元前551年，卒于公元前479年，正处于春秋末期向战国过渡的动荡时期
- **核心关切**：面对"礼崩乐坏"的局面，孔子立志**恢复周礼**、**推行仁政**，试图通过道德教化重建社会秩序
- **政治实践**：曾担任鲁国司寇（最高司法官），后因政治理想无法实现，带领弟子周游列国14年，寻求推行仁政的机会
- **教育贡献**：开创私学，提出"有教无类"，打破了贵族对教育的垄断

### 与本篇的关联

本篇所论述的**学习、修身、孝悌、礼信**等主题，正是孔子针对时代问题提出的解决之道：

- "学而时习之"：强调学习的重要性，培养能够担当重任的君子
- "孝弟也者，其为仁之本与"：从家庭伦理入手，重建社会秩序的根基
- "礼之用，和为贵"：提倡礼制规范，化解社会冲突
- "弟子入则孝，出则弟"：培养具有道德修养的治国人才

## 主要内容

本篇共16章，主要论述：
- 学习的重要性和方法
- 仁德修养的根本
- 孝悌之道
- 君子品格
- 处世之道

### 章节逻辑关联

| 章节范围 | 主题侧重 | 核心思想 |
|---------|---------|---------|
| 第1-4章 | 学习与修身 | 学习的快乐、自省、仁德修养 |
| 第5-8章 | 处世与为政 | 治国之道、修身原则、交友之道 |
| 第9-16章 | 孝悌礼信 | 孝道、礼制、诚信、人伦关系 |

### 易错点提示

**易混淆字词**：
- 弟（tì）同"悌"，敬爱兄长 ≠ 弟（dì）兄弟
- 省（xǐng）反省 ≠ 省（shěng）省份
- 说（yuè）通"悦" ≠ 说（shuō）说话
- 乐（lè）快乐 ≠ 乐（yuè）音乐

**易误解语句**：
- "无友不如己者"：主流解读"不要结交不如自己的人"，另一说"没有朋友是完全不如自己的"
- "贤贤易色"：重视贤者而轻视女色（易：轻视）
- "三年无改于父之道"："三年"指古代守丧制度（实为25个月），非单纯三年时间

### 孔门十哲

孔子弟子三千，其中贤人七十二，最优秀者称为"孔门十哲"，分为四科，各有侧重：

| 学科 | 核心能力 | 代表弟子 |
|------|---------|---------|
| **德行科** | 仁德修养、道德典范 | 颜回、闵子骞、冉伯牛、仲弓 |
| **言语科** | 辞令、外交、沟通 | 子贡（端木赐）、宰我 |
| **政事科** | 治国理政、政务实践 | 冉有、季路（子路） |
| **文学科** | 文献典籍、文化传承 | 子夏（卜商）、子游 |

**说明**：
- 本篇中出现的子贡属"言语科"，子夏属"文学科"
- 颜回虽未在本篇发言，但被孔子誉为最得意的弟子（德行科之首）

## 全文

子曰：「学而时习之，不亦说乎？有朋自远方来，不亦乐乎？人不知而不愠，不亦君子乎？」

有子曰：「其为人也孝弟，而好犯上者，鲜矣；不好犯上，而好作乱者，未之有也。君子务本，本立而道生。孝弟也者，其为仁之本与！」

子曰：「巧言令色，鲜矣仁！」

曾子曰：「吾日三省吾身：为人谋而不忠乎？与朋友交而不信乎？传不习乎？」

子曰：「道千乘之国，敬事而信，节用而爱人，使民以时。」

子曰：「弟子入则孝，出则弟，谨而信，泛爱众，而亲仁。行有余力，则以学文。」

子夏曰：「贤贤易色；事父母，能竭其力；事君，能致其身；与朋友交，言而有信。虽曰未学，吾必谓之学矣。」

子曰：「君子不重，则不威；学则不固。主忠信。无友不如己者。过则勿惮改。」

曾子曰：「慎终追远，民德归厚矣。」

子禽问于子贡曰：「夫子至于是邦也，必闻其政，求之与，抑与之与？」子贡曰：「夫子温、良、恭、俭、让以得之。夫子之求之也，其诸异乎人之求之与？」

子曰：「父在，观其志；父没，观其行；三年无改于父之道，可谓孝矣。」

有子曰：「礼之用，和为贵。先王之道，斯为美；小大由之。有所不行，知和而和，不以礼节之，亦不可行也。」

有子曰：「信近于义，言可复也。恭近于礼，远耻辱也。因不失其亲，亦可宗也。」

子曰：「君子食无求饱，居无求安，敏于事而慎于言，就有道而正焉，可谓好学也已。」

子贡曰：「贫而无谄，富而无骄，何如？」子曰：「可也。未若贫而乐，富而好礼者也。」子贡曰：「《诗》云：『如切如磋，如琢如磨』，其斯之谓与？」子曰：「赐也，始可与言《诗》已矣，告诸往而知来者。」

子曰：「不患人之不己知，患不知人也。」

## 对话重现

### 第1章
```
🎓 子（孔子）：
学而时习之，不亦说乎？
有朋自远方来，不亦乐乎？
人不知而不愠，不亦君子乎？
```

🔖 **重点发音**：说（yuè）乐（lè）
📖 **重点词汇**：
- 时习：时常温习
- 朋：志同道合的人（古代"朋"多指同门、同道，与"友"有别；"友"侧重情谊相投）
- 愠（yùn）：怨恨、生气
- 君子：有道德修养的人
🔗 **句型特点**：
- "不亦...乎"：反问句式，"不也是很...吗"（本篇常见，如第1、3、16章）
- "学而时习之"：省略了主语"（人）"

### 第2章
```
📚 有子：
其为人也孝弟，而好犯上者，鲜矣；
不好犯上，而好作乱者，未之有也。
君子务本，本立而道生。
孝弟也者，其为仁之本与！
```

🔖 **重点发音**：弟（tì）鲜（xiǎn）与（yú）
📖 **重点词汇**：
- 孝弟（tì）：孝顺父母，敬爱兄长
- 犯上：冒犯上级
- 鲜（xiǎn）：少
- 务本：致力于根本
- 仁：仁德、爱心
🔗 **句型特点**：
- "其为人也孝弟"："其...也"表示停顿，强调主语
- "未之有也"：宾语前置，正常语序是"未有之也"（否定句中代词"之"作宾语前置）
- "与（yú）"：句末语气词，表疑问或感叹，同"欤"

### 第3章
```
🎓 子（孔子）：
巧言令色，鲜矣仁！
```

🔖 **重点发音**：鲜（xiǎn）矣（yǐ）
📖 **重点词汇**：
- 巧言：花言巧语
- 令色："令"为形容词，美好的、和善的；"令色"指刻意装出的和善、讨好的脸色（体现虚伪做作，契合"鲜矣仁"的批判态度）
- 鲜（xiǎn）：少
- 矣（yǐ）：文言助词，用于句末，表示感叹或陈述语气，相当于"了"或"啊"
🔗 **句型特点**：
- "鲜矣仁"：定语后置+主谓倒装，完整为"巧言令色之人，仁鲜矣"（先省略中心词"人"，再主谓倒装）
- 省略句，完整意思是"巧言令色之人，鲜矣仁"（仁德很少啊！）

### 第4章
```
🧘 曾子：
吾日三省吾身：
为人谋而不忠乎？
与朋友交而不信乎？
传不习乎？
```

🔖 **重点发音**：省（xǐng）传（chuán）
📖 **重点词汇**：
- 三省（xǐng）：多次反省
- 谋：谋划、办事
- 忠：尽心尽力
- 信：诚信（此处"信"既指说话算数，也指交往中的真诚无欺，与第7章子夏"与朋友交，言而有信"的"信"同义，是儒家交友的核心准则）
- 传（chuán）：老师传授的知识
🔗 **句型特点**：
- "三省"：三表示多次，非确指三次
- 三句都是省略句，省略了"吾"
  - 完整："（吾）为人谋而不忠乎？"
  - 完整："（吾）与朋友交而不信乎？"
  - 完整："（吾）传（所）不习乎？"

### 第5章
```
🎓 子（孔子）：
道千乘之国，敬事而信，
节用而爱人，使民以时。
```

🔖 **重点发音**：乘（shèng）
📖 **重点词汇**：
- 道：治理
- 千乘（shèng）之国：拥有千辆兵车的诸侯国（一乘含兵车一辆、甲士三人、步卒七十二人；春秋时期千乘之国为中等诸侯国，实力较强）
- 敬事：严肃认真对待政事
- 节用：节省费用
- 使民以时：按恰当的时节役使百姓
💡 **文化背景**：
- 春秋时期"千乘之国"的兵力约为3000甲士+72000步卒，合计7.5万人左右，是中等诸侯国的军事规模
- 孔子的治理建议（敬事而信、节用爱人等）针对的是有一定实力但需注重民生的诸侯国
🔗 **句型特点**：
- "道"用作动词，"治理"之意
- "使民以时"：介词结构后置，正常语序是"以时使民"

### 第6章
```
🎓 子（孔子）：
弟子入则孝，出则弟，谨而信，
泛爱众，而亲仁。
行有余力，则以学文。
```

🔖 **重点发音**：弟（tì）
📖 **重点词汇**：
- 弟（tì）：同"悌"，敬爱兄长
- 谨：谨慎
- 泛爱：广泛爱护（"泛爱众"是儒家"仁爱"的延伸，指对众人有仁爱之心，而非无差别的平等之爱）
- 亲仁：亲近有仁德的人（与"泛爱众"形成层次，体现儒家"爱有差等"的伦理观）
- 文：文献典籍
🔗 **句型特点**：
- "入则孝，出则弟"：对偶句，入/出、孝/弟相对
- 省略主语"弟子"
- "入"指在家，"出"指外出
✨ **思想内涵**：
- "泛爱众"与"亲仁"体现儒家"爱有差等"的伦理观：先爱亲（孝悌），再爱众人（泛爱），特别亲近有仁德之人（亲仁），形成由近及远、有层次的仁爱体系

### 第7章
```
✍️ 子夏：
贤贤易色；
事父母，能竭其力；
事君，能致其身；
与朋友交，言而有信。
虽曰未学，吾必谓之学矣。
```

🔖 **重点发音**：易（yì）
📖 **重点词汇**：
- 贤贤：尊重贤者（第一个"贤"作动词用）
- 易（yì）：轻视（此处核心义）
- 致其身：奉献生命
- 言而有信：说话算数
🔗 **句型特点**：
- "贤贤易色"：第一个"贤"是动词（尊重），第二个"贤"是名词（贤者）；易，轻视；全句意为"重视贤者而轻视女色"

### 第8章
```
🎓 子（孔子）：
君子不重，则不威；学则不固。
主忠信。无友不如己者。
过则勿惮改。
```

🔖 **重点发音**：重（zhòng）惮（dàn）
📖 **重点词汇**：
- 重（zhòng）：庄重
- 威：威严
- 固：牢固
- 主忠信：以忠信为主
- 惮（dàn）：害怕
🔗 **句型特点**：
- "无友不如己者"："友"作动词用，"结交朋友"
- 省略句："（君子）无友不如己者"
💡 **多元解读**：
📌 语境提示：此句与"主忠信""过则勿惮改"并列，核心是倡导"以优秀者为友、以忠信为交友原则"，避免误解为"功利性排斥不如自己的人"
- 主流：不要结交道德学问上不如自己的人（此处"不如己"侧重道德、学问，核心是倡导向优秀者学习）
- 另一说：没有朋友是完全不如自己的（每个人都有可取之处）
✨ 综合理解：结合孔子"友直、友谅、友多闻"（《论语·季氏》）的交友观，两种解读均有经典依据，应理解为"择善而从，以友辅仁"

### 第9章
```
🧘 曾子：
慎终追远，民德归厚矣。
```

🔖 **重点发音**：无
📖 **重点词汇**：
- 慎终：谨慎对待父母的丧事
- 追远：追念祭祀远代祖先
- 厚：淳厚
🔗 **句型特点**：
- 省略主语："（君子/统治者）慎终追远"

### 第10章
```
❓ 子禽：
夫子至于是邦也，必闻其政，
求之与，抑与之与？

🗣️ 子贡：
夫子温、良、恭、俭、让以得之。
夫子之求之也，其诸异乎人之求之与？
```

🔖 **重点发音**：与（yú）诸（zhū）
📖 **重点词汇**：
- 夫子：古代对老师、有学问德行之人的尊称，类似现代的"老师"或"先生"
  - 在《论语》中：专门用来尊称孔子
  - 词源：夫（发语词，无实义）+ 子（对男子的尊称）= 夫子（老师、学者的尊称）
  - 在"夫子温、良、恭、俭、让以得之"中：子贡用"夫子"来称呼自己的老师孔子，表示尊敬。这句话的意思是"老师（孔子）以温和、善良、恭敬、节俭、谦让的品德获得了各国诸侯的信任"
- 邦：诸侯国
- 抑：还是
- 诸（zhū）：之乎的合音
- 温良恭俭让：温和、善良、恭敬、节俭、谦让
🔗 **句型特点**：
- "与（yú）"：句末语气词，表疑问或感叹，同"欤"
- "求之与，抑与之与"："与"是句末语气词，表示疑问
- "其诸...与"："其诸"是表示推测的语气词，"大概...吧"

### 第11章
```
🎓 子（孔子）：
父在，观其志；
父没，观其行；
三年无改于父之道，可谓孝矣。
```

🔖 **重点发音**：没（mò）
📖 **重点词汇**：
- 没（mò）：去世
- 行：行为
- 三年：古代三年之丧（实为25个月），体现对父亲的敬重
- 道：原则、做法（此处"父之道"特指父亲符合道义、仁德的处事原则，非指所有行为；若父亲之道有偏颇，无需固守）
🔗 **句型特点**：
- "父在，观其志"：省略了"观其志者"的主语
- 对偶句：父在/父没、观其志/观其行相对
💡 **文化背景**：
- "三年之丧"源于上古礼制，孔子推崇其核心是"思慕父母"的孝心表达
- 25个月是"满三年"的虚数表述（古代以农历计，25个月约为两年多，接近"三年"的情感认知）
- 体现儒家"以礼表孝"的伦理思想：通过外在礼制表达内在孝心
✨ **思想内涵**：
- 此处体现孔子"孝"的观念，但也强调"义以为上"（《论语·阳货》），非无条件盲从父亲的一切做法

### 第12章
```
📚 有子：
礼之用，和为贵。
先王之道，斯为美；小大由之。
有所不行，知和而和，
不以礼节之，亦不可行也。
```

🔖 **重点发音**：无
📖 **重点词汇**：
- 礼：礼制、礼仪
- 和：和谐、恰当
- 先王：古代圣王
- 斯：这
- 节：节制、规范
🔗 **句型特点**：
- "礼之用"："之"用于主谓之间取消句子独立性（主谓之间取消独立性句型）
- "斯为美"：斯，代词"这"

### 第13章
```
📚 有子：
信近于义，言可复也。
恭近于礼，远耻辱也。
因不失其亲，亦可宗也。
```

🔖 **重点发音**：复（fù）
📖 **重点词汇**：
- 信：诺言
- 义：道义、合理
- 复（fù）：实践、兑现
- 恭：恭敬
- 远（yuàn）：远离、避免
- 宗：尊崇、效法
🔗 **句型特点**：
- "因不失其亲"：因，依靠；指依靠亲近且有德行的人，不失去自己的亲族关系
- "亦可宗也"：也值得效法

### 第14章
```
🎓 子（孔子）：
君子食无求饱，居无求安，
敏于事而慎于言，
就有道而正焉，可谓好学也已。
```

🔖 **重点发音**：无
📖 **重点词汇**：
- 敏：勤勉、敏捷
- 慎：谨慎
- 就：靠近、追随
- 有道：有道德学问的人
- 正：匡正（此处特指以有道之人的言行匡正自己的思想、行为，"正"的对象是自身的品德与学问）
🔗 **句型特点**：
- "就有道而正焉"：省略主语"自己"，完整为"（君子）就有道而正（自己）焉"
- "正"使动用法，"使...端正"（使动用法的具体指向：使自己的品德与学问端正）

### 第15章
```
🗣️ 子贡：
贫而无谄，富而无骄，何如？

🎓 子（孔子）：
可也。未若贫而乐，富而好礼者也。

🗣️ 子贡：
《诗》云：『如切如磋，如琢如磨』，
其斯之谓与？

🎓 子（孔子）：
赐也，始可与言《诗》已矣，
告诸往而知来者。
```

🔖 **重点发音**：谄（chǎn）与（yú）
📖 **重点词汇**：
- 谄（chǎn）：巴结奉承
- 骄：骄傲
- 贫而乐：贫穷却乐于道
- 好礼：喜好礼制
- 切磋琢磨：出自《诗经·卫风·淇奥》，原指加工骨角玉石，此处比喻修养德行
- 赐：子贡的名
- 告诸往而知来者：告诉他过去的事就能推知未来的事
🔗 **句型特点**：
- "何如"：固定句式，"怎么样"
- "未若"：不如、比不上
- "与（yú）"：句末语气词，表疑问或感叹，同"欤"
- "其斯之谓与"：宾语前置，正常语序是"其谓之斯与"（大概说的就是这个吧）

### 第16章
```
🎓 子（孔子）：
不患人之不己知，患不知人也。
```

🔖 **重点发音**：无
📖 **重点词汇**：
- 患：担心
- 人之不己知：别人不了解自己（宾语前置）
🔗 **句型特点**：
- "不己知"：宾语前置，正常语序是"不知己"（否定句中代词"己"作宾语前置）
- "人之不己知"："之"取消句子独立性，"人之不己知"作为主谓短语作"患"的宾语
- 对偶句：人之不己知 / 不知人也
- 宾语前置规律：本篇多次出现否定句中代词作宾语前置（如第2章"未之有"、第16章"不己知"）

## 重要语句

- **学而时习之，不亦说乎** - 强调学习要不断温习实践
- **吾日三省吾身** - 提倡自我反省
- **君子务本，本立而道生** - 强调根本的重要性
- **礼之用，和为贵** - 礼的运用以和谐为贵
- **不患人之不己知，患不知人也** - 不要担心别人不了解自己，而要担心自己不了解别人
