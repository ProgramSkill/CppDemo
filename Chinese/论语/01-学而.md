# 01-学而

## 篇名说明

本篇以首章"子曰：学而时习之"的前两字"学而"命名。《论语》各篇皆以篇首前二三字命名。

## 主要内容

本篇共16章，主要论述：
- 学习的重要性和方法
- 仁德修养的根本
- 孝悌之道
- 君子品格
- 处世之道

### 章节逻辑关联

| 章节范围 | 主题侧重 | 核心思想 |
|---------|---------|---------|
| 第1-4章 | 学习与修身 | 学习的快乐、自省、仁德修养 |
| 第5-8章 | 处世与为政 | 治国之道、修身原则、交友之道 |
| 第9-16章 | 孝悌礼信 | 孝道、礼制、诚信、人伦关系 |

### 易错点提示

**易混淆字词**：
- 弟（tì）同"悌"，敬爱兄长 ≠ 弟（dì）兄弟
- 省（xǐng）反省 ≠ 省（shěng）省份
- 说（yuè）通"悦" ≠ 说（shuō）说话
- 乐（lè）快乐 ≠ 乐（yuè）音乐

**易误解语句**：
- "无友不如己者"：主流解读"不要结交不如自己的人"，另一说"没有朋友是完全不如自己的"
- "贤贤易色"：重视贤者而轻视女色（易：轻视）
- "三年无改于父之道"："三年"指古代守丧制度（实为25个月），非单纯三年时间

## 全文

子曰：「学而时习之，不亦说乎？有朋自远方来，不亦乐乎？人不知而不愠，不亦君子乎？」

有子曰：「其为人也孝弟，而好犯上者，鲜矣；不好犯上，而好作乱者，未之有也。君子务本，本立而道生。孝弟也者，其为仁之本与！」

子曰：「巧言令色，鲜矣仁！」

曾子曰：「吾日三省吾身：为人谋而不忠乎？与朋友交而不信乎？传不习乎？」

子曰：「道千乘之国，敬事而信，节用而爱人，使民以时。」

子曰：「弟子入则孝，出则弟，谨而信，泛爱众，而亲仁。行有余力，则以学文。」

子夏曰：「贤贤易色；事父母，能竭其力；事君，能致其身；与朋友交，言而有信。虽曰未学，吾必谓之学矣。」

子曰：「君子不重，则不威；学则不固。主忠信。无友不如己者。过则勿惮改。」

曾子曰：「慎终追远，民德归厚矣。」

子禽问于子贡曰：「夫子至于是邦也，必闻其政，求之与，抑与之与？」子贡曰：「夫子温、良、恭、俭、让以得之。夫子之求之也，其诸异乎人之求之与？」

子曰：「父在，观其志；父没，观其行；三年无改于父之道，可谓孝矣。」

有子曰：「礼之用，和为贵。先王之道，斯为美；小大由之。有所不行，知和而和，不以礼节之，亦不可行也。」

有子曰：「信近于义，言可复也。恭近于礼，远耻辱也。因不失其亲，亦可宗也。」

子曰：「君子食无求饱，居无求安，敏于事而慎于言，就有道而正焉，可谓好学也已。」

子贡曰：「贫而无谄，富而无骄，何如？」子曰：「可也。未若贫而乐，富而好礼者也。」子贡曰：「《诗》云：『如切如磋，如琢如磨』，其斯之谓与？」子曰：「赐也，始可与言《诗》已矣，告诸往而知来者。」

子曰：「不患人之不己知，患不知人也。」

## 对话重现

### 第1章
```
🎓 子（孔子）：
学而时习之，不亦说乎？
有朋自远方来，不亦乐乎？
人不知而不愠，不亦君子乎？
```

🔖 **重点发音**：说（yuè）乐（lè）
📖 **重点词汇**：
- 时习：时常温习
- 朋：志同道合的人（古代"朋"多指同门、同道，与"友"有别；"友"侧重情谊相投）
- 愠（yùn）：怨恨、生气
- 君子：有道德修养的人
🔗 **句型特点**：
- "不亦...乎"：反问句式，"不也是很...吗"（本篇常见，如第1、3、16章）
- "学而时习之"：省略了主语"（人）"

### 第2章
```
📚 有子：
其为人也孝弟，而好犯上者，鲜矣；
不好犯上，而好作乱者，未之有也。
君子务本，本立而道生。
孝弟也者，其为仁之本与！
```

🔖 **重点发音**：弟（tì）鲜（xiǎn）与（yú）
📖 **重点词汇**：
- 孝弟（tì）：孝顺父母，敬爱兄长
- 犯上：冒犯上级
- 鲜（xiǎn）：少
- 务本：致力于根本
- 仁：仁德、爱心
🔗 **句型特点**：
- "其为人也孝弟"："其...也"表示停顿，强调主语
- "未之有也"：宾语前置，正常语序是"未有之也"（否定句中代词"之"作宾语前置）
- "与（yú）"：句末语气词，表疑问或感叹，同"欤"

### 第3章
```
🎓 子（孔子）：
巧言令色，鲜矣仁！
```

🔖 **重点发音**：鲜（xiǎn）
📖 **重点词汇**：
- 巧言：花言巧语
- 令色："令"为形容词，美好的、和善的；"令色"指伪装和善的脸色
- 鲜（xiǎn）：少
🔗 **句型特点**：
- "鲜矣仁"：定语后置+主谓倒装，完整为"巧言令色之人，仁鲜矣"（先省略中心词"人"，再主谓倒装）
- 省略句，完整意思是"巧言令色之人，鲜矣仁"

### 第4章
```
🧘 曾子：
吾日三省吾身：
为人谋而不忠乎？
与朋友交而不信乎？
传不习乎？
```

🔖 **重点发音**：省（xǐng）传（chuán）
📖 **重点词汇**：
- 三省（xǐng）：多次反省
- 谋：谋划、办事
- 忠：尽心尽力
- 信：诚信
- 传（chuán）：老师传授的知识
🔗 **句型特点**：
- "三省"：三表示多次，非确指三次
- 三句都是省略句，省略了"吾"
  - 完整："（吾）为人谋而不忠乎？"
  - 完整："（吾）与朋友交而不信乎？"
  - 完整："（吾）传（所）不习乎？"

### 第5章
```
🎓 子（孔子）：
道千乘之国，敬事而信，
节用而爱人，使民以时。
```

🔖 **重点发音**：乘（shèng）
📖 **重点词汇**：
- 道：治理
- 千乘（shèng）之国：拥有千辆兵车的诸侯国（一乘含兵车一辆、甲士三人、步卒七十二人，千乘之国为中等诸侯国）
- 敬事：严肃认真对待政事
- 节用：节省费用
- 使民以时：按恰当的时节役使百姓
🔗 **句型特点**：
- "道"用作动词，"治理"之意
- "使民以时"：介词结构后置，正常语序是"以时使民"

### 第6章
```
🎓 子（孔子）：
弟子入则孝，出则弟，谨而信，
泛爱众，而亲仁。
行有余力，则以学文。
```

🔖 **重点发音**：弟（tì）
📖 **重点词汇**：
- 弟（tì）：同"悌"，敬爱兄长
- 谨：谨慎
- 泛爱：广泛爱护
- 亲仁：亲近有仁德的人
- 文：文献典籍
🔗 **句型特点**：
- "入则孝，出则弟"：对偶句，入/出、孝/弟相对
- 省略主语"弟子"
- "入"指在家，"出"指外出

### 第7章
```
✍️ 子夏：
贤贤易色；
事父母，能竭其力；
事君，能致其身；
与朋友交，言而有信。
虽曰未学，吾必谓之学矣。
```

🔖 **重点发音**：易（yì）
📖 **重点词汇**：
- 贤贤：尊重贤者（第一个"贤"作动词用）
- 易（yì）：轻视（此处核心义），指重视贤者而轻视女色
- 致其身：奉献生命
- 言而有信：说话算数
🔗 **句型特点**：
- "贤贤易色"：第一个"贤"是动词（尊重），第二个"贤"是名词（贤者）
- "贤贤易色"：尊重贤者而轻视女色（易：轻视）

### 第8章
```
🎓 子（孔子）：
君子不重，则不威；学则不固。
主忠信。无友不如己者。
过则勿惮改。
```

🔖 **重点发音**：重（zhòng）惮（dàn）
📖 **重点词汇**：
- 重（zhòng）：庄重
- 威：威严
- 固：牢固
- 主忠信：以忠信为主
- 惮（dàn）：害怕
🔗 **句型特点**：
- "无友不如己者"："友"作动词用，"结交朋友"
- 省略句："（君子）无友不如己者"
💡 **多元解读**：
- 主流：不要结交不如自己的人
- 另一说：没有朋友是完全不如自己的（每个人都有可取之处）

### 第9章
```
🧘 曾子：
慎终追远，民德归厚矣。
```

🔖 **重点发音**：无
📖 **重点词汇**：
- 慎终：谨慎对待父母的丧事
- 追远：追念祭祀远代祖先
- 厚：淳厚
🔗 **句型特点**：
- 省略主语："（君子/统治者）慎终追远"

### 第10章
```
❓ 子禽：
夫子至于是邦也，必闻其政，
求之与，抑与之与？

🗣️ 子贡：
夫子温、良、恭、俭、让以得之。
夫子之求之也，其诸异乎人之求之与？
```

🔖 **重点发音**：与（yú）诸（zhū）
📖 **重点词汇**：
- 夫子：对孔子的尊称
- 邦：诸侯国
- 抑：还是
- 诸（zhū）：之乎的合音
- 温良恭俭让：温和、善良、恭敬、节俭、谦让
🔗 **句型特点**：
- "与（yú）"：句末语气词，表疑问或感叹，同"欤"
- "求之与，抑与之与"："与"是句末语气词，表示疑问
- "其诸...与"："其诸"是表示推测的语气词，"大概...吧"

### 第11章
```
🎓 子（孔子）：
父在，观其志；
父没，观其行；
三年无改于父之道，可谓孝矣。
```

🔖 **重点发音**：没（mò）
📖 **重点词汇**：
- 没（mò）：去世
- 行：行为
- 三年：古代三年之丧（实为25个月），体现对父亲的敬重
- 道：原则、做法
🔗 **句型特点**：
- "父在，观其志"：省略了"观其志者"的主语
- 对偶句：父在/父没、观其志/观其行相对
💡 **文化背景**：
- "三年"非单纯时间，指古代"三年之丧"礼制（子女为父母守丧三年，实际为25个月）

### 第12章
```
📚 有子：
礼之用，和为贵。
先王之道，斯为美；小大由之。
有所不行，知和而和，
不以礼节之，亦不可行也。
```

🔖 **重点发音**：无
📖 **重点词汇**：
- 礼：礼制、礼仪
- 和：和谐、恰当
- 先王：古代圣王
- 斯：这
- 节：节制、规范
🔗 **句型特点**：
- "礼之用"："之"连接主语和动词，无实义
- "斯为美"：斯，代词"这"

### 第13章
```
📚 有子：
信近于义，言可复也。
恭近于礼，远耻辱也。
因不失其亲，亦可宗也。
```

🔖 **重点发音**：复（fù）
📖 **重点词汇**：
- 信：诺言
- 义：道义、合理
- 复（fù）：实践、兑现
- 恭：恭敬
- 远（yuàn）：远离、避免
- 宗：尊崇、效法
🔗 **句型特点**：
- "因不失其亲"：因，凭借；依靠不失去自己的亲族
- "亦可宗也"：也值得效法

### 第14章
```
🎓 子（孔子）：
君子食无求饱，居无求安，
敏于事而慎于言，
就有道而正焉，可谓好学也已。
```

🔖 **重点发音**：无
📖 **重点词汇**：
- 敏：勤勉、敏捷
- 慎：谨慎
- 就：靠近、追随
- 有道：有道德学问的人
- 正：匡正
🔗 **句型特点**：
- "就有道而正焉"：省略主语"自己"，完整为"（君子）就有道而正（自己）焉"
- "正"使动用法，"使...端正"

### 第15章
```
🗣️ 子贡：
贫而无谄，富而无骄，何如？

🎓 子（孔子）：
可也。未若贫而乐，富而好礼者也。

🗣️ 子贡：
《诗》云：『如切如磋，如琢如磨』，
其斯之谓与？

🎓 子（孔子）：
赐也，始可与言《诗》已矣，
告诸往而知来者。
```

🔖 **重点发音**：谄（chǎn）与（yú）
📖 **重点词汇**：
- 谄（chǎn）：巴结奉承
- 骄：骄傲
- 贫而乐：贫穷却乐于道
- 好礼：喜好礼制
- 切磋琢磨：加工骨角玉石，比喻修养德行
- 赐：子贡的名
- 告诸往而知来者：告诉他过去的事就能推知未来的事
🔗 **句型特点**：
- "何如"：固定句式，"怎么样"
- "未若"：不如、比不上
- "与（yú）"：句末语气词，表疑问或感叹，同"欤"
- "其斯之谓与"：宾语前置，正常语序是"其谓之斯与"（大概说的就是这个吧）

### 第16章
```
🎓 子（孔子）：
不患人之不己知，患不知人也。
```

🔖 **重点发音**：无
📖 **重点词汇**：
- 患：担心
- 人之不己知：别人不了解自己（宾语前置）
🔗 **句型特点**：
- "不己知"：宾语前置，正常语序是"不知己"（否定句中代词"己"作宾语前置）
- "人之不己知"："之"取消句子独立性，不翻译
- 对偶句：人之不己知 / 不知人也
- 宾语前置规律：本篇多次出现否定句中代词作宾语前置（如第2章"未之有"、第16章"不己知"）

## 重要语句

- **学而时习之，不亦说乎** - 强调学习要不断温习实践
- **吾日三省吾身** - 提倡自我反省
- **君子务本，本立而道生** - 强调根本的重要性
- **礼之用，和为贵** - 礼的运用以和谐为贵
- **不患人之不己知，患不知人也** - 不要担心别人不了解自己，而要担心自己不了解别人
