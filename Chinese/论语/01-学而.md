# 01-学而

## 篇名说明

本篇以首章"子曰：学而时习之"的前两字"学而"命名。《论语》各篇皆以篇首前二三字命名。

## 主要内容

本篇共16章，主要论述：
- 学习的重要性和方法
- 仁德修养的根本
- 孝悌之道
- 君子品格
- 处世之道

## 全文

子曰：「学而时习之，不亦说乎？有朋自远方来，不亦乐乎？人不知而不愠，不亦君子乎？」

有子曰：「其为人也孝弟，而好犯上者，鲜矣；不好犯上，而好作乱者，未之有也。君子务本，本立而道生。孝弟也者，其为仁之本与！」

子曰：「巧言令色，鲜矣仁！」

曾子曰：「吾日三省吾身：为人谋而不忠乎？与朋友交而不信乎？传不习乎？」

子曰：「道千乘之国，敬事而信，节用而爱人，使民以时。」

子曰：「弟子入则孝，出则弟，谨而信，泛爱众，而亲仁。行有余力，则以学文。」

子夏曰：「贤贤易色；事父母，能竭其力；事君，能致其身；与朋友交，言而有信。虽曰未学，吾必谓之学矣。」

子曰：「君子不重，则不威；学则不固。主忠信。无友不如己者。过则勿惮改。」

曾子曰：「慎终追远，民德归厚矣。」

子禽问于子贡曰：「夫子至于是邦也，必闻其政，求之与，抑与之与？」子贡曰：「夫子温、良、恭、俭、让以得之。夫子之求之也，其诸异乎人之求之与？」

子曰：「父在，观其志；父没，观其行；三年无改于父之道，可谓孝矣。」

有子曰：「礼之用，和为贵。先王之道，斯为美；小大由之。有所不行，知和而和，不以礼节之，亦不可行也。」

有子曰：「信近于义，言可复也。恭近于礼，远耻辱也。因不失其亲，亦可宗也。」

子曰：「君子食无求饱，居无求安，敏于事而慎于言，就有道而正焉，可谓好学也已。」

子贡曰：「贫而无谄，富而无骄，何如？」子曰：「可也。未若贫而乐，富而好礼者也。」子贡曰：「《诗》云：『如切如磋，如琢如磨』，其斯之谓与？」子曰：「赐也，始可与言《诗》已矣，告诸往而知来者。」

子曰：「不患人之不己知，患不知人也。」

## 对话重现

### 第1章
```
🎓 子（孔子）：
学而时习之，不亦说乎？
有朋自远方来，不亦乐乎？
人不知而不愠，不亦君子乎？
```

🔖 **重点发音**：说（yuè）乐（lè）
📖 **重点词汇**：
- 时习：时常温习
- 朋：志同道合的人
- 愠（yùn）：怨恨、生气
- 君子：有道德修养的人

### 第2章
```
📚 有子：
其为人也孝弟，而好犯上者，鲜矣；
不好犯上，而好作乱者，未之有也。
君子务本，本立而道生。
孝弟也者，其为仁之本与！
```

🔖 **重点发音**：弟（tì）鲜（xiǎn）与（yú）
📖 **重点词汇**：
- 孝弟（tì）：孝顺父母，敬爱兄长
- 犯上：冒犯上级
- 鲜（xiǎn）：少
- 务本：致力于根本
- 仁：仁德、爱心

### 第3章
```
🎓 子（孔子）：
巧言令色，鲜矣仁！
```

🔖 **重点发音**：鲜（xiǎn）
📖 **重点词汇**：
- 巧言：花言巧语
- 令色：伪装和善的脸色
- 鲜（xiǎn）：少

### 第4章
```
🧘 曾子：
吾日三省吾身：
为人谋而不忠乎？
与朋友交而不信乎？
传不习乎？
```

🔖 **重点发音**：省（xǐng）传（chuán）
📖 **重点词汇**：
- 三省（xǐng）：多次反省
- 谋：谋划、办事
- 忠：尽心尽力
- 信：诚信
- 传（chuán）：老师传授的知识

### 第5章
```
🎓 子（孔子）：
道千乘之国，敬事而信，
节用而爱人，使民以时。
```

🔖 **重点发音**：乘（shèng）
📖 **重点词汇**：
- 道：治理
- 千乘（shèng）之国：拥有千辆兵车的诸侯国
- 敬事：严肃认真对待政事
- 节用：节省费用
- 使民以时：按恰当的时节役使百姓

### 第6章
```
🎓 子（孔子）：
弟子入则孝，出则弟，谨而信，
泛爱众，而亲仁。
行有余力，则以学文。
```

🔖 **重点发音**：弟（tì）
📖 **重点词汇**：
- 弟（tì）：同"悌"，敬爱兄长
- 谨：谨慎
- 泛爱：广泛爱护
- 亲仁：亲近有仁德的人
- 文：文献典籍

### 第7章
```
✍️ 子夏：
贤贤易色；
事父母，能竭其力；
事君，能致其身；
与朋友交，言而有信。
虽曰未学，吾必谓之学矣。
```

🔖 **重点发音**：易（yì）
📖 **重点词汇**：
- 贤贤：尊重贤者（第一个"贤"作动词用）
- 易（yì）：改变、轻视
- 致其身：奉献生命
- 言而有信：说话算数

```

主忠信。无友不如己者。

```

🔖 **重点发音**：重（zhòng）惮（dàn）
📖 **重点词汇**：

- 威：威严
- 固：牢固
- 主忠信：以忠信为主
- 惮（dàn）：害怕

### 第9章
```

```

- 与（yú）：语气词

🔖 **重点发音**：复（fù）
📖 **重点词汇**：

🔖 **重点发音**：无
📖 **重点词汇**：
- 敏：勤勉、敏捷
- 慎：谨慎

### 第15章
```
🗣️ 子贡：
贫而无谄，富而无骄，何如？

🎓 子（孔子）：
可也。未若贫而乐，富而好礼者也。

🗣️ 子贡：
《诗》云：『如切如磋，如琢如磨』，
其斯之谓与？

🎓 子（孔子）：
赐也，始可与言《诗》已矣，
告诸往而知来者。
```

🔖 **重点发音**：谄（chǎn）与（yú）
📖 **重点词汇**：
- 谄（chǎn）：巴结奉承
- 骄：骄傲
- 贫而乐：贫穷却乐于道
- 好礼：喜好礼制
- 切磋琢磨：加工骨角玉石，比喻修养德行
- 赐：子贡的名
- 告诸往而知来者：告诉他过去的事就能推知未来的事

### 第16章
```
🎓 子（孔子）：
不患人之不己知，患不知人也。
```

🔖 **重点发音**：无
📖 **重点词汇**：
- 患：担心
- 人之不己知：别人不了解自己（宾语前置）

## 重要语句

- **学而时习之，不亦说乎** - 强调学习要不断温习实践
- **吾日三省吾身** - 提倡自我反省
- **君子务本，本立而道生** - 强调根本的重要性
- **礼之用，和为贵** - 礼的运用以和谐为贵
- **不患人之不己知，患不知人也** - 不要担心别人不了解自己，而要担心自己不了解别人
