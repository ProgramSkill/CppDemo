# 01-学而

## 篇名说明

本篇以首章"子曰：学而时习之"的前两字"学而"命名。《论语》各篇皆以篇首前二三字命名。

## 历史背景

《论语》成书于**春秋末期**（约公元前5世纪），这一时期的社会剧变深刻影响了孔子的思想与言论。

### 一、春秋末期的社会剧变：从制度崩塌到思想觉醒

#### （一）政治格局的深层瓦解

周王室衰微并非单纯的权力旁落，而是维系天下秩序的"封建亲戚，以藩屏周"制度彻底失效。西周时期，周天子通过分封制将土地与人民授予诸侯，诸侯对天子履行朝贡、述职、勤王等义务，形成"天子—诸侯—卿大夫—士"的层级统治。但到春秋末期，诸侯国实力此消彼长，齐、晋、楚、秦等大国通过兼并战争扩大疆域，"春秋五霸"相继崛起，周天子沦为名义上的共主，甚至需要依附强大诸侯生存。

更严重的是，诸侯国内部也出现权力下移。卿大夫阶层崛起，如鲁国的"三桓"（季孙氏、叔孙氏、孟孙氏）、晋国的"六卿"，逐渐架空诸侯权力，形成"陪臣执国命"的局面。这种"君不君、臣不臣"的层级崩塌，正是孔子所痛心的"礼崩乐坏"的核心表现。

#### （二）经济变革引发的社会结构变动

铁器的普及与牛耕技术的推广，是春秋末期生产力飞跃的关键。铁器农具相较于青铜农具更锋利、耐用，极大提高了耕作效率，使得大规模开垦私田成为可能。西周以来的井田制（土地国有，分配给农民耕种，公田收获归贵族）逐渐瓦解，私田所有者（包括诸侯、卿大夫、平民）开始按亩收税，鲁国的"初税亩"（公元前594年）便是典型标志，这一制度承认了私田的合法性。

经济基础的变革直接冲击了社会等级制度。原本"士农工商"的固定阶层界限被打破，部分平民通过开垦私田、经商致富，甚至跻身士阶层；而一些没落贵族则失去土地与特权，沦为平民。这种阶层流动使得传统的血缘等级制度松动，士阶层作为连接贵族与平民的中间力量迅速崛起。

#### （三）思想文化的转型：从官学垄断到私学勃兴

西周时期，教育由贵族垄断，官学的核心内容是"六艺"（礼、乐、射、御、书、数），目的是培养统治阶层的继承人。随着周王室衰微，掌管文化典籍的官员（如太史、卜官）流落民间，原本藏于官府的典籍文献得以扩散，为私学兴起提供了物质基础。

春秋末期的私学与官学有本质区别：官学以血缘为准入标准，私学则面向社会各阶层；官学服务于贵族统治，私学则以传授知识、培养品德、塑造治国人才为目标。孔子的私学是当时规模最大、影响最深远的，他提出"有教无类"，无论出身贵贱、地域远近，只要愿意学习，都可成为弟子。这种教育普及打破了贵族对知识的垄断，使得士阶层的队伍不断壮大，形成了"百家争鸣"的思想前提。

### 二、孔子的人生轨迹与思想溯源

#### （一）家世与早年经历对思想的影响

孔子出身于鲁国的没落贵族家庭，其先祖是宋国的贵族，后因战乱迁居鲁国。孔子的父亲叔梁纥曾担任鲁国的陬邑大夫，属于士阶层，但在孔子年幼时便去世，家庭陷入贫困。这种"贵族后裔+平民生活"的特殊经历，让孔子既熟悉周礼的核心内涵（贵族家庭的文化传承），又深刻体会到平民的生活疾苦（贫困的成长环境），这也是他后来提出"仁政""爱人"思想的重要根源。

孔子早年做过"委吏"（管理仓库）、"乘田"（管理畜牧）等基层官职，这些经历让他深入了解社会底层的运作与民众的需求，避免了脱离现实的空谈。30岁左右，孔子开始创办私学，收徒讲学，逐渐形成自己的思想体系。

#### （二）周游列国的政治实践与思想成熟

孔子51岁时担任鲁国司寇，摄相事，曾短暂推行自己的政治主张：削弱"三桓"势力，维护鲁君权威；整顿社会秩序，强调礼制规范。但由于触动了贵族的既得利益，孔子的改革遭到强烈反对，最终被迫辞职。此次政治实践的失败，让孔子认识到在单个诸侯国推行仁政的艰难，于是在55岁时带领弟子周游列国，先后游历卫、陈、蔡、宋、楚等国，历时14年。

周游列国期间，孔子多次面见诸侯，阐述自己的治国理念：以"仁"为核心，通过道德教化改善人心；以"礼"为准则，重建等级秩序；以"德治"代替"法治"，反对苛政与战争。但当时各国诸侯都忙于兼并战争，追求"霸道"而非"王道"，孔子的思想被视为"迂远而阔于事情"，始终未得到重用。

尽管政治理想未能实现，但周游列国的经历让孔子的思想更加成熟。他接触到不同地域的文化与社会现实，对"礼崩乐坏"的根源有了更深刻的认识，也对自己的思想进行了完善与调整。

#### （三）晚年归鲁与思想的系统化

68岁时，孔子在弟子冉求的帮助下回到鲁国，此时他已放弃仕途，专注于整理古代典籍与教育弟子。孔子晚年修订了《诗》《书》《礼》《乐》《易》《春秋》六部典籍，即"六经"，这些典籍成为儒家思想的重要载体，也为《论语》的成书奠定了基础。

在教育过程中，孔子的思想通过与弟子的问答逐渐系统化。《学而》篇作为《论语》的开篇，集中体现了孔子思想的核心框架：以"学习"为起点（学而时习之），以"修身"为核心（吾日三省吾身），以"孝悌"为根基（孝弟也者，其为仁之本与），以"礼信"为规范（礼之用，和为贵；与朋友交而不信乎），以"治国"为目标（道千乘之国，敬事而信）。

### 三、《学而》篇主题与时代需求的深度契合

#### （一）"学习"主题：应对士阶层的崛起需求

春秋末期，士阶层成为社会变革的核心力量，他们不再依赖血缘世袭，而是通过知识与能力获得诸侯的任用。因此，"学习"成为士阶层安身立命的关键。孔子强调"学而时习之"，并非单纯指书本知识的学习，而是包括道德修养、社会实践在内的全面学习——"学"是知识与品德的积累，"习"是将所学付诸实践，通过反复实践巩固知识、提升品德。

#### （二）"孝悌礼信"主题：重建社会秩序的迫切需要

面对"礼崩乐坏"的社会现实，孔子认为，社会秩序的崩塌源于家庭伦理的瓦解——"其为人也孝弟，而好犯上者，鲜矣；不好犯上，而好作乱者，未之有也"。在他看来，"孝"（孝顺父母）与"悌"（敬爱兄长）是家庭伦理的核心，而家庭是社会的基本单元，只要每个家庭都能遵守孝悌之道，社会的等级秩序自然能够重建。

"礼"与"信"则是社会交往的规范。春秋末期，诸侯之间、人与人之间的信任缺失，战乱与欺诈频发，孔子提出"礼之用，和为贵""信近于义，言可复也"，正是希望通过礼制规范化解社会冲突，通过诚信重建人际信任。

#### （三）"君子品格"主题：塑造理想的统治人才

孔子的最终目标是推行仁政，而仁政的实施需要具备高尚品德与治国能力的"君子"。《学而》篇中对君子品格的界定——"食无求饱，居无求安，敏于事而慎于言，就有道而正焉""贫而无谄，富而无骄""人不知而不愠"，正是针对当时统治阶层的弊病提出的。

孔子所倡导的君子，是兼具道德修养与责任担当的理想人才：他们不追求物质享受，专注于事业与品德；他们勤勉谨慎，善于向有道德的人学习；他们无论贫富都能坚守品德，不卑不亢；他们不担心别人不了解自己，只担心自己没有能力。

### 四、历史影响：《学而》篇对中国传统文化的奠基作用

《学而》篇作为《论语》的开篇，其思想不仅是孔子对春秋末期社会问题的回应，更奠定了中国传统文化的核心框架。"学而时习之"成为中国人的学习理念，强调学以致用、知行合一；"吾日三省吾身"成为自我修养的重要方法，影响了中国人的道德观念；"孝悌礼信"成为中华民族的传统美德，塑造了中国人的家庭伦理与社会交往规范；"君子品格"成为中国人的人格理想，指引着人们追求高尚的道德境界。

这些思想经过后世儒家学者的继承与发展，成为中国传统文化的核心精神，影响了中国两千多年的政治、教育、伦理与社会生活。即使在现代社会，《学而》篇中的许多思想依然具有重要的现实意义。

### 易错点提示

**易混淆字词**：
- 弟（tì）同"悌"，敬爱兄长 ≠ 弟（dì）兄弟
- 省（xǐng）反省 ≠ 省（shěng）省份
- 说（yuè）通"悦" ≠ 说（shuō）说话
- 乐（lè）快乐 ≠ 乐（yuè）音乐

**易误解语句**：
- "无友不如己者"：主流解读"不要结交不如自己的人"，另一说"没有朋友是完全不如自己的"
- "贤贤易色"：重视贤者而轻视女色（易：轻视）
- "三年无改于父之道"："三年"指古代守丧制度（实为25个月），非单纯三年时间

## 全文

子曰：「学而时习之，不亦说乎？有朋自远方来，不亦乐乎？人不知而不愠，不亦君子乎？」

有子曰：「其为人也孝弟，而好犯上者，鲜矣；不好犯上，而好作乱者，未之有也。君子务本，本立而道生。孝弟也者，其为仁之本与！」

子曰：「巧言令色，鲜矣仁！」

曾子曰：「吾日三省吾身：为人谋而不忠乎？与朋友交而不信乎？传不习乎？」

子曰：「道千乘之国，敬事而信，节用而爱人，使民以时。」

子曰：「弟子入则孝，出则弟，谨而信，泛爱众，而亲仁。行有余力，则以学文。」

子夏曰：「贤贤易色；事父母，能竭其力；事君，能致其身；与朋友交，言而有信。虽曰未学，吾必谓之学矣。」

子曰：「君子不重，则不威；学则不固。主忠信。无友不如己者。过则勿惮改。」

曾子曰：「慎终追远，民德归厚矣。」

子禽问于子贡曰：「夫子至于是邦也，必闻其政，求之与，抑与之与？」子贡曰：「夫子温、良、恭、俭、让以得之。夫子之求之也，其诸异乎人之求之与？」

子曰：「父在，观其志；父没，观其行；三年无改于父之道，可谓孝矣。」

有子曰：「礼之用，和为贵。先王之道，斯为美；小大由之。有所不行，知和而和，不以礼节之，亦不可行也。」

有子曰：「信近于义，言可复也。恭近于礼，远耻辱也。因不失其亲，亦可宗也。」

子曰：「君子食无求饱，居无求安，敏于事而慎于言，就有道而正焉，可谓好学也已。」

子贡曰：「贫而无谄，富而无骄，何如？」子曰：「可也。未若贫而乐，富而好礼者也。」子贡曰：「《诗》云：『如切如磋，如琢如磨』，其斯之谓与？」子曰：「赐也，始可与言《诗》已矣，告诸往而知来者。」

子曰：「不患人之不己知，患不知人也。」

## 对话重现

### 第1章
```
🎓 子（孔子）：
学而时习之，不亦说乎？
有朋自远方来，不亦乐乎？
人不知而不愠，不亦君子乎？
```

🔖 **重点发音**：说（yuè）乐（lè）
📖 **重点词汇**：
- 时习：时常温习
- 朋：志同道合的人（古代"朋"多指同门、同道，与"友"有别；"友"侧重情谊相投）
- 愠（yùn）：怨恨、生气
- 君子：有道德修养的人
🔗 **句型特点**：
- "不亦...乎"：反问句式，"不也是很...吗"（本篇常见，如第1、3、16章）
- "学而时习之"：省略了主语"（人）"

### 第2章
```
📚 有子：
其为人也孝弟，而好犯上者，鲜矣；
不好犯上，而好作乱者，未之有也。
君子务本，本立而道生。
孝弟也者，其为仁之本与！
```

🔖 **重点发音**：弟（tì）鲜（xiǎn）与（yú）
📖 **重点词汇**：
- 孝弟（tì）：孝顺父母，敬爱兄长
- 犯上：冒犯上级
- 鲜（xiǎn）：少
- 务本：致力于根本
- 仁：仁德、爱心
🔗 **句型特点**：
- "其为人也孝弟"："其...也"表示停顿，强调主语
- "未之有也"：宾语前置，正常语序是"未有之也"（否定句中代词"之"作宾语前置）
- "与（yú）"：句末语气词，表疑问或感叹，同"欤"

### 第3章
```
🎓 子（孔子）：
巧言令色，鲜矣仁！
```

🔖 **重点发音**：鲜（xiǎn）矣（yǐ）
📖 **重点词汇**：
- 巧言：花言巧语
- 令色："令"为形容词，美好的、和善的；"令色"指刻意装出的和善、讨好的脸色（体现虚伪做作，契合"鲜矣仁"的批判态度）
- 鲜（xiǎn）：少
- 矣（yǐ）：文言助词，用于句末，表示感叹或陈述语气，相当于"了"或"啊"
🔗 **句型特点**：
- "鲜矣仁"：定语后置+主谓倒装，完整为"巧言令色之人，仁鲜矣"（先省略中心词"人"，再主谓倒装）
- 省略句，完整意思是"巧言令色之人，鲜矣仁"（仁德很少啊！）

### 第4章
```
🧘 曾子：
吾日三省吾身：
为人谋而不忠乎？
与朋友交而不信乎？
传不习乎？
```

🔖 **重点发音**：省（xǐng）传（chuán）
📖 **重点词汇**：
- 三省（xǐng）：多次反省
- 谋：谋划、办事
- 忠：尽心尽力
- 信：诚信（此处"信"既指说话算数，也指交往中的真诚无欺，与第7章子夏"与朋友交，言而有信"的"信"同义，是儒家交友的核心准则）
- 传（chuán）：老师传授的知识
🔗 **句型特点**：
- "三省"：三表示多次，非确指三次
- 三句都是省略句，省略了"吾"
  - 完整："（吾）为人谋而不忠乎？"
  - 完整："（吾）与朋友交而不信乎？"
  - 完整："（吾）传（所）不习乎？"

### 第5章
```
🎓 子（孔子）：
道千乘之国，敬事而信，
节用而爱人，使民以时。
```

🔖 **重点发音**：乘（shèng）
📖 **重点词汇**：
- 道：治理
- 千乘（shèng）之国：拥有千辆兵车的诸侯国（一乘含兵车一辆、甲士三人、步卒七十二人；春秋时期千乘之国为中等诸侯国，实力较强）
- 敬事：严肃认真对待政事
- 节用：节省费用
- 使民以时：按恰当的时节役使百姓
💡 **文化背景**：
- 春秋时期"千乘之国"的兵力约为3000甲士+72000步卒，合计7.5万人左右，是中等诸侯国的军事规模
- 孔子的治理建议（敬事而信、节用爱人等）针对的是有一定实力但需注重民生的诸侯国
🔗 **句型特点**：
- "道"用作动词，"治理"之意
- "使民以时"：介词结构后置，正常语序是"以时使民"

### 第6章
```
🎓 子（孔子）：
弟子入则孝，出则弟，谨而信，
泛爱众，而亲仁。
行有余力，则以学文。
```

🔖 **重点发音**：弟（tì）
📖 **重点词汇**：
- 弟（tì）：同"悌"，敬爱兄长
- 谨：谨慎
- 泛爱：广泛爱护（"泛爱众"是儒家"仁爱"的延伸，指对众人有仁爱之心，而非无差别的平等之爱）
- 亲仁：亲近有仁德的人（与"泛爱众"形成层次，体现儒家"爱有差等"的伦理观）
- 文：文献典籍
🔗 **句型特点**：
- "入则孝，出则弟"：对偶句，入/出、孝/弟相对
- 省略主语"弟子"
- "入"指在家，"出"指外出
✨ **思想内涵**：
- "泛爱众"与"亲仁"体现儒家"爱有差等"的伦理观：先爱亲（孝悌），再爱众人（泛爱），特别亲近有仁德之人（亲仁），形成由近及远、有层次的仁爱体系

### 第7章
```
✍️ 子夏：
贤贤易色；
事父母，能竭其力；
事君，能致其身；
与朋友交，言而有信。
虽曰未学，吾必谓之学矣。
```

🔖 **重点发音**：易（yì）
📖 **重点词汇**：
- 贤贤：尊重贤者（第一个"贤"作动词用）
- 易（yì）：轻视（此处核心义）
- 致其身：奉献生命
- 言而有信：说话算数
🔗 **句型特点**：
- "贤贤易色"：第一个"贤"是动词（尊重），第二个"贤"是名词（贤者）；易，轻视；全句意为"重视贤者而轻视女色"

### 第8章
```
🎓 子（孔子）：
君子不重，则不威；学则不固。
主忠信。无友不如己者。
过则勿惮改。
```

🔖 **重点发音**：重（zhòng）惮（dàn）
📖 **重点词汇**：
- 重（zhòng）：庄重
- 威：威严
- 固：牢固
- 主忠信：以忠信为主
- 惮（dàn）：害怕
🔗 **句型特点**：
- "无友不如己者"："友"作动词用，"结交朋友"
- 省略句："（君子）无友不如己者"
💡 **多元解读**：
📌 语境提示：此句与"主忠信""过则勿惮改"并列，核心是倡导"以优秀者为友、以忠信为交友原则"，避免误解为"功利性排斥不如自己的人"
- 主流：不要结交道德学问上不如自己的人（此处"不如己"侧重道德、学问，核心是倡导向优秀者学习）
- 另一说：没有朋友是完全不如自己的（每个人都有可取之处）
✨ 综合理解：结合孔子"友直、友谅、友多闻"（《论语·季氏》）的交友观，两种解读均有经典依据，应理解为"择善而从，以友辅仁"

### 第9章
```
🧘 曾子：
慎终追远，民德归厚矣。
```

🔖 **重点发音**：无
📖 **重点词汇**：
- 慎终：谨慎对待父母的丧事
- 追远：追念祭祀远代祖先
- 厚：淳厚
🔗 **句型特点**：
- 省略主语："（君子/统治者）慎终追远"

### 第10章
```
❓ 子禽：
夫子至于是邦也，必闻其政，
求之与，抑与之与？

🗣️ 子贡：
夫子温、良、恭、俭、让以得之。
夫子之求之也，其诸异乎人之求之与？
```

🔖 **重点发音**：与（yú）诸（zhū）
📖 **重点词汇**：
- 夫子：古代对老师、有学问德行之人的尊称，类似现代的"老师"或"先生"
  - 在《论语》中：专门用来尊称孔子
  - 词源：夫（发语词，无实义）+ 子（对男子的尊称）= 夫子（老师、学者的尊称）
  - 在"夫子温、良、恭、俭、让以得之"中：子贡用"夫子"来称呼自己的老师孔子，表示尊敬。这句话的意思是"老师（孔子）以温和、善良、恭敬、节俭、谦让的品德获得了各国诸侯的信任"
- 邦：诸侯国
- 抑：还是
- 诸（zhū）：之乎的合音
- 温良恭俭让：温和、善良、恭敬、节俭、谦让
🔗 **句型特点**：
- "与（yú）"：句末语气词，表疑问或感叹，同"欤"
- "求之与，抑与之与"："与"是句末语气词，表示疑问
- "其诸...与"："其诸"是表示推测的语气词，"大概...吧"

### 第11章
```
🎓 子（孔子）：
父在，观其志；
父没，观其行；
三年无改于父之道，可谓孝矣。
```

🔖 **重点发音**：没（mò）
📖 **重点词汇**：
- 没（mò）：去世
- 行：行为
- 三年：古代三年之丧（实为25个月），体现对父亲的敬重
- 道：原则、做法（此处"父之道"特指父亲符合道义、仁德的处事原则，非指所有行为；若父亲之道有偏颇，无需固守）
🔗 **句型特点**：
- "父在，观其志"：省略了"观其志者"的主语
- 对偶句：父在/父没、观其志/观其行相对
💡 **文化背景**：
- "三年之丧"源于上古礼制，孔子推崇其核心是"思慕父母"的孝心表达
- 25个月是"满三年"的虚数表述（古代以农历计，25个月约为两年多，接近"三年"的情感认知）
- 体现儒家"以礼表孝"的伦理思想：通过外在礼制表达内在孝心
✨ **思想内涵**：
- 此处体现孔子"孝"的观念，但也强调"义以为上"（《论语·阳货》），非无条件盲从父亲的一切做法

### 第12章
```
📚 有子：
礼之用，和为贵。
先王之道，斯为美；小大由之。
有所不行，知和而和，
不以礼节之，亦不可行也。
```

🔖 **重点发音**：无
📖 **重点词汇**：
- 礼：礼制、礼仪
- 和：和谐、恰当
- 先王：古代圣王
- 斯：这
- 节：节制、规范
🔗 **句型特点**：
- "礼之用"："之"用于主谓之间取消句子独立性（主谓之间取消独立性句型）
- "斯为美"：斯，代词"这"

### 第13章
```
📚 有子：
信近于义，言可复也。
恭近于礼，远耻辱也。
因不失其亲，亦可宗也。
```

🔖 **重点发音**：复（fù）
📖 **重点词汇**：
- 信：诺言
- 义：道义、合理
- 复（fù）：实践、兑现
- 恭：恭敬
- 远（yuàn）：远离、避免
- 宗：尊崇、效法
🔗 **句型特点**：
- "因不失其亲"：因，依靠；指依靠亲近且有德行的人，不失去自己的亲族关系
- "亦可宗也"：也值得效法

### 第14章
```
🎓 子（孔子）：
君子食无求饱，居无求安，
敏于事而慎于言，
就有道而正焉，可谓好学也已。
```

🔖 **重点发音**：无
📖 **重点词汇**：
- 敏：勤勉、敏捷
- 慎：谨慎
- 就：靠近、追随
- 有道：有道德学问的人
- 正：匡正（此处特指以有道之人的言行匡正自己的思想、行为，"正"的对象是自身的品德与学问）
🔗 **句型特点**：
- "就有道而正焉"：省略主语"自己"，完整为"（君子）就有道而正（自己）焉"
- "正"使动用法，"使...端正"（使动用法的具体指向：使自己的品德与学问端正）

### 第15章
```
🗣️ 子贡：
贫而无谄，富而无骄，何如？

🎓 子（孔子）：
可也。未若贫而乐，富而好礼者也。

🗣️ 子贡：
《诗》云：『如切如磋，如琢如磨』，
其斯之谓与？

🎓 子（孔子）：
赐也，始可与言《诗》已矣，
告诸往而知来者。
```

🔖 **重点发音**：谄（chǎn）与（yú）
📖 **重点词汇**：
- 谄（chǎn）：巴结奉承
- 骄：骄傲
- 贫而乐：贫穷却乐于道
- 好礼：喜好礼制
- 切磋琢磨：出自《诗经·卫风·淇奥》，原指加工骨角玉石，此处比喻修养德行
- 赐：子贡的名
- 告诸往而知来者：告诉他过去的事就能推知未来的事
🔗 **句型特点**：
- "何如"：固定句式，"怎么样"
- "未若"：不如、比不上
- "与（yú）"：句末语气词，表疑问或感叹，同"欤"
- "其斯之谓与"：宾语前置，正常语序是"其谓之斯与"（大概说的就是这个吧）

### 第16章
```
🎓 子（孔子）：
不患人之不己知，患不知人也。
```

🔖 **重点发音**：无
📖 **重点词汇**：
- 患：担心
- 人之不己知：别人不了解自己（宾语前置）
🔗 **句型特点**：
- "不己知"：宾语前置，正常语序是"不知己"（否定句中代词"己"作宾语前置）
- "人之不己知"："之"取消句子独立性，"人之不己知"作为主谓短语作"患"的宾语
- 对偶句：人之不己知 / 不知人也
- 宾语前置规律：本篇多次出现否定句中代词作宾语前置（如第2章"未之有"、第16章"不己知"）

## 重要语句

- **学而时习之，不亦说乎** - 强调学习要不断温习实践
- **吾日三省吾身** - 提倡自我反省
- **君子务本，本立而道生** - 强调根本的重要性
- **礼之用，和为贵** - 礼的运用以和谐为贵
- **不患人之不己知，患不知人也** - 不要担心别人不了解自己，而要担心自己不了解别人
