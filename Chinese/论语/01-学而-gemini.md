<h2 style="color: #8B008B; font-size: 1.8em; margin-top: 0; padding-bottom: 8px; border-bottom: 1px solid #DDA0DD;">篇名说明</h2>
<p style="line-height: 1.8; margin-bottom: 15px; text-indent: 2em; color: #333;">本篇以首章<strong style="color: #A52A2A;">"子曰：学而时习之"</strong>的前两字<strong style="color: #A52A2A;">"学而"</strong>命名。《论语》各篇皆以篇首前二三字命名。</p>

<h3 style="color: #9370DB; margin-top: 30px; font-size: 1.4em;">易错点提示</h3>
<p style="line-height: 1.8; margin-bottom: 15px; text-indent: 2em; color: #333;"><strong style="color: #A52A2A;">易混淆字词</strong>：</p>
<ul style="list-style-type: disc; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;"><strong style="color: #000080;">弟（tì）</strong>同<strong style="color: #000080;">"悌"</strong>，敬爱兄长 ≠ 弟（dì）兄弟</li>
<li style="margin-bottom: 5px;"><strong style="color: #000080;">省（xǐng）</strong>反省 ≠ 省（shěng）省份</li>
<li style="margin-bottom: 5px;"><strong style="color: #000080;">说（yuè）</strong>通<strong style="color: #000080;">"悦"</strong> ≠ 说（shuō）说话</li>
<li style="margin-bottom: 5px;"><strong style="color: #000080;">乐（lè）</strong>快乐 ≠ 乐（yuè）音乐</li>
</ul>
<p style="line-height: 1.8; margin-bottom: 15px; text-indent: 2em; color: #333;"><strong style="color: #A52A2A;">易误解语句</strong>：</p>
<ul style="list-style-type: disc; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;"><strong style="color: #000080;">"无友不如己者"</strong>：主流解读"不要结交不如自己的人"，另一说"没有朋友是完全不如自己的"</li>
<li style="margin-bottom: 5px;"><strong style="color: #000080;">"贤贤易色"</strong>：重视贤者而轻视女色（易：轻视）</li>
<li style="margin-bottom: 5px;"><strong style="color: #000080;">"三年无改于父之道"</strong>："三年"指古代守丧制度（实为25个月），非单纯三年时间</li>
</ul>

<h2 style="color: #8B008B; font-size: 1.8em; margin-top: 0; padding-bottom: 8px; border-bottom: 1px solid #DDA0DD;">全文</h2>
<p style="line-height: 1.8; margin-bottom: 15px; text-indent: 2em; color: #333;">子曰：「学而时习之，不亦说乎？有朋自远方来，不亦乐乎？人不知而不愠，不亦君子乎？」</p>
<p style="line-height: 1.8; margin-bottom: 15px; text-indent: 2em; color: #333;">有子曰：「其为人也孝弟，而好犯上者，鲜矣；不好犯上，而好作乱者，未之有也。君子务本，本立而道生。孝弟也者，其为仁之本与！」</p>
<p style="line-height: 1.8; margin-bottom: 15px; text-indent: 2em; color: #333;">子曰：「巧言令色，鲜矣仁！」</p>
<p style="line-height: 1.8; margin-bottom: 15px; text-indent: 2em; color: #333;">曾子曰：「吾日三省吾身：为人谋而不忠乎？与朋友交而不信乎？传不习乎？」</p>
<p style="line-height: 1.8; margin-bottom: 15px; text-indent: 2em; color: #333;">子曰：「道千乘之国，敬事而信，节用而爱人，使民以时。」</p>
<p style="line-height: 1.8; margin-bottom: 15px; text-indent: 2em; color: #333;">子曰：「弟子入则孝，出则弟，谨而信，泛爱众，而亲仁。行有余力，则以学文。」</p>
<p style="line-height: 1.8; margin-bottom: 15px; text-indent: 2em; color: #333;">子夏曰：「贤贤易色；事父母，能竭其力；事君，能致其身；与朋友交，言而有信。虽曰未学，吾必谓之学矣。」</p>
<p style="line-height: 1.8; margin-bottom: 15px; text-indent: 2em; color: #333;">子曰：「君子不重，则不威；学则不固。主忠信。无友不如己者。过则勿惮改。」</p>
<p style="line-height: 1.8; margin-bottom: 15px; text-indent: 2em; color: #333;">曾子曰：「慎终追远，民德归厚矣。」</p>
<p style="line-height: 1.8; margin-bottom: 15px; text-indent: 2em; color: #333;">子禽问于子贡曰：「夫子至于是邦也，必闻其政，求之与，抑与之与？」子贡曰：「夫子温、良、恭、俭、让以得之。夫子之求之也，其诸异乎人之求之与？」</p>
<p style="line-height: 1.8; margin-bottom: 15px; text-indent: 2em; color: #333;">子曰：「父在，观其志；父没，观其行；三年无改于父之道，可谓孝矣。」</p>
<p style="line-height: 1.8; margin-bottom: 15px; text-indent: 2em; color: #333;">有子曰：「礼之用，和为贵。先王之道，斯为美；小大由之。有所不行，知和而和，不以礼节之，亦不可行也。」</p>
<p style="line-height: 1.8; margin-bottom: 15px; text-indent: 2em; color: #333;">有子曰：「信近于义，言可复也。恭近于礼，远耻辱也。因不失其亲，亦可宗也。」</p>
<p style="line-height: 1.8; margin-bottom: 15px; text-indent: 2em; color: #333;">子曰：「君子食无求饱，居无求安，敏于事而慎于言，就有道而正焉，可谓好学也已。」</p>
<p style="line-height: 1.8; margin-bottom: 15px; text-indent: 2em; color: #333;">子贡曰：「贫而无谄，富而无骄，何如？」子曰：「可也。未若贫而乐，富而好礼者也。」子贡曰：「《诗》云：『如切如磋，如琢如磨』，其斯之谓与？」子曰：「赐也，始可与言《诗》已矣，告诸往而知来者。」</p>
<p style="line-height: 1.8; margin-bottom: 15px; text-indent: 2em; color: #333;">子曰：「不患人之不己知，患不知人也。」</p>
</div>

<div style="margin-top: 30px; padding: 15px; background-color: #F9F9F9; border-left: 5px solid #8A2BE2; border-radius: 8px;">
<h2 style="color: #8B008B; font-size: 1.8em; margin-top: 0; padding-bottom: 8px; border-bottom: 1px solid #DDA0DD;">对话重现</h2>

<div style="background-color: #F8F8FF; border: 1px solid #DCDCDC; border-radius: 10px; padding: 20px; margin-bottom: 30px; box-shadow: 2px 2px 8px rgba(0,0,0,0.1);">
<h3 style="color: #9370DB; margin-top: 0; font-size: 1.4em;">第1章</h3>
<div style="background-color: #E6E6FA; border-left: 5px solid #9370DB; padding: 15px; margin-bottom: 15px; font-family: 'Georgia', serif; font-size: 1.1em; color: #2F4F4F; line-height: 1.6;">
<p style="margin: 0;">🎓 <strong style="color: #A52A2A;">子（孔子）</strong>：</p>
<p style="margin: 5px 0 0 20px;">学而时习之，不亦说乎？</p>
<p style="margin: 0 0 0 20px;">有朋自远方来，不亦乐乎？</p>
<p style="margin: 0 0 0 20px;">人不知而不愠，不亦君子乎？</p>
</div>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔖 重点发音</span>：说（yuè）乐（lè）</p>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">📖 重点词汇</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">时习</strong>：时常温习</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">朋</strong>：志同道合的人（古代"朋"多指同门、同道，与"友"有别；"友"侧重情谊相投）</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">愠（yùn）</strong>：怨恨、生气</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">君子</strong>：有道德修养的人</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔗 句型特点</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"不亦...乎"</strong>：反问句式，"不也是很...吗"（本篇常见，如第1、3、16章）</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"学而时习之"</strong>：省略了主语"（人）"</li>
</ul>
</div>

<div style="background-color: #F8F8FF; border: 1px solid #DCDCDC; border-radius: 10px; padding: 20px; margin-bottom: 30px; box-shadow: 2px 2px 8px rgba(0,0,0,0.1);">
<h3 style="color: #9370DB; margin-top: 0; font-size: 1.4em;">第2章</h3>
<div style="background-color: #E6E6FA; border-left: 5px solid #9370DB; padding: 15px; margin-bottom: 15px; font-family: 'Georgia', serif; font-size: 1.1em; color: #2F4F4F; line-height: 1.6;">
<p style="margin: 0;">📚 <strong style="color: #A52A2A;">有子</strong>：</p>
<p style="margin: 5px 0 0 20px;">其为人也孝弟，而好犯上者，鲜矣；</p>
<p style="margin: 0 0 0 20px;">不好犯上，而好作乱者，未之有也。</p>
<p style="margin: 0 0 0 20px;">君子务本，本立而道生。</p>
<p style="margin: 0 0 0 20px;">孝弟也者，其为仁之本与！</p>
</div>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔖 重点发音</span>：弟（tì）鲜（xiǎn）与（yú）</p>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">📖 重点词汇</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">孝弟（tì）</strong>：孝顺父母，敬爱兄长</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">犯上</strong>：冒犯上级</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">鲜（xiǎn）</strong>：少</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">务本</strong>：致力于根本</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">仁</strong>：仁德、爱心</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔗 句型特点</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"其为人也孝弟"</strong>："其...也"表示停顿，强调主语</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"未之有也"</strong>：宾语前置，正常语序是"未有之也"（否定句中代词"之"作宾语前置）</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"与（yú）"</strong>：句末语气词，表疑问或感叹，同"欤"</li>
</ul>
</div>

<div style="background-color: #F8F8FF; border: 1px solid #DCDCDC; border-radius: 10px; padding: 20px; margin-bottom: 30px; box-shadow: 2px 2px 8px rgba(0,0,0,0.1);">
<h3 style="color: #9370DB; margin-top: 0; font-size: 1.4em;">第3章</h3>
<div style="background-color: #E6E6FA; border-left: 5px solid #9370DB; padding: 15px; margin-bottom: 15px; font-family: 'Georgia', serif; font-size: 1.1em; color: #2F4F4F; line-height: 1.6;">
<p style="margin: 0;">🎓 <strong style="color: #A52A2A;">子（孔子）</strong>：</p>
<p style="margin: 5px 0 0 20px;">巧言令色，鲜矣仁！</p>
</div>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔖 重点发音</span>：鲜（xiǎn）矣（yǐ）</p>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">📖 重点词汇</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">巧言</strong>：花言巧语</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">令色</strong>："令"为形容词，美好的、和善的；"令色"指刻意装出的和善、讨好的脸色（体现虚伪做作，契合"鲜矣仁"的批判态度）</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">鲜（xiǎn）</strong>：少</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">矣（yǐ）</strong>：文言助词，用于句末，表示感叹或陈述语气，相当于"了"或"啊"</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔗 句型特点</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"鲜矣仁"</strong>：定语后置+主谓倒装，完整为"巧言令色之人，仁鲜矣"（先省略中心词"人"，再主谓倒装）</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">省略句</strong>，完整意思是"巧言令色之人，鲜矣仁"（仁德很少啊！）</li>
</ul>
</div>

<div style="background-color: #F8F8FF; border: 1px solid #DCDCDC; border-radius: 10px; padding: 20px; margin-bottom: 30px; box-shadow: 2px 2px 8px rgba(0,0,0,0.1);">
<h3 style="color: #9370DB; margin-top: 0; font-size: 1.4em;">第4章</h3>
<div style="background-color: #E6E6FA; border-left: 5px solid #9370DB; padding: 15px; margin-bottom: 15px; font-family: 'Georgia', serif; font-size: 1.1em; color: #2F4F4F; line-height: 1.6;">
<p style="margin: 0;">🧘 <strong style="color: #A52A2A;">曾子</strong>：</p>
<p style="margin: 5px 0 0 20px;">吾日三省吾身：</p>
<p style="margin: 0 0 0 20px;">为人谋而不忠乎？</p>
<p style="margin: 0 0 0 20px;">与朋友交而不信乎？</p>
<p style="margin: 0 0 0 20px;">传不习乎？</p>
</div>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔖 重点发音</span>：省（xǐng）传（chuán）</p>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">📖 重点词汇</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">三省（xǐng）</strong>：多次反省</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">谋</strong>：谋划、办事</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">忠</strong>：尽心尽力</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">信</strong>：诚信（此处"信"既指说话算数，也指交往中的真诚无欺，与第7章子夏"与朋友交，言而有信"的"信"同义，是儒家交友的核心准则）</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">传（chuán）</strong>：老师传授的知识</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔗 句型特点</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"三省"</strong>：三表示多次，非确指三次</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">三句都是省略句</strong>，省略了"吾"</li>
<li style="margin-bottom: 5px;">&nbsp;&nbsp;- 完整："（吾）为人谋而不忠乎？"</li>
<li style="margin-bottom: 5px;">&nbsp;&nbsp;- 完整："（吾）与朋友交而不信乎？"</li>
<li style="margin-bottom: 5px;">&nbsp;&nbsp;- 完整："（吾）传（所）不习乎？"</li>
</ul>
</div>

<div style="background-color: #F8F8FF; border: 1px solid #DCDCDC; border-radius: 10px; padding: 20px; margin-bottom: 30px; box-shadow: 2px 2px 8px rgba(0,0,0,0.1);">
<h3 style="color: #9370DB; margin-top: 0; font-size: 1.4em;">第5章</h3>
<div style="background-color: #E6E6FA; border-left: 5px solid #9370DB; padding: 15px; margin-bottom: 15px; font-family: 'Georgia', serif; font-size: 1.1em; color: #2F4F4F; line-height: 1.6;">
<p style="margin: 0;">🎓 <strong style="color: #A52A2A;">子（孔子）</strong>：</p>
<p style="margin: 5px 0 0 20px;">道千乘之国，敬事而信，</p>
<p style="margin: 0 0 0 20px;">节用而爱人，使民以时。</p>
</div>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔖 重点发音</span>：乘（shèng）</p>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">📖 重点词汇</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">道</strong>：治理</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">千乘（shèng）之国</strong>：拥有千辆兵车的诸侯国（一乘含兵车一辆、甲士三人、步卒七十二人；春秋时期千乘之国为中等诸侯国，实力较强）</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">敬事</strong>：严肃认真对待政事</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">节用</strong>：节省费用</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">使民以时</strong>：按恰当的时节役使百姓</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">💡 文化背景</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">春秋时期"千乘之国"的兵力</strong>约为3000甲士+72000步卒，合计7.5万人左右，是中等诸侯国的军事规模</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">孔子的治理建议</strong>（敬事而信、节用爱人等）针对的是有一定实力但需注重民生的诸侯国</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔗 句型特点</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"道"</strong>用作动词，"治理"之意</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"使民以时"</strong>：介词结构后置，正常语序是"以时使民"</li>
</ul>
</div>

<div style="background-color: #F8F8FF; border: 1px solid #DCDCDC; border-radius: 10px; padding: 20px; margin-bottom: 30px; box-shadow: 2px 2px 8px rgba(0,0,0,0.1);">
<h3 style="color: #9370DB; margin-top: 0; font-size: 1.4em;">第6章</h3>
<div style="background-color: #E6E6FA; border-left: 5px solid #9370DB; padding: 15px; margin-bottom: 15px; font-family: 'Georgia', serif; font-size: 1.1em; color: #2F4F4F; line-height: 1.6;">
<p style="margin: 0;">🎓 <strong style="color: #A52A2A;">子（孔子）</strong>：</p>
<p style="margin: 5px 0 0 20px;">弟子入则孝，出则弟，谨而信，</p>
<p style="margin: 0 0 0 20px;">泛爱众，而亲仁。</p>
<p style="margin: 0 0 0 20px;">行有余力，则以学文。</p>
</div>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔖 重点发音</span>：弟（tì）</p>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">📖 重点词汇</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">弟（tì）</strong>：同"悌"，敬爱兄长</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">谨</strong>：谨慎</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">泛爱</strong>：广泛爱护（"泛爱众"是儒家"仁爱"的延伸，指对众人有仁爱之心，而非无差别的平等之爱）</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">亲仁</strong>：亲近有仁德的人（与"泛爱众"形成层次，体现儒家"爱有差等"的伦理观）</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">文</strong>：文献典籍</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔗 句型特点</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"入则孝，出则弟"</strong>：对偶句，入/出、孝/弟相对</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">省略主语"弟子"</strong></li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"入"</strong>指在家，<strong style="color: #000080;">"出"</strong>指外出</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">✨ 思想内涵</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"泛爱众"与"亲仁"</strong>体现儒家"爱有差等"的伦理观：先爱亲（孝悌），再爱众人（泛爱），特别亲近有仁德之人（亲仁），形成由近及远、有层次的仁爱体系</li>
</ul>
</div>

<div style="background-color: #F8F8FF; border: 1px solid #DCDCDC; border-radius: 10px; padding: 20px; margin-bottom: 30px; box-shadow: 2px 2px 8px rgba(0,0,0,0.1);">
<h3 style="color: #9370DB; margin-top: 0; font-size: 1.4em;">第7章</h3>
<div style="background-color: #E6E6FA; border-left: 5px solid #9370DB; padding: 15px; margin-bottom: 15px; font-family: 'Georgia', serif; font-size: 1.1em; color: #2F4F4F; line-height: 1.6;">
<p style="margin: 0;">✍️ <strong style="color: #A52A2A;">子夏</strong>：</p>
<p style="margin: 5px 0 0 20px;">贤贤易色；</p>
<p style="margin: 0 0 0 20px;">事父母，能竭其力；</p>
<p style="margin: 0 0 0 20px;">事君，能致其身；</p>
<p style="margin: 0 0 0 20px;">与朋友交，言而有信。</p>
<p style="margin: 0 0 0 20px;">虽曰未学，吾必谓之学矣。</p>
</div>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔖 重点发音</span>：易（yì）</p>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">📖 重点词汇</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">贤贤</strong>：尊重贤者（第一个"贤"作动词用）</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">易（yì）</strong>：轻视（此处核心义）</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">致其身</strong>：奉献生命</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">言而有信</strong>：说话算数</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔗 句型特点</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"贤贤易色"</strong>：第一个"贤"是动词（尊重），第二个"贤"是名词（贤者）；易，轻视；全句意为"重视贤者而轻视女色"</li>
</ul>
</div>

<div style="background-color: #F8F8FF; border: 1px solid #DCDCDC; border-radius: 10px; padding: 20px; margin-bottom: 30px; box-shadow: 2px 2px 8px rgba(0,0,0,0.1);">
<h3 style="color: #9370DB; margin-top: 0; font-size: 1.4em;">第8章</h3>
<div style="background-color: #E6E6FA; border-left: 5px solid #9370DB; padding: 15px; margin-bottom: 15px; font-family: 'Georgia', serif; font-size: 1.1em; color: #2F4F4F; line-height: 1.6;">
<p style="margin: 0;">🎓 <strong style="color: #A52A2A;">子（孔子）</strong>：</p>
<p style="margin: 5px 0 0 20px;">君子不重，则不威；学则不固。</p>
<p style="margin: 0 0 0 20px;">主忠信。无友不如己者。</p>
<p style="margin: 0 0 0 20px;">过则勿惮改。</p>
</div>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔖 重点发音</span>：重（zhòng）惮（dàn）</p>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">📖 重点词汇</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">重（zhòng）</strong>：庄重</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">威</strong>：威严</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">固</strong>：牢固</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">主忠信</strong>：以忠信为主</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">惮（dàn）</strong>：害怕</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔗 句型特点</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"无友不如己者"</strong>："友"作动词用，"结交朋友"</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">省略句</strong>："（君子）无友不如己者"</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">💡 多元解读</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">📌 <strong style="color: #000080;">语境提示</strong>：此句与"主忠信""过则勿惮改"并列，核心是倡导"以优秀者为友、以忠信为交友原则"，避免误解为"功利性排斥不如自己的人"</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">主流</strong>：不要结交道德学问上不如自己的人（此处"不如己"侧重道德、学问，核心是倡导向优秀者学习）</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">另一说</strong>：没有朋友是完全不如自己的（每个人都有可取之处）</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">✨ 综合理解</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">结合孔子"友直、友谅、友多闻"</strong>（《论语·季氏》）的交友观，两种解读均有经典依据，应理解为"择善而从，以友辅仁"</li>
</ul>
</div>

<div style="background-color: #F8F8FF; border: 1px solid #DCDCDC; border-radius: 10px; padding: 20px; margin-bottom: 30px; box-shadow: 2px 2px 8px rgba(0,0,0,0.1);">
<h3 style="color: #9370DB; margin-top: 0; font-size: 1.4em;">第9章</h3>
<div style="background-color: #E6E6FA; border-left: 5px solid #9370DB; padding: 15px; margin-bottom: 15px; font-family: 'Georgia', serif; font-size: 1.1em; color: #2F4F4F; line-height: 1.6;">
<p style="margin: 0;">🧘 <strong style="color: #A52A2A;">曾子</strong>：</p>
<p style="margin: 5px 0 0 20px;">慎终追远，民德归厚矣。</p>
</div>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔖 重点发音</span>：无</p>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">📖 重点词汇</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">慎终</strong>：谨慎对待父母的丧事</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">追远</strong>：追念祭祀远代祖先</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">厚</strong>：淳厚</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔗 句型特点</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">省略主语</strong>："（君子/统治者）慎终追远"</li>
</ul>
</div>

<div style="background-color: #F8F8FF; border: 1px solid #DCDCDC; border-radius: 10px; padding: 20px; margin-bottom: 30px; box-shadow: 2px 2px 8px rgba(0,0,0,0.1);">
<h3 style="color: #9370DB; margin-top: 0; font-size: 1.4em;">第10章</h3>
<div style="background-color: #E6E6FA; border-left: 5px solid #9370DB; padding: 15px; margin-bottom: 15px; font-family: 'Georgia', serif; font-size: 1.1em; color: #2F4F4F; line-height: 1.6;">
<p style="margin: 0;">❓ <strong style="color: #A52A2A;">子禽</strong>：</p>
<p style="margin: 5px 0 0 20px;">夫子至于是邦也，必闻其政，</p>
<p style="margin: 0 0 0 20px;">求之与，抑与之与？</p>
<p style="margin: 15px 0 0 0;">🗣️ <strong style="color: #A52A2A;">子贡</strong>：</p>
<p style="margin: 5px 0 0 20px;">夫子温、良、恭、俭、让以得之。</p>
<p style="margin: 0 0 0 20px;">夫子之求之也，其诸异乎人之求之与？</p>
</div>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔖 重点发音</span>：与（yú）诸（zhū）</p>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">📖 重点词汇</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">夫子</strong>：古代对老师、有学问德行之人的尊称，类似现代的"老师"或"先生"</li>
<li style="margin-bottom: 5px;">&nbsp;&nbsp;- 在《论语》中：专门用来尊称孔子</li>
<li style="margin-bottom: 5px;">&nbsp;&nbsp;- 词源：夫（发语词，无实义）+ 子（对男子的尊称）= 夫子（老师、学者的尊称）</li>
<li style="margin-bottom: 5px;">&nbsp;&nbsp;- 在"夫子温、良、恭、俭、让以得之"中：子贡用"夫子"来称呼自己的老师孔子，表示尊敬。这句话的意思是"老师（孔子）以温和、善良、恭敬、节俭、谦让的品德获得了各国诸侯的信任"</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">邦</strong>：诸侯国</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">抑</strong>：还是</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">诸（zhū）</strong>：之乎的合音</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">温良恭俭让</strong>：温和、善良、恭敬、节俭、谦让</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔗 句型特点</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"与（yú）"</strong>：句末语气词，表疑问或感叹，同"欤"</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"求之与，抑与之与"</strong>："与"是句末语气词，表示疑问</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"其诸...与"</strong>："其诸"是表示推测的语气词，"大概...吧"</li>
</ul>
</div>

<div style="background-color: #F8F8FF; border: 1px solid #DCDCDC; border-radius: 10px; padding: 20px; margin-bottom: 30px; box-shadow: 2px 2px 8px rgba(0,0,0,0.1);">
<h3 style="color: #9370DB; margin-top: 0; font-size: 1.4em;">第11章</h3>
<div style="background-color: #E6E6FA; border-left: 5px solid #9370DB; padding: 15px; margin-bottom: 15px; font-family: 'Georgia', serif; font-size: 1.1em; color: #2F4F4F; line-height: 1.6;">
<p style="margin: 0;">🎓 <strong style="color: #A52A2A;">子（孔子）</strong>：</p>
<p style="margin: 5px 0 0 20px;">父在，观其志；</p>
<p style="margin: 0 0 0 20px;">父没，观其行；</p>
<p style="margin: 0 0 0 20px;">三年无改于父之道，可谓孝矣。</p>
</div>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔖 重点发音</span>：没（mò）</p>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">📖 重点词汇</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">没（mò）</strong>：去世</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">行</strong>：行为</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">三年</strong>：古代三年之丧（实为25个月），体现对父亲的敬重</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">道</strong>：原则、做法（此处"父之道"特指父亲符合道义、仁德的处事原则，非指所有行为；若父亲之道有偏颇，无需固守）</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔗 句型特点</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"父在，观其志"</strong>：省略了"观其志者"的主语</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">对偶句</strong>：父在/父没、观其志/观其行相对</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">💡 文化背景</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"三年之丧"</strong>源于上古礼制，孔子推崇其核心是"思慕父母"的孝心表达</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">25个月</strong>是"满三年"的虚数表述（古代以农历计，25个月约为两年多，接近"三年"的情感认知）</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">体现儒家"以礼表孝"</strong>的伦理思想：通过外在礼制表达内在孝心</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">✨ 思想内涵</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">此处体现孔子"孝"的观念</strong>，但也强调"义以为上"（《论语·阳货》），非无条件盲从父亲的一切做法</li>
</ul>
</div>

<div style="background-color: #F8F8FF; border: 1px solid #DCDCDC; border-radius: 10px; padding: 20px; margin-bottom: 30px; box-shadow: 2px 2px 8px rgba(0,0,0,0.1);">
<h3 style="color: #9370DB; margin-top: 0; font-size: 1.4em;">第12章</h3>
<div style="background-color: #E6E6FA; border-left: 5px solid #9370DB; padding: 15px; margin-bottom: 15px; font-family: 'Georgia', serif; font-size: 1.1em; color: #2F4F4F; line-height: 1.6;">
<p style="margin: 0;">📚 <strong style="color: #A52A2A;">有子</strong>：</p>
<p style="margin: 5px 0 0 20px;">礼之用，和为贵。</p>
<p style="margin: 0 0 0 20px;">先王之道，斯为美；小大由之。</p>
<p style="margin: 0 0 0 20px;">有所不行，知和而和，</p>
<p style="margin: 0 0 0 20px;">不以礼节之，亦不可行也。</p>
</div>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔖 重点发音</span>：无</p>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">📖 重点词汇</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">礼</strong>：礼制、礼仪</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">和</strong>：和谐、恰当</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">先王</strong>：古代圣王</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">斯</strong>：这</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">节</strong>：节制、规范</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔗 句型特点</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"礼之用"</strong>："之"用于主谓之间取消句子独立性（主谓之间取消独立性句型）</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"斯为美"</strong>：斯，代词"这"</li>
</ul>
</div>

<div style="background-color: #F8F8FF; border: 1px solid #DCDCDC; border-radius: 10px; padding: 20px; margin-bottom: 30px; box-shadow: 2px 2px 8px rgba(0,0,0,0.1);">
<h3 style="color: #9370DB; margin-top: 0; font-size: 1.4em;">第13章</h3>
<div style="background-color: #E6E6FA; border-left: 5px solid #9370DB; padding: 15px; margin-bottom: 15px; font-family: 'Georgia', serif; font-size: 1.1em; color: #2F4F4F; line-height: 1.6;">
<p style="margin: 0;">📚 <strong style="color: #A52A2A;">有子</strong>：</p>
<p style="margin: 5px 0 0 20px;">信近于义，言可复也。</p>
<p style="margin: 0 0 0 20px;">恭近于礼，远耻辱也。</p>
<p style="margin: 0 0 0 20px;">因不失其亲，亦可宗也。</p>
</div>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔖 重点发音</span>：复（fù）</p>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">📖 重点词汇</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">信</strong>：诺言</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">义</strong>：道义、合理</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">复（fù）</strong>：实践、兑现</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">恭</strong>：恭敬</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">远（yuàn）</strong>：远离、避免</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">宗</strong>：尊崇、效法</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔗 句型特点</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"因不失其亲"</strong>：因，依靠；指依靠亲近且有德行的人，不失去自己的亲族关系</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"亦可宗也"</strong>：也值得效法</li>
</ul>
</div>

<div style="background-color: #F8F8FF; border: 1px solid #DCDCDC; border-radius: 10px; padding: 20px; margin-bottom: 30px; box-shadow: 2px 2px 8px rgba(0,0,0,0.1);">
<h3 style="color: #9370DB; margin-top: 0; font-size: 1.4em;">第14章</h3>
<div style="background-color: #E6E6FA; border-left: 5px solid #9370DB; padding: 15px; margin-bottom: 15px; font-family: 'Georgia', serif; font-size: 1.1em; color: #2F4F4F; line-height: 1.6;">
<p style="margin: 0;">🎓 <strong style="color: #A52A2A;">子（孔子）</strong>：</p>
<p style="margin: 5px 0 0 20px;">君子食无求饱，居无求安，</p>
<p style="margin: 0 0 0 20px;">敏于事而慎于言，</p>
<p style="margin: 0 0 0 20px;">就有道而正焉，可谓好学也已。</p>
</div>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔖 重点发音</span>：无</p>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">📖 重点词汇</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">敏</strong>：勤勉、敏捷</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">慎</strong>：谨慎</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">就</strong>：靠近、追随</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">有道</strong>：有道德学问的人</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">正</strong>：匡正（此处特指以有道之人的言行匡正自己的思想、行为，"正"的对象是自身的品德与学问）</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔗 句型特点</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"就有道而正焉"</strong>：省略主语"自己"，完整为"（君子）就有道而正（自己）焉"</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"正"使动用法</strong>，"使...端正"（使动用法的具体指向：使自己的品德与学问端正）</li>
</ul>
</div>

<div style="background-color: #F8F8FF; border: 1px solid #DCDCDC; border-radius: 10px; padding: 20px; margin-bottom: 30px; box-shadow: 2px 2px 8px rgba(0,0,0,0.1);">
<h3 style="color: #9370DB; margin-top: 0; font-size: 1.4em;">第15章</h3>
<div style="background-color: #E6E6FA; border-left: 5px solid #9370DB; padding: 15px; margin-bottom: 15px; font-family: 'Georgia', serif; font-size: 1.1em; color: #2F4F4F; line-height: 1.6;">
<p style="margin: 0;">🗣️ <strong style="color: #A52A2A;">子贡</strong>：</p>
<p style="margin: 5px 0 0 20px;">贫而无谄，富而无骄，何如？</p>
<p style="margin: 15px 0 0 0;">🎓 <strong style="color: #A52A2A;">子（孔子）</strong>：</p>
<p style="margin: 5px 0 0 20px;">可也。未若贫而乐，富而好礼者也。</p>
<p style="margin: 15px 0 0 0;">🗣️ <strong style="color: #A52A2A;">子贡</strong>：</p>
<p style="margin: 5px 0 0 20px;">《诗》云：『如切如磋，如琢如磨』，</p>
<p style="margin: 0 0 0 20px;">其斯之谓与？</p>
<p style="margin: 15px 0 0 0;">🎓 <strong style="color: #A52A2A;">子（孔子）</strong>：</p>
<p style="margin: 5px 0 0 20px;">赐也，始可与言《诗》已矣，</p>
<p style="margin: 0 0 0 20px;">告诸往而知来者。</p>
</div>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔖 重点发音</span>：谄（chǎn）与（yú）</p>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">📖 重点词汇</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">谄（chǎn）</strong>：巴结奉承</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">骄</strong>：骄傲</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">贫而乐</strong>：贫穷却乐于道</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">好礼</strong>：喜好礼制</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">切磋琢磨</strong>：出自《诗经·卫风·淇奥》，原指加工骨角玉石，此处比喻修养德行</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">赐</strong>：子贡的名</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">告诸往而知来者</strong>：告诉他过去的事就能推知未来的事</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔗 句型特点</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"何如"</strong>：固定句式，"怎么样"</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"未若"</strong>：不如、比不上</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"与（yú）"</strong>：句末语气词，表疑问或感叹，同"欤"</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"其斯之谓与"</strong>：宾语前置，正常语序是"其谓之斯与"（大概说的就是这个吧）</li>
</ul>
</div>

<div style="background-color: #F8F8FF; border: 1px solid #DCDCDC; border-radius: 10px; padding: 20px; margin-bottom: 30px; box-shadow: 2px 2px 8px rgba(0,0,0,0.1);">
<h3 style="color: #9370DB; margin-top: 0; font-size: 1.4em;">第16章</h3>
<div style="background-color: #E6E6FA; border-left: 5px solid #9370DB; padding: 15px; margin-bottom: 15px; font-family: 'Georgia', serif; font-size: 1.1em; color: #2F4F4F; line-height: 1.6;">
<p style="margin: 0;">🎓 <strong style="color: #A52A2A;">子（孔子）</strong>：</p>
<p style="margin: 5px 0 0 20px;">不患人之不己知，患不知人也。</p>
</div>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔖 重点发音</span>：无</p>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">📖 重点词汇</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">患</strong>：担心</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">人之不己知</strong>：别人不了解自己（宾语前置）</li>
</ul>
<p style="margin-bottom: 10px;"><span style="font-weight: bold; color: #8B0000;">🔗 句型特点</span>：</p>
<ul style="list-style-type: none; margin-left: 20px; padding-left: 0; color: #333;">
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"不己知"</strong>：宾语前置，正常语序是"不知己"（否定句中代词"己"作宾语前置）</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">"人之不己知"</strong>："之"取消句子独立性，"人之不己知"作为主谓短语作"患"的宾语</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">对偶句</strong>：人之不己知 / 不知人也</li>
<li style="margin-bottom: 5px;">- <strong style="color: #000080;">宾语前置规律</strong>：本篇多次出现否定句中代词作宾语前置（如第2章"未之有"、第16章"不己知"）</li>
</ul>
</div>
