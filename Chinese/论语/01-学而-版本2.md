### ⚠️ 易错点提示

<div style="background: #fff3e0; border-left: 4px solid #ff9800; padding: 15px 20px; margin: 20px 0; border-radius: 0 5px 5px 0;">
  <h4 style="color: #e65100; margin-top: 0;">易混淆字词</h4>
  <ul style="margin-bottom: 15px;">
    <li>弟（tì）同"悌"，敬爱兄长 ≠ 弟（dì）兄弟</li>
    <li>省（xǐng）反省 ≠ 省（shěng）省份</li>
    <li>说（yuè）通"悦" ≠ 说（shuō）说话</li>
    <li>乐（lè）快乐 ≠ 乐（yuè）音乐</li>
  </ul>

  <h4 style="color: #e65100; margin-top: 0;">易误解语句</h4>
  <ul>
    <li>"无友不如己者"：主流解读"不要结交不如自己的人"，另一说"没有朋友是完全不如自己的"</li>
    <li>"贤贤易色"：重视贤者而轻视女色（易：轻视）</li>
    <li>"三年无改于父之道"："三年"指古代守丧制度（实为25个月），非单纯三年时间</li>
  </ul>

<div style="background: #fafafa; border: 2px solid #e0e0e0; border-radius: 10px; padding: 25px; margin: 20px 0; line-height: 2;">
  <p style="margin-bottom: 15px;">子曰：「学而时习之，不亦说乎？有朋自远方来，不亦乐乎？人不知而不愠，不亦君子乎？」</p>

  <p style="margin-bottom: 15px;">有子曰：「其为人也孝弟，而好犯上者，鲜矣；不好犯上，而好作乱者，未之有也。君子务本，本立而道生。孝弟也者，其为仁之本与！」</p>

  <p style="margin-bottom: 15px;">子曰：「巧言令色，鲜矣仁！」</p>

  <p style="margin-bottom: 15px;">曾子曰：「吾日三省吾身：为人谋而不忠乎？与朋友交而不信乎？传不习乎？」</p>

  <p style="margin-bottom: 15px;">子曰：「道千乘之国，敬事而信，节用而爱人，使民以时。」</p>

  <p style="margin-bottom: 15px;">子曰：「弟子入则孝，出则弟，谨而信，泛爱众，而亲仁。行有余力，则以学文。」</p>

  <p style="margin-bottom: 15px;">子夏曰：「贤贤易色；事父母，能竭其力；事君，能致其身；与朋友交，言而有信。虽曰未学，吾必谓之学矣。」</p>

  <p style="margin-bottom: 15px;">子曰：「君子不重，则不威；学则不固。主忠信。无友不如己者。过则勿惮改。」</p>

  <p style="margin-bottom: 15px;">曾子曰：「慎终追远，民德归厚矣。」</p>

  <p style="margin-bottom: 15px;">子禽问于子贡曰：「夫子至于是邦也，必闻其政，求之与，抑与之与？」子贡曰：「夫子温、良、恭、俭、让以得之。夫子之求之也，其诸异乎人之求之与？」</p>

  <p style="margin-bottom: 15px;">子曰：「父在，观其志；父没，观其行；三年无改于父之道，可谓孝矣。」</p>

  <p style="margin-bottom: 15px;">有子曰：「礼之用，和为贵。先王之道，斯为美；小大由之。有所不行，知和而和，不以礼节之，亦不可行也。」</p>

  <p style="margin-bottom: 15px;">有子曰：「信近于义，言可复也。恭近于礼，远耻辱也。因不失其亲，亦可宗也。」</p>

  <p style="margin-bottom: 15px;">子曰：「君子食无求饱，居无求安，敏于事而慎于言，就有道而正焉，可谓好学也已。」</p>

  <p style="margin-bottom: 15px;">子贡曰：「贫而无谄，富而无骄，何如？」子曰：「可也。未若贫而乐，富而好礼者也。」子贡曰：「《诗》云：『如切如磋，如琢如磨』，其斯之谓与？」子曰：「赐也，始可与言《诗》已矣，告诸往而知来者。」</p>

  <p style="margin-bottom: 0;">子曰：「不患人之不己知，患不知人也。」</p>
</div>

## 💬 对话重现

### 第1章

<div style="background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); color: white; padding: 20px; border-radius: 10px; margin: 20px 0;">
  <div style="font-size: 1.2em; margin-bottom: 10px;">🎓 <strong>子（孔子）</strong></div>
  <div style="line-height: 1.8;">
  学而时习之，不亦说乎？<br>
  有朋自远方来，不亦乐乎？<br>
  人不知而不愠，不亦君子乎？
  </div>
</div>

<div style="background: #e8f5e9; border-left: 4px solid #4caf50; padding: 15px 20px; margin: 15px 0; border-radius: 0 5px 5px 0;">
  <div style="margin-bottom: 10px;">
    <span style="background: #4caf50; color: white; padding: 3px 10px; border-radius: 15px; font-size: 0.9em;">🔖 重点发音</span>
    <span style="color: #2e7d32; font-weight: bold;">说（yuè）乐（lè）</span>
  </div>

  <div style="margin-bottom: 10px;">
    <span style="background: #4caf50; color: white; padding: 3px 10px; border-radius: 15px; font-size: 0.9em;">📖 重点词汇</span>
  </div>
  <ul style="margin: 5px 0;">
    <li>时习：时常温习</li>
    <li>朋：志同道合的人（古代"朋"多指同门、同道，与"友"有别；"友"侧重情谊相投）</li>
    <li>愠（yùn）：怨恨、生气</li>
    <li>君子：有道德修养的人</li>
  </ul>

  <div>
    <span style="background: #4caf50; color: white; padding: 3px 10px; border-radius: 15px; font-size: 0.9em;">🔗 句型特点</span>
  </div>
  <ul style="margin: 5px 0;">
    <li>"不亦...乎"：反问句式，"不也是很...吗"（本篇常见，如第1、3、16章）</li>
    <li>"学而时习之"：省略了主语"（人）"</li>
  </ul>
</div>

  <div style="margin-bottom: 10px;">
    <span style="background: #ff9800; color: white; padding: 3px 10px; border-radius: 15px; font-size: 0.9em;">🔖 重点发音</span>
    <span style="color: #e65100; font-weight: bold;">弟（tì）鲜（xiǎn）与（yú）</span>
  </div>

  <div style="margin-bottom: 10px;">
    <span style="background: #ff9800; color: white; padding: 3px 10px; border-radius: 15px; font-size: 0.9em;">📖 重点词汇</span>
  </div>
  <ul style="margin: 5px 0;">
    <li>孝弟（tì）：孝顺父母，敬爱兄长</li>
    <li>犯上：冒犯上级</li>
    <li>鲜（xiǎn）：少</li>
    <li>务本：致力于根本</li>
    <li>仁：仁德、爱心</li>
  </ul>

  <div>
    <span style="background: #ff9800; color: white; padding: 3px 10px; border-radius: 15px; font-size: 0.9em;">🔗 句型特点</span>
  </div>
  <ul style="margin: 5px 0;">
    <li>"其为人也孝弟"："其...也"表示停顿，强调主语</li>
    <li>"未之有也"：宾语前置，正常语序是"未有之也"（否定句中代词"之"作宾语前置）</li>
    <li>"与（yú）"：句末语气词，表疑问或感叹，同"欤"</li>
  </ul>
</div>

## ✨ 重要语句

<div style="display: grid; grid-template-columns: repeat(auto-fit, minmax(300px, 1fr)); gap: 20px; margin: 20px 0;">
  <div style="background: linear-gradient(135deg, #667eea, #764ba2); color: white; padding: 25px; border-radius: 10px; box-shadow: 0 4px 15px rgba(102,126,234,0.3);">
    <div style="font-size: 1.3em; font-weight: bold; margin-bottom: 15px; line-height: 1.6;">"学而时习之，不亦说乎"</div>
    <div style="opacity: 0.95; font-size: 0.95em; line-height: 1.6;">强调学习要不断温习实践。学习不仅仅是知识的积累，更是品德的修养。知行合一，才是真正的学习。</div>
  </div>

  <div style="background: linear-gradient(135deg, #f093fb, #f5576c); color: white; padding: 25px; border-radius: 10px; box-shadow: 0 4px 15px rgba(240,147,251,0.3);">
    <div style="font-size: 1.3em; font-weight: bold; margin-bottom: 15px; line-height: 1.6;">"吾日三省吾身"</div>
    <div style="opacity: 0.95; font-size: 0.95em; line-height: 1.6;">提倡自我反省，每天多次反省自己的行为：为人谋是否忠心？与朋友交往是否诚信？老师传授的知识是否温习？</div>
  </div>

  <div style="background: linear-gradient(135deg, #4facfe, #00f2fe); color: white; padding: 25px; border-radius: 10px; box-shadow: 0 4px 15px rgba(79,172,254,0.3);">
    <div style="font-size: 1.3em; font-weight: bold; margin-bottom: 15px; line-height: 1.6;">"君子务本，本立而道生"</div>
    <div style="opacity: 0.95; font-size: 0.95em; line-height: 1.6;">强调根本的重要性。君子致力于根本，根本确立了，道就会自然产生。孝悌就是仁的根本。</div>
  </div>

  <div style="background: linear-gradient(135deg, #43e97b, #38f9d7); color: white; padding: 25px; border-radius: 10px; box-shadow: 0 4px 15px rgba(67,233,123,0.3);">
    <div style="font-size: 1.3em; font-weight: bold; margin-bottom: 15px; line-height: 1.6;">"礼之用，和为贵"</div>
    <div style="opacity: 0.95; font-size: 0.95em; line-height: 1.6;">礼的运用以和谐为贵。礼制的最终目标是和谐，但如果只知道追求和谐而不用礼来节制，也是不可行的。</div>
  </div>

  <div style="background: linear-gradient(135deg, #fa709a, #fee140); color: white; padding: 25px; border-radius: 10px; box-shadow: 0 4px 15px rgba(250,112,154,0.3);">
    <div style="font-size: 1.3em; font-weight: bold; margin-bottom: 15px; line-height: 1.6;">"不患人之不己知，患不知人也"</div>
    <div style="opacity: 0.95; font-size: 0.95em; line-height: 1.6;">不要担心别人不了解自己，而要担心自己不了解别人。强调内在修养重于外在认可。</div>
  </div>
