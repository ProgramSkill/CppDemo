# 第81章 利而不害

## 原文

信言不美，美言不信。
善者不辩，辩者不善。
知者不博，博者不知。
圣人不积，既以为人己愈有，既以与人己愈多。
天之道，利而不害；圣人之道，为而不争。

## 译文

真实的话不漂亮，漂亮的话不真实。
善良的人不巧辩，巧辩的人不善良。
真正了解的人不广博，广博的人不了解。
圣人不私自积藏，他尽力帮助别人，自己反而更充足；他尽力给予别人，自己反而更丰富。
天道是利物而不害物，圣人的法则是施为而不争夺。

## 解析

这是《道德经》的最后一章，可以看作是全书的总结。

1. **真与美的辩证**：
   - 真实的话往往不悦耳动听
   - 悦耳动听的话往往不真实
   - 强调本质重于形式

2. **善与辩的关系**：
   - 真正善良的人不需要巧辩
   - 巧辩往往掩盖不善的意图

3. **知与博的取舍**：
   - 真正的"知"是深刻把握道的本质
   - 表面的"博"可能只是知识的堆砌

4. **圣人的给予之道**：
   - 不积藏私财，尽力帮助他人
   - 给予的越多，自己反而越富有
   - 这是利他与利己的统一

5. **天道与圣人**：
   - 天道：利万物而不害万物
   - 圣人：施为而不争夺
   - 这是道家最高境界的表述

这一章总结了《道德经》的核心价值观：真诚、善良、不争、利他，为全书画上了完美的句号。
