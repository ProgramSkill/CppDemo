# 00-诗经

## 书籍简介

《诗经》是中国古代第一部诗歌总集，也是世界文学史上最早的诗歌总集之一。全书共收录诗歌305篇（另有6篇"笙诗"有目无辞），故又称"诗三百"。这些诗歌创作于西周初年至春秋中叶（约公元前11世纪至前6世纪），跨越了五百多年的时间跨度，全面反映了周代社会的方方面面。

《诗经》与《尚书》《礼记》《周易》《春秋》合称"五经"，是儒家学派的核心经典之一，对中国文学、语言、思想、文化产生了深远影响。

**当代学术界共识**：
- 《诗经》的成书是一个长期过程
- 周代设有"采诗官"，到民间采集诗歌；各国诸侯也向王室献诗
- 乐官负责整理、配乐
- 孔子可能做过最终的整理、编纂、定本工作，并对《诗经》的传播和解读产生重大影响

## 文学手法：赋比兴

"赋、比、兴"是《诗经》的三种基本表现手法，也是中国古代诗歌创作的传统手法。

### 一、赋（铺陈直叙）

**定义**：直接铺陈叙述，直抒胸臆，不加掩饰地表达情感和描述事物。

**特点**：
- 叙事直接，语言质朴
- 情感真挚，直白表达
- 多用于《雅》《颂》，也见于《风》

**经典例证**：

| 诗篇 | 例句 | 简析 |
|------|------|------|
| 《氓》 | "氓之蚩蚩，抱布贸丝。匪来贸丝，来即我谋" | 直接叙述男子来求婚的经过 |
| 《采薇》 | "昔我往矣，杨柳依依。今我来思，雨雪霏霏" | 直抒昔往今来的心境变化 |
| 《击鼓》 | "死生契阔，与子成说。执子之手，与子偕老" | 直白表达爱情誓言 |
| 《蒹葭》 | "蒹葭苍苍，白露为霜。所谓伊人，在水一方" | 铺陈景物，引出思念 |

### 二、比（比喻类比）

**定义**：通过比喻和类比，用具体的事物说明抽象的道理或情感。

**特点**：
- 以此喻彼，形象生动
- 使抽象情感具象化
- 包含明喻、暗喻、借喻等多种形式

**经典例证**：

| 诗篇 | 例句 | 比喻类型 | 简析 |
|------|------|----------|------|
| 《氓》 | "桑之未落，其叶沃若" | 借喻 | 以桑叶润泽比喻女子年轻美貌 |
| 《氓》 | "桑之落矣，其黄而陨" | 借喻 | 以桑叶枯黄比喻女子年老色衰 |
| 《鹤鸣》 | "他山之石，可以攻玉" | 借喻 | 比喻任用贤能治理国家 |
| 《硕鼠》 | "硕鼠硕鼠，无食我黍" | 借喻 | 以硕鼠比喻贪婪的剥削者 |
| 《硕人》 | "手如柔荑，肤如凝脂" | 明喻 | 连用比喻描绘美人 |

### 三、兴（托物起兴）

**定义**：先言他物以引起所咏之词，即通过描写某一事物来引发对主题的联想和抒情。

**特点**：
- 触景生情，托物起兴
- 具有启发性和暗示性
- 往往在诗歌开头，营造氛围
- 是《诗经》中最具特色、最美妙的手法

**经典例证**：

| 诗篇 | 起兴句 | 主题 | 简析 |
|------|--------|------|------|
| 《关雎》 | "关关雎鸠，在河之洲" | 爱情追求 | 以雎鸠和鸣兴起对淑女的思念 |
| 《桃夭》 | "桃之夭夭，灼灼其华" | 婚姻祝福 | 以桃花盛开兴起对新婚女子的赞美 |
| 《采薇》 | "采薇采薇，薇亦作止" | 征人思乡 | 以采薇活动起兴，引出征戍之苦 |
| 《蒹葭》 | "蒹葭苍苍，白露为霜" | 相思之苦 | 以秋景起兴，营造凄清氛围 |
| 《风雨》 | "风雨凄凄，鸡鸣喈喈" | 相见之喜 | 以风雨鸡鸣起兴，引出见君子的喜悦 |

### 四、赋比兴的综合运用

在《诗经》中，赋、比、兴往往综合运用，形成丰富的艺术效果：

```
【综合运用实例】

《关雎》结构：
┌─────────────────────────────────────┐
│ 起兴（兴）：关关雎鸠，在河之洲      │
│           （雎鸠和鸣的景象）         │
├─────────────────────────────────────┤
│ 赋陈（赋）：窈窕淑女，君子好逑      │
│           （直接表达爱慕之情）       │
├─────────────────────────────────────┤
│ 比喻（比）：参差荇菜，左右流之      │
│           （以采摘荇菜比喻追求）     │
└─────────────────────────────────────┘

《氓》结构：
┌─────────────────────────────────────┐
│ 起兴（兴）：桑之未落，其叶沃若      │
│           （桑树繁茂的景象）         │
├─────────────────────────────────────┤
│ 叙事（赋）：直接叙述恋爱过程        │
├─────────────────────────────────────┤
│ 比喻（比）：以桑树荣枯比喻女子年华  │
└─────────────────────────────────────┘
```

### 三、语言特色

**1. 叠字的妙用**

《诗经》中大量使用叠字，据统计达200多个，极大地丰富了诗歌的表现力：

| 叠字类型 | 例词 | 例句 | 效果 |
|---------|------|------|------|
| **拟声叠字** | 关关、喈喈、丁丁 | "关关雎鸠" | 模拟声音，生动形象 |
| **状物叠字** | 天夭、沃若、霏霏 | "桃之夭夭" | 描绘形态，具体可感 |
| **抒情叠字** | 依依、切切、迟迟 | "杨柳依依" | 表达情感，细腻入微 |

**2. 双声叠韵**

- **双声词**：参差（cēn cī）、辗转（zhǎn zhuǎn）、栗烈（lì liè）
- **叠韵词**：窈窕（yǎo tiǎo）、虺隤（huī tuí）、须臾（xū yú）

**3. 虚词的运用**

《诗经》中大量使用虚词（之、乎、者、也、矣、焉、哉等），使语气舒缓，节奏和谐。

## 学习建议

### 一、阅读方法

1. **结合注释**：借助现代注本理解古文

3. **了解背景**：结合历史背景理解诗篇
4. **比较阅读**：对比不同诗篇的主题和手法
5. **专题研究**：按主题、手法、地域等分类研究
